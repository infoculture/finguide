# Открытые государственные финансы России

Книга-справочник об устройстве государственных финансов в России и о работе с открытыми финансовыми данными. Сайт собирается из этого репозитория с помощью [Docusaurus 3](https://docusaurus.io/).

## Структура репозитория

```
.
├── docusaurus.config.ts   # Конфигурация сайта Docusaurus
├── sidebars.ts            # Структура навигации/сайдбара
├── package.json           # Зависимости и npm-скрипты
├── tsconfig.json          # Настройки TypeScript
├── src/                   # Кастомные стили и компоненты
│   └── css/custom.css
├── static/                # Статические ассеты (PDF, изображения и т.п.)
│   └── assets/
└── wiki/                  # Контент книги в виде Markdown
    ├── README.md          # Главная страница
    ├── glossary.md        # Глоссарий
    ├── faq.md             # FAQ
    ├── intro/             # Введение
    ├── howto/             # С чего начать (нормативка, принципы)
    ├── power/             # Органы власти
    ├── gis/               # Государственные информационные системы
    ├── orgs/              # Государственные и муниципальные организации
    ├── opendata/          # Открытые данные
    └── addons/            # Приложения и справочники
```

## Требования

- Node.js >= 20
- npm >= 10

## Быстрый старт

```bash
# Установить зависимости
npm install

# Запустить локальный сервер разработки
npm start

# Открыть в браузере
# http://localhost:3000/opengovfinancesbook/
```

## Основные команды

| Команда | Описание |
| --- | --- |
| `npm start` | Запуск локального dev-сервера с горячей перезагрузкой |
| `npm run build` | Сборка production-версии в директорию `build/` |
| `npm run serve` | Локальный просмотр собранного сайта |
| `npm run clear` | Очистка кэша Docusaurus |
| `npm run typecheck` | Проверка типов TypeScript |
| `npm run deploy` | Публикация на GitHub Pages |

## Как редактировать контент

1. Откройте нужный Markdown-файл в каталоге `wiki/`.
2. Внесите изменения. Frontmatter (метаданные в начале файла) — опционален; минимально достаточно поля `title`.
3. Если нужен новый раздел — создайте подкаталог в `wiki/`, добавьте в него `README.md` (он станет индексной страницей раздела) и обновите `sidebars.ts` при необходимости.
4. Картинки, PDF и прочие файлы складывайте в `static/assets/` и ссылайтесь на них как `/assets/имя-файла`.

## Конфигурация

- **`docusaurus.config.ts`** — заголовок, URL-параметры, navbar, footer, тема Prism.
- **`sidebars.ts`** — структура и порядок навигации.
- **`src/css/custom.css`** — пользовательские стили.

## Деплой

По умолчанию настроен деплой на GitHub Pages с baseUrl `/opengovfinancesbook/`. Чтобы опубликовать обновления:

```bash
GIT_USER=<ваш_github_логин> npm run deploy
```
