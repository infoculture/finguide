# opengovfinancesbook

Репозиторий переведен на [Docusaurus](https://docusaurus.io/) для публикации wiki-контента.

## Структура репозитория

- `wiki/` — весь контент wiki (Markdown-файлы и разделы).
- `docusaurus.config.ts`, `sidebars.ts`, `src/` — инфраструктура отображения и навигации.
- `openspec/` и `AGENTS.md` — правила и служебные материалы по процессу разработки.

Такой разделенный подход позволяет развивать контент независимо от инфраструктуры сайта.

## Быстрый старт

1. Установить зависимости:

```bash
npm install
```

2. Запустить локальный сервер:

```bash
npm run start
```

3. Открыть wiki в браузере:

`http://localhost:3000/opengovfinancesbook/`

## Сборка production-версии

```bash
npm run build
```

Локальный просмотр production-сборки:

```bash
npm run serve
```

## Как расширять wiki

- Добавляйте или редактируйте статьи в `wiki/` с сохранением тематической структуры каталогов.
- Для новых крупных разделов создавайте подкаталоги в `wiki/`; они автоматически появятся в сайдбаре.
- Для изменения порядка/структуры верхнеуровневых блоков редактируйте `sidebars.ts`.
- Для настройки сайта (URL, navbar, поведение линков, тема) редактируйте `docusaurus.config.ts`.

## Полезные команды

- `npm run clear` — очистка кэша Docusaurus.
- `npm run start` — локальная разработка.
- `npm run build` — production-сборка.
- `npm run serve` — просмотр собранного сайта.
