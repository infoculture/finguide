# ngo_federal_list_2020.xlsx — методика

Источник: исследование **GovNGO** (ноябрь 2020), файл `ngo_federal_list_2020.xlsx`.

## Листы

| Лист | Охват | Ключевые поля |
|------|--------|----------------|
| **по ЕГРЮЛ** | 46 организаций | ИНН, дата регистрации, наименование, основание («Отметка в ЕГРЮЛ»), СО НКО, субсидии 2018–2020 |
| **по субсидиям** | 968 получателей; **40** с `госНКО = Да` | ИНН, отметка в ЕГРЮЛ, учредитель, госНКО, субсидии 2017–2020, всего |

## Целевой перечень для wiki

**Объединение** обоих листов по ИНН: **61** уникальная организация (46 + 40 − 25 пересечений).

Пересечение с каталогом **30 госкомпаний** (`federal_gov_orgs_links_table.xlsx`) по ИНН — только **Автодор**, **ВЭБ.РФ**, **Росатом**; для них обновляются существующие карточки в `organizations/state-sector` и `enterprises/`, без дублирования.

## Импорт в репозиторий

```bash
node scripts/build-federal-gos-ngo-registry.mjs   # пересобрать scripts/data/federal-gos-ngo-registry.json
npm run import:federal-gos-ngo
npm run import:federal-gos-ngo -- --dry-run
npm run import:federal-gos-ngo -- --only=nark
```

Канонический реестр: [`scripts/data/federal-gos-ngo-registry.json`](../scripts/data/federal-gos-ngo-registry.json).

**Срез данных:** 2017–2020 (субсидии); статусы и суммы не экстраполируются на текущий год без пересчёта.
