# Работа с региональными порталами открытых данных

## Зачем

Региональные порталы «Открытый бюджет» имеют разные структуры. Универсального API нет — приходится извлекать данные из HTML или скачивать XLS.

## Шаги

1. Найти раздел «Открытые данные» или «Открытый бюджет» на сайте региона.
2. Скачать XLS/CSV выгрузки.
3. Привести к единой схеме на стороне аналитика.

## Проверка результата

Сверьте итоги с официальной отчётностью, контрольными суммами или sanity-check из текста выше.


## Подводные камни

- Кодировки и формат дат могут различаться между источниками.
- Ограничения по rate limit — проверяйте документацию.
- Обновления могут приходить с задержкой по отношению к публикации.
