# API Банка России

## Зачем

ЦБ РФ предоставляет несколько API: ежедневные курсы валют (XML/JSON), статистику через сервисы SOAP/REST, банковскую отчётность XBRL.

## Шаги

1. Выбрать тип данных (курсы, статистика, отчётность).
2. Использовать соответствующую точку доступа.
3. Кешировать результаты на стороне клиента.

## Подводные камни

- Кодировки и формат дат могут различаться между источниками.
- Ограничения по rate limit — проверяйте документацию.
- Обновления могут приходить с задержкой по отношению к публикации.
