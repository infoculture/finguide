# How-to: доступ к данным

Практические инструкции по получению данных из ключевых источников. После выгрузки переходите к разделу [Анализ](/howto/analysis/) или [Автоматизация](/howto/automation/) для обработки.

## Для кого

- Аналитикам и разработчикам, которым нужен **воспроизводимый** путь к файлам и API без ручного копирования из PDF.
