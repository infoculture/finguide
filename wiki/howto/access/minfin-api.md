# API и открытые данные Минфина

## Зачем

Минфин публикует открытые данные через раздел /opendata/. Каждый датасет имеет паспорт с актуальной ссылкой на JSON или CSV.

## Шаги

1. Открыть реестр датасетов Минфина.
2. Скачать паспорт датасета.
3. Использовать ссылки на актуальные версии.

```python
import requests
# r = requests.get("https://...")
# r.raise_for_status()
# data = r.json()
```

## Подводные камни

- Кодировки и формат дат могут различаться между источниками.
- Ограничения по rate limit — проверяйте документацию.
- Обновления могут приходить с задержкой по отношению к публикации.
