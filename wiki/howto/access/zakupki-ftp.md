# Выгрузка XML с FTP ЕИС Закупок

## Зачем

ЕИС Закупки публикует ежедневные XML-архивы по регионам и типам документов через FTP. Это основной источник для масштабного анализа закупок.

## Шаги

1. Подключиться к ftp.zakupki.gov.ru.
2. Скачать архивы за нужный период по регионам.
3. Распаковать ZIP, обработать XML согласно схемам.
