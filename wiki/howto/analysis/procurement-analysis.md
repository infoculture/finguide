# Анализ государственных закупок

## Зачем

Анализ закупок строится на данных ЕИС (44-ФЗ и 223-ФЗ) с учётом ОКПД2, заказчика, ГРБС, поставщика.

## Шаги

1. Скачать XML с FTP ЕИС или использовать Clearspending.
2. Привести данные к табличному виду.
3. Сгруппировать по ОКПД2/региону/заказчику.
