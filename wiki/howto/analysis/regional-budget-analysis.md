# Анализ региональных бюджетов

## Зачем

Региональные бюджеты сравниваются через консолидированные данные Казначейства (форма 0503317) и региональные порталы.

## Шаги

1. Скачать 0503317 для нужных регионов.
2. Привести к единой структуре.
3. Учесть дотации/субвенции из федерального бюджета для корректного сравнения.
