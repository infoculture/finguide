# Расшифровка КБК

## Зачем

КБК — 20-значный код. Его можно разобрать программно, используя справочники Минфина по составляющим (главный администратор, раздел/подраздел, целевая статья, КВР).

## Шаги

1. Получить справочник КБК года из открытых данных Минфина.
2. Разбить код на составляющие по позициям.
3. Сопоставить каждой части название из справочника.
