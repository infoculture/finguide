# Шаблон страницы How-to

Служебная страница: при добавлении инструкции копируйте структуру ниже в `wiki/howto/<раздел>/<slug>.md`, заполните frontmatter и удалите неиспользуемые подсказки.

### Поля frontmatter

| Поле | Обязательно | Пример |
|------|-------------|--------|
| `title` | да | `"Работа с API …"` |
| `sidebar_label` | да | Короткая подпись |
| `description` | да | Одно-два предложения для поиска и RAG |
| `tags` | да | `[howto, access]` или `[howto, analysis]` + тема из таксономии |
| `last_updated` | да | `2026-05-12` |
| `content_type` | да | `howto` |
| `entity_type` | да | `howto` |
| `draft` | для черновиков | `true` |

### Структура тела

1. `# Заголовок`
2. `## Назначение и аудитория` (или `## Зачем`)
3. `## Входные данные`
4. `## Инструменты`
5. `## Шаги`
6. `## Воспроизводимый пример`
7. `## Проверка результата`
8. `## Ограничения и типовые ошибки` (или `## Подводные камни`)
9. `## Связанные страницы`

Проверка CI: `npm run lint:howto`.
