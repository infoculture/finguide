# ЕИС в сфере закупок

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ЕИС Закупки |
| Оператор | Федеральное казначейство |
| Официальный URL | https://zakupki.gov.ru/ |

## Описание

Единая информационная система в сфере закупок. Содержит сведения о закупках по 44-ФЗ и 223-ФЗ: извещения, заявки, контракты, реестры. Все данные публикуются в открытом доступе и доступны через FTP-выгрузки и API.

## Подсистемы / разделы

- Реестр контрактов 44-ФЗ
- Реестр контрактов 223-ФЗ
- Реестр недобросовестных поставщиков
- Реестр банковских гарантий

## Точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Веб-портал | https://zakupki.gov.ru/ | HTML |
| FTP XML-выгрузки | ftp://ftp.zakupki.gov.ru/ | XML/ZIP |
| API | https://api.zakupki.gov.ru/ | JSON/SOAP |
