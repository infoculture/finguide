# ГИИС «Электронный бюджет»

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Электронный бюджет |
| Оператор | Минфин России / Федеральное казначейство |
| Официальный URL | https://budget.gov.ru/ |

## Описание

Государственная интегрированная информационная система управления общественными финансами. Объединяет процессы планирования и исполнения бюджетов всех уровней, ведения реестров получателей субсидий, исполнения бюджета по доходам и расходам.

## Подсистемы / разделы

- Подсистема бюджетного планирования
- Подсистема управления расходами
- Подсистема ведения нормативной справочной информации (НСИ)
- Реестр участников бюджетного процесса
