# OpenNGO.ru — Финансирование НКО

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | OpenNGO |
| Оператор | НП «Информационная культура» |
| Официальный URL | https://openngo.ru/ |

## Описание

Информационная система, агрегирующая данные о государственном финансировании НКО — субсидии, гранты, госконтракты. Объединяет источники Минэкономразвития, Минюста, Фонда президентских грантов.

## Подсистемы / разделы

- Субсидии и гранты НКО
- Госконтракты с НКО
- Реестры НКО Минюста
