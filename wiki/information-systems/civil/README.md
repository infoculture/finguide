# Гражданские информационные системы

Независимые от государства проекты, агрегирующие, анализирующие и переиспользующие данные о государственных финансах.

- [Clearspending.ru](./clearspending.md) — анализ закупок
- [HubOfData](./hubofdata.md) — каталог датасетов
- [DataCatalogs.ru](./datacatalogs-ru.md) — реестр каталогов открытых данных
- [OpenNGO.ru](./open-ngo.md) — финансирование НКО
- [Президентские гранты](./pgrants.md) — `pgrants.ru`
