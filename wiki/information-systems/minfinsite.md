# Сайт Министерства финансов РФ

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | minfin.gov.ru |
| Оператор | Минфин России |
| Официальный URL | https://minfin.gov.ru/ |

## Описание

Официальный сайт Минфина России. Раздел «Открытые данные» содержит наборы по бюджетной классификации, государственному долгу, ФНБ, бюджетной росписи.

## Подсистемы / разделы

- Открытые данные Минфина
- Бюджетная роспись
- Государственный долг РФ
