# АИС «Финансы» (исторический)

|  Признак | Значение |
| :--- | :--- |

| Официальный URL | https://minfin.gov.ru/ |

## Описание

Закрытая ведомственная информационная система Минфина России для управления финансовыми потоками. Открытые данные не публикуются; информация о составе системы — в государственных контрактах и пресс-релизах.

## Подсистемы / разделы

- Управление финансовыми операциями Минфина (закрытый контур)
