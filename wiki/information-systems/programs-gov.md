# programs.gov.ru — Государственные программы РФ

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | programs.gov.ru |
| Оператор | Минэкономразвития России |
| Официальный URL | https://programs.gov.ru/ |

## Описание

Портал, содержащий полный перечень государственных программ РФ с их финансированием, целями, индикаторами, отчётами о реализации.

## Подсистемы / разделы

- Перечень госпрограмм
- Финансирование госпрограмм
- Целевые индикаторы и отчёты
