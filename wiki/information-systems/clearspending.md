# Clearspending.ru — Анализ госзакупок

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Clearspending |
| Оператор | Комитет гражданских инициатив / гражданское сообщество |
| Официальный URL | https://clearspending.ru/ |

## Описание

Гражданская информационная система, индексирующая данные ЕИС Закупок и предоставляющая удобный поиск, аналитику, визуализации и REST API для исследователей и журналистов.

## Подсистемы / разделы

- Контракты по 44-ФЗ
- Извещения о закупках
- Аналитика по поставщикам
