# ЕГРЮЛ / ЕГРИП — Единые государственные реестры

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ЕГРЮЛ/ЕГРИП |
| Оператор | ФНС России |
| Официальный URL | https://egrul.nalog.ru/ |

## Описание

Государственный реестр юридических лиц и индивидуальных предпринимателей. Сведения о регистрации, учредителях, ОКВЭД, изменениях. Доступны через сервис ФНС и через API.

## Подсистемы / разделы

- Карточка ЮЛ/ИП
- Сведения об учредителях и руководителях
- Информация о ликвидации, реорганизации
