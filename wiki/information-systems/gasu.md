# ГАС «Управление»

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ГАСУ |
| Оператор | Федеральное казначейство (оператор) |
| Официальный URL | https://gasu.gov.ru/ |

## Описание

Государственная автоматизированная информационная система «Управление» — единое аналитическое пространство для мониторинга деятельности органов власти, исполнения госпрограмм, нацпроектов, прогнозов социально-экономического развития.

## Подсистемы / разделы

- Мониторинг госпрограмм
- Мониторинг национальных проектов
- Прогнозы социально-экономического развития
