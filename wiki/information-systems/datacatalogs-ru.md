# DataCatalogs.ru — Реестр каталогов открытых данных

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | DataCatalogs.ru |
| Оператор | НП «Информационная культура» |
| Официальный URL | https://datacatalogs.ru/ |

## Описание

Реестр каталогов открытых данных российских органов власти. Помогает аналитикам ориентироваться в источниках, оценивать актуальность и полноту данных.

## Подсистемы / разделы

- Перечень каталогов открытых данных
- Машиночитаемые метаданные
- Оценка актуальности
