# Региональные информационные системы

## Назначение раздела

Здесь будут карточки **региональных и муниципальных** информационных систем, через которые публикуются бюджет, закупки, госпрограммы и смежные реестры. Пока отдельные ИС не вынесены в файлы, используйте связанные материалы ниже как маршрут «система → данные».

## Для кого

- Аналитикам, которым нужно понять, **где искать витрину** субъекта или МО до появления отдельной карточки ИС в wiki.
- Редакторам: новые карточки — по `wiki/information-systems/information-system-card-template.md` ([GitHub](https://github.com/infoculture/opengovfinancesbook/blob/master/wiki/information-systems/information-system-card-template.md)) и `AGENTS.md`; шаблон с `draft: true` в production не собирается.

## Типовые вопросы

**Чем региональная ИС отличается от регионального источника в `data-sources`?** ИС — операторская среда; **источник** — конкретные наборы, API и ограничения выгрузки. Карточки ИС и источников должны перекрёстно ссылаться друг на друга, когда связь однозначна.
