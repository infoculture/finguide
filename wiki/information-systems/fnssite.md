# Сайт ФНС России — статистика и данные

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | nalog.gov.ru |
| Оператор | ФНС России |
| Официальный URL | https://www.nalog.gov.ru/ |

## Описание

Раздел статистики и аналитики на сайте ФНС России. Публикует формы 1-НМ, 1-НДС, 7-НП и многие другие; реестр МСП; раздел открытых данных.

## Подсистемы / разделы

- Статистическая налоговая отчётность
- Реестр МСП
- Открытые данные ФНС
