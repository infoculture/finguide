# ЕМИСС — Единая межведомственная статистическая система

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ЕМИСС |
| Оператор | Росстат |
| Официальный URL | https://fedstat.ru/ |

## Описание

ЕМИСС — портал, объединяющий статистические данные федеральных органов исполнительной власти. Содержит более 6000 показателей: социально-экономических, демографических, отраслевых.

## Подсистемы / разделы

- Социально-экономическая статистика
- Финансовая статистика
- Региональные показатели

## Точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
