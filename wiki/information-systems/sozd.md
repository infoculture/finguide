# СОЗД — Система обеспечения законодательной деятельности

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | СОЗД |
| Оператор | Государственная Дума |
| Официальный URL | https://sozd.duma.gov.ru/ |

## Описание

Информационная система Государственной Думы, публикующая полный жизненный цикл законопроектов: тексты, заключения, стенограммы, голосования. Источник всех материалов по проекту федерального бюджета.

## Подсистемы / разделы

- Законопроекты и сопроводительные материалы
- Заключения комитетов
- Стенограммы и голосования
