# Информационные системы

Государственные и гражданские информационные системы, в которых публикуются и обрабатываются данные о государственных финансах России.

- [ГИИС «Электронный бюджет»](./giis-eb.md) — основной портал бюджета (`budget.gov.ru`)
- [ЕИС в сфере закупок](./zakupki.md) — `zakupki.gov.ru`
- [bus.gov.ru](./bus-gov.md) — отчётность государственных и муниципальных учреждений
- [ГАС «Управление»](./gasu.md) — мониторинг госпрограмм и нацпроектов
- [programs.gov.ru](./programs-gov.md) — государственные программы
- [ФЦП.gov.ru](./sistema-fcp.md) — архив федеральных целевых программ
- [Сайт Минфина](./minfinsite.md) — открытые данные Минфина
- [Портал Казначейства](./roskazna-portal.md) — отчёты Казначейства
