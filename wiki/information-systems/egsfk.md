# ЕГСФК — Единая государственная система финансового контроля

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ЕГСФК |
| Оператор | Счётная палата РФ |
| Официальный URL | https://ach.gov.ru/ |

## Описание

Информационная система Счётной палаты РФ для ведения единых классификаторов и реестров объектов финансового контроля.

## Подсистемы / разделы

- Классификаторы для аудиторской деятельности
- Реестр объектов контроля
