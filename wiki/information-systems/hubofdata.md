# HubOfData — каталог датасетов по госфинансам

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | HubOfData |
| Оператор | НП «Информационная культура» |
| Официальный URL | https://hubofdata.ru/ |

## Описание

Каталог открытых данных, агрегирующий датасеты Минфина, Казначейства, ФНС и других ведомств. Поддерживает CKAN API для машинного доступа.

## Подсистемы / разделы

- Датасеты Минфина
- Датасеты Казначейства
- Датасеты ФНС

## Точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Веб-интерфейс | https://hubofdata.ru/ | HTML |
| CKAN API | https://hubofdata.ru/api/3/ | JSON |
