# Портал Федерального казначейства

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | roskazna.gov.ru |
| Оператор | Федеральное казначейство |
| Официальный URL | https://roskazna.gov.ru/ |

## Описание

Официальный сайт Федерального казначейства. Публикует ежемесячные и годовые отчёты об исполнении федерального бюджета и консолидированных бюджетов субъектов РФ.

## Подсистемы / разделы

- Отчёты об исполнении бюджетов
- Долговые книги
- Открытые данные

## Точки доступа
