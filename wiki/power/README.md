# По уровням власти

Все бюджетные отношения в России можно разделить на 3 уровня власти:

* [Федеральная власть](fed/)
* [Региональная власть](reg) \(субъекты федерации\)
* [Местное самоуправление](mun) \(муниципальная власть\)

Местное самоуправление по [Конституции ](../howto/howtostart/constitution)в России не является частью государственной власти, но в рамках бюджетных отношений оно встроено в их структуру. 

Практически полностью полномочия и процессы на каждому из уровней власти описаны в [Бюджетном кодексе](../howto/howtostart/budkodeks).
