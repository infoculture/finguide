# Поиск по кодам ОКТМО, КБК и КОСГУ

## Зачем эта страница

В выгрузках встречаются «сырые» коды; здесь собраны канонические точки входа внутри книги без дублирования справочников ФНС и Минфина.

## ОКТМО и территории

- Обзор и связка с бюджетной территорией — [ОКАТО / ОКТМО](/reference/okato-oktmo).
- Учёт территории в расходных кодах — раздел [Коды расходов](/budget-classification/expense-codes).

## КБК и классификация расходов/доходов

- Термин — [КБК](/glossary/kbk).
- Структура расходного кода — [Обзор бюджетной классификации](/budget-classification/overview) и [Коды расходов](/budget-classification/expense-codes).
- Доходная классификация — [Коды доходов](/budget-classification/income-codes).

## КОСГУ и учёт учреждений

- Термин — [КОСГУ](/glossary/kosgu).
- Табличный справочный материал — [КОСГУ (справочно)](/budget-classification/kosgu-reference).

## Практическая проверка

Сверяйте редакцию кода с годом бюджета и приказами Минфина ([приказы по классификации](/legal/budget-classification-orders)).
