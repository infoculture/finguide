# Справочные материалы

Справочники дополняют [глоссарий](/glossary/) и [источники данных](/data-sources/): ориентиры по **кодам**, **ключам связки наборов** и **внешним стандартам**, без дублирования полных карточек терминов.

## Для кого раздел

- Аналитикам, которым нужна **навигация по кодам** (ОКТМО, КБК, КОСГУ, ОКПД2) и версиям классификаторов.
- Редакторам wiki: таксономия тегов и журнал обновлений — для согласованности корпуса.

## Типовые вопросы

**Чем этот раздел отличается от глоссария?** Здесь — таблицы, индексы и процессные обзоры; в [глоссарии](/glossary/) — определения терминов по одному.

**Где юридически значимые перечни кодов?** Карточки [бюджетной классификации](/budget-classification/) и НПА, например [приказы по классификации](/legal/budget-classification-orders).
