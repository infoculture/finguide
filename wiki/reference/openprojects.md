# Общественные проекты

К общественным (и смежным **неофициальным витринам**) в сфере государственных финансов относят инициативы по **мониторингу госрасходов**, **закупок** и **субсидий**, каталоги открытых данных и справочные сервисы поверх публичных реестров. Они **не** заменяют **ЕИС**, **ГИИС «Электронный бюджет»** и иные первичные контуры: сверяйте юридически значимые факты у операторов. Ниже — ориентиры для навигации; актуальность ссылок, лицензии и политика API проверяйте на стороне операторов.

| Название | Ссылка | Описание | Авторы / операторы |
| :--- | :--- | :--- | :--- |
| Госзатраты (Clearspending) | [clearspending.ru](https://clearspending.ru/) | Мониторинг государственных расходов: госконтракты и смежные срезы; есть публичный REST API закупочного контура | [Комитет гражданских инициатив](https://komitetgi.ru/), [АНО «Инфокультура»](https://infoculture.ru/) |
| Госзатраты.Субсидии | [sub.clearspending.ru](https://sub.clearspending.ru/) | Витрина по субсидиям в логике того же проекта; не реестр соглашений | Комитет гражданских инициатив, АНО «Инфокультура» |
| Hub of Data | [hubofdata.ru](https://hubofdata.ru/) | Гражданский каталог датасетов на CKAN; поиск и API метаданных по наборам (в т. ч. бюджет, закупки, ведомства) | [НП «Информационная культура»](https://infoculture.ru/) |
| DataCatalogs.ru | [datacatalogs.ru](https://datacatalogs.ru/) | Метакаталог порталов и реестров открытых данных | НП «Информационная культура» |
| OpenNGO | [openngo.ru](https://openngo.ru/) | Сквозной поиск по НКО: субсидии, гранты, госконтракты и справка по организациям | НП «Информационная культура» |
| Портал президентских грантов | [президентскиегранты.рф](https://президентскиегранты.рф/) | Конкурсная поддержка НКО, каталог открытых данных и витрины ФПГ | Фонд президентских грантов |
| Российские школы | [goodschools.ru](https://goodschools.ru/) | Данные по школам, в т. ч. бюджеты и расходы | Сергей Устинов |
| За честные закупки | [z4z.onf.ru](https://z4z.onf.ru/) | Мониторинг закупок | [Общероссийский народный фронт](https://onf.ru/) |

## Карточки источников в этой книге

Для части перечисленных витрин в wiki есть отдельные карточки источников и ИС — удобный старт для границ сервиса, форматов и оговорок относительно официальных регистров:

- [Clearspending.ru](/data-sources/civil/clearspending) и [ИС Clearspending](/information-systems/civil/clearspending) — закупки и API; поддомен субсидий.
- [Hub of Data](/data-sources/civil/hubofdata) и [ИС Hub of Data](/information-systems/civil/hubofdata) — CKAN-каталог и Action API.
- [DataCatalogs.ru](/data-sources/civil/datacatalogs-ru) и [ИС DataCatalogs.ru](/information-systems/civil/datacatalogs-ru) — метакаталог порталов ОД.
- [OpenNGO.ru](/data-sources/civil/open-ngo) и [ИС OpenNGO](/information-systems/civil/open-ngo) — агрегация по НКО.
- [Президентские гранты (pgrants)](/data-sources/civil/pgrants) и [ИС портала ФПГ](/information-systems/civil/pgrants) — грантовый контур и открытые наборы оператора.
