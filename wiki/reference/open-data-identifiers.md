# Идентификаторы в открытых наборах

Страница задаёт **сквозной** взгляд: какие **поля-ключи** чаще всего нужны, чтобы связать строки из разных публичных источников о госфинансах. Детали кодов организаций — в **[идентификаторах организаций](./organization-codes.md)**; территориальные — в **[ОКАТО / ОКТМО](./okato-oktmo.md)**.

## Субъекты и территория

| Поле (типичное имя) | Смысл | Где смотреть |
| :--- | :--- | :--- |
| **Код субъекта РФ** | Регион бюджета или административная принадлежность (формат зависит от набора) | [Наборы ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets), [отчёты Казначейства](/data-sources/federal/roskazna-reports) |
| **ОКТМО** | Территория муниципального образования | [ОКТМО в глоссарии](/glossary/oktmo), бюджетные и статистические наборы |
| **ОКАТО** | Устаревший аналог в legacy-файлах | См. [ОКАТО / ОКТМО](./okato-oktmo.md) |

## Организации

| Поле | Смысл | Типичные источники |
| :--- | :--- | :--- |
| **ИНН**, **КПП** | Налоговая идентификация, различение подразделений | [Закупки](/data-sources/federal/procurement), [субсидии](/data-sources/federal/subsidy), [bus.gov.ru (открытые данные)](/data-sources/federal/bus-opendata) |
| **ОГРН / ОГРНИП** | Регистрационный номер | [ЕГРЮЛ и ЕГРИП](/information-systems/federal/egrul), иногда в обогащённых выгрузках |
| **ОКПО** | Статистический код организации | Старые и отраслевые таблицы; реже в «свежих» закупочных API |

## Бюджетная классификация

| Поле | Смысл | Типичные источники |
| :--- | :--- | :--- |
| **КБК** (полный код) | Категория дохода / расхода / источника | [Отчёты Казначейства](/data-sources/federal/roskazna-reports), [витрины datamarts](/data-sources/federal/roskazna-datamarts), [«Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets) |
| **КОСГУ** | Экономическая классификация операций | Те же контуры плюс отчётность учреждений; см. [КОСГУ](/glossary/kosgu) |
| **КВР** | Вид расхода | План и исполнение в бюджетных наборах; см. [КВР](/glossary/kvr) |

Переходы между версиями справочников — **[версии классификаций](./classification-versions.md)**.

## Закупки и контракты

| Поле | Смысл | Типичные источники |
| :--- | :--- | :--- |
| **Реестровый номер контракта / извещения** | Устойчивый ключ документа в ЕИС | [Сведения о закупках](/data-sources/federal/procurement), [XML ЕИС](/data-sources/federal/zakupki-xml) |
| **Идентификатор версии** (если есть) | Различение правок XML | `zakupki-xml`, методика выгрузки |
| **ОКПД2** | Предмет закупки | [ОКПД2 в глоссарии](/glossary/okpd), [частые коды](./okpd-budgets.md) |
| **НМЦК, суммы** | Денежные показатели | Согласовать **базис** (с НДС / без, валюта) перед склейкой с бюджетом |

## Временной ключ

Почти всегда нужны **финансовый год** и **отчётный период** (месяц, квартал). При соединении **плана** ([ГИИС](/information-systems/federal/giis-eb)) и **исполнения** ([Казначейство](/information-systems/federal/roskazna-portal)) явно фиксируйте, **касса** это или **начисления**, и методику показателя.
