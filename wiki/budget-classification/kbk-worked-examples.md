# Разобранные примеры КБК

Ниже — **учебные** разборы: **наименования строк** и юридически значимые перечни кодов для анализа всегда берите из **приказов Минфина на конкретный финансовый год** и из [открытых справочников](/data-sources/federal/minfin-opendata). Цифровые строки иллюстрируют **геометрию** поля и типичные ошибки join, а не заменяют официальный справочник.

## 1. Расходный КБК: администратор, ФКР, целевая статья, КВР

**Код:** `07104010201081000110` (тот же пример, что в [«Расшифровке КБК»](/howto/analysis/kbc-decoding)).

| Фрагмент | Позиции | Смысл (типовая схема) |
| --- | --- | --- |
| `071` | 1–3 | Код главного администратора расходов (связка с [ГРБС](/glossary/grbs) — по методике года) |
| `04` | 4–5 | Раздел функциональной классификации расходов |
| `01` | 6–7 | Подраздел |
| `0201081001` | 8–17 | [Целевая статья](/budget-classification/target-articles) (10 знаков) |
| `110` | 18–20 | [КВР](/glossary/kvr) |

**Куда дальше:** [коды расходов](/budget-classification/expense-codes) → [форма 0503117](/reporting/0503117) / [0507011](/reporting/0507011) → [отчёты Казначейства](/data-sources/federal/roskazna-reports).

## 2. Доходный КБК: ГАДБ и ступени доходной классификации

**Код (иллюстрация разрядов):** `18210201010101010101` — длина 20 знаков; **подписи к подстрокам** сверяйте с таблицей в [«Кодах доходов»](/budget-classification/income-codes).

| Фрагмент | Позиции | Смысл (каркас доходного КБК) |
| --- | --- | --- |
| `182` | 1–3 | Главный администратор **доходов** (ГАДБ) |
| `1` | 4 | Группа доходов |
| `02` | 5–6 | Подгруппа |
| `01` | 7–8 | Статья |
| `010` | 9–11 | Подстатья |
| `10` | 12–13 | Элемент |
| `1010` | 14–17 | Подвид |
| `101` | 18–20 | Аналитическая группа подвида |

**Куда дальше:** [КБК в глоссарии](/glossary/kbk) → [АнКВ](/glossary/ankv) (если в наборе есть отдельные аналитические поля) → [Минфин — открытые данные](/data-sources/federal/minfin-opendata).

## 3. Межбюджетные трансферты: отчётные агрегаты и договорный след

**Полный 20-значный КБК** в законах о бюджете и в отчётности об исполнении кодирует операции того же каркаса, что и прочие расходы, но **смысл строки** определяется сочетанием кода, раздела формы и [гл. 16 БК РФ о трансфертах](/glossary/transfers).

**Иллюстрация расходной геометрии с «трансфертной» КВР-группой (проверяйте код по справочнику года):** `07201010101081000560` — последние три знака здесь выбраны как **учебный** ориентир на группу видов расходов «безвозмездные перечисления…» в терминологии КВР; фактическое наименование `560` **может отличаться по году**.

**Куда дальше:** [форма 0503387](/reporting/0503387) (виды трансфертов и ИФД в консолидированном бюджете субъекта) → [0503317](/reporting/0503317) → [реестр соглашений](/data-sources/federal/subsidy) для микроданных по отдельным соглашениям.

## 4. Источники финансирования дефицита (ИФД)

**Код (иллюстрация длины и необходимости типа строки):** `09010203040506070809` — **не** доход и **не** «обычный» расход в смысле функциональной классификации программных статей; трактовка разрядов задаётся **отдельной** схемой ИФД на год (см. [источники финансирования дефицита](/glossary/deficit-financing)).

**Куда дальше:** [отчётность по долгу](/reporting/public-debt-reporting) → [ФНБ и резервы](/reporting/fiscal-reserves-reporting) → [закон о бюджете](/legal/budgetlaw) (приложения по источникам).

## 5. Доход бюджета: другая тройка ГАДБ (иллюстрация)

**Код:** `11311212121121210121` — разбивка по позициям доходного КБК как в примере 2; первая тройка `113` **иллюстрирует** другой ГАДБ, чем в примере 2.

| Фрагмент | Позиции |
| --- | --- |
| `113` | 1–3 |
| `1` | 4 |
| `12` | 5–6 |
| `12` | 7–8 |
| `121` | 9–11 |
| `12` | 12–13 |
| `1210` | 14–17 |
| `121` | 18–20 |

**Куда дальше:** [налоговая и таможенная статистика как зеркало доходов](/reporting/tax-and-customs-revenue-statistics) → [сайт ФНС](/information-systems/federal/fnssite) при стыковке с налоговыми базами (вне бюджетной классификации, но рядом по задаче).

## 6. КОСГУ рядом с КБК в учёте (не продолжение 20 знаков)

**Пример:** трёхзначная статья расходов вида **`244`** (подстатья в разделе **2xx** — см. логику разделов в [справочнике КОСГУ](/budget-classification/kosgu-reference)); точное наименование и допустимые сочетания с [КВР](/glossary/kvr) — только из **перечня на ваш год** и [таблицы соответствия](/budget-classification/kvr-kosgu-mapping).

**Смысл для аналитика:** в выгрузках учреждений и [сметах](/reporting/budget-estimates) КОСГУ стоит **отдельным столбцом**; его нельзя подставлять в хвост расходного КБК. Сверка с бюджетным контуром — через [таблицу КВР и КОСГУ](/budget-classification/kvr-kosgu-mapping).

**Куда дальше:** [ПФХД](/reporting/pfhd) → [бухотчётность учреждений](/reporting/institution-financial-statements).
