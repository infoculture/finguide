# Обзор бюджетной классификации

## Описание

Бюджетная классификация — группировка доходов, расходов и источников финансирования дефицита бюджетов всех уровней, обеспечивающая сопоставимость показателей. Состоит из четырёх частей: классификация доходов, расходов, источников финансирования дефицита и операций сектора государственного управления (КОСГУ).

## Ключевые элементы

- КБК — 20-значный код
- Структура: главный администратор + вид/группа + статья + элемент + подвид + аналитика
- Утверждается приказами Минфина России ежегодно
