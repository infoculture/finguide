# Бюджетная классификация

Группировка доходов, расходов и источников финансирования дефицита бюджетов всех уровней. Материалы раздела помогают **читать коды** в выгрузках Минфина, Казначейства, портала закупок и отчётности учреждений, не подменяя юридические определения текстами wiki.

## Для кого раздел

- Аналитики открытых данных и разработчики, которым нужно разобрать **20-значный КБК**, КОСГУ и целевые статьи в таблицах.
- Редакторы карточек источников и отчётности, сверяющие формулировки с **официальными перечнями** по году.
