# Справочник КОСГУ 2024–2026

## Описание

КОСГУ группирует операции сектора государственного управления по экономическому смыслу. Ежегодно Минфин публикует обновлённый справочник с разъяснениями применения статей и подстатей.

## Ключевые элементы

- Группа 100 — Доходы
- Группа 200 — Расходы
- Группа 300 — Поступление нефинансовых активов
- Группа 500/600/700 — Финансовые активы и обязательства
