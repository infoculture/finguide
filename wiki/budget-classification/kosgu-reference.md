# Справочник КОСГУ 2024–2026

## Описание

**[КОСГУ](/glossary/kosgu)** — часть бюджетной классификации, которая кодирует операции сектора государственного управления по **экономическому содержанию** (доходы, расходы, движение активов и обязательств). Перечень подстатей и разъяснения по применению утверждаются Минфином России **на каждый финансовый год** (и плановый период); при анализе рядов за несколько лет храните **версию справочника на год**, а не один «универсальный» файл.

Ниже — логика **разделов** по первой цифре кода; точные наименования и состав кодов всегда сверяйте с **нормативным актом на выбранный год** и с выгрузками из раздела «Связанные источники».

## Структура кода

В учётных формах и открытых выгрузках чаще всего встречается **трёхзначный** код КОСГУ (статья и подстатья в рамках раздела). **Первая цифра** задаёт раздел (группу операций по экономическому смыслу), **вторая и третья** — детализацию по официальному перечню на год.

## Разделы (по первой цифре кода)

| Первая цифра | Укрупнённый смысл раздела |
| :--- | :--- |
| **1** | Доходы (**1xx**) |
| **2** | Расходы (**2xx**) |
| **3** | Поступление нефинансовых активов (**3xx**) |
| **4** | Выбытие нефинансовых активов (**4xx**) |
| **5** | Поступление финансовых активов (**5xx**) |
| **6** | Выбытие финансовых активов (**6xx**) |
| **7** | Обязательства (**7xx**) |

Границы групп и наименования подстатей в конкретном году задаются приказами; таблица отражает **типовую структуру** классификатора, а не замену официальному справочнику.

## Связь с КВР и КБК

**[КВР](/glossary/kvr)** — три знака **внутри** расходного **[КБК](/glossary/kbk)**. **КОСГУ** в 20-значный КБК **не подставляется**: это отдельный классификатор для **бюджетного учёта** и отчётности учреждений. Сопоставление «вид расхода бюджета ↔ подстатья учёта» выполняют только по **[таблице соответствия КВР и КОСГУ](/budget-classification/kvr-kosgu-mapping)** на соответствующий год.
