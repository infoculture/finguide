# СГФ-2014 — Соответствие классификаций МВФ

## Описание

Руководство МВФ по статистике государственных финансов 2014 (СГФ/GFSM 2014) описывает международный стандарт. Минфин публикует таблицы соответствия российских КБК и категорий СГФ-2014 для целей международной отчётности.

## Ключевые элементы

- Таблицы соответствия КБК ↔ GFSM 2014
- Применение для международных сопоставлений
- Используется в данных Всемирного банка и МВФ по России
