# Коды доходов

## Описание

Структура кода доходов: главный администратор (3 знака) + группа (1) + подгруппа (2) + статья (2) + подстатья (3) + элемент бюджета (2) + код подвида (4) + аналитическая группа (3). Полный справочник — в Приказе Минфина РФ.

## Ключевые элементы

- Главный администратор доходов (ГАДБ)
- Виды доходов: налоговые, неналоговые, безвозмездные
- Подвиды доходов
- Аналитические группы
