# Коды доходов

## Описание

**Коды доходов бюджета** — это перечень и правила построения **доходного** [КБК](/glossary/kbk): тоже **20 знаков**, но **разметка позиций другая**, чем у расходного или у кода источников финансирования дефицита. Перечень действующих значений и расшифровка разрядов задаются **приказами Минфина России** на соответствующий **финансовый год** (ориентир по НПА — [приказы по бюджетной классификации](/legal/budget-classification-orders)). В плановых и отчётных таблицах тип кода (доход / расход / источники финансирования дефицита) нужно брать из метаданных набора или смысла формы, а не только из длины строки.

Разбор по разрядам (позиции в 20-значной строке, считая слева):

| Позиции | Смысл |
| :---: | :--- |
| 1–3 | Код **главного администратора доходов бюджета** (ГАДБ) |
| 4 | **Группа** доходов (в т.ч. налоговые, неналоговые, безвозмездные поступления) |
| 5–6 | **Подгруппа** доходов |
| 7–8 | **Статья** доходов |
| 9–11 | **Подстатья** доходов |
| 12–13 | **Элемент** дохода бюджета |
| 14–17 | **Подвид** дохода бюджета |
| 18–20 | **Аналитическая группа** подвида доходов бюджета |

В открытых выгрузках и учёте рядом с доходным КБК может встречаться отдельная аналитика ([АнКВД](/glossary/ankv)); при сверке с планом по одному только «голому» КБК часть строк может не сходиться.

## Ключевые элементы

- **ГАДБ** — административный верх иерархии в **доходной** классификации; не подменяет без проверки справочника код **главного распорядителя** по расходам ([ГРБС](/glossary/grbs)).
- **Группы и подгруппы доходов** — верхняя экономическая группировка поступлений в бюджет.
- **Статья, подстатья, элемент, подвид** — последовательное уточнение содержания дохода до уровня, на котором закон о бюджете и роспись обычно ведут учёт и публичную отчётность.
- **Аналитическая группа подвида** — завершающие три знака каркаса доходного КБК по методике года.

## Связанные источники

- [Минфин — открытые данные классификации](/information-systems/federal/minfinsite) — витрина НСИ и наборов по бюджетной классификации на сайте ведомства.
- [Открытые данные Минфина России](/data-sources/federal/minfin-opendata) — машиночитаемые справочники и срезы по **КБК доходов** и смежным справочникам по годам (удобная отправная точка для расшифровки кодов в скриптах).
- [Отчёты Федерального казначейства](/data-sources/federal/roskazna-reports) — формы и выгрузки **исполнения бюджета** с разрезом по КБК, в т.ч. доходные строки (методика и поля зависят от формы и периода).
- [ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb) — проект закона о бюджете, роспись и плановые срезы с доходными КБК (доступность разделов зависит от роли пользователя).
- [Приказы по бюджетной классификации](/legal/budget-classification-orders) — каноническая нормативная база структуры и перечней кодов на финансовый год.

Пошаговый разбор позиций кода в данных — в материале [«Расшифровка КБК»](/howto/analysis/kbc-decoding); общий контекст четырёх частей классификации — в [обзоре бюджетной классификации](/budget-classification/overview).

## Связанные термины

- [КБК](/glossary/kbk) — единый 20-значный реквизит для доходов, расходов и источников финансирования дефицита; для доходов используется отдельная схема разрядов.
- [АнКВ](/glossary/ankv) — **АнКВД** и связанная аналитика в учёте и отчётности рядом с доходным КБК.
- [ГРБС](/glossary/grbs) — роль главного распорядителя по **расходам**; для доходов параллельным понятием выступает **главный администратор доходов** (первые три знака доходного КБК — по справочнику года).
- [КОСГУ](/glossary/kosgu) — классификация **операций** бюджетного учёта; в таблицах может стоять рядом с КБК, но **не** является продолжением или заменой доходного кода.
