# О базе знаний

«Открытые государственные финансы России» — это структурированный справочник, объединяющий
описание бюджетной системы РФ, ключевых органов власти, государственных информационных систем,
источников открытых данных и практических инструкций по работе с ними. Стартовая страница книги —
[Государственные финансы](/).

## Цели

- Объяснить устройство государственных финансов России простым языком.
- Описать каждый источник данных по единому шаблону: оператор, точки доступа, форматы, периодичность.
- Дать практические рецепты для аналитиков, журналистов, исследователей и разработчиков.
- Сформировать корпус знаний, пригодный для использования ИИ-ассистентом (RAG).

## Аудитория

- Аналитики и исследователи, работающие с бюджетными данными.
- Журналисты, которым нужны проверяемые источники о госрасходах.
- Разработчики, интегрирующие государственные API и открытые данные.
- ИИ-системы, использующие базу знаний как источник для ответов на вопросы пользователей.

## Принципы

- **Один файл — одна основная сущность.** Термин, организация, источник, форма отчётности, информационная система или инструкция изолированы в отдельной странице.
- **Машиночитаемый frontmatter.** Поля вроде `title`, `tags`, при необходимости `source_url`, `last_updated` поддерживают навигацию и индексацию.
- **Q&A-секции.** В конце ключевых страниц — типовые вопросы и краткие ответы (по шаблонам карточек разделов).
- **Открытость.** Контент распространяется по лицензии MIT; правки и дополнения приветствуются через Pull Request и [обсуждения в репозитории](https://github.com/infoculture/opengovfinancesbook/issues).

## Структура и навигация

Материалы сгруппированы по одиннадцати разделам (введение, глоссарий, организации, информационные системы,
бюджетная система, бюджетная классификация, источники данных, how-to, нормативная база, отчётность,
справочники). Обзор назначения каждого раздела — в документе [Как пользоваться базой знаний](/intro/how-to-use).
Крупные структурные изменения фиксируются в [История изменений](/intro/changelog).

## Развитие проекта

После перехода на Docusaurus 3 и выравнивания структуры (май 2026) приоритет — **наполнение и выверка**
карточек по единым шаблонам, **расширение** региональных и гражданских источников в каталоге данных,
практические **how-to** с воспроизводимыми примерами доступа и анализа, а также постепенное оформление
блоков «частые вопросы» там, где они усиливают страницу.
