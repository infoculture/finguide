# Нижний Новгород: открытый бюджет города (budgetnn.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Город Нижний Новгород** (муниципальное образование) — **отдельный** портал открытого бюджета и разделы **администрации города** |
| Тип доступа | **Публичные** HTML-витрины; **открытые данные** администрации — отдельным разделом на **admgor.nnov.ru** |
| Основные форматы | **HTML** на **budgetnn.ru**; на **opendata** администрации — наборы в **CSV** и др. по паспортам |
| Оператор | **Администрация города Нижнего Новгорода**; портал **budgetnn.ru** позиционируется как официальная витрина **открытого бюджета** города |
| Режим доступа | Открытое чтение публичных разделов |

Коротко: **городской** портал **«Открытый бюджет»** с разделами планирования, исполнения и сопутствующими сервисами; для **единого федерального** поиска по бюджетам МО и выгрузок по регламенту портала используйте **[муниципальные бюджеты на `budget.gov.ru`](/data-sources/regional/municipal-budgets)** и **[наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** с фильтром по **ОКТМО** территории города.

## Описание

**budgetnn.ru** агрегирует материалы **бюджетного процесса** (в т.ч. планирование, документы, исполнение, муниципальный долг в публичной части по мере размещения). Раздел **[открытые данные](https://admgor.nnov.ru/opendata)** на сайте **admgor.nnov.ru** даёт **каталог наборов** для машиночитаемых срезов по тематикам администрации; состав наборов **не обязан** совпадать с полным пакетом бюджетных PDF на портале бюджета.

## Оператор

**Администрация города Нижнего Новгорода** — https://admgor.nnov.ru/

## URL и точки доступа

| Назначение | URL |
| --- | --- |
| Портал «Открытый бюджет города» | https://budgetnn.ru/ |
| Бюджетный процесс (пример стабильного подраздела) | https://budgetnn.ru/o-byudzhete/byudzhetnyj-protsess |
| Открытые данные администрации | https://admgor.nnov.ru/opendata |

Глубокие URL на **admgor.nnov.ru** (например отчёты об исполнении) **меняются** при редизайне; закладывайте **корень** разделов «Экономика и финансы» / «Бюджет» или **поиск по сайту** администрации.

## Качество и верификация

| Измерение | Значение |
| --- | --- |
| Дата проверки (`last_verified`) | 2026-05-14 |
| Полнота | **Частичная относительно `budget.gov.ru`** — сильная **городская** витрина; федеральный портал остаётся якорём для сверки полей и годов |
| Машиночитаемость | **HTML** на **budgetnn.ru**; **CSV** — по наборам на **opendata** администрации |

## Объекты данных и показатели

- **Пояснительные и аналитические материалы** по бюджету города, **сравнения** и сервисы на **budgetnn.ru** (по актуальному меню).
- **Наборы открытых данных** на **admgor.nnov.ru/opendata** — по паспортам; тематика шире, чем только бюджет.
- **Связка с федеральным контуром** — через **[муниципальные бюджеты](/data-sources/regional/municipal-budgets)** и **[ОКТМО](/glossary/oktmo)** города Нижнего Новгорода.

## Периодичность обновления

По мере публикаций администрации и обновлений портала **budgetnn.ru**; для выгрузок **opendata** — даты в **метаданных** наборов.

## Ограничения и особенности

- Не подменяйте **юридически значимый** текст решений о бюджете **только** инфографикой портала без сверки с **официальными** документами на сайте МО и при необходимости с **федеральным** порталом.
- **Консолидированный бюджет Нижегородской области** и **закупки** — **иные** уровни; не смешивайте с бюджетом **города** без проверки **ОКТМО** и полномочий.

## Примеры доступа

Проверка корня портала открытого бюджета:

```bash
curl -fsSL -A "finguide-budgetnn-check/1.0" -o /dev/null -w "%{http_code}\n" \
  "https://budgetnn.ru/"
```

Проверка раздела открытых данных администрации:

```bash
curl -fsSL -A "finguide-admgor-opendata-check/1.0" -o /dev/null -w "%{http_code}\n" \
  "https://admgor.nnov.ru/opendata"
```

## Частые вопросы

**В:** Где скачать закон о бюджете города в PDF?

**О:** Ищите в разделах **планирования** на **budgetnn.ru** и на **admgor.nnov.ru** в блоке бюджета; при отсутствии стабильной глубокой ссылки используйте **поиск** по сайту администрации и сверку даты документа.
