# Брянская область: открытый бюджет и «бюджет для граждан» (bryanskoblfin.ru/open)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Брянская область, консолидированный региональный бюджет (агрегаты и разделы витрины) |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML; при наличии — вложения PDF/XLS по ссылкам из разделов |

Коротко: **главная «гражданская» витрина** по областному бюджету: показатели, доходы и расходы, долговая нагрузка, трансферты, государственные программы и инвестиции. Для юридически значимых текстов законов о бюджете и проектов см. [Брянская областная Дума: бюджетный процесс](/data-sources/regional/bryansk-oblast-regional-duma-budget); для сырья по исполнению и методике — [сайт департамента финансов](/data-sources/regional/bryansk-oblast-dept-finance).

## Описание

Раздел **«Открытый бюджет»** на домене департамента финансов дублирует и раскрывает темы «бюджет для граждан»: наглядные блоки по доходам, расходам, дефициту/профициту, долгу, межбюджетным трансфертам, программам и инвестициям. Дополнительно на портале Правительства области публикуется упрощённый раздел «Простой бюджет»: https://www.bryanskobl.ru/simple-budget — его удобно использовать как альтернативную точку входа для обзорных показателей.

## Объекты данных и показатели

- **Агрегированные ряды** по консолидированному бюджету области: доходы, расходы, источники финансирования дефицита, долг.
- **Трансферты** и **государственные программы** — в составе тематических блоков витрины (структура по годам уточняется на сайте).
- Связка с **КБК/функциональными разделами** и муниципальным уровнем — при переходе к детальным отчётам на сайте финоргана и в нормативных текстах.

## Периодичность обновления

Определяется регламентом публикаций департамента финансов; привязка к **этапам бюджетного цикла** (проект, поправки, отчёт об исполнении). Детали см. на https://bryanskoblfin.ru/ в разделах отчётности.

## Ограничения и особенности

- Витрина ориентирована на **объяснение** показателей, а не на выгрузку полных машинных наборов строк исполнения; для воспроизводимых CSV/JSON по региону сопоставляйте с федеральными открытыми продуктами и отчётностью казначейства.
- Перед массовым скрейпингом проверяйте `robots.txt` и правила сайта.
