# Брянская область: департамент финансов (bryanskoblfin.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Брянская область; областной бюджет, межбюджетные отношения, отчётность |
| Тип доступа | Публичный официальный сайт финансового органа |
| Основные форматы | HTML; типовые вложения отчётности PDF и XLSX по ссылкам из разделов |
| Оператор | Департамент финансов Брянской области |
| Режим доступа | Открытый просмотр и скачивание опубликованных материалов |

Коротко: **главный бюджетный вход** субъекта: материалы по составлению и исполнению областного бюджета, долг, трансферты, отчёты, методика. Витрина «открытого бюджета» вынесена в отдельную карточку: [открытый бюджет](/data-sources/regional/bryansk-oblast-open-budget). Нормативные тексты — через [региональные НПА](/data-sources/regional/bryansk-oblast-regional-npa) и [сайт Думы](/data-sources/regional/bryansk-oblast-regional-duma-budget).

## Описание

На сайте департамента обычно публикуются: **основные характеристики бюджета**, проект на очередной год и поправки, **отчёты об исполнении**, сведения о **государственном долге** и **межбюджетных отношениях**, методические письма и разъяснения для заказчиков бюджетных средств. Раздел `/open` дублирует тему гражданской витрины и вынесен в отдельную карточку для навигации.

## Объекты данных и показатели

- **Бюджетные показатели** по годам: доходы, расходы, источники покрытия дефицита.
- **Отчётность об исполнении** консолидированного бюджета (в форматах, размещённых на сайте).
- **Межбюджетные трансферты** и методические материалы для муниципальных образований (по разделам сайта).

## Периодичность обновления

Волнами в течение **бюджетного цикла** (внесение проекта, чтения, исполнение, годовой отчёт); оперативные новости — по мере публикации.
