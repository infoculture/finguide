# Московская область: единый портал закупок (ЕПЗ, easuz.mosreg.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональный** портал **закупок** Московской области |
| Тип доступа | Публичный **веб-интерфейс** |
| Основные форматы | **HTML**; **XML/пакеты** федеральной контрактной системы — в **[ЕИС](/information-systems/federal/zakupki)** |
| Оператор | Оператор **ЕПЗ** субъекта (официальный региональный контур закупок) |
| Режим доступа | Открытое чтение; участие в процедурах — по регламенту |

Коротко: **региональная витрина** закупок области; для **сквозной** аналитики по **44/223-ФЗ** и **единым** идентификаторам контрактов используйте **[ЕИС](/information-systems/federal/zakupki)** и агрегаторы вроде **[Clearspending](/data-sources/civil/clearspending)**.

## Описание

**easuz.mosreg.ru** — точка входа для **закупок** в интересах **Московской области** и связанных заказчиков. Для **сопоставления с бюджетом** используйте **[МЭФ](/data-sources/regional/moscow-oblast-mef)** и коды **КБК/заказчика**.

## Оператор

**Единый портал закупок Московской области** — канонический вход: https://easuz.mosreg.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Портал закупок | https://easuz.mosreg.ru/ | HTML |
| ЕИС (федеральный контур) | https://zakupki.gov.ru/epz/main/public/home.html | HTML, выгрузки |

## Объекты данных и показатели

- **Извещения, протоколы, контракты** в региональном UI (набор полей зависит от процедуры).
- **Ключи:** реестровый номер контракта/извещения, **ИНН** заказчика, **КБК** (если раскрыт).
- **Федеральный след** — дублируется или дополняется в **[ЕИС](/information-systems/federal/zakupki)**.

## Периодичность обновления

**Онлайн** в ходе закупочных процедур.

## Ограничения и особенности

- Не все закупки **регионального** уровня проходят только через **ЕПЗ**; часть публикуется **напрямую** в **ЕИС**.
- Для **массовых выгрузок** ориентируйтесь на **официальные** каналы **ЕИС**, а не на парсинг HTML.
