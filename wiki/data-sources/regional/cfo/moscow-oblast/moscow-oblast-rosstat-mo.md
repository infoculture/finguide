# Московская область: территориальный орган Росстата (77.rosstat.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Территориальный орган Росстата** по **Московской области** |
| Тип доступа | Публичный **веб-портал** |
| Основные форматы | **HTML**, **XLSX** (в ряде разделов); федеральные **OD** — **[каталог Росстата](/data-sources/federal/rosstat-opendata)** |
| Оператор | **Территориальный орган Федеральной службы государственной статистики** по региону |
| Режим доступа | Открытое чтение |

Коротко: **официальная статистика** по области (демография, экономика, социалька); для **сводных финансовых рядов РФ** см. **[финансы России](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf)** и **[консолидированные бюджеты](/data-sources/regional/consolidated-budgets)** как методический мост к **бюджетной** отчётности.

## Описание

Поддомен **77.rosstat.gov.ru** — типовая **точка входа** для **Московской области**: региональные **комментарии**, **срезы** и ссылки на **официальные** публикации. Для **сопоставления с бюджетом** используйте согласованные **коды** (ОКВЭД, ОКФС, ОКПО и др.) и **[МЭФ](/data-sources/regional/moscow-oblast-mef)**.

## Оператор

**Территориальный орган Росстата по Московской области** — канонический вход: https://77.rosstat.gov.ru/Moskovskaya_oblast

## Объекты данных и показатели

- **Официальные статистические показатели** по субъекту (витрины и файлы по разделам сайта).
- **Метаданные:** период, единица измерения, территория (**Московская область**).
- **Не** путать с **административной** отчётностью **МЭФ** — разные **методики**.

## Периодичность обновления

По **календарю** публикаций Росстата (**ежемесячно**, **ежеквартально**, **ежегодно** — в зависимости от блока).

## Ограничения и особенности

- **Московская область** и **Москва** — **разные** территории в статистике Росстата.
- Часть показателей доступна **только** в **федеральных** комплектах таблиц — проверяйте **[OD](/data-sources/federal/rosstat-opendata)**.

curl -fsSL -A "finguide-check/1.0" -o /dev/null -w "%{http_code}\n" \
