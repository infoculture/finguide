# Тамбовская область: Тамбовстат (территориальный орган Росстата, 68.rosstat.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** служба (ТОР) на территории Тамбовской области |
| Тип доступа | Региональный официальный сайт статистики |
| Основные форматы | HTML; официальные таблицы — XLSX/PDF по ссылкам |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Тамбовской области |
| Режим доступа | Открытый |

Коротко: **официальная статистика** по региону (ВРП, демография, цены, отраслевые блоки). Для **налоговых** административных рядов сверяйте с [УФНС](/data-sources/regional/tambov-oblast-ufns-nalog-rn68) и [налоговой статистикой ФНС](/data-sources/federal/nalog-statistics).

## Описание

Тамбовстат публикует **региональные срезы** официальной статистики: оперативные индикаторы, пресс-релизы, методические пояснения. При **бюджетном анализе** такие ряды помогают объяснять динамику доходов и расходов [областного бюджета](/data-sources/regional/tambov-oblast-minfin-budget-for-citizens), но **не** заменяют бюджетную отчётность.

## Объекты данных и показатели

- **Социально-экономические показатели** региона (структура по разделам сайта).
- **Цены, население, ВРП** — в публикуемых формах Росстата.
- Ключи связки: **год**, **классификаторы отраслей/товаров** (по методологии Росстата), **территория** (область).

## Периодичность обновления

По **календарю** публикаций официальной статистики (оперативные данные и годовые сборники).

## Ограничения и особенности

- Пересечение с **[налоговой статистикой](/data-sources/federal/nalog-statistics)** требует внимания к **определениям** показателей (налоговая база vs национальные счета).
