# Костромская область: Территориальный фонд ОМС (ТФОМС)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Костромская область; **внебюджетный** контур ОМС |
| Тип доступа | Публичный сайт ТФОМС |
| Основные форматы | HTML; PDF и табличные вложения — по разделам |
| Оператор | Территориальный фонд обязательного медицинского страхования Костромской области |
| Режим доступа | Открытый просмотр |

Коротко: **официальная витрина** ТФОМС: тарифы на ОМС, программы обязанного страхования, отчётность и разъяснения. Для **сводной бюджетной аналитики** субъекта фонд дополняет картину **внебюджетных** расходов на здравоохранение и должен сопоставляться с консолидированной отчётностью и материалами [Департамента финансов](/data-sources/regional/kostroma-oblast-department-of-finance).

## Описание

ТФОМС публикует материалы по **страховым тарифам**, **программам ОМС**, отчётам о расходовании средств и нормативным актам фонда. При анализе **консолидированного** бюджета субъекта учитывайте, что значительная часть расходов на медицину идёт **вне** основного бюджета через ОМС.

## Объекты данных и показатели

- **Тарифы и нормативы** ОМС по региону.
- **Отчёность фонда**: годовые и периодические отчёты (часто PDF).
- **Связка с бюджетом**: дотации бюджета субъекта фонду и обратные трансферты — через материалы финансового органа и отчёты об исполнении бюджета.

## Периодичность обновления

По мере принятия актов и публикации отчётности; новости — чаще.

## Ограничения и особенности

- Канонический вход нормализован на **HTTPS**; старые ссылки с `http://` могут перенаправляться.
