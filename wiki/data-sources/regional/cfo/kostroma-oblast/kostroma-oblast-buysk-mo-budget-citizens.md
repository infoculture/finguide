## Описание

Карточка фиксирует **типичный** для муниципалитетов формат «бюджет для граждан»: наглядное изложение доходов и расходов **местного бюджета** в PDF. Для **областного** контура см. [бюджет для граждан на сайте Департамента финансов](/data-sources/regional/kostroma-oblast-budget-for-citizens-depfin).

## Оператор

Буйский муниципальный округ — см. канонический `source_url` в шапке (прямая ссылка на файл).

## Объекты данных и показатели

- **Агрегированные показатели** местного бюджета в гражданской подаче.
- **ОКТМО** муниципалитета — для связки с региональными и федеральными рядами.
- **Пробелы**: отсутствие единого API; стабильность URL не гарантируется.

## Периодичность обновления

Как минимум **ежегодно** при принятии местного бюджета и корректировках.

## Ограничения и особенности

- Длинные URL с `iblock` часто **меняются** при выгрузке новой версии файла; для устойчивых ссылок в исследованиях сохраняйте копию с фиксацией даты скачивания.
