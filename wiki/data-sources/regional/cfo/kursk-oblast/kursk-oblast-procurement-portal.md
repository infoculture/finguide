# Курская область: информационная система в сфере закупок (zak.imkursk.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный контур закупок для нужд субъекта и подведомственных заказчиков (в объёме витрины) |
| Тип доступа | Специализированный веб-портал закупок |
| Основные форматы | HTML; при наличии — XML/выгрузки по правилам портала |
| Оператор | Оператор региональной ИС закупок (zak.imkursk.ru) |
| Режим доступа | Открытый для просмотра публичных разделов |

Коротко: **региональная** витрина, через которую проходят закупки, не сводимые к единому федеральному сценарию ЕИС в полном объёме; для **44-ФЗ/223-ФЗ** и контрактного реестра по РФ по-прежнему опирайтесь на ЕИС.

## Описание

Портал публикует **региональные** извещения, протоколы и итоги процедур для заказчиков Курской области. Для **сквозной** аналитики по крупным федеральным закупкам и **единому** контрактному следу используйте [ЕИС](/data-sources/federal/procurement) и связанные XML-выгрузки.

## Объекты данных и показатели

- Извещение о закупке, протокол, контракт/договор (в модели данных портала).
- Ключи: **реестровые номера** (если дублируются в ЕИС), ИНН заказчика, предмет закупки.

## Периодичность обновления

Непрерывно в рабочее время размещения процедур.

## Ограничения и особенности

- Технические ограничения (капча, rate limit) — по факту на портале; для массовых выгрузок сверяйте с [XML ЕИС](/data-sources/federal/zakupki-xml).
