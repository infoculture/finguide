# Курская область: источники данных

Коротко: эта страница перечисляет **опубликованные** карточки источников данных по субъекту. Публичные URL карточек имеют вид `/data-sources/regional/<slug>`; имя файла в репозитории совпадает с `<slug>`.

## Карточки источников

- [Минфин области: нормативные приказы (раздел Правительства)](/data-sources/regional/kursk-oblast-dept-finance-normative-orders) — приказы и НПА по бюджетному процессу и ТФОМС.
- [Курская областная Дума](/data-sources/regional/kursk-oblast-legislature-duma) — законы об областном бюджете, проекты, материалы слушаний.
- [Раздел финансов и «Бюджет для граждан» (сайт Правительства)](/data-sources/regional/kursk-oblast-open-budget) — экономика и бюджет на adm2.rkursk.ru; см. также архивный опрос по формату «Бюджет для граждан».
- [Бюджет для граждан (город Курск, Госуслуги)](/data-sources/regional/kursk-city-open-budget-gosuslugi) — упрощённые материалы по бюджету города на муниципальном сайте.
- [Информационная система закупок области](/data-sources/regional/kursk-oblast-procurement-portal) — региональный контур госзакупок (zak.imkursk.ru).
- [Интернет-портал правовой информации](/data-sources/regional/kursk-oblast-regional-npa-portal) — региональный банк НПА (kurskpravo.ru).
- [Контрольно-счётная палата области](/data-sources/regional/kursk-oblast-accounting-chamber) — контрольные материалы и заключения по бюджету.
- [Курскстат (ТО Росстата)](/data-sources/regional/kursk-oblast-rosstat-territorial) — официальная территориальная статистика.
- [Территориальный фонд ОМС](/data-sources/regional/kursk-oblast-tfoms) — финансовая отчётность и закупки фонда.
- [УФК по Курской области](/data-sources/regional/kursk-oblast-ufk) — сайт территориального органа Росказны.
- [УФНС по Курской области](/data-sources/regional/kursk-oblast-ufns) — региональная страница ФНС (код rn46).
