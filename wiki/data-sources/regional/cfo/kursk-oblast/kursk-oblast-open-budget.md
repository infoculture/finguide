# Курская область: раздел финансов и материалы «Бюджет для граждан» (Правительство)

Коротко: **навигационный центр** по финансовой тематике на портале Правительства: отсюда уходят ссылки на социальные сети Минфина, КСП и др.; отдельно зафиксирован **архивный опрос** «Бюджет для граждан» к закону об областном бюджете на 2024 год (не путать с постоянной инфографикой «открытого бюджета»).

## Описание

Исходная таблица исследования указывала URL `…/residents/polls/15/` — это **страница опроса** «Бюджет для граждан» к закону об областном бюджете на 2024 год (на момент проверки доступна с **завершающим слэшем**). Как **устойчивый вход** по теме «финансы региона» в карточке зафиксирован раздел **«Экономика → Финансы»**, откуда видны актуальные перекрёстные ссылки (в т.ч. на КСП). Для **муниципального** «бюджета для граждан» города Курска см. отдельную карточку [на Госуслугах](/data-sources/regional/kursk-city-open-budget-gosuslugi).

## Объекты данных и показатели

- Новости и страницы Правительства по бюджетной политике.
- Опрос как **артефакт обратной связи** по дизайну гражданской отчётности (не таблица исполнения бюджета).

## Периодичность обновления

Раздел финансов обновляется **нерегулярно**; опрос — **разовый** документ периода подготовки закона об бюджете 2024.

## Ограничения и особенности

- Для **машиночитаемых рядов исполнения** опирайтесь на федеральные и региональные открытые наборы (см. связанные страницы), а не на этот раздел как на единственный источник.
