# Курская область: интернет-портал правовой информации

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный контур Курской области |
| Тип доступа | Веб-портал правовой информации |
| Основные форматы | HTML, вложения PDF/DOC по документам |
| Оператор | Оператор портала правовой информации Курской области (официальная публикация НПА субъекта) |
| Режим доступа | Открытый |

Коротко: **официальное** размещение нормативных актов Курской области — удобная отправная точка для поиска закона об областном бюджете, постановлений Правительства и ведомственных актов до углубления в предметные разделы сайта Думы или Минфина.

## Описание

Портал агрегирует **нормативную базу субъекта**: законы Курской области, указы и постановления Губернатора, постановления и распоряжения Правительства, муниципальные правовые акты (в объёме, который оператор включает в базу), а также иные документы по мере их включения оператором. Для **бюджетного цикла** на практике связка обычно такая: проекты и сопроводительные материалы — на сайте [Курской областной Думы](/data-sources/regional/kursk-oblast-legislature-duma), **официальные тексты после принятия** — через этот портал и разделы Правительства; приказы финансового блока — также в карточке [нормативных приказов Минфина](/data-sources/regional/kursk-oblast-dept-finance-normative-orders).

| Федеральное зеркало официального опубликования | https://publication.pravo.gov.ru/ | HTML (поиск по реквизитам федеральных актов; региональные акты сверяйте по порталу субъекта) |

## Объекты данных и показатели

- **Документ**: НПА субъекта РФ (закон, постановление, распоряжение, приказ и т.д.) с реквизитами и файлом.
- **Связки**: номер и дата акта; предметные теги (бюджет, ОМС, закупки) — по полям поиска портала.
- **Пробелы**: единый машиночитаемый справочник «все акты в JSON» на портале не предполагается; для массовой обработки выгружайте PDF и нормализуйте вне портала.

## Периодичность обновления

По мере **официального** принятия и регистрации актов; для срочных сверок используйте дату размещения на карточке документа.

## Ограничения и особенности

- Не путайте с федеральным **[publication.pravo.gov.ru](https://publication.pravo.gov.ru/)**: там иной контур опубликования; региональные акты Курской области ищите на kurskpravo.ru.
- Перед автоматизированным скачиванием проверьте `robots.txt` и типовые ограничения на частоту запросов.
