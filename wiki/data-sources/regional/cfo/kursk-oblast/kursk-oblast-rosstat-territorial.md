# Курская область: Курскстат (территориальный орган Росстата)

| Тип доступа | Сайт территориального органа Росстата (код 46) |
| Основные форматы | HTML, XLSX, PDF (по разделам) |

## Описание

Сайт **46.rosstat.gov.ru** публикует **территориальные** срезы показателей, методические комментарии и официальные пресс-выпуски по Курской области. Для **федеральных** открытых наборов и поиска форм используйте витрины [Росстата](/data-sources/federal/rosstat-opendata) и [Showdata](/data-sources/federal/rosstat-showdata-finder).

## Оператор

Территориальный орган Росстата по Курской области (Курскстат).

## Объекты данных и показатели

- Пресс-выпуски по ключевым формам (социум, промышленность, доходы населения).
- Региональные таблицы (год, показатель, единица измерения — по файлам).
- Ключ связки: **код 46** (ТО), **ОКТМО/регион** в федеральных формах.

## Периодичность обновления

По **календарю** статистического наблюдения; крупные блоки — в помесячном и квартальном разрезе в зависимости от формы.

## Ограничения и особенности

- Статистика **не заменяет** бюджетную отчётность; для исполнения бюджета сверяйте [консолидированные бюджеты](/data-sources/regional/consolidated-budgets) и [УФК](/data-sources/regional/kursk-oblast-ufk).
