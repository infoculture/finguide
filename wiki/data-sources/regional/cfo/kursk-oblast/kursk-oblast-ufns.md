| :--- | :--- |

## Описание

URL **rn46** — установленный ФНС код регионального управления для Курской области. Страница даёт **навигацию** к сервисам и справочникам; для аналитики по налоговым сборам и юридически значимым выгрузкам подключайте [аналитический портал](/data-sources/federal/analytic-nalog-gov-ru) и [открытые наборы](/data-sources/federal/bo-nalog-gov-ru) (паспорта на `nalog.gov.ru`).

## Оператор

УФНС России по Курской области — https://www.nalog.gov.ru/rn46/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Региональная страница | https://www.nalog.gov.ru/rn46/ | HTML |
| Каталог открытых данных ФНС | https://www.nalog.gov.ru/opendata/ | HTML, CSV |

## Объекты данных и показатели

- Новости и пресс-релизы УФНС.
- Ссылки на **федеральные** справочники (ИНН, адреса инспекций и т.д.).
- Для бюджетной аналитики: агрегаты по налогам — через федеральные статформы и аналитический портал, а не через HTML главной УФНС.

## Периодичность обновления

Разная по подсистемам; открытые наборы — по **паспорту** каждого набора.

## Ограничения и особенности

- Автоматические HEAD/бот-запросы к `nalog.gov.ru` могут получать **403**; для скриптов используйте официальные endpoint’ы открытых данных и соблюдайте robots.txt.
