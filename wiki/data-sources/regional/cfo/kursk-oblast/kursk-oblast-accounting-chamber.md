# Курская область: контрольно-счётная палата

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный орган ВГФК |
| Тип доступа | Официальный сайт КСП |
| Основные форматы | HTML, PDF отчётов |
| Оператор | Контрольно-счётная палата Курской области |

Коротко: материалы **контроля и экспертизы** по использованию бюджетных средств — отчёты о проверках, заключения на проекты бюджета, мониторинги.

## Описание

На сайте публикуются **результаты контрольных мероприятий**, информационные сообщения и аналитика по эффективности бюджетных расходов. На странице финансов Правительства встречается также альтернативное имя домена **ksp46.ru** — при расхождении приоритетно используйте тот URL, который открывается у читателя и совпадает с официальной ссылкой на портале Правительства.

## Оператор

Контрольно-счётная палата Курской области.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Основной сайт | https://ksp.kursk.ru/ | HTML |
| Альтернативный домен (встречается в ссылках Правительства) | http://ksp46.ru/ | HTML |

## Объекты данных и показатели

- Отчёт о результатах контрольного мероприятия (объект, период, выводы).
- Заключение на проект закона об областном бюджете.
- Ключи: **наименование проверяемого**, год, тип расходов (по тексту отчёта).

## Периодичность обновления

По мере **плановых и внеплановых** проверок и годового цикла экспертизы бюджета.

## Ограничения и особенности

- КСП не является источником **первичных** бюджетных проводок; для кассы и лимитов сверяйтесь с [УФК](/data-sources/regional/kursk-oblast-ufk) и федеральными витринами.
