# Курская область: официальный сайт Курской областной Думы

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный законодательный орган |
| Тип доступа | Официальный парламентский сайт |
| Основные форматы | HTML, PDF повесток и законопроектов |
| Оператор | Курская областная Дума |
| Режим доступа | Открытый |

Коротко: **законодательная стадия** бюджетного цикла — закон (и изменения) об областном бюджете, проекты, экспертные материалы и протокольная информация.

## Описание

Сайт Думы — ключевой вход для **проектов** законов об областном бюджете, графиков чтений, приложений с таблицами доходов/расходов и материалов слушаний. **Официально опубликованные** тексты законов после подписания также сверяйте через [портал правовой информации](/data-sources/regional/kursk-oblast-regional-npa-portal).

## Объекты данных и показатели

- Законопроект об областном бюджете на очередной год (таблицы приложений).
- Протокольные и информационные материалы о слушаниях.
- Ключи связки: **номер законопроекта**, финансовый год, стадия рассмотрения.

## Периодичность обновления

Пик активности — **осенне-зимний** период принятия бюджета; вне сессии обновления точечные.

## Ограничения и особенности

- Структура меню может меняться; при устаревшей ссылке используйте поиск по сайту Думы.
