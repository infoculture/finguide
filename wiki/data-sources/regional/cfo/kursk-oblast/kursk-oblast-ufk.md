# Курская область: Управление Федерального казначейства

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональное представительство федерального органа кассового обслуживания |
| Тип доступа | Официальный сайт УФК |
| Основные форматы | HTML, PDF отчётности |
| Оператор | Управление Федерального казначейства по Курской области |
| Режим доступа | Открытый |

Коротко: **касса и отчётность** по исполнению бюджетов в части, обслуживаемой Казначейством, плюс региональные новости и разъяснения УФК.

## Описание

Сайт УФК дублирует и локализует **федеральные** сервисы Казначейства: новости территориального управления, разделы для клиентов бюджетной сферы, ссылки на **отчётность** и открытые данные. Для **агрегированных** машиночитаемых рядов используйте федеральные витрины Росказны и «Электронного бюджета».

## Объекты данных и показатели

- Публикации об исполнении бюджета в части кассы (в объёме, раскрываемом для граждан).
- Справочная информация для участников бюджетного процесса.

## Периодичность обновления

Отчётность — по **федеральному календарю**; новости УФК — по мере событий.

## Ограничения и особенности

- Персонифицированные данные по счетам **не** являются открытыми; не ожидайте полной детализации как в внутренних учётных системах.
