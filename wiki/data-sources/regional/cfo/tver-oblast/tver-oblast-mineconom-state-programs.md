## Описание

Раздел **gos-programmy-to** публикует структурированную информацию о государственных программах: перечни, паспорта, отчёты о реализации. Для анализа **расходов** удобно сопоставлять показатели программ с бюджетной классификацией в материалах Минфина и открытых бюджетных отчётах.

## Объекты данных и показатели

- Государственные программы: цели, индикаторы, мероприятия, ответственные органы.
- Связка с бюджетом: через коды и суммы в паспортах/отчётах; унифицированного открытого API в карточке нет.

## Периодичность обновления

По мере утверждения новых редакций программ и годовой отчётности.

## Ограничения и особенности

- Полнота показателей по годам может отличаться по программам.
