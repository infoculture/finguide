# Тверская область: УФНС России (региональная страница nalog.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональное представительство федерального органа (Тверская область) |
| Тип доступа | Публичный веб-портал (в составе nalog.gov.ru) |
| Основные форматы | HTML; федеральные открытые данные ФНС — преимущественно CSV/XML по наборам на nalog.gov.ru |
| Оператор | Управление Федеральной налоговой службы по Тверской области |
| Режим доступа | Открытый |

Коротко: **региональный вход** к сервисам и новостям ФНС для Тверской области; для **массовых выгрузок** (справочники инспекций, реестры и т.д.) используйте федеральный каталог наборов на nalog.gov.ru и соответствующие карточки в разделе `wiki/data-sources/federal/nalog/`.

## Описание

URL **nalog.gov.ru/rn69** ведёт на региональную витрину: контакты, новости, разъяснения и переходы к федеральным онлайн-сервисам. Для **бюджетной** аналитики регионального бюджета налоговые ряды полезны как контекст доходов; плановые и кассовые показатели бюджета субъекта смотрите у [Минфина области](/data-sources/regional/tver-oblast-ministry-of-finance).

## Объекты данных и показатели

- Региональные новости и сервисы ФНС; переходы к федеральным личным кабинетам и справочникам.
- Машиночитаемые наборы (ЕГРЮЛ/адреса инспекций и др.) — через федеральный каталог, не через отдельный «API региона 69».

## Периодичность обновления

Разная по типам материалов; для open data — по паспортам конкретных наборов на nalog.gov.ru.

## Ограничения и особенности

- Часть функциональности требует учётной записи налогоплательщика (для карточки не разбираем внутренние кабинеты).
