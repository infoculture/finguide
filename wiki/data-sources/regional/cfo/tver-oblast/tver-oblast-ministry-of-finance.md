# Тверская область: Министерство финансов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный финансовый орган субъекта РФ |
| Тип доступа | Публичный веб-сайт и файловые выкладки по разделам |
| Основные форматы | HTML; отчёты и формы — PDF, иногда XLSX |
| Оператор | Министерство финансов Тверской области |

Коротко: **главный региональный вход** по бюджету, долговой политике (в части раскрытия на сайте), отчётности и методическим материалам. Связанные витрины: [открытый бюджет](/data-sources/regional/tver-oblast-open-budget-portal), [региональные НПА](/data-sources/regional/tver-oblast-minfin-regional-npa), [УФК](/data-sources/regional/tver-oblast-ufk-roskazna).

## Описание

Сайт **tverfin.ru** концентрирует материалы областного бюджета: проект, исполнение, отчёты, методические рекомендации для учреждений и органов местного самоуправления. Для **официальных текстов законов** используйте [pravo.gov.ru](/data-sources/regional/tver-oblast-pravo-gov-regional-acts); для **казначейского исполнения** — публичные разделы [УФК](/data-sources/regional/tver-oblast-ufk-roskazna).

Министерство финансов Тверской области — канонический вход: https://www.tverfin.ru/

## Объекты данных и показатели

- Бюджетные законы и подзаконные акты (часть на сайте, канон опубликования — pravo.gov.ru).
- Отчёты об исполнении бюджета, материалы к «бюджету для граждан».
- Ключи: финансовый год, КБК/КОСГУ в таблицах (если публикуются), ведомственная структура расходов.

## Периодичность обновления

По бюджетному календарю и законодательству о раскрытии информации о бюджете.
