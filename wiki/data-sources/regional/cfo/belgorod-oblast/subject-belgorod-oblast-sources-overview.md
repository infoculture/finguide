# Белгородская область: источники данных

Коротко: эта страница перечисляет **опубликованные** карточки источников данных по субъекту. Публичные URL карточек имеют вид `/data-sources/regional/<slug>` и не зависят от вложенности файла в папках округа в репозитории.

## Карточки источников

- [Минфин Белгородской области (сайт)](/data-sources/regional/belgorod-oblast-minfin-beldepfin) — главный финансовый орган: бюджет, долг, отчётность.
- [«Бюджет для граждан» (раздел Минфина)](/data-sources/regional/belgorod-oblast-budget-for-citizens-beldepfin) — популярные материалы по областному бюджету.
- [Официальное опубликование НПА (zakonbelregion.ru)](/data-sources/regional/belgorod-oblast-official-legal-acts-zakonbelregion) — тексты законов и иных актов области.
- [Белгородская областная Дума](/data-sources/regional/belgorod-oblast-legislature-belduma) — законодательный процесс и проекты, включая бюджет.
- [Портал Губернатора и Правительства (belregion.ru)](/data-sources/regional/belgorod-oblast-government-portal-belregion) — общий портал органов исполнительной власти.
- [УФНС России по Белгородской области](/data-sources/regional/belgorod-oblast-fns-ufns-belgorod) — региональная витрина ФНС (код 31).
- [Белгородстат (ТО Росстата)](/data-sources/regional/belgorod-oblast-rosstat-belgorodobl) — официальная региональная статистика.
- [УФК по Белгородской области](/data-sources/regional/belgorod-oblast-federal-treasury-belgorod) — региональный сайт Федерального казначейства.
- [Портал госзакупок (Белгосзаказ)](/data-sources/regional/belgorod-oblast-procurement-portal-belgoszakaz) — региональный контур закупок.
- [ТФОМС Белгородской области (belfoms.ru)](/data-sources/regional/belgorod-oblast-tfoms-belfoms) — обязательное медицинское страхование.
- [Контрольно-счётная палата области](/data-sources/regional/belgorod-oblast-control-chamber-belksp) — внешний финансовый контроль.
