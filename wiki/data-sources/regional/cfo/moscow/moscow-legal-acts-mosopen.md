# Город Москва: система правовых актов (MosOpen / mosopen.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Город Москва** — **нормативные правовые акты** субъекта и органов города |
| Тип доступа | Публичный **банк НПА** |
| Основные форматы | **HTML**, **PDF** (в зависимости от документа) |
| Оператор | **Правительство Москвы** (система **mosopen.ru**) |
| Режим доступа | Открытое чтение |

Коротко: **канонический поиск** текстов **законов города**, **постановлений** и иных актов для связки с **бюджетом** и **госпрограммами**. Агрегированные **цифры бюджета** удобнее смотреть на **[«Открытом бюджете»](/data-sources/regional/moscow-open-budget)**.

## Описание

**mosopen.ru** используют для **проверки редакции** закона о бюджете, постановлений о предоставлении субсидий и иных финансовых НПА города. Для **проектов** законов МГД см. также сайт **[Московской городской Думы](/data-sources/regional/moscow-city-duma)**.

## Оператор

**Система правовых актов города Москвы** — https://www.mosopen.ru/documents (при недоступности **HTTPS** следуйте редиректу с **HTTP**).

## Объекты данных и показатели

- **НПА** с реквизитами (вид, номер, дата, статус).
- **Приложения** к актам (таблицы лимитов и т.п.) — в составе карточки документа.
- **Связка с бюджетом**: номер и дата акта для сопоставления с **плановыми** и **отчётными** материалами на **budget.mos.ru** и в **ГИИС «Электронный бюджет»**.
