# Город Москва и Московская область: территориальный орган Росстата (77.rosstat.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Территориальный орган Росстата** по **Москве и Московской области** (официальная статистика) |
| Тип доступа | Региональный портал **77.rosstat.gov.ru** |
| Основные форматы | **HTML**, **XLSX** (в публикациях) |
| Оператор | **Управление Федеральной службы государственной статистики по г. Москве и Московской области** |
| Режим доступа | Открытый доступ |

Коротко: **официальная статистика** по **двум субъектам** (город Москва и Московская область) в одном территориальном контуре **«77»**. Для **бюджетного исполнения** не подменяет **[казначейство](/data-sources/regional/moscow-ufk-roskazna)** и **ГИИС «Электронный бюджет»** — сверяйте **предмет показателя**.

## Качество и верификация

Опорные определения: **[«Измерения качества источника данных»](/reference/data-quality-dimensions)**.

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки (`last_verified`) | 2026-05-15 |
| Полнота (`data_completeness`) | **По статистике** — зависит от наблюдений Росстата; **не** полный финансовый учёт города |
| Машиночитаемость (`machine_readability`) | **Смешанная** — таблицы и файлы по отдельным выпускам |
| Юридическая значимость (`legal_significance`) | **Официальная** статистика Росстата |

## Описание

Портал **Мосстата** публикует **региональные срезы** официальной статистики. При анализе **только Москвы** внимательно фильтруйте **географию** показателей: часть таблиц относится к **агрегату «Москва + область»**, часть — к раздельным территориям (см. метаданные выпуска).

**Росстат** (территориальное управление) — https://77.rosstat.gov.ru/

## Объекты данных и показатели

- **Официальные статистические показатели** по методологии Росстата.
- **Календарь** и **анонсы** публикаций.
- **Не является** налоговой отчётностью — см. **[УФНС](/data-sources/regional/moscow-ufns-rn77)**.

## Периодичность обновления

По **календарю** статистических наблюдений Росстата; уточняйте дату в конкретном выпуске.
