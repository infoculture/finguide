# Город Москва: УФНС России по г. Москве (nalog.gov.ru/rn77)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Территориальное управление ФНС** по **городу Москва** (код региона **77**) |
| Тип доступа | **Публичный** раздел на общем домене **nalog.gov.ru** |
| Основные форматы | **HTML**; электронные сервисы и личные кабинеты — по правилам ФНС |
| Оператор | **УФНС России по г. Москве** |
| Режим доступа | Открытые разделы; персональные данные — через идентификацию |

Коротко: **региональный вход** к **сервисам и новостям** ФНС для налогоплательщиков Москвы. Для **агрегированной налоговой статистики** и **открытых наборов** используйте **[налоговую статистику ФНС](/data-sources/federal/nalog-statistics)** и карточку **[ФНС](/organizations/fns)**.

## Описание

Сайт **УФНС** дублирует часть навигации **федерального** портала с фильтром по **Москве**: инспекции, региональные новости, сервисы записи и т.д. Для **связки с бюджетом** используйте **ИНН/КПП** и налоговые **коды доходов бюджета** из методички Минфина, а не только HTML-страницы УФНС.

## Объекты данных и показатели

- **Сервисы и календарь отчётности** для налогоплательщиков на территории Москвы.
- **Ссылки** на федеральные **открытые данные** и **налоговую статистику**.
- **Не содержит** полного зеркала всех CSV ФНС — ищите набор по каталогу на **nalog.gov.ru**.

## Периодичность обновления

**По мере** обновления федерального портала и публикаций УФНС.
