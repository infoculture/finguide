| :--- | :--- |
| Уровень и охват | Региональный контур Рязанской области |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML; отдельные вложения (PDF, XLSX) по ссылкам внутри разделов |

Коротко: главная **официальная** витрина финансового блока области — от бюджетной документации до смежных разделов (в т.ч. навигация к открытому бюджету и нормативным актам на том же домене).

## Описание

Сайт Минфина Рязанской области — базовый вход для поиска **проектов и текстов** областного бюджета, методических материалов, ссылок на закупки и смежные сервисы. Для узких задач в wiki вынесены отдельные карточки: [банк НПА на сайте Минфина](/data-sources/regional/ryazan-oblast-minfin-normative-acts) и [«Открытый бюджет»](/data-sources/regional/ryazan-oblast-open-budget).

## Объекты данных и показатели

- Проекты и пояснительные записки к закону об областном бюджете (как правило, вложения и HTML-страницы по годам).
- Подборки нормативных актов по финансам и бюджету (см. отдельную карточку [НПА на сайте Минфина](/data-sources/regional/ryazan-oblast-minfin-normative-acts)).
- Материалы раздела «Открытый бюджет» (см. [отдельную карточку](/data-sources/regional/ryazan-oblast-open-budget)).
- Связка с другими наборами: год, вид документа (план/исполнение), при наличии — КБК/ОКТМО в таблицах вложений (нужна ручная выверка по конкретному файлу).

## Периодичность обновления

По событиям бюджетного цикла и решениям правительства области; для конкретных разделов уточняйте дату публикации на странице документа.

## Ограничения и особенности

- При автоматической загрузке возможны предупреждения TLS о цепочке сертификата; в браузере сайт может открываться без ошибок. Для скриптов может потребоваться явная настройка доверия к цепочке или ручная загрузка файлов.
- Не путать с федеральной витриной «Электронный бюджет» — см. [наборы budget.gov.ru](/data-sources/federal/budget-gov-ru-datasets) для сверки федерального контура.
