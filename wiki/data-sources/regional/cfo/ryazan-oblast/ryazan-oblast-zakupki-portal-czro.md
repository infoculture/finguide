# Рязанская область: портал государственных закупок (czro.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный контур закупок Рязанской области |
| Тип доступа | Веб-витрина; часть материалов может выгружаться в XML/CSV (зависит от настроек площадки) |
| Основные форматы | HTML, XML, CSV (по доступным выгрузкам) |

Коротко: **региональная** точка доступа к материалам закупок для нужд Рязанской области; дополняет федеральную интеграцию с ЕИС.

## Описание

Портал **czro.ru** используется заказчиками и поставщиками для работы с региональным контуром закупок Рязанской области. Для федерального реестра контрактов, единой информационной системы и агрегированной статистики см. карточки [Закупки: обзор федерального контура](/data-sources/federal/procurement) и [Статистика ЕИС](/data-sources/federal/zakupki-eis-statistics).

Региональная витрина закупок — канонический URL: https://czro.ru/

## Объекты данных и показатели

- Извещения, протоколы, проекты контрактов (в модели данных площадки).
- Идентификаторы закупок и контрактов для связки с [ЕИС](/data-sources/federal/procurement) (при наличии в выгрузках).
- Финансовые лимиты и обеспечение заявок — в документации конкретной процедуры.

## Периодичность обновления

Оперативно по мере размещения закупочной документации.

## Ограничения и особенности

- Для единой сквозной аналитики по стране часто удобнее начинать с федерального контура ЕИС, а региональный портал использовать для региональных особенностей и дублей документации.
