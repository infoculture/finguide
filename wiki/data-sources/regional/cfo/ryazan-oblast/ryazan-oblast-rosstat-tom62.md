# Рязанская область: территориальный орган Росстата (ТОМ №62)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональная официальная статистика по Рязанской области (федеральный орган в региональном представительстве) |
| Тип доступа | Публичный веб-портал ТОМ |
| Основные форматы | HTML, публикации в форматах таблиц и PDF (в зависимости от выпуска) |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Рязанской области |
| Режим доступа | Открытый |

Коротко: **официальный** региональный вход к статистике по Рязанской области: демография, экономика, финансы субъектов и муниципалитетов, инвестиции — для макроанализа и обоснования показателей бюджета.

## Описание

ТОМ №62 агрегирует **региональные официальные статистические данные** по Рязанской области. Для загрузки машиночитаемых наборов и поиска показателей по федеральному контуру используйте [открытые данные Росстата](/data-sources/federal/rosstat-opendata) и [ShowData / finder](/data-sources/federal/rosstat-showdata-finder).

## Оператор

Территориальный орган Федеральной службы государственной статистики по Рязанской области — https://62.rosstat.gov.ru/

## Объекты данных и показатели

- Региональные показатели ВРП, инвестиций, демографии, рынка труда (по линии опубликованных форм).
- Муниципальная статистика и сопоставления по ОКТМО — в пределах опубликованных таблиц.
- Финансовые показатели сектора государственного управления и смежные ряды — по мере публикации в официальных коммуникациях Росстата.

## Периодичность обновления

По календарю официального статистического наблюдения (месячные, квартальные, годовые выпуски).

## Ограничения и особенности

- Часть детальных рядов доступна только через федеральные инструменты или платные/закрытые сервисы — уточняйте по конкретному показателю.
