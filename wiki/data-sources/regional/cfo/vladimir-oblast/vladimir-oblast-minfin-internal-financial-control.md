# Владимирская область: внутренний финансовый контроль (Минфин)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Внутренний** финансовый контроль органов **исполнительной** власти субъекта |
| Тип доступа | Раздел сайта Минфина области |

| Оператор | Министерство финансов Владимирской области (ОВФК) |

## Описание

Раздел посвящён **ОВФК** и мерам внутреннего финансового контроля при исполнении областного бюджета. Нормативная база и **законы** о бюджете публикуются на сайте [Законодательного собрания](/data-sources/regional/vladimir-oblast-legislative-assembly-laws).
