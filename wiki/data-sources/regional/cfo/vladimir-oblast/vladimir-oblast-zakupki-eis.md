# ЕИС закупок: вход для анализа закупок

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** система; данные по заказчикам **всех** регионов |
| Тип доступа | Веб-портал и **СОИ** (машиночитаемые комплекты по правилам оператора) |
| Основные форматы | HTML, XML (через СОИ) |
| Оператор | Федеральная контрактная система (ЕИС) |
| Режим доступа | Открытый (для публичных сведений) |

Коротко: основной **официальный** источник по закупкам **44-ФЗ** и **223-ФЗ** для заказчиков на территории Владимирской области и других субъектов. Карточка заведена в **областном** каталоге wiki как типичный вход аналитика по региону; юридически источник **федеральный**.

## Описание

Для выборки по **Владимирской области** используйте фильтры заказчика, региональные признаки в извещениях и реестровые идентификаторы. Архитектура выгрузок и переход с FTP на СОИ раскрыты в карточке ИС **[ЕИС](/information-systems/federal/zakupki)** и в обзоре **[сведения о закупках](/data-sources/federal/procurement)**.

## Объекты данных и показатели

- **Извещения**, **контракты**, изменения, исполнение — в моделях ЕИС.
- Ключи связки: **реестровый номер закупки/контракта**, **ИНН** заказчика и поставщика, **ОКПД2**, регион.

## Периодичность обновления

Онлайн-обновление по мере публикации участниками.

## Ограничения и особенности

- Массовые выгрузки — только в рамках **СОИ** и учётных записей по регламенту оператора (см. ИС ЕИС).
