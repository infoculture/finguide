# Смоленская область: территориальный орган Росстата

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Официальная статистика** по Смоленской области (ТО Росстата, код 67) |
| Тип доступа | Региональный поддомен rosstat.gov.ru |
| Основные форматы | HTML; табличные вложения по мере публикации |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Смоленской области |
| Режим доступа | Открытый |

Коротко: **региональный вход** к официальной статистике и комментариям Смоленскстата. Для **машиночитаемых массивов и ЕМИСС** используйте федеральную витрину [Росстат и открытые данные](/data-sources/federal/rosstat-opendata).

## Описание

Сайт ТО Росстата полезен для **контекста по региону**: оперативные сводки, ссылки на публикации, иногда региональные таблицы. Для **сопоставления с бюджетными рядами** см. также федеральные наборы по финансам сектора госуправления, например [консолидированный бюджет РФ (статсводка)](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf).

## Объекты данных и показатели

- **Региональные сводки** социально-экономического положения (в формате публикаций ТО).
- **Ссылки на федеральные** базы показателей и классификаторы.
- Ключи: **код субъекта 67**, **ОКТМО** / территориальные коды в таблицах Росстата, **период** наблюдения.

## Периодичность обновления

Согласована с графиком официальной статистики и региональными пресс-релизами.

## Ограничения и особенности

- Сетевой доступ к поддоменам Росстата иногда нестабилен для простых HTTP-клиентов; при сбоях используйте федеральный [ЕМИСС/открытые данные](/data-sources/federal/rosstat-opendata).

## Примеры доступа

```bash
curl -sI -A "Mozilla/5.0" --max-time 25 "https://67.rosstat.gov.ru/" | head -n 5
```
