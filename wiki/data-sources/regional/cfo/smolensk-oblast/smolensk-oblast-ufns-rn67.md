# Смоленская область: УФНС России по Смоленской области

| :--- | :--- |
| Уровень и охват | **Федеральный орган** на территории Смоленской области (код 67) |

## Описание

Раздел `rn67` используют для **локальных новостей**, расписаний, региональных статистических подборок и ссылок на единые федеральные сервисы. Для **сопоставления с бюджетом области** после получения налоговых агрегатов см. [Минфин Смоленской области](/data-sources/regional/smolensk-oblast-ministry-of-finance).

## Объекты данных и показатели

- **Региональные** публикации и справочные материалы УФНС.
- **Статистика и аналитика** в структуре сайта ФНС (подразделы могут меняться).
- Ключи: **код субъекта 67**, **налоговые периоды**, классификаторы в таблицах ФНС.

## Периодичность обновления

По мере выхода отчётов и изменений законодательства; федеральные релизы статистики — по календарю ФНС.

## Ограничения и особенности

- Прямой `curl` без заголовка браузера иногда получает **403**; для проверки используйте браузер или `-A "Mozilla/5.0"`.

## Примеры доступа

```bash
curl -sI -A "Mozilla/5.0" --max-time 20 "https://www.nalog.gov.ru/rn67" | head -n 5
```
