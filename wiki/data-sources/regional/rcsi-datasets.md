# data.rcsi.science — машиночитаемые региональные данные

## Описание

Каталог машиночитаемых датасетов по региональной экономике, бюджетам и социальной сфере. Альтернативный источник для исследовательских целей.

## Оператор

РНИИИС — https://data.rcsi.science/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Каталог | https://data.rcsi.science/ | HTML/CSV |

## Состав данных

- Региональные бюджетные показатели
- Социально-экономические показатели регионов
