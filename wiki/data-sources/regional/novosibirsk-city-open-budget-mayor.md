# Новосибирск: бюджет для граждан и открытые данные мэрии (novo-sibirsk.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Город Новосибирск** (муниципальное образование) — публичные разделы **Департамента финансов и налоговой политики мэрии** и **портал открытых данных** |
| Тип доступа | **Публичные** HTML-витрины и **каталог наборов** открытых данных |
| Основные форматы | **HTML**; на портале opendata — **CSV** и иные выгрузки по наборам (см. паспорта) |
| Оператор | **Мэрия города Новосибирска** (официальный сайт **novo-sibirsk.ru**), в частности департамент финансов и налоговой политики |
| Режим доступа | Открытое чтение публичных разделов |

Коротко: **городской** вход к **«Бюджету для граждан»** и к **открытым данным** инфраструктуры и смежных тем; для **единого федерального** поиска по бюджетам МО и машиночитаемых срезов по регламенту портала используйте **[муниципальные бюджеты на `budget.gov.ru`](/data-sources/regional/municipal-budgets)** и **[наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** с фильтром по **ОКТМО** территории города.

## Описание

Раздел **«Бюджет для граждан»** на сайте мэрии предназначен для **доступного** изложения бюджетной политики города (цели, приоритеты, ключевые показатели в понятной форме). **Портал открытых данных** **opendata.novo-sibirsk.ru** публикует **тематические наборы** (в т.ч. вне бюджетной тематики); для аналитики бюджета сверяйте **название и паспорт набора**, год и **территориальную привязку**.

## Оператор

**Мэрия города Новосибирска** — https://novo-sibirsk.ru/

## URL и точки доступа

| Назначение | URL |
| --- | --- |
| Бюджет для граждан (Депфин мэрии) | https://novo-sibirsk.ru/dep/finance/budget/people/ |
| Корень раздела финансов и бюджета | https://novo-sibirsk.ru/dep/finance/docs/ |
| Портал открытых данных мэрии | http://opendata.novo-sibirsk.ru/ (см. ограничения по **HTTPS**) |

Структура подразделов на **novo-sibirsk.ru** может меняться при обновлении сайта; при потере глубокой ссылки начните с **корня** домена и раздела «Департаменты».

## Объекты данных и показатели

- **Пояснительные материалы и показатели** «бюджет для гражданина» (агрегированный уровень, не полный дубль казначейских форм).
- **Наборы открытых данных** города — по паспортам на **opendata** (перечень и тематика **меняются**).
- **Связка с федеральным контуром** — через **[муниципальные бюджеты](/data-sources/regional/municipal-budgets)** и **[ОКТМО](/glossary/oktmo)** территории Новосибирска.

## Периодичность обновления

По мере публикаций мэрии и выгрузок на **opendata**; для срезов в контуре **ГИИС «Электронный бюджет»** — даты в **метаданных наборов** на `budget.gov.ru`.

## Ограничения и особенности

- Не подменяйте **юридически значимый** пакет документов местного бюджета **только** HTML-витриной без сверки с **официальными** текстами решений на сайте МО и с **федеральным** порталом при необходимости.
- **HTTPS** для **opendata.novo-sibirsk.ru** при проверке из командной строки может давать ошибку цепочки сертификата; для **ручной** проверки доступности используйте браузер или `curl -k` осознанно. По состоянию на дату проверки **HTTP**-корень отвечал **200**.
- **Консолидированный бюджет Новосибирской области** и **закупки** — **иные** уровни и операторы; не смешивайте с муниципальным бюджетом **города** без проверки **ОКТМО** и полномочий.

## Примеры доступа

Проверка раздела «Бюджет для граждан» (HTTPS):

```bash
curl -fsSL -A "finguide-nsk-budget-people-check/1.0" -o /dev/null -w "%{http_code}\n" \
  "https://novo-sibirsk.ru/dep/finance/budget/people/"
```

Проверка корня портала opendata (при ошибке сертификата на HTTPS — **HTTP**-корень):

```bash
curl -fsSL -A "finguide-nsk-opendata-check/1.0" -o /dev/null -w "%{http_code}\n" \
  "http://opendata.novo-sibirsk.ru/"
```
