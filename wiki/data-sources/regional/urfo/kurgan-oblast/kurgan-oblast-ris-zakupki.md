## Описание

Портал **zakupki.45fin.ru** — типовая **региональная информационная система (РИС)** в сфере закупок: реестр процедур, разделы для заказчика и поставщика, нормативные ссылки. Для сопоставления с **консолидированным бюджетом** и планами-графиками используйте материалы департамента финансов; для федерального контура — ЕИС.

## Объекты данных и показатели

- **Процедуры закупок**: извещения, протоколы, договоры (как файлы на страницах процедур).
- **Ключи связки**: реестровый номер контракта/извещения (при дублировании с ЕИС), ИНН заказчика, код **ОКПД2**, суммы лотов.
- **Пробелы**: нет гарантии единого открытого API; для выгрузок больших массивов предпочтительны федеральные каналы ЕИС.

## Периодичность обновления

По мере публикации заказчиками (непрерывно в активных закупках).

## Ограничения и особенности

- Часть закупок публикуется **только в ЕИС**; региональная РИС может выполнять роль удобной региональной «витрины».
- Не полагайтесь на веб-скрапинг без проверки `robots.txt` и нагрузочных ограничений.
