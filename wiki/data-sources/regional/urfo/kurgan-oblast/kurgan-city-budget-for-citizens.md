# Курган: «Бюджет для граждан» (Администрация города Кургана)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Бюджет муниципального образования **город Курган** (ОКТМО города) |
| Тип доступа | Публичный раздел официального сайта Администрации |

Коротко: **муниципальный «Бюджет для граждан»** для столицы области: наглядные материалы по **местному бюджету**. Для **областного** бюджета используйте сайт **департамента финансов**; для **муниципальных НПА** — **реестр** на kurganobl.ru и сайт конкретного ОМС.

## Описание

Раздел размещён в структуре сайта города и предназначен для **прозрачности местного бюджета**: показатели доходов и расходов, справочные материалы, иногда — **отчёты об исполнении**. Для **сводной аналитики** по субъекту РФ сочетайте с **областным** «Бюджетом для граждан» и федеральными наборами при необходимости консолидации.

## Объекты данных и показатели

- **Агрегированные показатели** бюджета города (доходы/расходы).
- **Отчёты об исполнении** местного бюджета (по мере публикации).
- **Ключи связки**: **ОКТМО** города Кургана, бюджетный год, коды **КБК/КОСГУ** (если приводятся).
- **Пробелы**: деталь первички и казначейские обороты — в федеральном контуре казначейства, не на городской витрине.

## Периодичность обновления

При подготовке **проекта местного бюджета**, **уточнений** и **годовой отчётности**.

## Ограничения и особенности

- Карточка описывает **только город Курган**; другие ОМС области имеют собственные сайты.
