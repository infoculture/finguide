# Ханты-Мансийский автономный округ — Югра: источники данных

Коротко: перечислены **опубликованные** карточки источников по субъекту. Публичный URL каждой карточки — `/data-sources/regional/<slug>`, где `<slug>` совпадает с **именем файла** без `.md` (путь в репозитории может включать округ и субъект, URL — плоский).

## Карточки источников

- [Департамент финансов Югры (depfin.admhmao.ru)](/data-sources/regional/khmao-yugra-depfin) — бюджетная политика, исполнение бюджета, ссылки на отчётность и смежные материалы.
- [Открытый бюджет и «бюджет для граждан» (раздел Депфина)](/data-sources/regional/khmao-yugra-depfin-open-budget-citizens) — объяснение планирования и исполнения окружного бюджета в доступной форме.
- [Портал открытых данных Югры (data.admhmao.ru)](/data-sources/regional/khmao-yugra-open-data-portal) — каталог наборов, в том числе с финансовой тематикой.
- [Портал государственных программ Югры (gp-portal.admhmao.ru)](/data-sources/regional/khmao-yugra-state-programs-portal) — цели, объёмы финансирования и материалы по реализации госпрограмм округа.
- [Раздел «Правовые акты» на admhmao.ru](/data-sources/regional/khmao-yugra-admhmao-legal-acts) — официальная публикация НПА органов власти Югры, включая бюджетное законодательство.
- [Дума ХМАО — Югры (dumahmao.ru)](/data-sources/regional/khmao-yugra-duma-dumahmao) — законы о бюджете, постановления и парламентские материалы по финансовым вопросам.
- [Счётная палата ХМАО — Югры (sphmao.ru)](/data-sources/regional/khmao-yugra-accounts-chamber-sphmao) — результаты внешнего муниципального финансового контроля и аудиторские материалы.
- [ТФОМС ХМАО — Югры (ofoms.ru)](/data-sources/regional/khmao-yugra-tfoms-ofoms) — отчётность и публичные сведения территориального фонда ОМС.
- [УФНС России по ХМАО — Югре (nalog.gov.ru/rn86)](/data-sources/regional/khmao-yugra-ufns-nalog-rn86) — региональные разделы и сервисы ФНС по субъекту.
- [Управление Федерального казначейства по ХМАО — Югре](/data-sources/regional/khmao-yugra-roskazna-hantymansiysk) — казначейское сопровождение и публичные разделы по округу.
- [Росстат: раздел по ХМАО — Югре (Тюменьстат)](/data-sources/regional/khmao-yugra-rosstat-tumen-hmao-section) — официальная статистика по округу на сайте межрегионального управления.
- [ЕИС (zakupki.gov.ru): вход с привязкой к субъекту](/data-sources/regional/khmao-yugra-eis-zakupki-gov-ru-regional-entry) — федеральная витрина закупок; карточка фиксирует удобную точку входа для Югры.
- [Портал «Закупки Югры» (zakupki.admhmao.ru)](/data-sources/regional/khmao-yugra-procurement-zakupki-admhmao) — региональная ИС и витрина закупок округа, дополняющая ЕИС.
- [Электронный бюджет Ханты-Мансийского района (hmrn.ru)](/data-sources/regional/khmao-yugra-khanty-mansi-district-electronic-budget) — муниципальный пример бюджетной витрины внутри округа.
