## Описание

**74.rosstat.gov.ru** — территориальная витрина **Росстата** для Челябинской области: отчёты по экономике, демографии, отраслям. Используется для **контекста** при анализе бюджетных рядов (ВРП, занятость, цены и т.д.), но не заменяет [бюджетную отчётность Минфина](/data-sources/regional/chelyabinsk-oblast-minfin).

## Объекты данных и показатели

- Социально-экономические показатели по Челябинской области.
- Региональные сборники и таблицы (форматы по разделам).
- Ключи связки: год, ОКТМО/территория (если указаны), код показателя Росстата.

## Периодичность обновления

По **официальному календарю** публикаций Росстата (ежемесячно, ежеквартально, ежегодно — по показателю).
