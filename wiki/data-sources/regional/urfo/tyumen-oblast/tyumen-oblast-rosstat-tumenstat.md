## Описание

Тюменьстат сопровождает **сбор и распространение** официальной статистики по трём субъектам. Для аналитики **государственных финансов** на федеральном уровне удобны тематические файлы Росстата, например [консолидированный бюджет РФ](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf). Институциональный контекст — [Росстат (организация)](/organizations/rosstat).

## Объекты данных и показатели

- **Пресс-релизы** и краткие итоги по ключевым показателям по регионам зоны ответственности.
- **Календарь** и **формы** статистического учёта для респондентов.
- **Не путать** «областной» аналитический запрос с данными **ХМАО и ЯНАО** — сайт обслуживает три субъекта.

## Периодичность обновления

Определяется **статистическим** календарём и планом публикаций Росстата по конкретным формам и разделам.

## Ограничения и особенности

- Для **одной только Тюменской области** часть материалов может быть **смешана** с публикациями по ХМАО и ЯНАО в рамках одного территориального управления.
