# Республика Карелия: УФНС России по Республике Карелия (nalog.gov.ru/rn10)

| Признак | Значение |
| :--- | :--- |

| Тип доступа | Публичный сайт УФНС в составе nalog.gov.ru |
| Основные форматы | HTML; переходы в федеральные сервисы и формы |

| Режим доступа | Открытый просмотр; часть сервисов — после авторизации |

Коротко: **региональный вход в контур ФНС** для налогоплательщиков: локальные новости, разъяснения, расписания и ссылки на единые федеральные инструменты. Для аналитики по **налоговым поступлениям в разрезе субъектов** чаще используют открытую статистику ФНС и бюджетные материалы Минфина.

## Описание

Сайт УФНС дублирует типовую структуру территориальных управлений ФНС: **разъяснения**, контакты, региональные новости и маршруты к **электронным сервисам** (проверка задолженности, запись на приём, личный кабинет и т.д.). Для сопоставления с бюджетом субъекта используйте материалы [Минфина Республики Карелия](/data-sources/regional/republic-of-karelia-minfin); для статистики по налогам — федеральные открытые разделы ФНС.

## Объекты данных и показатели

- **Сервисные объекты**: формы, инструкции, календарь отчётности, новости.
- **Персональные данные** по конкретным налогоплательщикам — только через авторизованные сервисы, не для открытой аналитики.
- **Связка** с **ОКТМО/КПП** и региональными реестрами при построении цепочек «организация — налог — бюджет».

## Периодичность обновления

Непрерывно для новостей и сервисов; отчётные календари обновляются по нормативному графику.

## Ограничения и особенности

- Автоматизированные запросы к порталу ФНС могут ограничиваться политиками WAF; для воспроизводимых рядов используйте публичные **статистические** продукты ФНС.
- Ответ **403** для некоторых автоматических клиентов не означает закрытость для пользователя в браузере.
