# Республика Карелия: Карелиястат (территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Республика Карелия; официальная региональная статистика в контуре Росстата |
| Тип доступа | Официальный сайт территориального органа (код 10) |
| Основные форматы | HTML; пресс-выпуски и вложения (PDF/XLSX) |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Республике Карелия |
| Режим доступа | Открытый просмотр |

Коротко: **региональный вход Росстата** для оперативных сообщений и региональных обзоров; для **воспроизводимых открытых рядов** и API ориентируйтесь на [федеральные витрины открытых данных Росстата](/data-sources/federal/rosstat-opendata) и тематические сборники (например [«Финансы России»](/data-sources/federal/rosstat-finances-of-russia)).

## Описание

Карелиястат публикует **региональные пресс-релизы**, календарь статработ, краткие комментарии к данным по Республике Карелия и навигацию к **единым федеральным** инструментам Росстата. Для финансово-бюджетных показателей важно различать **официальную статистику** и **бюджетную отчётность** Минфина субъекта.

## Объекты данных и показатели

- **Региональные статистические наблюдения** в публикуемых формах (социально-экономика, демография, отраслевые блоки).
- **Финансовые блоки** — часто как агрегаты в пресс-выпусках; за машинными рядами идите на федеральные наборы.
- **Ключи связки**: код субъекта 10, ОКТМО, классификаторы Росстата.

## Периодичность обновления

По календарю Росстата и при публикации внеочередных оперативных оценок.

## Ограничения и особенности

- Региональный сайт может быть менее стабилен для автоматизации, чем федеральные API.
- Сопоставление с **бюджетом** требует осторожности в определениях показателей (СНС, казначейские кассы, отчётность Минфина).
