# Калининградская область: Управление Федерального казначейства

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральный** орган на **территории** Калининградской области — **УФК** (Росказна) |
| Тип доступа | Публичный **портал** территориального управления |
| Основные форматы | **HTML**; **PDF** и иные вложения по ссылкам |
| Оператор | **Управление Федерального казначейства по Калининградской области** |
| Режим доступа | Открытое чтение публичных разделов |

Коротко: **официальный** региональный **вход** к материалам **Росказны** для **39 региона**; для **массовых** открытых **казначейских** наборов и **методологии** используйте федеральные витрины (см. связанные страницы) и **[Минфин области](/data-sources/regional/kaliningrad-oblast-minfin)** для **регионального** бюджетного контура.

## Описание

Сайт **kaliningrad.roskazna.gov.ru** — типичная **точка входа** к **региональным** контактам, **новостям** и **справочной** информации УФК. Для **аналитики** казначейского **исполнения** и **открытых** выгрузок используйте **[портал Росказны](/information-systems/federal/roskazna-portal)** и карточку **[отчётности и открытых разделов](/data-sources/federal/roskazna-reports)** на федеральном уровне.

## Оператор

**Управление Федерального казначейства по Калининградской области** — канонический вход: https://kaliningrad.roskazna.gov.ru/

| --- | --- | --- |

## Объекты данных и показатели

- **Новости** и **анонсы** территориального управления.
- **Организационная** информация (режим работы, контакты).
- Возможные **отчёты** и **материалы** в **PDF**, если вынесены в публичные разделы (перечень **не** фиксирован в карточке).

## Периодичность обновления

По мере **публикаций** на сайте управления; **единой** периодичности для всех типов материалов нет.

## Ограничения и особенности

- Региональный сайт **не** заменяет **федеральные** **реестры** и **массовые** **open data**-витрины Росказны.
- **URL** внутренних разделов могут меняться при обновлении CMS.

curl -fsSL --connect-timeout 20 -A "finguide-roskazna39-check/1.0" -o /dev/null -w "%{http_code}\n" \
