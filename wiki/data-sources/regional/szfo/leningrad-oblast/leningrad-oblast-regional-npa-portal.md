# Ленинградская область: Портал правовых актов (lenoblpravo.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональный** справочно-правовой контур **НПА** субъекта |
| Тип доступа | Публичный **веб-портал** с поиском |
| Основные форматы | **HTML**; **PDF/DOC** вложения по карточкам актов |
| Оператор | **Органы государственной власти Ленинградской области** (публикация через **lenoblpravo.ru**) |
| Режим доступа | Открытое чтение публичных разделов |

Коротко: **центральный** вход для **поиска** и **чтения** региональных НПА, включая **законы о бюджете**, **постановления** и иные акты. Для **стадии законопроекта** и **парламентских** материалов параллельно используйте **[Законодательное собрание](/data-sources/regional/leningrad-oblast-legislative-assembly)**; для **федерального** официального опубликования см. **[ОИППИ pravo.gov.ru](/information-systems/federal/pravo-gov-ru)**.

## Описание

**lenoblpravo.ru** — типичный **региональный правовой портал** с **классификаторами**, **поиском** и **карточками** документов. Для **бюджетного** анализа важны **законы** и **подзаконные** акты о **расходах**, **налогах** субъекта и **межбюджетных** отношениях.

## Оператор

**Правовой портал Ленинградской области** — канонический вход (**HTTPS**): https://lenoblpravo.ru/

| Назначение | URL | Формат |

## Объекты данных и показатели

- **Карточки** НПА: вид, номер, дата, статус, **вложения** с полным текстом.
- **Классификация** по органам и темам (включая **финансы** и **бюджет**).
- **Не** содержит **первичных** учётных данных **исполнения** бюджета по платежам.

## Периодичность обновления

**По мере** официального **внесения** актов в базу; всплески — после **пакетов** постановлений и **бюджетного** цикла.

## Ограничения и особенности

- **Юридические** тонкости **вступления в силу** и **официального** опубликования уточняйте по **тексту** акта и **регламентам** субъекта; **федеральные** акты ищите в **[ОИППИ](/information-systems/federal/pravo-gov-ru)**.
