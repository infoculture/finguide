# Ленинградская область: Петростат (территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** служба (**Росстат**), **территориальный** орган по **СПб** и **Ленинградской области** (**код 78**) |
| Тип доступа | Публичный **веб-портал** (`78.rosstat.gov.ru`) |
| Основные форматы | **HTML**; **табличные** вложения (**XLSX** и др.) по ссылкам разделов |

## Описание

**Петростат** ведёт **государственный статистический учёт** на территории **двух** субъектов (**47** и **41** в иной нумерации не путать: **код 78** — **исторический** номер **территориального** органа **СПб+ЛО**). На **78.rosstat.gov.ru** публикуются **региональные** **обзоры**, **выпуски** и **ссылки** на **формы**; **сквозная** **машинная** выгрузка всех рядов **не** гарантируется на **одном** URL.

## Объекты данных и показатели

- **Региональные** статистические **обзоры** и **таблицы** (в т.ч. по **финансам организаций** — по мере публикации).
- **Метаданные** и **пояснения** к формам внутри разделов.
- **Не** путать с **бюджетным исполнением** по **казначейским** лицевым счетам.

## Периодичность обновления

По **официальному** календарю статистических **операций** и **релизов**; **месячные** и **годовые** циклы.

## Ограничения и особенности

- Некоторые **TLS-клиенты** (например, узкие сборки `curl`) могут ругаться на **цепочку сертификатов** хоста **`78.rosstat.gov.ru`**; в **браузере** обычно открывается штатно — при **массовой** загрузке проверяйте окружение.
- Для **единых** **скачиваемых** наборов по стране ориентируйтесь на **[rosstat.gov.ru/opendata](/data-sources/federal/rosstat-opendata)**.
