# Ленинградская область: Электронный магазин (закупки малого объёма)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональный** контур **закупок** малого объёма и **сопоставимых** процедур |
| Тип доступа | Публичный **веб-интерфейс** (каталог предложений и процедур) |
| Основные форматы | **HTML** |
| Оператор | **Государственный заказчик** / **оператор площадки** в логике **lenreg.ru** (см. импрессум на сайте) |
| Режим доступа | Публичное чтение карточек; **участие** — по правилам площадки |

Коротко: **региональная** альтернатива/дополнение к **федеральному** **ЕИС** для **узкого** класса закупок. Для **сквозной** аналитики **контрактов** и **планов-графиков** используйте **[ЕИС](/information-systems/federal/zakupki)**; для **бюджетных** сумм и **КБК** — **[«Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** и **[Комитет финансов](/data-sources/regional/leningrad-oblast-finance-committee)**.

## Описание

**zakupki.lenreg.ru** — **электронный магазин** для типовых **малых** закупок: **карточки** товаров/услуг, **извещения**, **результаты**. Это **не** полная копия **ЕИС**, а **региональный** операционный слой.

## Оператор

**Электронный магазин Ленинградской области** — канонический вход: https://zakupki.lenreg.ru/

## URL и точки доступа

| Назначение | URL | Формат |
| --- | --- | --- |
| Публичная витрина | https://zakupki.lenreg.ru/ | HTML |

## Объекты данных и показатели

- **Карточки** закупок/предложений: предмет, **НМЦК**, сроки, **заказчик** (в объёме, предусмотренном интерфейсом).
- **История** процедур в границах доступного **архива** площадки.
- **Склейка** с **ЕИНН**, **ИНН** и **идентификаторами** контрактов ЕИС — **вручную** или через **федеральные** выгрузки.

## Периодичность обновления

**Непрерывно** в рабочее время размещения процедур.

## Ограничения и особенности

- Площадка может быть за **антибот-фронтом**; для **технических** проверок используйте **идентифицируемый** User-Agent.
- Для **юридически** значимых **реестровых** реквизитов в **федеральном** контурe ориентируйтесь на **[ЕИС](/information-systems/federal/zakupki)**.
