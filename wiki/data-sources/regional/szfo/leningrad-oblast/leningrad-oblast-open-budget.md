# Ленинградская область: Открытый бюджет

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональный** контур **областного** бюджета для **граждан** и **аналитиков** |
| Тип доступа | Публичный **тематический портал** |
| Основные форматы | **HTML** (часто с **JavaScript**-визуализациями) |
| Оператор | **Комитет финансов Ленинградской области** (типичная практика размещения «открытого бюджета») |
| Режим доступа | Открытое чтение публичных разделов |

Коротко: **специализированная** витрина **«бюджет для граждан»**: агрегированные показатели, графики и пояснения без обязательства дублировать полный **казначейский** учёт. Для **сырых** открытых наборов и **сквозной** методики комбинируйте с **[наборами «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)** и сайтом **[Комитета финансов](/data-sources/regional/leningrad-oblast-finance-committee)**.

## Описание

**budget.lenobl.ru** — типичный **«Open Budget»**-портал: **наглядность** планов и исполнения областного бюджета, разделы для неспециалистов. Для **юридически** значимых текстов законов о бюджете используйте **[Законодательное собрание](/data-sources/regional/leningrad-oblast-legislative-assembly)** и **[банк НПА](/data-sources/regional/leningrad-oblast-regional-npa-portal)**.

## Объекты данных и показатели

- **Агрегированные** доходы и расходы областного бюджета по **разделам**/**группам** показателей (конкретные срезы зависят от редакции портала).
- **Пояснительные** записки, **инфографика**, иногда **скачиваемые** файлы (при наличии).
- **Не** заменяет **ЕИС**, **казначейские** реестры и **налоговую** отчётность.

## Периодичность обновления

Обычно **обновляется** вокруг **утверждения** бюджета, **уточнений** и **отчётов** об исполнении; точные даты — в новостной ленте **Комитета финансов**.

## Ограничения и особенности

- Публичный **HEAD/GET** без браузерного `User-Agent` может получать **403** от защитного фронта; для скриптов задавайте **идентифицируемый** User-Agent и уважайте нагрузку.
- Показатели **согласуются** с официальными документами, но **могут отличаться** по детализации от федеральных **open data**-выгрузок из‑за **округлений** и **срезов**.
