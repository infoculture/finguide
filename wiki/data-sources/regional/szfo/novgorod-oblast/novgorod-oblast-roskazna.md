# Новгородская область: Управление Федерального казначейства (Росказна)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный орган на территории Новгородской области (казначейское обслуживание бюджетов) |
| Тип доступа | Официальный региональный сайт Росказны |
| Основные форматы | HTML, PDF (инструкции, отчётность — по разделам) |
| Оператор | Управление Федерального казначейства по Новгородской области |
| Режим доступа | Открытый для общедоступных разделов; часть сервисов — для участников бюджетного процесса |

Коротко: **казначейский** контур исполнения бюджетов на регионе: публичные сведения об УФК, разделы для контрагентов и открытые материалы, связанные с казначейским сопровождением. Для **ведомственной** бюджетной отчётности субъекта — [Минфин области](/data-sources/regional/novgorod-oblast-minfin); для федеральной аналитики — [ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb) и [наборы budget.gov.ru](/data-sources/federal/budget-gov-ru-datasets).

## Описание

Сайт **novgorod.roskazna.gov.ru** дублирует типовую структуру территориальных управлений: новости, нормативные документы по казначейскому сопровождению, ссылки на федеральные сервисы. Для понимания **консолидации** бюджетов субъекта и муниципалитетов см. [обзор консолидированных бюджетов](/data-sources/regional/consolidated-budgets).

## Объекты данных и показатели

- **Новости** и разъяснения по казначейским процедурам на регионе.
- Ссылки на **федеральные** сервисы Росказны (единый клиент, реестры — по актуальной навигации).
- Публичные материалы, связанные с **исполнением** бюджетов (в части, раскрываемой для открытого доступа).

## Периодичность обновления

Непрерывно для новостного контура; нормативные документы — по мере выпуска.

## Ограничения и особенности

- Детальные **платёжные** и **лицевые** данные не являются открытыми персональными наборами; для агрегатов исполнения бюджета субъекта сопоставляйте с публикациями Минфина.
