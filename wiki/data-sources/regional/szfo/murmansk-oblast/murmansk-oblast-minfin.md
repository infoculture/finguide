# Мурманская область: Министерство финансов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Субъект РФ — **Мурманская область** (региональный бюджет) |
| Тип доступа | Официальный сайт финансового органа субъекта |
| Основные форматы | HTML, **PDF**, **XLSX** (в типовых разделах бюджета и отчётности) |
| Оператор | Министерство финансов Мурманской области |
| Режим доступа | Открытый |

Коротко: **главный региональный вход** для проекта и исполнения **областного** бюджета, долга и сопутствующей отчётности. Для **муниципального** разреза и «бюджета для граждан» используйте [портал B4U](/data-sources/regional/murmansk-oblast-b4u-budget-portal); для **законов о бюджете** — также [Областную Думу](/data-sources/regional/murmansk-oblast-regional-duma).

## Описание

На сайте Минфина обычно публикуются **материалы к закону об областном бюджете**, **отчёты об исполнении**, сведения по **государственному долгу** и **межбюджетным отношениям**, а также **нормативные акты** ведомства. Для юридически значимых текстов законов и постановлений сверяйтесь с [официальным опубликованием](/data-sources/regional/murmansk-oblast-pravo-gov-npa).

## Объекты данных и показатели

- **Объекты:** показатели **областного** бюджета (план, уточнения, исполнение), приложения к бюджету, отчётность, методические материалы.
- **Ключи связки:** **финансовый год**, **КБК/КОСГУ** (в таблицах по мере публикации), **ОКТМО** для муниципальных срезов при их наличии на сайте.
- **Пробелы:** детализация может отличаться от [федеральных открытых наборов ГИИС](/data-sources/federal/budget-gov-ru-datasets) — сверяйте методики.

## Периодичность обновления

Пик обновлений — календарь **бюджетного процесса** (внесение проекта, трёхчтение, годовая отчётность) и оперативные публикации ведомства.

## Ограничения и особенности

- Доступ для скриптов может ограничиваться (HTTP **403**); загрузку документов планируйте с паузами и с учётом `robots.txt`.
- Не смешивайте **консолидированный** бюджет с **только областным**: уточняйте заголовки таблиц и методички.
