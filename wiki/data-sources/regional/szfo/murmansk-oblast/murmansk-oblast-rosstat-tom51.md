# Мурманская область: Росстат (территориальный орган, код 51)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Субъект РФ — **Мурманская область** (официальная статистика) |
| Тип доступа | Сайт **территориального органа** Федеральной службы государственной статистики |
| Основные форматы | HTML; типовые **табличные** публикации (в т.ч. XLSX через связанные витрины Росстата) |
| Оператор | Территориальный орган Росстата (условно «Мурманскстат») |
| Режим доступа | Открытый |

Коротко: **официальные социально-экономические ряды** по региону (ВРП, демография, отрасли и др.) для **контекста** бюджетных и налоговых показателей. Для **исполнения бюджета по КБК** используйте [Минфин](/data-sources/regional/murmansk-oblast-minfin) и [ГИИС](/data-sources/federal/budget-gov-ru-datasets).

## Описание

Территориальный сайт **дополняет** федеральные витрины Росстата: пресс-релизы по региону, разделы с **краткими сводками** и ссылками на **официальные базы данных** статистики. Для **единой машинной выгрузки** по многим формам ориентируйтесь на [ЕМИСС](/information-systems/federal/emiss).

## Объекты данных и показатели

- **Объекты:** региональные статистические показатели (макро, демография, отрасли — по опубликованным разделам).
- **Ключи связки:** **финансовый год/период**, классификаторы Росстата в таблицах; для склейки с бюджетом — **согласование методик**, не «автоматический join».
- **Пробелы:** не заменяет [бюджетную отчётность](/data-sources/regional/murmansk-oblast-minfin) и [ЕИС](/information-systems/federal/zakupki).

## Периодичность обновления

По **статистическому календарю** и мере готовности форм; для конкретной формы смотрите дату публикации на витрине.

## Ограничения и особенности

- Часть **детальных таблиц** может жить в **[ЕМИСС](/information-systems/federal/emiss)** глубже, чем на региональной «обложке» сайта.
- Сопоставление с [ФНС](/data-sources/regional/murmansk-oblast-ufns-rn51) и бюджетом требует явной методической оговорки.
