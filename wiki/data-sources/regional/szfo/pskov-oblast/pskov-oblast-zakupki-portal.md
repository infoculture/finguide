# Псковская область: портал государственных и муниципальных закупок

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Регион и муниципалитеты Псковской области (закупки вне единого федерального реестра, публикуемые на региональной витрине) |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML; стабильные открытые API в карточке не зафиксированы |
| Оператор | Оператор региональной системы закупок (витрина `zakupki.pskov.ru`) |
| Режим доступа | Открытый просмотр (условия автоматизированного сбора уточняйте по `robots.txt` и правилам сайта) |

Коротко: точка входа для **региональных и муниципальных** закупок на территории Псковской области; для федерального контура и части совместных процедур по-прежнему нужен [ЕИС](/information-systems/federal/zakupki).

## Описание

Портал агрегирует материалы по закупкам для органов власти и подведомственных учреждений Псковской области и муниципальных образований. Для связки с федеральным контуром контрактов используйте идентификаторы извещений и реестровые номера, если они присутствуют в карточке процедуры.

## Оператор

Региональная витрина закупок — канонический вход: https://zakupki.pskov.ru/

## Объекты данных и показатели

- Извещения, протоколы, разъяснения документации, итоги процедур (в объёме размещения на портале).
- Возможные ключи связки: реестровый номер контракта/процедуры (если указан), ИНН заказчика, коды по ОКПД2, сроки исполнения контракта.
- Пробелы: нестабильность вёрстки, необходимость ручного обхода разделов, отсутствие единого открытого API в описании витрины.

## Периодичность обновления

По событиям закупок; пик активности привязан к календарю планирования закупок и квартальным отчётным циклам.

## Ограничения и особенности

- Не путать с единой федеральной витриной: часть крупных процедур может вестись только в ЕИС.
- Перед массовым парсингом проверьте нагрузку на портал и ограничения оператора.
