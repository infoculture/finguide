# Псковская область: контрольно-счётная палата

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Внешний аудит и контроль за использованием публичных средств в Псковской области |
| Тип доступа | Публичный сайт КСП |
| Основные форматы | HTML, PDF (заключения, отчёты) |
| Оператор | Контрольно-счётная палата Псковской области |
| Режим доступа | Открытый |

Коротко: **аналитико-контрольный** слой поверх первичной бюджетной отчётности: заключения по результатам проверок, оценки эффективности и рисков; полезен для приоритизации тем при разборе расходов, но не заменяет первичные учётные данные.

## Описание

Материалы КСП дополняют картину исполнения бюджета качественными выводами и рекомендациями. Для количественной сверки используйте отчётность [Минфина](/data-sources/regional/pskov-oblast-minfin) и закупочные витрины.

## Объекты данных и показатели

- Заключения по результатам контрольных мероприятий, годовые отчёты палаты, аналитические записки.
- Связка: ссылки на проверенные объекты (муниципалитеты, учреждения) и периоды; для денежных сумм сверяйте с первичными документами заказчика.
- Пробелы: неполнота охвата (не каждая трата проходит отдельную публикацию).

## Периодичность обновления

По мере проведения проверок и утверждения отчётности палаты.

## Ограничения и особенности

- Интерпретация выводов КСП должна учитывать статус документа (проект vs утверждённое заключение) и дату проверки.
