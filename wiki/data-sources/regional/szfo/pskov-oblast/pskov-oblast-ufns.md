# Псковская область: УФНС России (официальный сайт, код региона 60)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Территориальное управление ФНС по Псковской области (`rn60` на сайте nalog.gov.ru) |
| Тип доступа | Публичный веб-портал (часть единого сайта ФНС) |
| Основные форматы | HTML; федеральные открытые наборы ФНС — отдельными разделами и каталогом данных |
| Оператор | УФНС России по Псковской области |
| Режим доступа | Открытый; отдельные сервисы могут требовать учётной записи налогоплательщика |

Коротко: **официальный** вход УФНС для региона; для аналитики по сборам и массовым выгрузкам чаще используют федеральные открытые наборы и ЕМИСС, а региональный сайт — для локальных новостей, адресов, региональных подборок ссылок.

## Описание

Страница `rn60` дублирует общую структуру сайта ФНС с региональным контекстом: контакты, пресс-релизы, разъяснения и переходы к федеральным сервисам. Для воспроизводимых рядов по налогам и сборам ориентируйтесь на федеральные наборы и статистику Росстата.

## Объекты данных и показатели

- Новости и методические материалы УФНС; переходы к сервисам для ЮЛ и ФЛ.
- Для статистики регистрации бизнеса и смежных федеральных файлов см. типовой федеральный источник [Статистика госрегистрации ФНС](/data-sources/federal/nalog-regstats) (структура URL по другим регионам аналогична шаблону `rnXX`).
- Пробелы: не все показатели представлены как готовые машинные выгрузки на уровне одной страницы УФНС.

## Периодичность обновления

Разная по разделам: новости — непрерывно; справочники — по мере изменения НПА на федеральном уровне.
