# Псковская область: Псковстат (территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Официальная статистика по Псковской области (через поддомен `60` Росстата) |
| Тип доступа | Публичный региональный сайт территориального органа |
| Основные форматы | HTML; отдельные таблицы и вложения могут быть в XLSX/PDF (зависит от публикации) |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Псковской области |
| Режим доступа | Открытый |

Коротко: **официальный** региональный вход Росстата для пресс-выпусков и региональных коммуникаций; для длинных сопоставимых рядов и API чаще используют [ЕМИСС и открытые данные Росстата](/data-sources/federal/rosstat-opendata).

## Качество и верификация

Ориентиры измерений: [«Измерения качества источника данных»](/reference/data-quality-dimensions).

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки (`last_verified`) | 2026-05-15 |
| Полнота (`data_completeness`) | частичная — не все показатели дублируются в удобных машинных выгрузках на региональной витрине |
| Машиночитаемость (`machine_readability`) | смешанный: HTML + файлы вложений; SDMX/API на уровне этой карточки не фиксировались |
| Юридическая значимость (`legal_significance`) | официальный оператор Росстата |
| Задержка обновления (`update_lag`) | зависит от вида показателя (оперативные оценки vs годовые отчёты) |
| Глубина архива (`archive_depth`) | определяется витриной Росстата и архивом пресс-выпусков |
| Лицензия (`license_or_terms`) | условия использования официальной статистики — см. федеральные правила Росстата |

## Описание

Сайт территориального органа публикует региональные результаты официального статистического учёта, методические комментарии и сопутствующие материалы. Для финансовой аналитики полезны разделы по ВРП, доходам населения, демографии и занятости как контекст для налоговой базы и расходов бюджета.

## Объекты данных и показатели

- Показатели социально-экономического развития региона (в объёме официальных статистических форм).
- Связка с бюджетом: макро- и демографические ряды как **контекст** для доходов и расходов; прямые таблицы казначейского исполнения здесь обычно не публикуются.
- Для машинной обработки длинных рядов используйте федеральные каналы: [Росстат: ЕМИСС и открытые данные](/data-sources/federal/rosstat-opendata), [«Финансы России»](/data-sources/federal/rosstat-finances-of-russia).

## Периодичность обновления

От оперативных пресс-выпусков до годовых статистических сборников в зависимости от показателя.

## Ограничения и особенности

- Территориальный сайт может указывать на федеральные инструменты поиска таблиц; закладывайте время на навигацию между региональной и федеральной витринами.
