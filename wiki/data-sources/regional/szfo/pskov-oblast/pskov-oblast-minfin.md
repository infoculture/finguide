# Псковская область: Министерство финансов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Исполнение и сопровождение областного бюджета, региональный финансовый контроль |
| Тип доступа | Официальный сайт министерства (смешанный: HTML-разделы и файлы отчётности) |

## Описание

На сайте публикуются проекты бюджета и пояснительные записки, графические материалы исполнения, нормативные акты министерства, сведения о государственном долге субъекта (если раскрываются), а также разделы для граждан. Для юридически значимых формулировок бюджетных лимитов используйте также [банк НПА](/data-sources/regional/pskov-oblast-npa-bank-pravo).

## Объекты данных и показатели

- Плановые и отчётные показатели областного бюджета, своды по исполнению, ведомственные разрезы (в объёме публикаций).
- Связка с законодательным контуром: [областное Собрание](/data-sources/regional/pskov-oblast-regional-assembly) — инициатива и принятие закона о бюджете.
- Пробелы: фрагментированность файлов по разделам; необходимость ручной инвентаризации ссылок по годам.

## Периодичность обновления

Привязана к бюджетному и отчётному календарю (месячные, квартальные, годовые публикации).

## Ограничения и особенности

- Для машинного сбора предпочитайте прямые ссылки на конкретные XLSX/PDF после инвентаризации раздела «Бюджет» за выбранный год.
