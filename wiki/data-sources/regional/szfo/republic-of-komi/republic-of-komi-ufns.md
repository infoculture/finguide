# Республика Коми: УФНС России (официальный сайт, код региона 11)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Территориальное управление ФНС по Республике Коми (`rn11` на сайте nalog.gov.ru) |
| Тип доступа | Публичный веб-портал (часть единого сайта ФНС) |
| Основные форматы | HTML; федеральные открытые наборы ФНС — отдельными разделами и каталогом данных |

Коротко: **официальный** вход УФНС для региона; для аналитики по сборам и массовым выгрузкам чаще используют федеральные открытые наборы ФНС и ЕМИСС, а региональный раздел — для локальных новостей, контактов и региональных подборок ссылок.

## Описание

Страница `rn11` повторяет общую структуру сайта ФНС с региональным контекстом: контакты, пресс-релизы, разъяснения и переходы к федеральным сервисам. Для воспроизводимых рядов по налогам и сборам ориентируйтесь на федеральные наборы и статистику Росстата.

## Объекты данных и показатели

- Новости и методические материалы УФНС; переходы к сервисам для ЮЛ и ФЛ.
- Для статистики регистрации бизнеса и смежных федеральных файлов см. [Статистика госрегистрации ФНС](/data-sources/federal/nalog-regstats).
- Пробелы: не все показатели представлены как готовые машинные выгрузки на уровне одной страницы УФНС.

## Периодичность обновления

Разная по разделам: новости — непрерывно; справочники — по мере изменения НПА на федеральном уровне.
