# Республика Коми: портал инициативного бюджетирования («Народный бюджет»)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Республика Коми (участие граждан в отборе/приоритизации проектов) |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML |
| Оператор | Органы власти РК (конкретный куратор раздела — по странице «Контакты» на портале) |
| Режим доступа | Открытый для просмотра; участие — по правилам конкурса |

Коротко: **не бухгалтерское исполнение бюджета**, а витрина **участия**: проекты, статусы, итоги голосований. Для агрегированных финансовых рядов по бюджету субъекта используйте [Минфин РК](/data-sources/regional/republic-of-komi-minfin) и [федеральные открытые данные](/data-sources/regional/republic-of-komi-budget-gov-ru-open-data).

## Описание

HTTP-вход `http://ib.rkomi.ru/` перенаправляет на HTTPS-версию. Портал предназначен для **прозрачности** этапов отбора проектов инициативного бюджетирования.

## Оператор

Портал инициативного бюджетирования Республики Коми — канонический вход: https://ib.rkomi.ru/

## Объекты данных и показатели

- Карточки проектов: описание, территория, статус отбора, результаты голосования (в объёме, опубликованном на сайте).
- Связка с бюджетом: суммы по проектам (если раскрыты), финансовый год, иногда муниципальный контур — по правилам конкретного сезона.

## Периодичность обновления

Сезонная: пики активности в периоды приёма заявок и голосования.

## Ограничения и особенности

- Для сопоставления с кассовым исполнением и КБК используйте казначейские и бюджетные отчётные контуры, а не только этот портал.
