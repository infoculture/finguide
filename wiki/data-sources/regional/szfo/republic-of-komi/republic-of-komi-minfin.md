# Республика Коми: Министерство финансов Республики Коми

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Финансовый орган Республики Коми |
| Тип доступа | Официальный публичный портал |
| Основные форматы | HTML, PDF; периодически XLSX/Zip вложения к отчётности |
| Оператор | Министерство финансов Республики Коми |
| Режим доступа | Открытый |

Коротко: **первый региональный вход** к материалам бюджета субъекта: проекты и пояснительные записки, отчёты об исполнении, методические указания. Для **машинных** срезов по единой модели «Электронного бюджета» дополнительно используют [федеральный каталог](/data-sources/regional/republic-of-komi-budget-gov-ru-open-data).

## Описание

Портал Минфина РК агрегирует **региональный бюджетный контур** для граждан и специалистов: законодательные инициативы по бюджету (часто со ссылками на парламент), исполнение, субсидии и иные разделы по мере ведения сайта.

## Объекты данных и показатели

- Проекты и материалы к бюджету; годовая и промежуточная отчётность об исполнении.
- Нормативные и методические документы финансового блока субъекта.
- Ключи для стыковки: финансовый год, коды бюджетной классификации (в таблицах отчётности), наименование программы/госзадания — по структуре конкретного файла.

## Периодичность обновления

Нерегулярно по разделам; пик — подготовка и отчётность по бюджету.
