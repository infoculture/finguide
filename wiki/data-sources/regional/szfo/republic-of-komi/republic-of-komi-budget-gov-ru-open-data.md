# Республика Коми: открытые данные ГИИС «Электронный бюджет» (федеральный портал, срезы по субъекту)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральная витрина открытых данных с фильтрацией/измерениями по субъектам РФ, включая Республику Коми |
| Тип доступа | Каталог наборов и файловые выгрузки |
| Основные форматы | CSV, JSON (по наборам), HTML каталога |
| Оператор | Минфин России / ГИИС «Электронный бюджет» (федеральный контур) |
| Режим доступа | Открытый |

Коротко: **не региональный сайт**, а федеральный канал, через который получают воспроизводимые ряды по бюджету субъекта (планы, исполнение, справочники участников — в объёме конкретного набора). Региональные публикации Минфина РК см. в [карточке Минфина РК](/data-sources/regional/republic-of-komi-minfin).

## Описание

Раздел «Открытые данные» на `budget.gov.ru` — основной вход к **паспортизированным** наборам ГИИС «Электронный бюджет». Для Республики Коми в прикладных выгрузках ищут идентификаторы субъекта/участников и нужный финансовый год; состав полей задаётся схемой набора.

## Оператор

ГИИС «Электронный бюджет» (федеральная витрина) — канонический вход к каталогу: https://www.budget.gov.ru/открытые-данные

## Объекты данных и показатели

- Зависят от набора: строки планирования и исполнения, справочники, реестры соглашений и др. (см. [обзор федеральных наборов](/data-sources/federal/budget-gov-ru-datasets)).
- Типичные ключи: финансовый год, КБК/коды классификации (в объёме набора), идентификаторы участников бюджетного процесса.
- Пробелы: не каждый региональный показатель дублируется отдельным CSV; часть аналитики остаётся в HTML/PDF на сайте Минфина субъекта.

## Периодичность обновления

Индивидуально по каждому набору (указано в паспорте на портале).
