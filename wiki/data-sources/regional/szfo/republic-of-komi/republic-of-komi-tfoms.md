# Республика Коми: Территориальный фонд ОМС (ТФОМС Республики Коми)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный фонд ОМС (отдельный бюджет фонда вне консолидированного бюджета субъекта, но связан с соцполитикой региона) |
| Тип доступа | Официальный публичный портал |
| Основные форматы | HTML, PDF отчётности и нормативных документов |
| Оператор | ТФОМС Республики Коми |
| Режим доступа | Открытый |

Коротко: **официальный вход** к материалам фонда ОМС: ПГГ, тарифная политика, отчёты и разъяснения. Для **сводной бюджетной** аналитики субъекта дополнительно используют [Минфин РК](/data-sources/regional/republic-of-komi-minfin).

## Описание

Сайт ТФОМС — типовая структура для страховых фондов: новости, нормативные акты, разделы для застрахованных и медорганизаций, отчётность. Для **финансового анализа** важны годовые отчёты, бюджет фонда и материалы по тарифам.

ТФОМС Республики Коми — канонический вход: https://komifoms.ru/

## Объекты данных и показатели

- Бюджет фонда и отчётность о его исполнении (в форматах, опубликованных на сайте).
- ПГГ, тарифы, реестры/перечни в объёме раскрытия.
- Ключи: финансовый год, разделы затрат на ОМС, связка с региональными НПА (см. [Банк НПА](/data-sources/regional/republic-of-komi-npa-portal)).

## Периодичность обновления

По мере нормативных изменений и отчётных кампаний фонда.

## Ограничения и особенности

- Показатели ОМС не заменяют консолидированную отчётность Республики Коми по «объединённому» бюджету субъекта.
