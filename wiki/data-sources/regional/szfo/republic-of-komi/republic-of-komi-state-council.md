# Республика Коми: Государственный Совет Республики Коми

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Законодательный орган Республики Коми |
| Тип доступа | Официальный публичный портал парламента |
| Основные форматы | HTML, PDF (тексты НПА и материалы заседаний) |
| Оператор | Государственный Совет Республики Коми |

Коротко: **законодательный след** по бюджету и смежным темам: проекты и принятые законы, сопутствующие документы заседаний. Для машинных рядов исполнения бюджета обычно сочетают с [Минфином РК](/data-sources/regional/republic-of-komi-minfin), [федеральным «Электронным бюджетом»](/data-sources/federal/budget-gov-ru-datasets) и [УФК](/data-sources/regional/republic-of-komi-treasury-roskazna).

## Описание

Сайт предназначен для опубликования повесток, стенограмм и нормативных текстов, проходящих через парламент. Для **поиска и подборки** уже зарегистрированных актов удобно параллельно использовать [Банк правовых актов РК](/data-sources/regional/republic-of-komi-npa-portal), если там полнее индексация.

## Объекты данных и показатели

- Законопроекты и законы (включая бюджетный цикл: проект закона о бюджете, поправки).
- Повестки, протоколы и информационные материалы к заседаниям.
- Связка с аналитикой: номер закона, дата принятия, вид акта; для стыковки с исполнением — через КБК/год в федеральных и региональных открытых наборах.

## Периодичность обновления

По мере заседаний и официального опубликования документов; пики активности — бюджетная сессия.

## Ограничения и особенности

- У части региональных сайтов цепочка TLS может требовать актуального корневого хранилища в автоматизированных клиентах.
- Массовая выгрузка «всех законов за период» одним API, как правило, отсутствует.
