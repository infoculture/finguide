# Вологодская область: Законодательное Собрание

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Субъект РФ — **Вологодская область** (законодательная власть) |
| Тип доступа | Официальный сайт регионального парламента |
| Основные форматы | HTML, PDF (стенограммы, повестки, законопроекты) |

Коротко: **законодательный этап** бюджетного процесса: проекты законов об областном бюджете, сопутствующие документы и ход рассмотрения.

## Описание

Сайт парламента полезен для **проектных** редакций, **поправок** и **контекста** принятия бюджетного закона. Для **официально опубликованных** редакций НПА после подписания используйте [портал правовой информации области](/data-sources/regional/vologda-oblast-npa-portal).

## Объекты данных и показатели

- **Законопроекты** о бюджете и сопутствующие акты.
- **Повестки** и **результаты** голосований (если публикуются).
- Связка: **номер законопроекта**, **сессия**, **дата** заседания.

## Периодичность обновления

Пик активности в **бюджетную сессию** и при рассмотрении отчёта об исполнении бюджета.
