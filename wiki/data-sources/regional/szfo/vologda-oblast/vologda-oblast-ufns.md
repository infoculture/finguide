# Вологодская область: сайт УФНС (регион 35)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональное** представительство ФНС; часть материалов — **общероссийские** с фильтром по субъекту |
| Тип доступа | Веб-портал ФНС России (`rn35`) |
| Основные форматы | HTML; статистические и отчётные файлы — XLSX, XML по конкретным разделам |
| Оператор | Управление ФНС России по Вологодской области (в составе единого сайта nalog.gov.ru) |
| Режим доступа | Открытый для публичных разделов |

Коротко: **точка входа** к территориальным новостям УФНС и к **налоговой статистике** и аналитическим публикациям ФНС с привязкой к **коду субъекта 35**.

## Описание

URL `https://www.nalog.gov.ru/rn35/` — стандартный **региональный корень** на общем портале ФНС. Для **сопоставления налоговых индикаторов** с бюджетным исполнением используйте в связке [консолидированные бюджеты](/data-sources/regional/consolidated-budgets) и [аналитический портал ФНС](/information-systems/federal/analytic-nalog) (общероссийский контур, не только 35 регион).

## Оператор

УФНС России по Вологодской области (через единый домен ФНС) — канонический вход: https://www.nalog.gov.ru/rn35/

## Объекты данных и показатели

- **Региональные** новости, объявления и разъяснения УФНС.
- Ссылки на **формы налоговой статистики** и отчётность (структура — по меню ФНС).
- Ключ связки: **код субъекта 35**, **налоговые периоды**, коды **ОКТМО** в табличных формах (если применимо).

## Периодичность обновления

Зависит от **календаря** публикаций ФНС по формам и от оперативных новостей УФНС.

## Ограничения и особенности

- Региональный раздел **не** заменяет **единый федеральный** каталог открытых данных ФНС: для машинного доступа к наборам см. **[ППД ФНС](/information-systems/federal/vpd-nalog)**.
