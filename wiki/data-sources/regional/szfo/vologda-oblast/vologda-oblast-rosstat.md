# Вологодская область: Вологдастат (ТО Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональное** представительство Росстата; показатели — в **федеральной** методологии |
| Тип доступа | Официальный сайт ТО Росстата (`35.rosstat.gov.ru`) |

Коротко: **региональный вход** к официальной статистике по Вологодской области: оперативные сообщения ТО, навигация к **ЕМИСС** и **открытым данным** Росстата.

## Описание

Сайт территориального органа дополняет **федеральные** витрины **локальным контекстом** (пресс-релизы по региону, контакты, разъяснения). Для **скрещивания** с бюджетом используйте [консолидированные бюджеты](/data-sources/regional/consolidated-budgets) и макропоказатели из ЕМИСС.

## Объекты данных и показатели

- **Региональные** оперативные индикаторы и комментарии ТО (по мере публикации).
- **Ссылки** на федеральные базы данных официальной статистики.
- Ключ связки: **код 35** (Вологодская область), **ОКТМО**, период.

## Периодичность обновления

Согласно **календарю** статистических наблюдений и графику публикаций Росстата.

## Ограничения и особенности

- Сайт ТО **может не содержать** всех машиночитаемых выгрузок; основной **каталог наборов** — на федеральных ресурсах Росстата.
