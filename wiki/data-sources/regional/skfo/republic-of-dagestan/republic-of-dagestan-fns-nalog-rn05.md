# Республика Дагестан: УФНС России по Республике Дагестан (nalog.gov.ru/rn05)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Республика Дагестан** — **территориальное** управление **ФНС России** |
| Тип доступа | Публичный **раздел** на **федеральном** домене **nalog.gov.ru** |
| Основные форматы | **HTML**; **таблицы**/**файлы** — по **конкретным** сервисам и **разделам** |
| Оператор | **Управление Федеральной налоговой службы по Республике Дагестан** |
| Режим доступа | Открытое чтение публичных разделов |

Коротко: **региональный** вход **ФНС** для **новостей**, **электронных** сервисов и **локальных** **справочных** материалов. Для **агрегированной налоговой статистики** и **машиночитаемых** наборов используйте **[налоговую статистику ФНС](/data-sources/federal/nalog-statistics)** и **каталог** **[открытых данных ФНС](https://www.nalog.gov.ru/rn77/opendata/)** (федеральный контур с **реестром** наборов).

## Описание

Путь **`/rn05/`** соответствует **региональному** **коду** **УФНС** по **Республике Дагестан**. Страница **объединяет** **навигацию** по **сервисам** и **локальным** материалам; **налоговая** **отчётность** юридических лиц и **микроданные** **как правило** живут в **федеральных** **кабинетах** и **каталогах** **открытых данных**, а не в **статическом** **HTML** одной **лендинговой** страницы.

## Объекты данных и показатели

- **Новости** и **разъяснения** **УФНС** для **налогоплательщиков**.
- **Сервисные** ссылки на **федеральные** **инструменты** ФНС.
- **Связка** с **бюджетом** — **опосредованная** (через **налоговые** **базы** и **агрегаты**), для **прямой** **сводки** по **субъекту** используйте **[Минфин РД](/data-sources/regional/republic-of-dagestan-minfin-minfinrd)**.

## Периодичность обновления

По мере **обновлений** **ФНС** и **региональных** **сообщений**; для **OD** — по **паспорту** набора.
