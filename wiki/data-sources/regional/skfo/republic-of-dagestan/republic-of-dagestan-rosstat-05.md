# Республика Дагестан: территориальный орган Росстата (Дагестанстат)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Республика Дагестан** — **региональный** контур **государственной статистики** |
| Тип доступа | Публичный **сайт** территориального органа **Росстата** |
| Основные форматы | **HTML**; отдельные **таблицы** — **XLSX** и др. (по разделам) |
| Оператор | **Территориальный орган Федеральной службы государственной статистики по Республике Дагестан** |
| Режим доступа | Открытое чтение публичных материалов |

Коротко: **региональная** витрина **Дагестанстата** для **официальных** статистических **коммуникаций** и **таблиц** по субъекту. Для **единой** федеральной базы временных рядов и **API** используйте **[ЕМИСС](/information-systems/federal/emiss)**; для **бюджетных** рядов — **[наборы «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** и **[Минфин РД](/data-sources/regional/republic-of-dagestan-minfin-minfinrd)**.

## Описание

Сайт **05.rosstat.gov.ru** публикует **региональные** итоги и **аналитические** справки по **Республике Дагестан** в логике **официальной статистики**. Для **склейки** с **бюджетом** используйте **согласованные** показатели (ВРП, инвестиции, демография и т. п.) и **не** подменяйте **факт** исполнения бюджета **статистическими** оценками без методической сверки.

**Территориальный орган Федеральной службы государственной статистики по Республике Дагестан** — https://05.rosstat.gov.ru/

## Объекты данных и показатели

- **Официальные** региональные **коммуникации** Росстата (пресс-релизы, комментарии к данным).
- **Табличные** материалы по **ключевым** блокам СЭР (наличие **XLSX** зависит от раздела).
- **Ключи связки** с федеральным контуром — **коды ОКТМО**, **наименование** субъекта, **классификаторы** показателей ЕМИСС.

## Периодичность обновления

По **календарю** Росстата и **типу** показателя (оперативные / годовые); для **единой** картины сверяйте дату **релиза** в **ЕМИСС**.

## Ограничения и особенности

- **Статистика** **не** является **бухгалтерским** исполнением бюджета: для **кассы** и **лимитов** используйте **бюджетную** отчётность.
- **URL** и **меню** региональных сайтов Росстата периодически **обновляются** — закладывайте **поиск** по **сайту** и **федеральному** каталогу.
