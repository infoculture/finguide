# Ставропольский край: Ставропольстат (Территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Ставропольский край** — **официальная** **статистика** **Росстата** |
| Тип доступа | Публичный **сайт** **ТОР** |
| Основные форматы | **HTML**; **XLSX**/… **в** **публикациях** |
| Оператор | **Территориальный** орган **Федеральной** **службы** **государственной** **статистики** |
| Режим доступа | Открытый |

Коротко: **региональный** **вход** к **официальным** **статистическим** **данным** **по** **краю**, включая **ряды**, **смежные** с **финансами** **региона**. **Бюджетное** **исполнение** **по** **КБК** — через **[Минфин](/data-sources/regional/stavropol-krai-minfin-mfsk)** и **федеральные** **витрины**.

## Описание

**Ставропольстат** публикует **своды** **по** **экономике**, **населению**, **социальной** **сфере**. **Показатели** **бюджета** **как** **учётного** **регистра** **здесь** **не** **заменяют** **казначейские** и **бюджетные** **ИС**.

- **Региональные** **статистические** **показатели** **и** **комментарии** **к** **ним**.
- **Сопоставление** с **бюджетом** — через **одинаковые** **периоды** и **определения** **показателей** **в** **метаданных**.
