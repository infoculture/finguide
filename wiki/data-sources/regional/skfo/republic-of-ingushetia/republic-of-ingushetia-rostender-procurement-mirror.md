# Республика Ингушетия: Ростендер — агрегатор закупок по региону (неофициально)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Республика Ингушетия** — **гражданский** агрегатор **закупок** (не **ЕИС**) |
| Тип доступа | Публичный **веб**-интерфейс третьей стороны |
| Основные форматы | **HTML** |
| Оператор | **Ростендер** (коммерческий/гражданский сервис) |
| Режим доступа | Открытый просмотр (см. условия сайта) |

Коротко: **вспомогательная** витрина для **поиска** и **мониторинга** закупок по фильтру региона. Для **юридически значимых** и **машинных** данных используйте **[официальный XML-контур закупок](/data-sources/federal/zakupki-xml)** и **[статистику ЕИС](/data-sources/federal/zakupki-eis-statistics)**.

## Описание

**Ростендер** предоставляет **удобную** навигацию по **закупкам** с привязкой к **региону**. Перед **юридически значимыми** выводами **сверяйте** карточки закупок с **официальной** записью в **ЕИС** и учитывайте **возможные** задержки и **ошибки** индексации.
