## Описание

Раздел **26.rosstat.gov.ru** относится к **Северо-Кавказстату**. **Папка** с идентификатором в **URL** (`folder/23783`) агрегирует **материалы** по **Карачаево-Черкесской Республике**: **ВРП**, **демография**, **рынок труда**, **зарплата** и др. **Постоянство** **числового** id **папки** при реорганизации сайта **не гарантировано** — при **поломке** ссылки ищите **аналог** через **навигацию** с **главной** страницы управления.

## Объекты данных и показатели

- **Показатели** социально-экономического развития субъекта в **публикациях** **Росстата** (состав **папки** может меняться).
- **Табличные** вложения — по **ссылкам** из страниц раздела.
- **Ключи связки** с бюджетной аналитикой: **год**, **классификаторы** показателей, **ОКТМО** (в **федеральных** наборах).

## Периодичность обновления

Зависит от **календаря** **Росстата** и **региональных** **выпусков**; **крупные** обновления — **ежеквартально**/**ежегодно** для **макропоказателей** (оценочно).

## Ограничения и особенности

- Часть **клиентов** может потребовать **обновления** **корневых** сертификатов для **HTTPS**.
- **Идентификатор** **папки** в **URL** может **измениться** при **редизайне** сайта.
