# Кабардино-Балкарская Республика: Территориальный орган Росстата по СКФО (26.rosstat.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **ТОФ Росстата** по **Северо-Кавказскому ФО** (несколько субъектов РФ, включая **КБР**) |
| Тип доступа | Официальный **региональный** сайт Росстата (код **26**) |
| Основные форматы | HTML; отдельные таблицы и вложения (**XLSX** и др.) |
| Оператор | Территориальный орган Федеральной службы государственной статистики по СКФО |
| Режим доступа | Открытый |

Коротко: **официальная статистика** и **комментарии** по социально-экономическому положению **округа**; для **КБР** используйте разделы с **региональным** срезом. Машиночитаемые **массовые** выгрузки удобнее дублировать через **[ЕМИСС / открытые данные Росстата](/data-sources/federal/rosstat-opendata)**.

## Описание

Сайт **26.rosstat.gov.ru** ориентирован на **СКФО** целиком: в таблицах и обзорах могут фигурировать **несколько** субъектов. Для **сопоставления** с бюджетными рядами КБР используйте [Минфин](/data-sources/regional/kabardino-balkar-republic-minfin) и [консолидированные бюджеты](/data-sources/regional/consolidated-budgets) как методический контекст.

Территориальный орган Росстата по Северо-Кавказскому федеральному округу — канонический вход: https://26.rosstat.gov.ru/

## Объекты данных и показатели

- **Региональные** показатели по субъектам СКФО (ВРП, демография, рынок труда и др. — по наличию в публикациях).
- **Комментарии** и **краткие отчёты** ТОФ.
- **Связка** с бюджетом: через **макро**- и **социальные** ряды, а не через бюджетную отчётность Минфина.

## Периодичность обновления

Зависит от **календаря** официальной статистики и типа выпуска (месяц, квартал, год).

## Ограничения и особенности

- **Зона** ТОФ шире одной республики: всегда проверяйте **географический** признак в таблице.
