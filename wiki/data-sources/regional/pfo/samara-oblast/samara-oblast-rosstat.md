# Самарская область: Территориальный орган Росстата — Самарастат (63.rosstat.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** служба на территории субъекта; публикует **региональные** сводки |
| Тип доступа | Официальный сайт **ТОР** на поддомене **63.rosstat.gov.ru** |
| Основные форматы | HTML (таблицы, публикации) |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Самарской области |
| Режим доступа | Открытый просмотр |

Коротко: **региональный** вход к **официальной статистике** по области; для **массовых CSV/SDMX** ориентируйтесь на **[ЕМИСС и открытые данные Росстата](/data-sources/federal/rosstat-opendata)** и тематические витрины (например **[«Финансы России»](/data-sources/federal/rosstat-finances-of-russia)**).

## Описание

В карточке зафиксирован вход в раздел **«Финансы»** (`/finance`) как типичная отправная точка для **финансово-хозяйственной** статистики по региону. Навигация по остальным тематическим блокам — с корня **https://63.rosstat.gov.ru/**.

## Оператор

Территориальный орган Росстата по Самарской области — канонический вход к разделу: https://63.rosstat.gov.ru/finance

## Объекты данных и показатели

- **Публикации** и **коммуникации** по официальной статистике по Самарской области.
- **Финансово-экономические** показатели в HTML-таблицах (конкретный перечень меняется со временем).
- **Ключи связки** с федеральными рядами: коды **ОКВЭД**, **ОКТМО** (где применимо), календарный **год**.

## Периодичность обновления

По **официальным релизам** Росстата и **календарю** публикаций (см. также федеральные анонсы на **rosstat.gov.ru**).

## Ограничения и особенности

- При проверке из **curl** с верификацией TLS на окружении агента возможна **ошибка цепочки сертификатов**; страница открывается в браузере. Для скриптов при необходимости используйте обновлённое хранилище CA или проверку в браузере.

## Примеры доступа

```bash
# при ошибке TLS в среде CI проверьте ответ в браузере
curl -skSL -o /dev/null -w "%{http_code}\n" --max-time 25 -A "Mozilla/5.0" "https://63.rosstat.gov.ru/finance"
```
