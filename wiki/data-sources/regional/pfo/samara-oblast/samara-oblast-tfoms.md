# Самарская область: Территориальный фонд ОМС (samtfoms.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональный** **внебюджетный** фонд (ОМС) на территории Самарской области |
| Тип доступа | Официальный **веб-сайт** с разделами документов и новостей |
| Основные форматы | HTML, **PDF** (отчёты и приказы, по мере публикаций) |

Коротко: **официальный** источник по **финансово-правовым** материалам ТФОМС (отчётность, бюджет фонда, тарифная политика — в объёме публикаций на сайте). ## Описание

Сайт ТФОМС используют для **юридически значимых** копий **нормативных** и **отчётных** документов фонда. Для **машинной** аналитики по **ОМС** на федеральном уровне часто нужны **дополнительные** федеральные витрины и реестры.

## Объекты данных и показатели

- **Отчёты** о финансовых результатах и **исполнении** бюджета фонда (как публикуются).
- **Нормативные** акты регионального уровня в зоне компетенции ТФОМС.
- **Ключи связки** с региональным бюджетом: **субвенции**, **дотации**, **год**.

## Периодичность обновления

По мере **утверждения** отчётности и **публикации** документов.
