## Описание

Региональный сайт **penza.roskazna.gov.ru** полезен для **контактов**, **новостей** и **разъяснений** по казначейскому обслуживанию. Агрегированные **открытые данные** по казначейскому контуру удобнее собирать из **федеральных** витрин (см. связанные страницы).

## Объекты данных и показатели

- **Новости** и **методические** материалы по казначейскому сопровождению.
- **Контакты** регионального управления.
- **Не** полноценная замена **открытых выгрузок** по всем лицевым счетам и формам — для этого используйте **ГИИС «Электронный бюджет»** и **наборы каталога** на **budget.gov.ru**.

## Периодичность обновления

По мере публикации новостей и нормативных изменений на стороне оператора.

## Ограничения и особенности

- При проверке из **строгого** TLS-клиента (например, `curl` без доверенной цепочки) возможны предупреждения о сертификате; в браузере сайт обычно открывается штатно.

## Примеры доступа

```bash
curl -fsSk -o /dev/null -w "%{http_code}\n" --max-time 25 -A "Mozilla/5.0" "https://penza.roskazna.gov.ru/" \
  || echo "при ошибке TLS проверьте в браузере"
```
