## Описание

Портал **penzagz.pnzreg.ru** отражает **региональный** слой закупочной информации. При построении воспроизводимых цепочек «**закупка → контракт → оплата**» сверяйтесь с **ЕИС** и казначейскими витринами.

## Оператор

Государственные закупки Пензенской области — канонический вход: http://penzagz.pnzreg.ru/

## Объекты данных и показатели

- **Извещения**, **документация** и сопутствующие материалы **регионального** контура (по мере размещения на портале).
- **Связка** с федеральным контуром: реестровые номера закупок/контрактов **ЕИС** (при наличии на карточке).
- **Пробелы**: без отдельной выгрузки API в карточке не фиксируется — уточняйте по сайту.

## Периодичность обновления

По мере размещения закупочных процедур и обновления разделов оператором.

## Ограничения и особенности

- На момент проверки **2026-05-14** доступ по **HTTPS** с некоторых TLS-стеков мог не устанавливаться (устаревший протокол на стороне сервера); **HTTP**-запрос с типовым User-Agent возвращал **200**. Предпочтительный канал уточняйте по редиректам с главной страницы.
