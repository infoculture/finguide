# Республика Тыва: УФНС России по Республике Тыва

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный орган, региональная **витрина** присутствия на территории Республики Тыва |
| Тип доступа | Публичный веб-раздел на `nalog.gov.ru` |

## Описание

Региональная страница `rn17` полезна для **контекста** (локальные объявления, пресс-релизы, контакты). Связь с **консолидированным бюджетом** — опосредованная (налоги как доходы бюджетов разных уровней): для бюджетных таблиц ориентируйтесь на [Минфин](/data-sources/regional/republic-of-tyva-minfin) и федеральные бюджетные наборы.
