# Республика Тыва: Росстат — Красноярскстат (раздел по Республике Тыва)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральная статистика с территориальной привязкой к Республике Тыва (через управление на три субъекта) |
| Тип доступа | Публичный веб-портал Росстата |
| Основные форматы | HTML, PDF, иногда XLSX |
| Оператор | Управление Федеральной службы государственной статистики по Красноярскому краю, Республике Хакасия и Республике Тыва |
| Режим доступа | Открытый |

Коротко: официальная статистика по региону — сборники, краткие оперативные выпуски, материалы о социально-экономическом положении; полезно как **макроконтекст** к бюджетным рядам, а не как замена бюджетной отчётности.

## Описание

Сайт **Красноярскстата** обслуживает сразу три субъекта РФ; карточка фиксирует **точку входа** в материалы по Республике Тыва (`folder/28176` на момент вычитки). Показатели здесь не заменяют [республиканский бюджетный контур](/data-sources/regional/republic-of-tyva-minfin), но позволяют сопоставлять динамику ВРП, демографии, рынка труда и др. с бюджетными агрегатами.

## Оператор

Управление Росстата по Красноярскому краю, Республике Хакасия и Республике Тыва — канонический URL раздела: https://24.rosstat.gov.ru/folder/28176

## Объекты данных и показатели

- Статистические сборники и бюллетени с региональным срезом по Тыве.
- Оперативные информационные материалы (структура и периодичность — по оглавлению сайта).
- Не путать с строками **консолидированной бюджетной отчётности**: иные системы классификации и единицы.

## Периодичность обновления

Задаётся календарём Росстата и типом продукта (оперативный обзор vs годовой сборник).

## Ограничения и особенности

- При смене структуры портала Росстата идентификатор папки (`folder/…`) может измениться — сверяйте с поиском по сайту `rosstat.gov.ru`.
- Один сайт на три субъекта: внимательно проверяйте заголовок публикации на соответствие региону.
