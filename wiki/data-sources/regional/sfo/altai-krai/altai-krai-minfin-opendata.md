# Алтайский край: открытые данные Министерства финансов (minfin.alregn.ru/opendata)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Финансовый орган Алтайского края, раздел **открытых данных** |
| Тип доступа | Веб-каталог и **файловые** вложения по ссылкам из карточек наборов |
| Основные форматы | HTML; состав файлов по наборам (CSV, XLSX, ZIP и др.) уточняйте в паспортах на портале |
| Оператор | Министерство финансов Алтайского края |
| Режим доступа | Открытый |

Коротко: **региональный** каталог **открытых наборов** Минфина края; для **сводного федерального контура** и **форм отчётности** параллельно используйте **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** и **[каталог budget.gov.ru](/data-sources/federal/budget-gov-ru-datasets)**.

## Описание

Раздел **https://minfin.alregn.ru/opendata** агрегирует **именованные наборы** (паспорта, файлы, периодичность) в рамках полномочий Минфина. Для задач **сопоставления с консолидированным бюджетом** и **единым классификатором** опирайтесь на федеральные наборы и НСИ в **Электронном бюджете**, а региональный каталог используйте как **локальную** витрину субъекта.

Министерство финансов Алтайского края — канонический вход: https://minfin.alregn.ru/opendata

## Объекты данных и показатели

- **Именованные наборы** в каталоге (состав и ключи зависят от паспорта каждого набора).
- Типичные связки для аналитики: **финансовый год**, **КБК/коды расходов** (если присутствуют в файле), **ведомство**.
- Пробелы: не все показатели исполнения дублируются в открытых файлах — при расхождениях сверяйте **ГИИС «Электронный бюджет»**.

## Периодичность обновления

Индивидуально по каждому набору (см. паспорт на портале).

```bash
curl -fsSL -o /dev/null -w "%{http_code}\n" --max-time 25 "https://minfin.alregn.ru/opendata"
```
