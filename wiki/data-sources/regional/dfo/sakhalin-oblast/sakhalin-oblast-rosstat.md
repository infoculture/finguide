# Сахалинская область: Территориальный орган Росстата — Сахалинстат (65.rosstat.gov.ru)

| Основные форматы | **HTML**, **XLSX**, **PDF** |

## Описание

**Сахалинстат** публикует **региональные** **коммуникации** и **выгрузки**, дополняя **федеральные** витрины. Для **сопоставления** с **бюджетом** используйте **[Минфин](/data-sources/regional/sakhalin-oblast-minfin)** и **федеральные** **бюджетные** ряды.

## Объекты данных и показатели

- **Индексы** производства, **ИПЦ**, **зарплата**, **торговля**, **инвестиции** — по **методологии** Росстата.
- **Ключи**: **период**, **классификатор** отрасли/товара, **единица** измерения.

## Периодичность обновления

По **календарю** **Росстата** для каждого **показателя**.
