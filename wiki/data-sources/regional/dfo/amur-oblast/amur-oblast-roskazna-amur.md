# Амурская область: Управление Федерального казначейства по Амурской области

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональное управление Федерального казначейства (операции казначейского обслуживания) |
| Тип доступа | Публичный веб-сайт УФК |
| Основные форматы | HTML |
| Оператор | Управление Федерального казначейства по Амурской области |

Коротко: **официальный региональный вход Росказны** для пользователей и контрагентов; для выгрузки открытых бюджетных наборов уровня РФ ориентируйтесь на каталог **budget.gov.ru** и карточки ГИИС «Электронный бюджет».

## Качество и верификация

Заполняйте по определениям на странице **[«Измерения качества источника данных»](/reference/data-quality-dimensions)**.

## Описание

Сайт УФК по Амурской области содержит **организационную и сервисную** информацию: контакты, разделы для получателей бюджетных средств, новости территориального управления. Карточка фиксирует этот вход как часть **регионального контура** казначейского обслуживания.

## Оператор

Управление Федерального казначейства по Амурской области — канонический вход: https://amur.roskazna.gov.ru/

## Объекты данных и показатели

- Новости и методические сообщения территориального органа.
- Ссылки на федеральные сервисы Росказны (по мере интеграции интерфейса).
- Для строк исполнения бюджета и справочников счетов — федеральные открытые наборы и внутренние кабинеты (вне объёма этой карточки).

## Периодичность обновления

Нерегулярно для новостей; сервисные разделы — по мере обновления федеральных компонентов.

## Ограничения и особенности

- Не путать с **единым каталогом открытых данных** бюджетной сферы: он ведётся на уровне ГИИС и портала **data.gov.ru** / **budget.gov.ru**.
