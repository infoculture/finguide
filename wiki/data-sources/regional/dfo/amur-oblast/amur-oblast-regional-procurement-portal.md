# Амурская область: Портал государственных закупок (gz.amurobl.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Закупки для нужд Амурской области (региональная витрина) |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML |
| Оператор | Исполнительные органы Амурской области (оператор региональной витрины) |

Коротко: **региональный интерфейс** к материалам закупок субъекта; для **единой юридически значимой** копии извещений, протоколов и контрактов используйте **ЕИС** и XML-выгрузки.

## Описание

Портал **gz.amurobl.ru** предназначен для публикации материалов по **государственным закупкам** Амурской области: планы, отчёты, разъяснения, ссылки на процедуры — в рамках компетенции субъекта. Для воспроизводимого анализа контрактов на всей территории РФ опирайтесь на **федеральный ЕИС**.

## Объекты данных и показатели

- Региональные планы-графики и отчётность (если размещены на портале).
- Справочная информация для заказчиков субъекта.
- Идентификаторы закупок и контрактов для связки с ЕИС (при наличии на страницах).

## Периодичность обновления

По мере проведения закупочных процедур и администрирования портала.

## Ограничения и особенности

- Технические сбои доступа возможны; при недоступности региональной витрины используйте **ЕИС** как устойчивый канал.
