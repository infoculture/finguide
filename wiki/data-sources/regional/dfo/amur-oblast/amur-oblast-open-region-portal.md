# Амурская область: Портал открытых данных «Открытый регион» (or28.amurobl.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Открытые данные Амурской области (региональный каталог) |
| Тип доступа | Публичный портал открытых данных субъекта |
| Основные форматы | HTML, CSV, JSON, XLSX (по конкретным наборам) |
| Оператор | Исполнительные органы Амурской области (портал «Открытый регион») |

Коротко: **региональный каталог машиночитаемых наборов** субъекта; приоритетный вход, если нужен локальный паспорт набора и прямые файлы, опубликованные областью.

## Качество и верификация

Заполняйте по определениям на странице **[«Измерения качества источника данных»](/reference/data-quality-dimensions)**.

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки (`last_verified`) | 2026-05-15 |

## Описание

«Открытый регион» — это **точка агрегации** открытых наборов Амурской области: бюджет, транспорт, медицина, образование и др., в зависимости от наполнения. Для финансовой тематики пересекается с **Открытым бюджетом**, но может содержать **табличные наборы**, которых нет на гражданской витрине.

## Объекты данных и показатели

- Именованные наборы открытых данных с паспортами (ключи, периодичность, ответственный орган).
- Пересечение с федеральным порталом: часть наборов может дублироваться на data.gov.ru с тем же идентификатором или связанными метаданными.

## Периодичность обновления

Индивидуально по каждому набору (см. паспорт на портале).

## Ограничения и особенности

- Техническая доступность и полнота каталога зависят от зрелости региональной политики открытых данных.
- Для федерального поиска и API каталога используйте **[data.gov.ru](/data-sources/federal/data-gov-ru)**.
