# Амурская область: Министерство финансов Амурской области

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный финансовый орган Амурской области |
| Тип доступа | Официальный веб-сайт (документы, отчёты, справочники) |

| Оператор | Министерство финансов Амурской области |

Коротко: **главный региональный источник** по циклу областного бюджета: подготовка, исполнение, отчётность и методические разъяснения; связывайте с Открытым бюджетом (наглядность) и pravo.gov.ru (официальное опубликование законов).

## Описание

Сайт Минфина Амурской области агрегирует **операционные материалы** бюджетного процесса: проекты закона о бюджете, пояснительные записки, отчёты об исполнении, своды по долговой политике, методические письма. Это первая остановка для аналитика после поиска **юридически значимого текста** закона на портале опубликования.

## Объекты данных и показатели

- Проекты и материалы к закону об областном бюджете.
- Отчёты об исполнении бюджета (табличные вложения — по годам).
- Нормативные акты финансового блока (часть дублируется в опубликовании на pravo.gov.ru).

## Периодичность обновления

Пиковые обновления — вокруг рассмотрения бюджета и отчётных кампаний; вне цикла — точечные публикации.
