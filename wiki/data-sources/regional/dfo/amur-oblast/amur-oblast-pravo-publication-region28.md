| Признак | Значение |
| :--- | :--- |
| Уровень и охват | НПА Амурской области в официальном опубликовании |
| Тип доступа | Публичный федеральный портал (выделенный блок субъекта) |
| Основные форматы | HTML, PDF (в зависимости от карточки документа) |

## Описание

Блок **region28** на портале опубликования правовых актов предназначен для документов **Амурской области**. Это ключевой вход для поиска закона об областном бюджете, изменений к нему, постановлений Правительства области по финансовым вопросам и смежных актов.

## Оператор

Федеральный портал правовой информации — канонический URL блока субъекта: https://publication.pravo.gov.ru/documents/block/region28

## Объекты данных и показатели

- Нормативные правовые акты субъекта (метаданные публикации, текст, история версий — по возможностям портала).
- Для бюджетного аналитика: законы о бюджете, кодексы и постановления с финансовыми нормами.

## Периодичность обновления

По мере официального опубликования новых актов и версий.

## Ограничения и особенности

- Это **нормативный** контур, а не витрина исполнения бюджета в цифрах.
- Для машинной обработки больших массивов актов может потребоваться отдельная загрузка PDF и парсинг с учётом структуры портала.
