# Амурская область: Амурстат (территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный контур Амурской области (официальная статистика) |
| Тип доступа | Публичный веб-портал территориального органа Росстата |
| Основные форматы | HTML; отдельные таблицы и вложения (xlsx/csv) встречаются в составе публикаций |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Амурской области (Амурстат) |
| Режим доступа | Открытый |

Коротко: отправная точка для **официальных региональных статистических рядов** и методологических пояснений; для бюджетного исполнения и первичных учётных реестров смотрите профильные витрины Минфина области и федеральных систем.

## Качество и верификация

Заполняйте по определениям на странице **[«Измерения качества источника данных»](/reference/data-quality-dimensions)**.

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки (`last_verified`) | 2026-05-15 |
| Полнота (`data_completeness`) | частичная для «финансового следа» — портал ориентирован на статистику, а не на полный контур госфинансов |
| Машиночитаемость (`machine_readability`) | преимущественно HTML; машиночитаемость зависит от конкретного раздела/вложения |
| Юридическая значимость (`legal_significance`) | официальный оператор федеральной статистики на уровне субъекта |
| Задержка обновления (`update_lag`) | зависит от вида публикации (оперативные сводки, годовые сборники) |
| Глубина архива (`archive_depth`) | типично несколько лет в открытых разделах; детали — по рубрикатору сайта |
| Лицензия (`license_or_terms`) | условия перепубликации и ссылки на первоисточник — на сайте Росстата |

## Описание

Сайт Амурстата публикует **региональные статистические материалы** по Амурской области: демография, экономика, социальная сфера, отраслевые блоки. Для аналитики открытых финансовых данных портал полезен как **контекст** (макро- и отраслевые ряды), но не как источник строк казначейского исполнения или контрактного реестра.

## Объекты данных и показатели

- Агрегированные показатели по региону (население, ВРП и отраслевые блоки, социальная сфера — по рубрикам сайта).
- Публикации с табличными вложениями; состав форматов нужно смотреть по конкретному материалу.
- Связка с бюджетными рядами: косвенная (контекст экономики и социальной нагрузки), без ключей уровня договора или лицевого счёта.

## Периодичность обновления

Зависит от вида продукта Росстата (оперативные данные, годовые сборники, внеплановые выпуски). Точные даты обновления разделов — на страницах соответствующих публикаций.

## Ограничения и особенности

- Статистические показатели не дублируют **операционные** данные казначейства, закупок или налоговой отчётности юрлиц.
- При сопоставлении с бюджетом учитывайте методологические разрывы (классификаторы, охват, пересмотры рядов).
