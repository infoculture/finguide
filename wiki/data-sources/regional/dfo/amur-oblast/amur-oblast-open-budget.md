# Амурская область: Открытый бюджет (ob.amurobl.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный бюджет Амурской области (гражданско-аналитическая витрина) |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML; возможны вложения и экспорты в отдельных разделах |
| Оператор | Исполнительные органы Амурской области (портал проекта «Открытый бюджет») |
| Режим доступа | Открытый |

Коротко: **главная гражданская витрина** по структуре и исполнению областного бюджета в наглядном виде; для юридически значимых текстов НПА и «узких» форм отчётности сверяйтесь с Минфином области и официальным опубликованием актов.

## Описание

Портал объединяет **аналитические представления** бюджета: доходы и расходы, разрезы по направлениям, материалы по государственным программам (часть вынесена на отдельный URL — см. связанную карточку). Удобен для объяснительной аналитики и обзорных срезов.

## Оператор

Проект «Открытый бюджет» Амурской области — канонический вход: https://ob.amurobl.ru/

## Объекты данных и показатели

- Агрегированные показатели бюджета и аналитические разрезы (конкретный перечень рубрик — на сайте).
- Материалы по государственным программам (см. также отдельную страницу раздела «Аналитика»).
- Ключи связки: коды бюджетной классификации, программы, год — по мере раскрытия на портале.

## Периодичность обновления

Привязана к бюджетному циклу и решениям оператора о публикации обновлений на портале.

## Ограничения и особенности

- Доступность отдельных разделов может меняться из-за технических работ или переноса материалов.
- Для воспроизводимых рядов «как в официальной отчётности» используйте сверку с **Минфином области** и федеральными наборами.
