# Камчатский край: Министерство финансов Камчатского края (minfin.kamgov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Региональный** финансовый орган **Камчатского края** |
| Тип доступа | **Публичный** **портал** с **разделами** по **бюджету**, **долгу**, **закупкам** и **контролю** |

| Оператор | **Министерство финансов Камчатского края** |

## Описание

Сайт **Минфина** концентрирует **информацию** о **бюджетном** процессе, **государственном** **долге**, **закупках** и **внутреннем** **финансовом** **контроле**. **Часть** **материалов** может **дублироваться** или **дополняться** на **общем** **портале** **правительства** (**kamgov.ru**).

## Оператор

**Министерство финансов Камчатского края** — канонический вход: https://minfin.kamgov.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Официальный сайт Минфина | https://minfin.kamgov.ru/ | HTML, PDF |
| Портал правительства (раздел Минфина) | https://kamgov.ru/minfin/ | HTML |

## Объекты данных и показатели

- **Документы** бюджетного цикла: **проекты** и **утверждённые** акты, **отчёты** об **исполнении**, **методические** материалы.
- **Закупки** и **контроль** — по **отдельным** **рубрикам** портала.
- **Ключи связки**: **финансовый год**, **КБК/КОСГУ** (если присутствуют в **таблицах**), **номер** **краевого** закона.

## Периодичность обновления

**Сезонная** активность вокруг **подготовки** и **утверждения** бюджета; **квартальная/годовая** **отчётность** об **исполнении** — по **календарю** субъекта.

## Ограничения и особенности

- **Два** **логических** входа (**minfin.kamgov.ru** и **kamgov.ru/minfin**) требуют **явной** **фиксации** **источника** при **цитировании**.
- **Массовое** скачивание без **пауз** может упираться в **WAF/CDN**.
