## Описание

Раздел сайта Минэкономразвития края посвящён **государственным программам субъекта**: типично публикуются паспорта, планы мероприятий и отчётные материалы. Для сопоставления с **федеральным** контуром программ используйте [programs.gov.ru](/information-systems/federal/programs-gov) и терминологию [государственных программ](/glossary/gov-programs); для бюджетных рядов — [открытые данные «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets) при необходимости сверки с федеральной витриной.

## Объекты данных и показатели

- Документы и страницы по **государственным программам Хабаровского края** (паспорта, перечни мероприятий, отчётность по реализации — по мере размещения).
- Связка с бюджетом: через отсылки к законам о бюджете и приложениям (см. [Законодательную Думу](/data-sources/regional/khabarovsk-krai-legislative-duma), [Минфин](/data-sources/regional/khabarovsk-krai-minfin)).

## Периодичность обновления

Обновляется по мере утверждения и корректировок программ и отчётности; точный регламент см. на сайте оператора.

## Ограничения и особенности

- URL раздела может меняться при редизайне портала; при 404 проверьте навигацию с главной https://minec.khabkrai.ru/ .
- Машиночитаемые наборы в этом разделе не гарантированы; для массовой выгрузки программно-целевых показателей ориентируйтесь на федеральные и краевые открытые данные в других витринах.
