# Хабаровский край: Управление Росстата по Хабаровскому краю, Магаданской области, Еврейской автономной области и Чукотскому АО (ТУ Росстата № 27)

## Описание

Сайт **27.rosstat.gov.ru** публикует **территориальные** материалы Росстата по зоне ТУ № 27. При анализе показателей **именно Хабаровского края** проверяйте разбивку по субъектам и методические примечания, чтобы не смешать ряды с Магаданской областью, ЕАО и Чукоткой. Для сопоставимых **федеральных** открытых наборов используйте [rosstat-opendata](/data-sources/federal/rosstat-opendata).

## Оператор

Управление Федеральной службы государственной статистики по Хабаровскому краю, Магаданской области, Еврейской автономной области и Чукотскому автономному округу — канонический вход: https://27.rosstat.gov.ru/

## Объекты данных и показатели

- **Социально-экономические показатели** по субъектам в зоне ТУ (в публикациях сайта).
- Справочная информация о **статистическом наблюдении** и релизах.

## Периодичность обновления

По календарю публикаций официальной статистики (месячные, квартальные, годовые блоки — в зависимости от показателя).

## Ограничения и особенности

- Один территориальный орган обслуживает **несколько** субъектов РФ: фильтруйте таблицы по нужному региону.
- Для машинной обработки больших рядов предпочтительны федеральные инструменты ([ЕМИСС/ОД](/data-sources/federal/rosstat-opendata), [Show Data](/data-sources/federal/rosstat-showdata-finder)).
