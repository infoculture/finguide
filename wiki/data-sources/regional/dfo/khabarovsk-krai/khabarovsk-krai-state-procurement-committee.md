## Описание

Портал **gz.khabkrai.ru** предназначен для публикации **региональных** материалов по организации закупок для нужд Хабаровского края: разъяснения, новости, методические документы, иногда агрегированная аналитика. Для построения реестра контрактов, извещений и протоколов используйте федеральную [ЕИС](/information-systems/federal/zakupki) и [XML-архив](/data-sources/federal/zakupki-xml); для сводной статистики ЕИС — [zakupki-eis-statistics](/data-sources/federal/zakupki-eis-statistics).

- Не путать с полным юридическим архивом закупок: первичные документы закупки ищите в ЕИС по реестровым номерам.
