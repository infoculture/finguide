# Магаданская область: Официальный интернет-портал правовой информации (раздел субъекта)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный портал с **региональным срезом** по НПА Магаданской области |
| Тип доступа | Публичный веб-интерфейс реестра официального опубликования |
| Основные форматы | HTML, PDF (официальные тексты актов) |
| Оператор | Федеральный орган власти (оператор ОИП); акты — органы Магаданской области |
| Режим доступа | Открытый |

Коротко: каноническая **федеральная** витрина официального опубликования региональных НПА (включая законы о бюджете), если акт зарегистрирован в установленном порядке. Для **хода принятия** и проектов законов используйте сайт областной Думы; для **бюджетных таблиц** — Минфин и «Открытый бюджет».

## Описание

Ссылка `pravo.gov.ru/links/magadan_obl` ведёт в **контур ОИП** с фильтром по субъекту. Это ключевой источник для юридически значимых редакций законов Магаданской области, постановлений Правительства и иных актов, прошедших регистрацию. Для бюджетного анализа начинайте с **закона о бюджете** и связанных актов в этом контуре, затем переходите к числовым приложениям на сайте Минфина.

## Объекты данных и показатели

- Зарегистрированные НПА Магаданской области с реквизитами опубликования.
- Полные тексты актов и их редакции (в пределах функционала ОИП).
- Связка с бюджетом: законы о бюджете, изменения, сопутствующие постановления.

## Периодичность обновления

По мере регистрации и опубликования новых актов; пик в период бюджетного сезона.

## Ограничения и особенности

- Проекты законов и «рабочие» версии до официального опубликования в ОИП **могут отсутствовать** — смотрите сайт Думы.
- Для машинной обработки массовых выгрузок ориентируйтесь на **официальные инструменты** портала pravo.gov.ru, а не на парсинг страницы-агрегатора.
