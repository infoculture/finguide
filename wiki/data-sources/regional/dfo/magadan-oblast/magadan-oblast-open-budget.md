# Магаданская область: Открытый бюджет (openbudget.49gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный портал; часть разделов касается **областного и муниципального** уровня |
| Тип доступа | Публичный веб-портал (интерактивные отчёты и материалы для граждан) |
| Основные форматы | HTML; при наличии экспорта — таблицы (CSV/XLSX) **по факту кнопок на сайте** |
| Оператор | Правительство Магаданской области / уполномоченный орган (портал домена `49gov.ru`) |
| Режим доступа | Открытый |

Коротко: витрина **«бюджет для граждан»**: понятные срезы по доходам и расходам, исполнению, долгу и (при наличии на сайте) муниципальному контуру. Для аудиторской точности первичных таблиц закона о бюджете по-прежнему сверяйтесь с **Минфином** и текстами НПА.

## Описание

Портал «Открытый бюджет» предназначен для **наглядного доступа** к бюджетным параметрам и исполнению. Типичные задачи: быстро сравнить годы, получить разрезы по функциям/ведомствам (если включены в модель), оценить долговой блок. Для построения воспроизводимых длинных рядов комбинируйте экспорты (если есть) с первичными PDF/XLSX на сайте Минфина.

## Объекты данных и показатели

- Агрегированные показатели областного бюджета: доходы, расходы, дефицит/источники финансирования.
- Исполнение бюджета по доступным срезам периода.
- Государственный и муниципальный долг — в объёме, вынесенном на портал.
- Муниципальный контур — если раздел реально заполнен по ОКТМО/муниципалитетам.

## Периодичность обновления

Обычно следует за официальной отчётностью Минфина области; точные даты обновления модулей смотрите на портале.

## Ограничения и особенности

- Показатели могут быть **округлены или агрегированы** для наглядности — не заменяют без сверки первичные бюджетные таблицы.
- Наличие машиночитаемых выгрузок меняется от версии к версии витрины; не закладывайте на стабильный JSON без проверки.
