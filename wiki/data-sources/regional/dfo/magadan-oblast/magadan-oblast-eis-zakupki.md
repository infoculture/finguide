# Магаданская область: закупки в ЕИС (федеральная витрина)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральная система; данные по закупкам **всех** заказчиков, в т.ч. Магаданской области |
| Тип доступа | Публичный веб-интерфейс и федеральные выгрузки (XML/инструменты ЕИС) |
| Основные форматы | HTML, XML (официальные выгрузки по правилам ЕИС) |
| Оператор | Единая информационная система в сфере закупок (оператор — федеральный контур) |
| Режим доступа | Открытый |

Коротко: это **не региональный** сайт, а **федеральная** точка правды по контрактной системе (44-ФЗ) и связанным режимам публикации в ЕИС. Для Магаданской области все публичные закупочные следы, которые законом должны попасть в ЕИС, ищутся здесь с отбором по заказчику, региону или коду ОКТМО.

## Описание

Карточка в разделе региональных источников отражает **типичный сценарий аналитика**: «найти закупки и контракты именно по Магаданской области». Юридически и технологически это тот же **ЕИС**, что и для всей страны: региональный контур отдельного «API Магадана» не выделяется. Используйте фильтры заказчика, региональные реестры учреждений и далее — федеральные выгрузки.

## Оператор

Федеральный оператор ЕИС (см. карточку информационной системы ниже) — канонический вход: https://zakupki.gov.ru/

- Карточка ИС: [ЕИС в сфере закупок](/information-systems/federal/zakupki) — официальное описание границ системы и каналов доступа.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Главная ЕИС | https://zakupki.gov.ru/ | HTML |
| Поиск и реестры | разделы «Закупки», «Контракты», «Реестры» на zakupki.gov.ru | HTML / выгрузки по правилам ЕИС |

## Объекты данных и показатели

- Извещения, протоколы, контракты, изменения контрактов (в пределах режимов 44-ФЗ и иных подлежащих публикации в ЕИС).
- Идентификаторы заказчика, ИНН, реестровые номера контрактов, суммы, предмет закупки (для связки с бюджетом — через заказчика и КБК в первичных документах, если раскрыты).
- Для закупок по 223-ФЗ и иным режимам — проверяйте, обязана ли конкретная закупка находиться в ЕИС.

## Периодичность обновления

Оперативная публикация по мере размещения заказчиками; федеральные регламенты сроков раскрытия задают нижнюю границу задержки.

## Ограничения и особенности

- Не все расходы бюджета субъекта проходят через контракты, публикуемые в ЕИС.
- Для стабильных конвейеров предпочтительны **официальные XML/инструменты** (см. федеральные карточки), а не скрейпинг HTML.
