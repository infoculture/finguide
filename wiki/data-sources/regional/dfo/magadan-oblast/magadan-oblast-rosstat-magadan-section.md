# Магаданская область: Росстат (Магаданская область в составе межрегионального управления)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральная статистика с **региональным срезом** по Магаданской области (ведение — межрегиональное управление Росстата) |
| Тип доступа | Публичный веб-раздел |
| Основные форматы | HTML, прилагаемые таблицы (часто XLSX/PDF — по конкретным публикациям) |
| Оператор | Федеральная служба государственной статистики (через МУ по ДФО) |
| Режим доступа | Открытый |

Коротко: **официальная статистика** по региону в разрезе методологии Росстата: демография, экономика, социальная сфера и др. — в объёме, вынесенном на сайт управления. Для **машиночитаемых массивов** по стране и сопоставимых рядов используйте федеральные витрины ЕМИСС и открытых данных Росстата; эта страница — про **региональный контент** межрегионального управления.

## Описание

Магаданская область обслуживается **Межрегиональным управлением Федеральной службы государственной статистики по Хабаровскому краю, Магаданской области, Еврейской автономной области и Чукотскому автономному округу** (сайт `27.rosstat.gov.ru`). Папка `folder/23554` использовалась в исходном обзоре как вход в материалы по Магаданской области; при смене структуры сайта обновите `source_url` в карточке, сохранив поиск по официальному сайту управления.

Для финансовых показателей субъекта (бюджет, долг) сочетайте Росстат с **Минфином** и «Открытым бюджетом»: это разные методологии и объекты.

## Оператор

Межрегиональное управление Росстата по указанным субъектам — базовый домен: https://27.rosstat.gov.ru/ (раздел по Магаданской области уточняйте по навигации сайта; исходная точка обзора: https://27.rosstat.gov.ru/folder/23554)

## Объекты данных и показатели

- Региональные статистические сборники, бюллетени и сопутствующие таблицы (по мере публикации).
- Показатели социально-экономического развития в официальных определениях Росстата.
- Не является источником **бюджетного исполнения** и казначейских рядов.

## Периодичность обновления

По расписанию формирования официальной статистики; отдельные разделы могут обновляться эпизодически.

## Ограничения и особенности

- URL вложенных папок на rosstat.gov.ru **исторически менялись**; при 404 ищите эквивалент через главную `27.rosstat.gov.ru` или федеральный каталог.
- Сопоставление с бюджетными таблицами только после проверки **определений показателей**.
