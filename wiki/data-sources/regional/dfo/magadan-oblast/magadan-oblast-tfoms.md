# Магаданская область: Территориальный фонд ОМС

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный внебюджетный фонд в сфере ОМС |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML, PDF (документы и отчёты — по факту размещения) |

| Режим доступа | Открытый |

Коротко: источник по **программе государственных гарантий**, тарифной политике ОМС, регламентам и отчётности ТФОМС в части, подлежащей раскрытию на официальном сайте. Для связки с **консолидированным бюджетом** субъекта используйте материалы Минфина и «Открытого бюджета»: там отражаются трансферты и агрегаты, а ТФОМС даёт **отраслевой** контекст.

## Описание

Сайт ТФОМС публикует материалы, ориентированные на застрахованных лиц, страховые медицинские организации и поставщиков услуг: программа ОМС, методические письма, отчёты о реализации программы и финансовые ведомости — **в объёме, который фонд выкладывает публично**. Для аналитики бюджетных трансфертов на здравоохранение сопоставляйте публикации фонда с **областным бюджетом**.

## Объекты данных и показатели

- Территориальная программа ОМС, изменения и разъяснения.
- Отчёты и аналитические материалы фонда (структура и периодичность — по сайту).
- Тарифы, нормативы, реестры/перечни в части, отнесённой к компетенции ТФОМС (по факту публикации).

## Периодичность обновления

Новости — по мере событий; нормативные изменения — по мере принятия; годовая и промежуточная отчётность — по календарю фонда и требованиям отчётности.

## Ограничения и особенности

- Финансовые детали могут быть **рассредоточены** по PDF и разделам; для единой картины по бюджету субъекта начинайте с Минфина, затем уточняйте отраслевые детали на сайте ТФОМС.
