# Приморский край: запись в DataCatalogs.ru о портале открытых данных

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Гражданский** метакаталог о **региональной** витрине |
| Тип доступа | Запись в реестре каталогов |
| Основные форматы | HTML |
| Оператор | DataCatalogs.ru (Инфокультура и сообщество) |

Коротко: **вторичная** навигационная точка. Для загрузки наборов и паспортов данных используйте **[официальный портал края](/data-sources/regional/primorsky-krai-opendata-portal)**.

## Описание

Страница в [DataCatalogs.ru](/information-systems/civil/datacatalogs-ru) помогает **найти** официальный каталог открытых данных Приморского края среди других региональных порталов. Для анализа бюджетных рядов после нахождения портала работайте уже с [официальной витриной](/data-sources/regional/primorsky-krai-opendata-portal).

## Объекты данных и показатели

- **Метаданные** о портале (URL, описание, теги — по состоянию записи).
- **Не** содержит самих бюджетных наборов; строки датасетов — на `opendata.primorsky.ru`.

## Периодичность обновления

Определяется поддержкой записи в DataCatalogs.ru; не синхронизирована автоматически с официальным порталом.
