# Приморский край: Приморскстат (территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** статистика в **региональном** изложении (код субъекта 25) |
| Тип доступа | Официальный сайт ТОР Росстата |
| Основные форматы | HTML, таблицы **XLSX**/документы по ссылкам |
| Оператор | Территориальный орган Федеральной службы государственной статистики по Приморскому краю |
| Режим доступа | Открытый |

Коротко: **официальные** региональные статистические данные и методические комментарии. Для **административной** бюджетной отчётности субъекта используйте [Минфин края](/data-sources/regional/primorsky-krai-minfin); для сопоставления с **федеральной** статистикой — [ЕМИСС и открытые данные Росстата](/data-sources/federal/rosstat-opendata).

## Описание

Сайт публикует **официальные** срезы по экономике региона, **демографию**, отраслевые блоки; разделы, касающиеся **финансов** субъектов и секторов, появляются в составе официальных статистических компиляций. Прямое сопоставление с **казначейским** исполнением бюджета требует осторожности из‑за разных методик и периодов.

## Объекты данных и показатели

- **Официальные статистические показатели** по Приморскому краю.
- Публикации в разрезе **отраслей**, **периодов**, иногда **муниципальных образований** (по форме конкретного сборника).
- Ключи: **код субъекта 25**, **период**, классификаторы отраслей/форм собственности — по метаданным таблиц.

## Периодичность обновления

По **календарю** публикации официальной статистики (оперативные данные, годовые сборники — см. анонсы на сайте).

## Ограничения и особенности

- Статистические показатели **не заменяют** бюджетную первичку из казначейства и бюджетной отчётности.
