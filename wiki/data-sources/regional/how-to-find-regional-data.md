# Как искать региональные бюджетные данные

| Признак | Значение |
| :--- | :--- |
| Уровень | Субъекты РФ и муниципалитеты |
| Тип доступа | Преимущественно открытые порталы и файловые выгрузки |
| Назначение | Найти официальный источник исполнения бюджета субъекта и связанных открытых наборов |

Коротко: это **не карточка одного URL**, а **рецепт**, как не потеряться между сайтом субъекта, зеркалами «Открытого бюджета» и федеральными сводами.

## Описание

Страница обобщает типовой порядок действий при поиске региональных публикаций (план, исполнение, долг, адресные программы). Используйте её вместе с обзором [региональных источников](/data-sources/regional/) и методическими материалами [муниципальных бюджетов](/data-sources/regional/municipal-budgets).

## Типовой алгоритм

1. Найдите официальный домен органа исполнительной власти субъекта (министерство финансов / финуправление) и раздел «Бюджет» / «Открытый бюджет».
2. Зафиксируйте **год** и **ведомственную принадлежность** набора (консолидированный бюджет vs отдельный фонд или ведомство).
3. Сверьте терминологию полей с [глоссарием](/glossary/) (КБК, КОСГУ, территориальные коды — см. [индекс кодов](/reference/classification-codes-index)).
4. Для межрегиональных сравнений используйте вспомогательные федеральные своды ([краткая информация об исполнении консолидированных бюджетов субъектов](/data-sources/federal/minfin-subbud-execute)), понимая ограничения детализации.

## Ограничения

Форматы и периодичность публикаций различаются; часть разрезов доступна только в PDF. Федеральные агрегаты не заменяют региональный первоисточник при юридически значимых выводах.
