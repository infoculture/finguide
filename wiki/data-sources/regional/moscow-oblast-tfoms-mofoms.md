# Территориальный фонд ОМС Московской области (МОФОМС)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **ТФОМС** **Московской области** (субъект РФ); портал — первичный вход к **нормативке**, **реестрам** и **сервисам** ОМС в границах области |
| Тип доступа | Публичный **веб-портал** с разделами для граждан и участников программы ОМС |
| Основные форматы | **HTML**, **PDF** |
| Оператор | **Территориальный фонд обязательного медицинского страхования Московской области** |
| Режим доступа | Открытое чтение публичных разделов; личные кабинеты — по правилам фонда и ЕПГУ |

Коротко: **официальный сайт ТФОМС** для **Московской области**; для **бюджета и исполнения** в машиночитаемом виде используйте **[«Электронный бюджет»](/information-systems/federal/giis-eb)** и **[0503317](/reporting/0503317)** по **консолидированному** бюджету области, не подменяя их публикациями только для гражданского контура полисов.

## Описание

**МОФОМС** — территориальный фонд ОМС субъекта «Московская область». Модель **ОМС** и роль **ФФОМС** — см. **[ФФОМС](/organizations/ffoms)** и **[навигатор по региональному звену ОМС](/data-sources/regional/regional-extrabudgetary-health-overview)**.

**Не путать:** **[МГФОМС](/data-sources/regional/moscow-city-tfoms-mgfoms)** относится к **городу Москва** (отдельный субъект РФ); юридический адрес и зона деятельности **МОФОМС** — **Московская область**.

## Объекты данных и показатели

- **Нормативные и организационные документы** фонда (разделы «О фонде», «Документы» — по актуальной структуре меню).
- **Реестры** участников ОМС (медицинские и страховые организации) — для связки с другими реестрами по **ИНН/ОГРН**.
- **Бюджет фонда** — в **[региональных бюджетах](/data-sources/federal/regionalnye-byudzhety)** и **[ГИИС](/information-systems/federal/giis-eb)**; исполнение в связке с **[консолидированными бюджетами](/data-sources/regional/consolidated-budgets)**.

## Периодичность обновления

По мере публикаций фонда и нормативных актов; для отчётности исполнения — регламент **0503317** и календарь субъекта.

## Ограничения и особенности

- Регион **«Москва»** и **«Московская область»** — **разные** субъекты РФ; карточки ТФОМС не взаимозаменяемы.
- Редиректы между `http`/`https` и вариантами домена — проверяйте **фактический** URL в браузере.

## Частые вопросы

**В:** Полис, выданный в Москве, относится к МОФОМС?

**О:** Зависит от **места регистрации** и правил страхования; для аналитики бюджета ориентируйтесь на **субъект учёта** фонда и реквизиты в полисе, а не на почтовый адрес сайта.
