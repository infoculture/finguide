# Севастополь: Территориальный фонд ОМС города Севастополя

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Город федерального значения; **внебюджетный** фонд ОМС на территории Севастополя |
| Тип доступа | Публичный корпоративный портал ТФОМС |
| Основные форматы | HTML; отчёты часто как PDF/DOCX во вложениях |
| Оператор | Территориальный фонд обязательного медицинского страхования города Севастополя |
| Режим доступа | Открытый |

Коротко: официальный вход для **отчётности ТФОМС** и материалов о **расходовании средств ОМС**; для анализа **консолидированного бюджета города** используйте вместе с бюджетными витринами и отчётностью финоргана.

## Описание

ТФОМС ведёт учёт и раскрытие информации в рамках **обязательного медицинского страхования**. Для финансового анализа города важно не смешивать **средства ОМС** с **бюджетом муниципального/городского уровня** без явной методики консолидации.

## Объекты данных и показатели

- Отчёты о деятельности фонда, документы по тарифам и программам ОМС (где опубликованы).
- Показатели расходования средств ОМС в форме, определённой оператором.
- Связка с бюджетом города — через **отдельные трансферты/субсидии** (ищите в [бюджетных](/data-sources/regional/sevastopol-legislative-assembly-budget-laws) и [исполнительных](/data-sources/regional/sevastopol-finance-department) витринах).

## Периодичность обновления

По событиям и отчётным датам фонда; типично квартальная/годовая отчётность в открытых разделах.

## Ограничения и особенности

- Это **не** портал Департамента финансов: классификация расходов и показатели другие (ОМС vs бюджетные статьи).
- Для машинной обработки чаще приходится извлекать данные из **PDF/таблиц** в отчётах, а не из API.
