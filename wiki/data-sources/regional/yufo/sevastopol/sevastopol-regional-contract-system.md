# Севастополь: региональная контрактная система (РКС, sevzakaz.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Город федерального значения; **региональный** контур публикаций по закупкам |
| Тип доступа | Публичный веб-портал РКС |
| Основные форматы | HTML; выгрузки — по функционалу портала |
| Оператор | Оператор региональной контрактной системы города Севастополя |

Коротко: **региональная** витрина закупок Севастополя; для **полного федерального следа** по процедурам в ЕИС используйте [XML-выгрузки ЕИС](/data-sources/federal/zakupki-xml) и карточку [ЕИС как ИС](/information-systems/federal/zakupki).

## Описание

РКС дублирует или дополняет федеральный контур **zakupki.gov.ru** для части процедур и заказчиков города. На практике аналитики часто **склеивают** региональный портал с **ЕИС** по реестровым номерам закупок и контрактов.

## Объекты данных и показатели

- Карточки **закупок**, **контрактов**, планы-графики (если размещаются на РКС).
- Ключи связки: **реестровый номер** контракта/процедуры, **ИНН заказчика**, КБК/коды закупки — по мере доступности в интерфейсе.

## Периодичность обновления

В реальном времени по мере публикаций участниками.

## Ограничения и особенности

- Не все закупки субъекта могут проходить через **только** РКС — сверяйте с **ЕИС**.
- Для RAG и машинного парсинга стабильнее опираться на **XML ЕИС**, а РКС использовать как региональный контекст.
