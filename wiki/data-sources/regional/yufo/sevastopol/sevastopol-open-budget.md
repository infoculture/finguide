# Севастополь: портал «Открытый бюджет» (бюджет для граждан)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Город федерального значения; публичная бюджетная витрина для граждан |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML; возможны вложения (PDF, XLSX) в материалах |
| Оператор | Исполнительные органы города (витрина «Открытый бюджет» / бюджет для граждан) |
| Режим доступа | Открытый |

Коротко: **главная гражданско-ориентированная** витрина по бюджету Севастополя: планы, исполнение и пояснения; для **юридически точных** формулировок и полного текста НПА используйте [раздел бюджетных законов](/data-sources/regional/sevastopol-legislative-assembly-budget-laws) и [банк законов](/data-sources/regional/sevastopol-legislative-assembly-law-bank).

## Описание

Портал **ob.sev.gov.ru** предназначен для раскрытия бюджетной информации в доступной форме (бюджет для граждан, исполнение, разъяснения). Для аналитики открытых данных полезно сопоставлять показатели здесь с **отчётами об исполнении** на сайте [Департамента финансов](/data-sources/regional/sevastopol-finance-department) и с **федеральными** витринами при консолидации.

## Оператор

Публичная бюджетная витрина города Севастополя — https://ob.sev.gov.ru/

## Объекты данных и показатели

- Агрегированные **доходы и расходы** бюджета, материалы «бюджет для граждан».
- **Год**, **раздел/статья/ЦСР** (в терминах отображаемой классификации), пояснительные записки (если размещены).
- Для машинной обработки: проверяйте наличие **файловых вложений** и повторяемых таблиц; структура страниц может меняться.

## Периодичность обновления

Обычно привязана к **бюджетному циклу** и публикации отчётов об исполнении; уточнять по датам на портале.

## Ограничения и особенности

- Гражданская подача может **упрощать** детализацию относительно первичных форм отчётности.
- Не заменяет **ЕИС** по контрактам — закупки см. [РКС](/data-sources/regional/sevastopol-regional-contract-system) и [XML ЕИС](/data-sources/federal/zakupki-xml).
