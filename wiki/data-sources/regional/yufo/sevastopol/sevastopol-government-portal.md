# Севастополь: официальный портал органов государственной власти (sev.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Город федерального значения; **корневой** портал исполнительной власти и администрации |
| Тип доступа | Публичный веб-портал |
| Основные форматы | HTML |
| Оператор | Исполнительные органы государственной власти города Севастополя |
| Режим доступа | Открытый |

Коротко: **навигационный хаб** к департаментам и проектам; для **бюджетных рядов** первично идите на [fin.sev.gov.ru](/data-sources/regional/sevastopol-finance-department) и [ob.sev.gov.ru](/data-sources/regional/sevastopol-open-budget).

## Описание

Корневой домен **sev.gov.ru** обычно используют для **общей информации**, новостей и ссылок на профильные сайты департаментов. В таблице исходного обзора отмечено «частично найдено» для прямых финансовых документов — это ожидаемо: **детализация бюджета** чаще лежит на поддоменах.
