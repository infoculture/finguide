# Севастополь: УФНС России по городу Севастополю (nalog.gov.ru/rn92)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Город федерального значения; **территориальное** управление ФНС России |
| Тип доступа | Публичный федеральный портал (региональный раздел **rn92**) |
| Основные форматы | HTML; часть сервисов — формы и PDF |
| Оператор | Управление Федеральной налоговой службы по городу Севастополю |
| Режим доступа | Открытый; часть сервисов — после идентификации пользователя |

Коротко: **налоговый и административный** контур доходов; для **консолидированного бюджета города** в целом сверяйте с [Департаментом финансов](/data-sources/regional/sevastopol-finance-department) и бюджетными витринами.

## Описание

Региональный раздел **rn92** повторяет стандартную структуру сайта ФНС: электронные сервисы, новости, контакты, разъяснения. Для анализа **сводных бюджетных поступлений** городского уровня часто требуется связка с отчётностью **финоргана**, а не только новости УФНС.

## Объекты данных и показатели

- Администрирование **налогов и сборов**, региональные новости и методические материалы.
- Связка с **открытой отчётностью организаций** — через федеральные инструменты ([ГИР БО](/data-sources/federal/bo-nalog-gov-ru)), не через «бюджет для граждан» напрямую.

## Периодичность обновления

Непрерывно по мере обновления федерального портала и публикаций УФНС.

## Ограничения и особенности

- Это **не** портал Департамента финансов: показатели и задачи другие.
- Код **92** соответствует региональному контуру ФНС для Севастополя.
