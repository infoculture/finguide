# Севастополь: Контрольно-счётная палата города Севастополя

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Город федерального значения; **внешний муниципальный финансовый контроль** |
| Тип доступа | Публичный корпоративный портал |

| Оператор | Контрольно-счётная палата города Севастополя |

Коротко: **заключения и отчёты КСП** по результатам проверок, включая **внешнюю проверку годового отчёта об исполнении бюджета**; для **первичных рядов исполнения** используйте [Департамент финансов](/data-sources/regional/sevastopol-finance-department).

## Описание

КСП публикует материалы **внешнего контроля** качества и достоверности отчётности об исполнении бюджета и смежных вопросов. Для аналитики открытых данных эти документы дают **независимую** оценку, но не заменяют **первичные** таблицы исполнения.

## Объекты данных и показатели

- Заключения по **внешней проверке** годового отчёта об исполнении бюджета.
- Иные отчёты и информационные материалы КСП (по факту публикации).
- Связка: **финансовый год**, предмет проверки, ссылки на **нарушения/риски** (в тексте заключений).

## Периодичность обновления

Привязана к **циклу проверок** и утверждению заключений.

## Ограничения и особенности

- Тексты заключений — **экспертная оценка** контрольного органа, не первичный учёт.
