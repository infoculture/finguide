# Севастополь: записи в каталоге DataCatalogs об открытых бюджетных данных

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Гражданский каталог** (DataCatalogs), описывающий или собирающий ссылки на материалы по **бюджету Севастополя** |
| Тип доступа | Веб-карточка каталога |
| Основные форматы | HTML |
| Оператор | Проект DataCatalogs (не орган власти Севастополя) |
| Режим доступа | Открытый |

Коротко: **вторичный** навигатор к описаниям наборов и документов; для официальных цифр и НПА опирайтесь на [Открытый бюджет](/data-sources/regional/sevastopol-open-budget), [Департамент финансов](/data-sources/regional/sevastopol-finance-department) и [Законодательное Собрание](/data-sources/regional/sevastopol-legislative-assembly-law-bank).

## Описание

В исходном обзоре репозитория эта точка доступа была заведена как «портал открытых данных»; фактически URL ведёт на **запись в каталоге DataCatalogs**, который **сам по себе не является** оператором бюджетных данных города. Карточку сохраняем как **навигационный** ориентир с явным предупреждением о статусе.

## Объекты данных и показатели

- Метаданные и **ссылки** на наборы/документы по бюджету (по состоянию карточки на DataCatalogs).
- Для анализа: всегда **поднимайте первоисточник** (госоператор) и фиксируйте дату скачивания.

## Периодичность обновления

Определяется обновлением записи на DataCatalogs, не бюджетным оператором.

## Ограничения и особенности

- Риск **устаревших** ссылок и **неполной** метаинформации.
- Не использовать как единственное доказательство показателей в отчётах без сверки с официальными порталами.
