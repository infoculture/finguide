# Крымстат: территориальный орган Росстата (код 82)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** служба (территориальный орган) с публикациями по **Республике Крым** и **Севастополю** |
| Тип доступа | Официальный сайт территориального органа Росстата |
| Основные форматы | HTML; типовые выгрузки Росстата — XLSX, CSV (через связанные сервисы) |
| Оператор | Управление Федеральной службы государственной статистики по Республике Крым и г. Севастополю |
| Режим доступа | Открытый просмотр |

Коротко: **региональная** витрина **официальной статистики** для южного контура: демография, экономика, социальная сфера — с оговоркой, что **один** территориальный орган обслуживает **два** субъекта Федерации; при сравнении с бюджетом субъекта разделяйте **коды территорий** и методические примечания Росстата.

## Описание

Сайт **82.rosstat.gov.ru** — вход в **публичные материалы** территориального органа Росстата с **кодом 82**. Для машинной выгрузки показателей по стандартным формам часто удобнее перейти к **федеральным** инструментам поиска таблиц ([Поиск данных](/data-sources/federal/rosstat-showdata-finder)) и открытым наборам ([Росстат: открытые данные](/data-sources/federal/rosstat-opendata)), а региональный сайт использовать для **контекста** и пресс-релизов.

## Объекты данных и показатели

- **Региональные своды** по показателям официальной статистики с разрезами, доступными для публикации.
- **Пресс-выпуски** и краткие комментарии к ключевым индикаторам.
- Сопоставление с **бюджетными** рядами — через **согласованные классификаторы** и периоды; бюджетные агрегаты см. у [Минфина](/data-sources/regional/republic-of-crimea-minfin).

## Периодичность обновления

По **статистическому календарю**: ежемесячная, квартальная и годовая отчётность в зависимости от формы.

## Ограничения и особенности

- **Два субъекта** в одном Управлении: внимательно разделяйте **Республику Крым** и **Севастополь** в таблицах и методичках.
- Для больших исторических рядов используйте **ЕМИСС**/федеральные банки данных в дополнение к региональной ленте новостей.
