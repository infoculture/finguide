# Краснодарский край: Краснодарстат (Территориальный орган Росстата)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Краснодарский край** (и **Республика Адыгея** в контуре **ТОР**) — **официальная** статистика **Росстата** |
| Тип доступа | Публичный **сайт** **территориального** органа |
| Основные форматы | **HTML**; **XLSX** и др. **в** публикациях |
| Оператор | **Управление Федеральной службы государственной статистики** по Краснодарскому краю и Республике Адыгея (**Краснодарстат**) |
| Режим доступа | Открытый |

Коротко: **региональный** вход к **официальным** **статистическим** данным, включая ряды, **смежные** с **финансами** региона. **Бюджетное** исполнение по **КБК** — через **[Минфин](/data-sources/regional/krasnodar-krai-minfin-krasnodar)** и **федеральные** витрины.

Заполняйте по определениям на странице **[«Измерения качества источника данных»](/reference/data-quality-dimensions)**.

| Полнота (`data_completeness`) | **частичная** — набор таблиц **зависит** от **программы** статнаблюдения |

## Описание

**Краснодарстат** публикует **своды** по **экономике**, **населению**, **социальной** сфере. **Показатели** бюджета как **учётного** регистра **здесь** **не** заменяют **казначейские** и **бюджетные** ИС.
