# Херсонская область: Электронный бюджет (раздел «Регионы»)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** витрина с **региональным** разрезом (все субъекты, включая Херсонскую область при наличии рядов) |
| Тип доступа | Публичный веб-портал ГИИС «Электронный бюджет» |
| Основные форматы | HTML (интерактивные разделы); массовые выгрузки — через федеральный каталог наборов |
| Оператор | Единый портал бюджетной системы Российской Федерации «Электронный бюджет» |
| Режим доступа | Открытый просмотр публичных разделов |

Коротко: **агрегирующая** федеральная витрина для сравнения и навигации по **региональным** бюджетам. Для **первичных** материалов субъекта используйте [Минфин Херсонской области](/data-sources/regional/kherson-oblast-minfin); для **машиночитаемых** наборов — [каталог budget.gov.ru](/data-sources/federal/budget-gov-ru-datasets).

## Описание

В исходной таблице источников для Херсонской области указан **единый федеральный** портал **budget.gov.ru** как канал категории «Открытый бюджет / бюджет для граждан». Отдельного регионального портала «Бюджет для граждан» на домене субъекта в таблице **не** зафиксировано — аналитикам следует комбинировать эту витрину с [сайтом Минфина](/data-sources/regional/kherson-oblast-minfin).

## Объекты данных и показатели

- **Сводные** показатели бюджетов субъектов РФ (при наличии на витрине).
- **Навигация** к разделам открытого бюджета и сравнительной информации.
- Детальные **машиночитаемые** ряды — в [федеральном каталоге наборов](/data-sources/federal/budget-gov-ru-datasets).

## Периодичность обновления

Привязана к **циклам** публикации данных на федеральной платформе.

## Ограничения и особенности

- Исходное исследование отмечало **технические сложности** с прямым доступом к региональному разделу — перед автоматизацией проверяйте актуальный URL и доступность из вашей сети.
- Показатели новых субъектов могут **заполняться с задержкой** относительно «старых» регионов.
