# Ростовская область: Министерство финансов (minfin.donland.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный финансовый орган субъекта РФ |
| Тип доступа | Официальный веб-портал |
| Основные форматы | HTML, PDF |
| Оператор | Министерство финансов Ростовской области |
| Режим доступа | Открытый |

Коротко: базовый источник по областному бюджету и исполнению. Для гражданской визуализации используйте [Открытый бюджет](/data-sources/regional/rostov-oblast-open-budget), для официально опубликованных НПА — [pravo.donland.ru](/data-sources/regional/rostov-oblast-pravo-donland).

## Объекты данных и показатели

- Закон об областном бюджете и изменения (в связке с парламентским и правовым контуром).
- Отчёты об исполнении, материалы по госпрограммам, методические документы.
- Ключи связки: год, КБК/разделы расходов, ОКТМО (когда присутствуют в таблицах).

## Периодичность обновления

По календарю бюджетного процесса и по мере публикации документов.
