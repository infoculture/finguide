# Донецкая Народная Республика: Донецкстат (территориальный орган Росстата по ДНР)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Региональный орган официальной статистики в контуре ДНР |
| Тип доступа | Публичный веб-сайт на выделенном поддомене `*.rosstat.gov.ru` |
| Основные форматы | HTML; таблицы и вложения по ссылкам публикаций |
| Оператор | Донецкстат (ТОФ Росстата по ДНР) |
| Режим доступа | Открытое ознакомление с публичными материалами |

Коротко: **официальная региональная статистика** для ДНР: демография, экономика, социальная сфера и отраслевые своды, которые удобно сопоставлять с бюджетными документами **[Минфина ДНР](/data-sources/regional/donetsk-people-republic-minfin)** и федеральными **[госфинансовыми рядами Росстата](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf)**.

## Описание

Донецкстат публикует **региональные** результаты статистического наблюдения и методические пояснения. Для **единой межведомственной статистики** и сопоставимых временных рядов по РФ в целом используйте **[ЕМИСС](/information-systems/federal/emiss)** и связанные федеральные карточки; для **госфинансов** — отдельные наборы Росстата (см. связанные страницы).

## Оператор

Донецкстат (территориальный орган Федеральной службы государственной статистики по Донецкой Народной Республике) — канонический вход: https://80.rosstat.gov.ru/

## Объекты данных и показатели

- **Официальные статистические публикации** по субъекту (перечень — по меню сайта).
- **Метаданные** (единицы измерения, периодичность, классификаторы) — внутри конкретных таблиц и сборников.
- Связка с **бюджетными агрегатами** выполняется осознанно: статистика и административная отчётность по бюджету отвечают разным регламентам.

## Периодичность обновления

Зависит от вида наблюдения: от оперативных пресс-релизов до годовых сборников.

## Ограничения и особенности

- Числовой **IP-хост** в URL (`80.rosstat.gov.ru`) может быть менее привычен для ссылок; фиксируйте канонический `source_url` из frontmatter при цитировании.
- Для **машинной обработки** предпочитайте федеральные open data там, где показатель действительно дублируется в паспорте набора.
