# Запорожская область: Территориальный фонд ОМС

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Запорожская область: **внебюджетный** контур ОМС (отдельный бюджет ТФОМС) |
| Тип доступа | Официальный сайт ТФОМС |
| Основные форматы | HTML, PDF |
| Оператор | Территориальный фонд обязательного медицинского страхования Запорожской области |
| Режим доступа | Открытый просмотр |

Коротко: **отдельный бюджетный контур** обязательного медицинского страхования: тарифы, договоры, отчётность и нормативные материалы ТФОМС. С **областным бюджетом** сопоставляйте через [Минфин](/data-sources/regional/zaporozhye-oblast-minfin) и федеральные ряды при наличии перекрёстных показателей.

## Описание

**tfomszo.ru** — типовой портал ТФОМС: информация об ОМС, новости, документы и отчёты в публичном доступе. Для **консолидированной** бюджетной аналитики субъекта ТФОМС дополняет, но не заменяет [региональный бюджет](/data-sources/regional/zaporozhye-oblast-minfin).

Территориальный фонд обязательного медицинского страхования Запорожской области — https://tfomszo.ru/

## Объекты данных и показатели

- **Бюджет ТФОМС** и связанные нормативные акты (в объёме публикаций).
- **Тарифы** и условия финансирования медицинской помощи.
- **Отчётность** фонда — при размещении на сайте.

## Периодичность обновления

Связана с **циклом планирования** бюджета ТФОМС и отчётными периодами ОМС.

## Ограничения и особенности

- Бюджет ТФОМС **не сводится** автоматически с областным бюджетом на одной витрине — нужна явная стыковка показателей.
