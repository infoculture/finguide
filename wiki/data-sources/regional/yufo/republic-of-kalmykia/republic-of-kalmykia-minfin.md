# Республика Калмыкия: Министерство финансов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Республика Калмыкия** — **региональный** финансовый орган |
| Тип доступа | Публичный **официальный** сайт |
| Основные форматы | **HTML**; **PDF** и другие **вложения** по разделам |
| Оператор | **Министерство финансов** Республики Калмыкия |
| Режим доступа | Открытый |

Коротко: главный **ведомственный** вход к **материалам** по **республиканскому бюджету**, **исполнению** и **межбюджетным** связям. Популяризированный слой «бюджет для граждан» вынесен в отдельную карточку: [бюджет для граждан (Минфин РК)](/data-sources/regional/republic-of-kalmykia-budget-for-citizens-minfinrk).

## Описание

На сайте обычно публикуются **официальные** документы и справочная информация по **бюджетному циклу** республики: **проекты** закона о бюджете, материалы по **исполнению**, **межбюджетным** отношениям. В таблице-источнике отмечались **перебои доступности** — закладывайте **зеркала** (например сохранённые копии НПА через [Народный Хурал](/data-sources/regional/republic-of-kalmykia-hural-parliament) или [Правительство РК](/data-sources/regional/republic-of-kalmykia-government-kalmregion)) и сверку с [федеральными наборами](/data-sources/federal/budget-gov-ru-datasets) там, где возможно.

## Объекты данных и показатели

- **Документы** по **планированию** и **исполнению** республиканского бюджета (в публикуемой части сайта).
- **Методические** и **информационные** материалы **Минфина** республики.
- **Связка** с **КБК** и федеральными рядами **исполнения** — через [наборы «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets) и [казначейский контур](/data-sources/regional/republic-of-kalmykia-ufk-roskazna).

## Периодичность обновления

**Нерегулярно** по **событиям** бюджетного процесса и **отчётным** датам; точные сроки — по датам публикаций в соответствующих разделах сайта.

## Ограничения и особенности

- **TLS**: при **автоматических** проверках из CLI возможна ошибка **цепочки сертификата**; в браузере сайт может открываться штатно.
- **Полнота**: не все **таблицы** и **выгрузки** с сайта удобно **воспроизводить** без ручного копирования.

```bash
curl -fsSLk -o /dev/null -w "%{http_code}\n" --max-time 25 -A "finguide-minfin-rk-check/1.0" \
  "https://www.minfinrk.ru/"
```
