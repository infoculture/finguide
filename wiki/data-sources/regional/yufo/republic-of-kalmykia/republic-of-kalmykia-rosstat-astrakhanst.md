# Республика Калмыкия: территориальный орган Росстата

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Территориальное** отделение Росстата (**Астраханьстат**) — **объединённая** зона **Астраханской области** и **Республики Калмыкия** |
| Тип доступа | Публичный **сайт** **территориального** органа |
| Основные форматы | **HTML**; **PDF** |
| Оператор | **Управление** Федеральной службы государственной статистики по **Астраханской области и Республике Калмыкия** |
| Режим доступа | Открытый |

Коротко: **официальный** **региональный** вход Росстата для **Калмыкии** через **объединённое** управление **с Астраханской областью**. **Машинные** массивы «**как у Росстата в целом**» ищите в [федеральной витрине открытых данных](/data-sources/federal/rosstat-opendata).

## Качество и верификация

Заполняйте по определениям на странице **[«Измерения качества источника данных»](/reference/data-quality-dimensions)**.

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки (`last_verified`) | 2026-05-15 (**GET** главной с **`curl -k`**, **HTTP 200**; без `-k` возможна ошибка TLS в части окружений) |
| Полнота (`data_completeness`) | **частичная** — сайт **ТОР** дополняет, но **не** всегда **заменяет** **федеральный** каталог |
| Машиночитаемость (`machine_readability`) | **HTML/PDF** на сайте; **API ЕМИСС** — в **[rosstat-opendata](/data-sources/federal/rosstat-opendata)** |
| Юридическая значимость (`legal_significance`) | **официальный** **территориальный** орган **федеральной** службы |
| Задержка обновления (`update_lag`) | зависит от **графиков** **региональных** и **федеральных** релизов |
| Глубина архива (`archive_depth`) | по **архиву** публикаций на сайте **30.rosstat.gov.ru** |
| Лицензия (`license_or_terms`) | условия **использования** материалов **Росстата** |

## Описание

Для **аналитики** по **республике** сайт **Астраханьстата** полезен как **региональный** контекст: **выпуски**, **комментарии**, **таблицы** в **зоне** **двух** субъектов. При **построении** **рядов** с **долгим** горизонтом **сверяйте** **коды** **ОКТМО** и **территориальную** принадлежность наблюдения. **Бюджетную** детализацию **не** смешивайте **напрямую** со **статистикой** без **методического** моста — см. [Минфин РК](/data-sources/regional/republic-of-kalmykia-minfin).

## Объекты данных и показатели

- **Региональные** публикации и **комментарии** **Росстата** по **территории** **зоны** управления.
- **Ссылки** на **федеральные** **классификаторы** и **методологию** статистического **наблюдения**.
- **Сшивка** с **федеральными** **машинными** наборами через [ЕМИСС/открытые данные](/data-sources/federal/rosstat-opendata).

## Периодичность обновления

**По календарю** статистических **выпусков**; **точные** даты — на сайте **ТОР** и на **федеральной** витрине.

## Ограничения и особенности

- **Один** **ТОР** **обслуживает** **два** субъекта — **всегда** проверяйте **географическую** **разбивку** **таблиц**.
- **TLS**: в **CLI** может понадобиться **`-k`**.

## Примеры доступа

```bash
curl -fsSLk -o /dev/null -w "%{http_code}\n" --max-time 25 -A "finguide-astrakhanst-check/1.0" \
  "https://30.rosstat.gov.ru/"
```
