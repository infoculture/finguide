# Республика Калмыкия: УФНС России

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральная** налоговая служба на территории **Республики Калмыкия** |
| Тип доступа | Публичный **федеральный** портал (**региональный** префикс URL) |
| Основные форматы | **HTML** |
| Оператор | **Управление ФНС России** по Республике Калмыкия |
| Режим доступа | Открытый для **большинства** справочных разделов; отдельные **сервисы** — по правилам ФНС |

Коротко: **официальный** **региональный** вход на **nalog.gov.ru** для новостей **УФНС** и **сервисов** в **контуре** ФНС. Для **сопоставления** с **бюджетом** субъекта используйте [Минфин РК](/data-sources/regional/republic-of-kalmykia-minfin) и [федеральные бюджетные наборы](/data-sources/federal/budget-gov-ru-datasets), помня, что **налоговая** и **бюджетная** отчётность **различаются** по **оператору** и **методике**.

## Описание

Страница **rn08** на портале ФНС традиционно используется как **региональная** «**точка входа**» для **УФНС** по **Калмыкии**. В аналитике **госфинансов** её имеет смысл сочетать с **[Минфином](/data-sources/regional/republic-of-kalmykia-minfin)** (**бюджетные** доходы и **агрегаты**) и с **[Росстатом](/data-sources/regional/republic-of-kalmykia-rosstat-astrakhanst)** (**социально-экономический** контекст). Вводные по **налогам** — [«Налог»](/glossary/tax) и [«Налоговый орган»](/glossary/tax-authority) в глоссарии.

Управление Федеральной налоговой службы по Республике Калмыкия — раздел https://www.nalog.gov.ru/rn08/

## Объекты данных и показатели

- **Новости** и **анонсы** УФНС по региону.
- **Сервисы** и **формы** электронного взаимодействия (**при** **указании** на портале).
- **Ссылки** на **федеральные** справочники и **методические** материалы **ФНС**.

## Периодичность обновления

**Непрерывно** для **сервисов**; **новости** — **по событиям**.

## Ограничения и особенности

- **Коммерческое** и **исследовательское** **скачивание** может попадать под **ограничения** роботов и **правила** ФНС.
- **Налоговая** статистика в **открытом** виде для **аналитики** часто требует **отдельных** федеральных **наборов**, а не только HTML **региональной** главной.
