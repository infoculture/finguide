## Описание

Раздел `/documents/opendata` предназначен для раскрытия наборов в соответствии с региональной практикой открытых данных: сведения о доходах, отчётность, регламентированные перечни — **по фактическому содержанию**. Перед массовым парсингом сверяйте **роботы**, ограничения частоты и наличие прямых ссылок на файлы.

Официальный портал Правительства Республики Адыгея — канонический вход к разделу: https://www.adygheya.ru/documents/opendata

## Объекты данных и показатели

- Каталог наборов открытых данных региона (структура — по странице).
- Типичные темы: сведения о доходах и имуществе должностных лиц, отчёты органов власти, открытые данные смежных реестров — по перечню на витрине.
- Ключи связки: **идентификатор набора**, **период**, **ведомство-ответчик**.

## Периодичность обновления

Индивидуально для каждого набора (см. карточку набора при наличии).

## Ограничения и особенности

- Наборы могут дублировать или пересекаться с материалами [Минфина](/data-sources/regional/republic-of-adygea-minfin); сверяйте оператора набора и методику.
- Исторический импорт использовал HTTP `adygheya.ru`; канонический вход выше — по HTTPS на официальном хосте.
