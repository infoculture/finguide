# Муниципальные бюджеты (единый портал бюджетной системы)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Местные бюджеты** муниципальных образований (ГО, МР, сельские поселения и др.), опубликованные в публичной части **единого портала** `budget.gov.ru`; охват по конкретным наборам и разделам нужно проверять по метаданным |
| Тип доступа | Веб-разделы портала, каталог и файлы **открытых данных**, REST **API** там, где это раскрыто оператором (см. федеральную карточку наборов) |
| Основные форматы | HTML, CSV, JSON (в т.ч. каталог наборов `list.json`) |
| Оператор | Минфин России / Федеральное казначейство (публичный контур **ГИИС «Электронный бюджет»**) |
| Режим доступа | Открытый для опубликованных разделов и наборов |

Коротко: **официальный федеральный вход** к сведениям о бюджетах МО в контуре **«Электронного бюджета»** — удобно для поиска по территории и для **машиночитаемых** срезов после фильтрации наборов по паспортам; для полноты пакета документов часто приходится **дополнять** сайтами самих МО.

## Описание

Карточка описывает **публичный слой** единого портала бюджетной системы (`budget.gov.ru`) в части **муниципальных бюджетов**: проекты и законы о местном бюджете, отчёты об исполнении, плановые и исполнительные срезы там, где они выложены в открытом доступе. Сюда **не** входят внутренние кабинеты участников бюджетного процесса и полнота данных **внутри** карточек документов в ИС.

По требованиям раскрытия бюджетной информации сведения о муниципальных бюджетах должны дублироваться на портале; на практике **полнота, форматы и сроки** загрузки по МО различаются, поэтому для юридически значимых формулировок и вложений полезна сверка с **[сайтами муниципальных образований](/data-sources/federal/municipalnye-byudzhety)**.

## Оператор

**Минфин России** и **Федеральное казначейство** — операторы публичного контура; канонический вход: https://budget.gov.ru/

- Карточка ИС: **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** — назначение портала, подсистемы и раздел **«Ограничения и актуальность»** про смену состава открытых наборов и про **региональный и муниципальный** уровни публикаций.

## URL и точки доступа

Часть «коротких» URL на стороне портала может отвечать **404** в зависимости от конфигурации; надёжнее заходить с главной страницы в раздел **«Данные и сервисы»** или использовать прямые технические URL из таблицы ниже (как в карточке **[наборов данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)**).

## Объекты данных и показатели

Состав полей задаётся **паспортом каждого набора** в каталоге открытых данных. Типично для муниципальной тематики ищут:

- **Документы бюджетного процесса** местного уровня (проект, закон о бюджете, изменения, отчёт об исполнении) — в HTML-разделах портала и/или в связанных наборах.
- **Плановые и исполнительные показатели** в разрезе кодов классификации и территории; важно не смешивать **план**, **роспись** и **факт** без проверки метаданных набора.
- **Ключи связки**: **ОКТМО** территории МО, при необходимости **ИНН** участников, **КБК** и смежные коды в таблицах; наименование МО в выгрузке сверяйте со **[справочником территорий и реестром](/information-systems/federal/munbase)**.

Пробелы: не каждый МО представлен одинаково полно в машиночитаемых наборах; часть материалов остаётся только в **PDF/HTML** на **[сайтах МО](/data-sources/federal/municipalnye-byudzhety)**.

## Периодичность обновления

Периодичность **различается по наборам** (план, исполнение, реестры); дата версии и регламент публикации указаны в **метаданных набора** в каталоге открытых данных — см. раздел **«Периодичность обновления»** на странице **[наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)**.

Общие предупреждения о смене URL и полей и о том, что **муниципальные** публикации имеют **собственную** логику наполнения, см. в разделе **«Ограничения и актуальность»** карточки **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)**.

## Ограничения и особенности

- Лицензия и условия переработки — в **паспорте набора** на витрине открытых данных (как правило, режим открытых данных с указанием источника).
- Публичные выгрузки **не обязаны** повторять полную внутреннюю карточку документа в ИС.
- Показатели **казначейского исполнения** в той же методике могут расходиться с бюджетными срезами портала — для факта сверяйте **[отчёты и открытые слои Федерального казначейства](/data-sources/federal/roskazna-reports)**.
- Для методики отчётности местного уровня см. **[отчётность об исполнении местных бюджетов](/reporting/municipal-budget-execution)**.

## Примеры доступа

Каталог наборов отдаёт JSON без авторизации (как на странице федерального источника по наборам). Дальше отбирайте идентификаторы наборов по заголовку/паспорту и скачивайте файлы по ссылкам из метаданных.

```bash
curl -sS "https://budget.gov.ru/epbs/opendata/list.json" -o opendata-catalog.json
```
