# Гражданские и исследовательские источники данных

Независимые от государства проекты, агрегирующие и переиспользующие данные о госфинансах. Карточки описывают **витрины и выгрузки** гражданского сектора; для юридически значимых копий документов и регламентных срезов опирайтесь на [федеральные](/data-sources/federal/) и [региональные](/data-sources/regional/) первоисточники.

## Для кого

- Исследователям, журналистам и активистам, которым нужны **агрегаты и поиск** поверх официальных порталов.
- Разработчикам, работающим с CKAN, API и готовыми выгрузками проектов сообщества.

## Быстрые входы

| Задача | Карточка |
| --- | --- |
| Аналитика закупок без сборки XML ЕИС | [Clearspending.ru](./clearspending.md) |
| Поиск датасетов по госфинансам (CKAN) | [Hub of Data](./hubofdata.md) |
| Найти официальный каталог ведомства или региона | [DataCatalogs.ru](./datacatalogs-ru.md) |
| НКО: гранты, контракты, поддержка | [OpenNGO.ru](./open-ngo.md) |
| Президентские гранты (официальные открытые данные конкурсов) | [Президентские гранты](./pgrants.md) |
| Методика по НКО | [Анализ финансирования НКО](/howto/analysis/ngo-funding) |

## Типовые вопросы

**Чем гражданский источник отличается от ЕИС или data.gov.ru?** Удобство поиска и склейки; **не** подмена официальной публикации и условий лицензии первоисточника.

**Где международные официальные витрины?** Раздел [Международные источники](/data-sources/international/).

Новые карточки источников — по шаблону `wiki/data-sources/data-source-card-template.md` ([GitHub](https://github.com/infoculture/finguide/blob/master/wiki/data-sources/data-source-card-template.md)) и `AGENTS.md` (шаблон с `draft: true` не в production-сборке).
