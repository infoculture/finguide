# Code for Russia — датасеты на GitHub

## Описание

Открытые датасеты и инструменты для работы с госданными, поддерживаемые сообществом разработчиков.

## Оператор

Сообщество Code for Russia — https://github.com/codeforrussia

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| GitHub | https://github.com/codeforrussia | CSV/JSON/Code |

## Состав данных

- Датасеты на GitHub
- Скрипты обработки и парсинга

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).
