# HubOfData

## Описание

Каталог датасетов Минфина, Казначейства, ФНС и других ведомств. Поддерживает CKAN API.

## Оператор

НП «Информационная культура» — https://hubofdata.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Веб-интерфейс | https://hubofdata.ru/ | HTML |
| CKAN API | https://hubofdata.ru/api/3/ | JSON |

## Состав данных

- Датасеты Минфина
- Датасеты Казначейства
- Датасеты ФНС

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).

## Ограничения и особенности

- Лицензия: открытые данные (как правило, разрешена свободная переработка с указанием источника).
- Объёмы и форматы могут меняться; рекомендуется отслеживать страницу источника.
