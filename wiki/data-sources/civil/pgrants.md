# pgrants.ru — Президентские гранты

## Описание

Информация обо всех конкурсах и победителях. Данные о финансировании НКО.

## Оператор

Фонд президентских грантов — https://президентскиегранты.рф/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Веб-интерфейс | https://президентскиегранты.рф/public/ | HTML |
| Открытые данные | https://президентскиегранты.рф/public/api/ | JSON |

## Состав данных

- Заявки и победители
- Отчёты о реализации проектов

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).
