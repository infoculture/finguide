# Clearspending.ru

## Описание

Гражданская система анализа закупок РФ. Удобный поиск по контрактам, REST API, аналитика по поставщикам и заказчикам.

## Оператор

Гражданское сообщество / Информационная культура — https://clearspending.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Веб-интерфейс | https://clearspending.ru/ | HTML |
| REST API | https://clearspending.ru/opendata/ | JSON |

## Состав данных

- Контракты и извещения
- Поставщики и заказчики
- Аналитические индикаторы

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).

- Лицензия: открытые данные (как правило, разрешена свободная переработка с указанием источника).
