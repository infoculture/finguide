# Минфин: краткая информация об исполнении бюджетов государственных внебюджетных фондов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральный уровень**: краткие своды по исполнению **бюджетов государственных внебюджетных фондов** ([**ГВФ**](/glossary/extrabudgetary-funds)) РФ — в трёх периодичностях публикации (месяц, квартал, год), в **млрд руб.** |
| Тип доступа | **Файлы по ссылке**: HTML-страница раздела «Статистика → Бюджеты государственных внебюджетных фондов» со списком документов и кнопкой «Скачать» для каждого **XLSX** |
| Основные форматы | **XLSX**; обёртка — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (типовое скачивание без регистрации с официальной страницы) |

Коротко: **официальные агрегированные таблицы исполнения бюджетов [ГВФ](/glossary/extrabudgetary-funds)** от Минфина — удобная **макро**-точка входа для динамики доходов и расходов фондов **отдельно от [федерального бюджета](/data-sources/federal/minfin-fedbud-execute)**; для **регламентированной отчётности**, разреза по **[КБК](/glossary/kbk)** и плановому контуру используйте **[отчётность по бюджетам ГВФ](/reporting/extrabudgetary-funds-reporting)**, **[ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** и материалы операторов фондов.

## Описание

Карточка описывает витрину **[Краткая информация об исполнении бюджетов государственных внебюджетных фондов](https://minfin.gov.ru/ru/statistics/outbud/execute/)**: на странице перечислены документы с датами публикации и ссылками на выгрузки. По состоянию на актуальные публикации на сайте доступны три типа файлов:

- **ежемесячная** краткая информация (показатели **накопленно с начала года**, млрд руб.);
- **ежеквартальная** краткая информация (также **накопленно с начала года**, млрд руб.);
- **ежегодная** краткая информация (итоги по **финансовому году** в млрд руб. — конкретный год см. в заголовке документа и в файле).

Сюда **не** входят **[открытые наборы Минфина](/data-sources/federal/minfin-opendata)** из каталога с `list.csv` и паспортами `meta.csv` — раздел **outbud/execute** — отдельная **статистическая** подборка на сайте ведомства, без включения в машинный реестр открытых данных того же формата.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/statistics/outbud/execute/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Статистика» и вложений к документам.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Страница раздела (список документов, «Скачать» для каждого файла) | https://minfin.gov.ru/ru/statistics/outbud/execute/ | HTML; вложения **XLSX** |
| Родительский раздел «Бюджеты государственных внебюджетных фондов» | https://minfin.gov.ru/ru/statistics/outbud/ | HTML (навигация по смежным материалам) |

Прямые URL файлов **XLSX** при обновлении вложений **меняются**; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** в карточке документа на странице раздела.

## Объекты данных и показатели

Состав строк и столбцов задаётся **опубликованными XLSX**; по смыслу заголовков страницы это:

- **Сводные показатели исполнения бюджетов [государственных внебюджетных фондов](/glossary/extrabudgetary-funds)** в **млрд руб.** — доходы, расходы и связанные блоки в агрегированном виде; при необходимости разделения **бюджета СФР**, **ФФОМС** и иных фондов ориентируйтесь на строки и подписи в файле на дату скачивания.
- **Накопленный с начала года** итог — для **ежемесячной** и **ежеквартальной** серий (проверяйте подписи листов и шапку таблицы в файле).
- **Годовая** серия — для сопоставления с параметрами **законов о бюджетах ГВФ** и иными официальными сводами за **финансовый год**.

**Пробелы**: это **не** построчный учёт по **[КБК](/glossary/kbk)** и не замена **[форм исполнения](/reporting/extrabudgetary-funds-reporting)** и отчётности операторов фондов; для детального контура комбинируйте **[наборы «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)** (с явным выбором типа бюджета «бюджет ГВФ») и **[федеральные бюджеты и открытые данные Минфина](/data-sources/federal/minfin-opendata)**.

## Периодичность обновления

Публикации привязаны к **выходу новых редакций документов** на странице: у каждой карточки документа указаны даты «Опубликовано» и «Изменено». **Ежемесячный** файл обновляется чаще, чем **квартальный** и **годовой**; стабильного API под выгрузку нет — перед массовой загрузкой проверяйте страницу.

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Автоматические запросы к сайту иногда упираются в **ограничения доступа**; для парсинга используйте опубликованные пользователем ссылки после проверки в браузере и учитывайте политику сайта.
- Показатели Минфина в этих XLSX **не обязаны** построчно совпадать с **[открытыми наборами Минфина](/data-sources/federal/minfin-opendata)** или **[срезами ГИИС](/data-sources/federal/budget-gov-ru-datasets)** — сверяйте методику, заголовки и дату файла.

## Примеры доступа

Сохранить HTML-страницу раздела для ручного или последующего разбора ссылок «Скачать»:

```bash
curl -sS "https://minfin.gov.ru/ru/statistics/outbud/execute/" -o minfin-outbud-execute.html
```

## Не путать с

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — **реестр наборов** с `list.csv` и паспортами, а не список трёх XLSX на странице статистики **outbud/execute**.
- **[Минфин: исполнение федерального бюджета (fedbud/execute)](/data-sources/federal/minfin-fedbud-execute)** — **иной предмет** (федеральная казна), хотя формат публикаций похож; цифры **не складывать** без методики как «единый бюджет государства».
- **[Минфин: исполнение консолидированного бюджета РФ и ГВФ (conbud/execute)](/data-sources/federal/minfin-conbud-execute)** — **консолидированный** свод РФ **и** ГВФ в одной логике; **outbud** даёт **отдельно** краткую информацию по **бюджетам ГВФ**, без замены задачи «свод страны» из **conbud**.

## Частые вопросы

**В:** Можно ли по этим XLSX восстановить исполнение по каждому **[КБК](/glossary/kbk)** по каждому фонду?

**О:** Обычно **нет** или **не полностью**: файлы **краткой информации** агрегируют показатели. Для разреза по **[КБК](/glossary/kbk)** и формам используйте **[отчётность по бюджетам ГВФ](/reporting/extrabudgetary-funds-reporting)** и **[наборы ГИИС](/data-sources/federal/budget-gov-ru-datasets)** с корректным типом бюджета.

**В:** Почему сумма по блокам ГВФ в файле Минфина не совпадает с выгрузкой «Электронного бюджета»?

**О:** Различаются **определение показателя**, **момент среза**, **методика свёртки** и **граница агрегации** (краткая сводка против машиночитаемого набора). Сверяйте заголовки документов, тип бюджета в наборе и дату файла в обоих источниках.
