# Минфин: дополнительные нефтегазовые доходы федерального бюджета

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный уровень: **информационное сообщение** Минфина о **формировании и использовании дополнительных нефтегазовых доходов** федерального бюджета (ряды по годам в приложении) |
| Тип доступа | **Файлы по ссылке**: HTML-страница раздела «Статистика → Федеральный бюджет» с блоком «Скачать» и табличной выгрузкой **XLSX** |
| Основные форматы | **XLSX** (информационное сообщение с таблицами); обёртка страницы — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (без регистрации для скачивания с официальной страницы) |

Коротко: **официальная сводка по дополнительным нефтегазовым доходам** и их использованию в логике **федерального бюджета** — точка сопоставления с **[ФНБ](/glossary/nwf)**, **[источниками финансирования дефицита](/glossary/deficit-financing)** и плановыми параметрами из **[закона о федеральном бюджете](/legal/budgetlaw)**; не заменяет **[оперативное исполнение](/glossary/treasury-execution)** по **[Казначейству](/data-sources/federal/roskazna-reports)**.

## Описание

Карточка описывает витрину **[Информация о дополнительных нефтегазовых доходах федерального бюджета](https://minfin.gov.ru/ru/statistics/fedbud/oil/)**: публикуется **информационное сообщение** со сведениями о формировании и использовании таких доходов (в актуальной редакции страницы — в том числе **горизонт 2018–2026 гг.**). Данные отдаются **одним файлом XLSX** с страницы документа.

Сюда **не** входят полные **[открытые наборы Минфина](/data-sources/federal/minfin-opendata)** с паспортами `meta.csv` и машинным каталогом `list.csv`; этот раздел — **отдельная статистическая подборка** под нефтегазовую тематику, а не универсальный реестр датасетов.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/statistics/fedbud/oil/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Статистика» и вложений к информационным сообщениям.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |

| Родительский раздел статистики по федеральному бюджету | https://minfin.gov.ru/ru/statistics/fedbud/ | HTML (навигация по смежным материалам) |

Прямой URL файла **XLSX** может меняться при обновлении вложения; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** на странице документа.

## Объекты данных и показатели

Состав столбцов и строк задаётся **опубликованным XLSX**; по смыслу источника это:

- **Дополнительные нефтегазовые доходы** федерального бюджета — показатели **формирования** и **направлений использования** в разрезе, зафиксированном в таблице сообщения (сопоставляйте с **[ФНБ](/glossary/nwf)** и нормами **[289-ФЗ](/legal/nwf-289fz)** по смыслу, не подменяя юридическую трактовку одной выгрузкой).
- **Годовой горизонт** в заголовке документа — проверяйте соответствие **финансовому году** и редакции **[закона о федеральном бюджете](/legal/budgetlaw)** на дату анализа.

**Пробелы**: это **не** пообъектный учёт по **[КБК](/glossary/kbk)** в формате **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)**; для построчной сверки с планом и исполнением нужны **[наборы «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)** и **[отчётность Казначейства](/data-sources/federal/roskazna-reports)**.

## Периодичность обновления

Обновление привязано к **публикации нового информационного сообщения** и файла на странице; даты «Опубликовано» / «Изменено» указаны в карточке документа на сайте. Типового ежедневного API здесь нет — проверяйте страницу перед скачиванием.

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Некоторые автоматические запросы к сайту могут получать **страницу ограничения доступа** вместо контента — для парсинга учитывайте политику сайта и используйте **опубликованную пользователем ссылку «Скачать»** после проверки в браузере.
- Показатели сообщения **не обязаны** совпадать построчно с **[агрегированными наборами открытых данных Минфина](/data-sources/federal/minfin-opendata)** — сверяйте методику и заголовок документа.

## Не путать с

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — **каталог наборов** с идентификаторами `7710168360-*` и `list.csv`, а не единичная XLSX-витрина по одной теме.

## Частые вопросы

**В:** Достаточно ли этого файла для проверки кассового исполнения по нефтегазовым поступлениям?

**О:** Для **юридически значимой** сверки кассы опирайтесь на **[отчётность Казначейства](/data-sources/federal/roskazna-reports)** и **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)**; информационное сообщение даёт **официальную методическую сводку** по дополнительным нефтегазовым доходам и их использованию в логике сообщения, а не полный реестр платежей.

**В:** Где искать те же темы в машиночитаемом каталоге Минфина?

**О:** Просматривайте **[сводный `list.csv`](https://minfin.gov.ru/opendata/list.csv)** и тематические наборы в **[открытых данных Минфина](/data-sources/federal/minfin-opendata)**; раздел **fedbud/oil** может **не дублироваться** тем же составом полей.
