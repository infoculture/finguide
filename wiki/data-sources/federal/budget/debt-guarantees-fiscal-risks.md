# Государственный долг, гарантии и фискальные риски: навигатор наборов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Российская Федерация, субъекты и муниципальные образования по набору |
| Тип доступа | Паспорта и версии наборов ГИИС «Электронный бюджет» |
| Основные форматы | JSON, CSV |
| Оператор | Минфин России |
| Режим доступа | Открытый |

Коротко: навигатор объединяет канонические паспорта долга, гарантий и связанных инвестиционных обязательств, не превращая каждый из них в один показатель «фискального риска».

## Качество и верификация

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки | 2026-07-16 |
| Полнота | Частичная: наборы раскрывают разные виды обязательств |
| Машиночитаемость | JSON/CSV по паспорту |
| Юридическая значимость | Официальная публикация |
| Задержка обновления | Проверяется отдельно по паспорту |
| Глубина архива | Проверяется по версиям набора |
| Лицензия | Не указана единым условием |

## Описание

Для федерального долга используйте отдельный набор, для государственных и муниципальных гарантий — набор гарантий, для программ гарантий субъектов — соответствующие региональные паспорта. ГЧП/МЧП и бюджетные инвестиции показывают условные или будущие обязательства только в опубликованном составе и не равны долгу автоматически.

## Оператор

[Минфин России](/organizations/minfin); система — [ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb).

## Объекты данных и показатели

- долговое обязательство, вид и валюта долга;
- гарантия, принципал, гарант и объём обязательства в доступном составе;
- программа гарантий и уровень бюджета;
- проект ГЧП/МЧП как отдельный аналитический контур.

## Периодичность обновления

Определяется паспортом каждого набора; дата паспорта не всегда равна дате последнего факта.

## Ограничения и особенности

- гарантия не равна исполненному расходу;
- объём проекта ГЧП не равен гарантированному обязательству бюджета;
- уровни бюджета, валюты и даты оценки нельзя агрегировать без нормализации.
