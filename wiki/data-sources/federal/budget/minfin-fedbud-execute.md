# Минфин: краткая информация об исполнении федерального бюджета

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный уровень: **краткие своды по исполнению федерального бюджета** в разрезе трёх периодичностей публикации (месяц, квартал, год) |
| Тип доступа | **Файлы по ссылке**: HTML-страница раздела «Статистика → Федеральный бюджет» со списком документов и кнопкой «Скачать» для каждого **XLSX** |
| Основные форматы | **XLSX**; обёртка — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (типовое скачивание без регистрации с официальной страницы) |

Коротко: **официальные агрегированные таблицы исполнения федерального бюджета** от Минфина (в выражении **млрд руб.**) — удобная **макро**-точка входа для обзора динамики доходов и расходов; для **регламентированной отчётности по формам**, **[КБК](/glossary/kbk)** и кассового контура используйте **[Казначейство](/data-sources/federal/roskazna-reports)** и **[ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)**.

## Описание

Карточка описывает витрину **[Краткая информация об исполнении федерального бюджета](https://minfin.gov.ru/ru/statistics/fedbud/execute/)**: на странице перечислены документы с датами публикации и ссылками на выгрузки. По состоянию на актуальные публикации на сайте доступны три типа файлов:

- **ежемесячная** краткая информация (показатели **накопленно с начала года**, млрд руб.);
- **ежеквартальная** краткая информация (также **накопленно с начала года**, млрд руб.);
- **ежегодная** краткая информация (итоги по **финансовому году** в млрд руб. — конкретный год см. в заголовке документа и в файле).

Сюда **не** входят **[открытые наборы Минфина](/data-sources/federal/minfin-opendata)** из каталога с `list.csv` и паспортами `meta.csv` — раздел **fedbud/execute** — отдельная **статистическая** подборка на сайте ведомства, без включения в машинный реестр открытых данных того же формата.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/statistics/fedbud/execute/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Статистика» и вложений к документам.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Страница раздела (список документов, «Скачать» для каждого файла) | https://minfin.gov.ru/ru/statistics/fedbud/execute/ | HTML; вложения **XLSX** |
| Родительский раздел статистики по федеральному бюджету | https://minfin.gov.ru/ru/statistics/fedbud/ | HTML (навигация по смежным материалам) |

Прямые URL файлов **XLSX** при обновлении вложений **меняются**; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** в карточке документа на странице раздела.

## Объекты данных и показатели

Состав строк и столбцов задаётся **опубликованными XLSX**; по смыслу заголовков страницы это:

- **Сводные показатели исполнения федерального бюджета** в **млрд руб.** — доходы, расходы и связанные статьи в агрегированном виде (конкретный перечень — в файле на дату скачивания).
- **Накопленный с начала года** итог — для **ежемесячной** и **ежеквартальной** серий (проверяйте подписи листов и шапку таблицы в файле).
- **Годовая** серия — для сопоставления с параметрами **[закона о федеральном бюджете](/legal/budgetlaw)** и иными официальными сводами за **финансовый год**.

**Пробелы**: это **не** построчный учёт по **[КБК](/glossary/kbk)** и не замена **[форм исполнения Казначейства](/data-sources/federal/roskazna-reports)**; для сопоставления с планом и детальным исполнением нужны **[наборы «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)** и регламентированные отчёты оператора казначейского сопровождения.

## Периодичность обновления

Публикации привязаны к **выходу новых редакций документов** на странице: у каждой карточки документа указаны даты «Опубликовано» и «Изменено». **Ежемесячный** файл обновляется чаще, чем **квартальный** и **годовой**; точного API под выгрузку нет — перед массовой загрузкой проверяйте страницу.

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Автоматические запросы к сайту иногда упираются в **ограничения доступа**; для парсинга используйте опубликованные пользователем ссылки после проверки в браузере и учитывайте политику сайта.
- Показатели Минфина в этих XLSX **не обязаны** построчно совпадать с **[открытыми наборами Минфина](/data-sources/federal/minfin-opendata)** или **[срезами ГИИС](/data-sources/federal/budget-gov-ru-datasets)** — сверяйте методику, заголовки и дату файла.

- **[Минфин: краткая информация об исполнении бюджетов государственных внебюджетных фондов (outbud/execute)](/data-sources/federal/minfin-outbud-execute)** — **только бюджеты [ГВФ](/glossary/extrabudgetary-funds)** в макро-**XLSX**; не тождественны **fedbud/execute** (федеральный бюджет) и **conbud/execute** (консолидированный РФ и ГВФ).

## Не путать с

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — **реестр наборов** с `list.csv` и паспортами, а не список трёх XLSX на странице статистики **fedbud/execute**.
- **[Отчёты Федерального казначейства](/data-sources/federal/roskazna-reports)** — **иной оператор** и иной уровень детализации (формы исполнения, регламент); свод Минфина удобен для обзора, но не подменяет казначейскую отчётность.

## Частые вопросы

**В:** Можно ли по этим XLSX восстановить исполнение по каждому **[КБК](/glossary/kbk)**?

**О:** Обычно **нет**: файлы **краткой информации** агрегируют показатели на уровне сводных блоков. Для разреза по **[КБК](/glossary/kbk)** и формам используйте **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)** и **[Казначейство](/data-sources/federal/roskazna-reports)**.

**В:** Почему цифра из месячного файла не совпадает со строкой в отчёте Казначейства?

**О:** Различаются **определение показателя**, **момент среза** (касса / учётный период), **методика свёртки** и **источник первичных данных**. Сверяйте заголовки документов и пояснения к таблицам в обоих источниках.
