# Наборы данных ГИИС «Электронный бюджет» (budget.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный контур **ГИИС «Электронный бюджет»**; в каталоге открытых данных встречаются также сводные материалы по субъектам и муниципалитетам — по каждому набору отдельно проверяйте метаданные и тип бюджета |
| Тип доступа | Витрина наборов и файловые выгрузки; каталог наборов в JSON; REST API к ряду объектов (описание на портале) |
| Основные форматы | CSV, JSON (в т.ч. каталог `list.json` и метаданные наборов по стандарту портала открытых данных) |
| Оператор | Минфин России / Федеральное казначейство (см. карточку ИС) |
| Режим доступа | Открытый для опубликованных наборов и документации API |

Коротко: **машиночитаемые** срезы планирования и исполнения бюджета, реестров соглашений и справочников участников — основной федеральный вход для сверки **КБК**, **ИНН**, периода и типа показателя (план/факт) перед стыковкой с казначейскими отчётами и закупками.

## Описание

Карточка описывает **публичный слой открытых данных** единого портала бюджетной системы `budget.gov.ru`: перечень наборов, паспорта, прямые ссылки на файлы и программный доступ там, где он раскрыт оператором. Сюда **не** входят закрытые кабинеты участников бюджетного процесса и полнота внутренних карточек документов в ИС.

Для **каждой записи** актуального `list.json` (кроме наборов, у которых уже есть отдельная тематическая страница в wiki) добавлены короткие карточки-заготовки с каноническими URL паспорта: публичный путь вида **`/data-sources/federal/opendata-<identifier>`** (в сайдбаре — внутри раздела **«Бюджет и исполнение»**, подгруппа **ГИИС «Электронный бюджет» (budget.gov.ru)**, далее свёрнутые группы **«Наборы каталога EPBS (7710168360-…)»** и **«Наборы каталога EPBS (7710568760-…)»** по префиксу идентификатора в каталоге). Перегенерация списка файлов — скриптом **`scripts/generate-budget-gov-opendata-dataset-cards.mjs`** после обновления каталога на стороне портала.

Полный читательский перечень: **[индекс паспортов открытых данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-opendata-index)**. Он обеспечивает входящую навигацию к каждой массово созданной карточке и регенерируется командой `npm run write:import-family-hubs`.

Контекст портала, подсистемы и типовые ограничения — в **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)**. Практические шаги по API — в **[руководстве по API budget.gov.ru](/howto/access/budget-gov-api)**.

## Оператор

**Минфин России** и **Федеральное казначейство** — операторы публичного контура; канонический вход: https://budget.gov.ru/

- Карточка ИС: **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** — официальная витрина и тот же контур открытых наборов, что и эта страница источника.

## Объекты данных и показатели

Заполняйте состав полей по **паспорту конкретного набора** из каталога; типичные сущности каталога:

- **Исполнение бюджета** — агрегированные суммы по доходам/расходам и кодам классификации; важно не смешивать **кассовый факт**, **план** и **роспись** без проверки метаданных набора.
- **Бюджетная роспись и плановые лимиты** — срезы по распорядителям и кодам; не обязаны воспроизводить смету отдельного учреждения.
- **Реестр соглашений** (субсидии, бюджетные инвестиции, межбюджетные трансферты и смежные документы — по актуальной структуре портала) — записи с **ИНН**, суммами, **КБК**, номером соглашения; для логики полей см. отдельную страницу источника по субсидиям.
- **Участники бюджетного процесса** — справочные идентификаторы организаций в бюджетном контуре; **поиск и карточки** — в **[реестре участников и неучастников бюджетного процесса](/data-sources/federal/orgsinfo)** (тот же портал, см. строку в таблице URL выше); для джойна с контрактами и учреждениями используйте **[ИНН](/glossary/inn)**, коды участника и **[КБК](/glossary/kbk)**.

В элементах каталога поле `link` к `meta.json` может быть относительным или с префиксом `null/` — базовый URL и правило сборки ссылки берите из **текущей** витрины и паспорта набора.

## Периодичность обновления

Периодичность **различается по наборам** (исполнение, роспись, реестры); актуальные дата версии и регламент публикации указаны в **метаданных набора** в каталоге открытых данных. Общие предупреждения о смене состава полей и URL — в разделе **«Ограничения и актуальность»** карточки **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)**.

## Ограничения и особенности

- Лицензия и условия переработки — в паспорте каждого набора на витрине открытых данных (как правило, режим открытых данных с указанием источника).
- Публичные выгрузки **не обязаны** повторять полную внутреннюю карточку документа в ИС.
- Некоторые клиенты получают **403** на `list.json` / `meta.json` без типичного заголовка **`User-Agent`** браузера; в скриптах задайте осмысленный User-Agent (см. примеры в карточках наборов группы **opendata-*** и в скрипте генерации).
- Региональный и муниципальный уровни имеют **собственные** публикации; не трактуйте федеральный набор как полный охват всех уровней без явного признака в метаданных.
- Показатели **казначейского исполнения** в полном объёме и с той же методикой могут расходиться с бюджетными срезами — для факта сверяйте **[отчёты и открытые слои Федерального казначейства](/data-sources/federal/roskazna-reports)**.

## Примеры доступа

Каталог наборов отдаёт JSON без авторизации (проверено запросом `GET`). Файл **объёмный** — удобнее сначала сохранить на диск, затем разбирать локально.

```bash
curl -sS "https://budget.gov.ru/epbs/opendata/list.json" -o opendata-catalog.json
```

Разбор сохранённого каталога на Python (стандартная библиотека):

```python
import json

with open("opendata-catalog.json", encoding="utf-8") as f:
    catalog = json.load(f)
meta = catalog.get("meta", [])
print("datasets:", len(meta))
if meta:
    print("example:", meta[0].get("identifier"), "-", meta[0].get("title"))
```

- **[Портал открытых данных РФ](/information-systems/federal/data-gov-ru)** — федеральный каталог-агрегатор; часть наборов дублирует `budget.gov.ru` — сверяйте поставщика и версию.

## Не путать с

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — другой оператор витрины и другой перечень наборов; пересечения по темам не означают идентичность полей.
- **[Портал ГМУ (bus.gov.ru)](/data-sources/federal/bus-opendata)** — уровень **учреждения** (ФХД, госзадание); данные сшиваются с бюджетным контуром по **ИНН** и году, но не подменяют друг друга.
