# Минфин: краткая информация об исполнении консолидированного бюджета РФ и ГВФ

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный уровень: **краткие своды по исполнению [консолидированного бюджета](/glossary/consolidated-budget) Российской Федерации** вместе с **[бюджетами государственных внебюджетных фондов](/glossary/extrabudgetary-funds)** (в терминологии заголовков документов на сайте — «консолидированный бюджет РФ и ГВФ»), в **млрд руб.** и трёх периодичностях публикации (месяц, квартал, год) |
| Тип доступа | **Файлы по ссылке**: HTML-страница раздела «Статистика → Консолидированный бюджет Российской Федерации» со списком документов и кнопкой «Скачать» для каждого **XLSX** |
| Основные форматы | **XLSX**; обёртка — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (типовое скачивание без регистрации с официальной страницы) |

Коротко: **официальные агрегированные таблицы** исполнения **сводного федерального контура** (консолидированный бюджет РФ **и** ГВФ) от Минфина — удобная **макро**-точка входа для сопоставления с **[федеральным бюджетом](/data-sources/federal/minfin-fedbud-execute)** и **[открытыми наборами про исполнение](/data-sources/federal/minfin-opendata)**; для **регламентированной отчётности по формам**, **[КБК](/glossary/kbk)** и кассового контура используйте **[Казначейство](/data-sources/federal/roskazna-reports)** и **[ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)**.

## Описание

Карточка описывает витрину **[Краткая информация об исполнении консолидированного бюджета Российской Федерации](https://minfin.gov.ru/ru/statistics/conbud/execute/)**: на странице перечислены документы с датами публикации и ссылками на выгрузки. По состоянию на актуальные публикации на сайте доступны три типа файлов:

- **ежемесячная** краткая информация (показатели **накопленно с начала года**, млрд руб.);
- **ежеквартальная** краткая информация (также **накопленно с начала года**, млрд руб.);
- **ежегодная** краткая информация (итоги по **финансовому году** в млрд руб. — конкретный год см. в заголовке документа и в файле).

Сюда **не** входят **[открытые наборы Минфина](/data-sources/federal/minfin-opendata)** из каталога с `list.csv` и паспортами `meta.csv` — раздел **conbud/execute** — отдельная **статистическая** подборка на сайте ведомства, без включения в машинный реестр открытых данных того же формата.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/statistics/conbud/execute/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Статистика» и вложений к документам.

| --- | --- | --- |

| Родительский раздел «Консолидированный бюджет Российской Федерации» | https://minfin.gov.ru/ru/statistics/conbud/ | HTML (навигация по смежным материалам) |

Прямые URL файлов **XLSX** при обновлении вложений **меняются**; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** в карточке документа на странице раздела.

## Объекты данных и показатели

Состав строк и столбцов задаётся **опубликованными XLSX**; по смыслу заголовков страницы это:

- **Сводные показатели исполнения консолидированного бюджета РФ** с учётом **государственных внебюджетных фондов** в **млрд руб.** — доходы, расходы и связанные агрегаты в логике **федеральной консолидации** (конкретный перечень статей — в файле на дату скачивания).
- **Накопленный с начала года** итог — для **ежемесячной** и **ежеквартальной** серий (проверяйте подписи листов и шапку таблицы в файле).
- **Годовая** серия — для сопоставления с **[законом о федеральном бюджете](/legal/budgetlaw)**, материалами по **[ГВФ](/glossary/extrabudgetary-funds)** и иными официальными сводами за **финансовый год** (методика **может отличаться** от показателей только **федерального бюджета**).

**Пробелы**: это **не** построчный учёт по **[КБК](/glossary/kbk)** и не замена **[форм исполнения Казначейства](/data-sources/federal/roskazna-reports)**; граница «**только ФБ**» vs «**консолидированный РФ + ГВФ**» принципиальна при сравнении с **[fedbud/execute](/data-sources/federal/minfin-fedbud-execute)**.

## Периодичность обновления

Публикации привязаны к **выходу новых редакций документов** на странице: у каждой карточки документа указаны даты «Опубликовано» и «Изменено». **Ежемесячный** файл обновляется чаще, чем **квартальный** и **годовой**; стабильного API под выгрузку нет — перед массовой загрузкой проверяйте страницу.

См. также общие замечания по обновлению публичного сайта в разделе **«Ограничения и актуальность»** карточки **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)**.

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Автоматические запросы к сайту иногда упираются в **ограничения доступа**; для парсинга используйте опубликованные пользователем ссылки после проверки в браузере и учитывайте политику сайта.
- Показатели Минфина в этих XLSX **не обязаны** построчно совпадать с **[открытыми наборами Минфина](/data-sources/federal/minfin-opendata)**, **[срезами ГИИС](/data-sources/federal/budget-gov-ru-datasets)** и **[формами Казначейства](/data-sources/federal/roskazna-reports)** — сверяйте методику, заголовки и дату файла.

## Примеры доступа

Сохранить HTML-страницу раздела для ручного или последующего разбора ссылок «Скачать»:

```bash
curl -sS "https://minfin.gov.ru/ru/statistics/conbud/execute/" -o minfin-conbud-execute.html
```

## Не путать с

- **[Минфин: исполнение федерального бюджета (fedbud/execute)](/data-sources/federal/minfin-fedbud-execute)** — **федеральный бюджет** как отдельный объект; цифры **консолидированного** свода **шире** и **не тождественны** «только ФБ» без проверки методики.
- **[Минфин: исполнение консолидированных бюджетов субъектов (subbud/execute)](/data-sources/federal/minfin-subbud-execute)** — **региональный** разрез по субъектам, а не **свод РФ** в логике **conbud**.
- **[Минфин: исполнение бюджетов государственных внебюджетных фондов (outbud/execute)](/data-sources/federal/minfin-outbud-execute)** — **только бюджеты [ГВФ](/glossary/extrabudgetary-funds)**; **conbud** включает **консолидированный бюджет РФ** и ГВФ в **одной** макро-сводке — объекты и суммы **не взаимозаменяемы** без заголовков файла.
- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — **реестр наборов** с `list.csv` и паспортами, а не список трёх XLSX на странице статистики **conbud/execute**.

## Частые вопросы

**В:** Чем этот раздел отличается от **fedbud/execute**?

**О:** **fedbud** публикует краткую информацию об исполнении **федерального бюджета**; **conbud** — об исполнении **консолидированного бюджета РФ** с включением **ГВФ** в формулировке документов на странице. Сопоставляйте заголовки файлов и шапки таблиц, а не предполагайте равенство сумм.

**В:** Можно ли по этим XLSX восстановить исполнение по каждому **[КБК](/glossary/kbk)**?

**О:** Обычно **нет**: файлы **краткой информации** агрегируют показатели. Для разреза по **[КБК](/glossary/kbk)** и формам используйте **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)** и **[Казначейство](/data-sources/federal/roskazna-reports)**.
