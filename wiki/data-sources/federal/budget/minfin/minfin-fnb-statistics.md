# Минфин: данные о движении средств и результатах управления ФНБ

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный уровень: **официальная выгрузка** Минфина по **движению средств** и **результатам управления** средствами **[Фонда национального благосостояния (ФНБ)](/glossary/nwf)** |
| Тип доступа | **Файл по ссылке**: HTML-страница раздела «Статистика → Суверенные фонды → ФНБ» с блоком «Скачать» и табличной выгрузкой **XLSX** |
| Основные форматы | **XLSX**; обёртка страницы — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (без регистрации для скачивания с официальной страницы) |

Коротко: **агрегированная официальная таблица** по ФНБ для сопоставления с **[открытыми наборами Минфина](/data-sources/federal/minfin-opendata)**, **[дополнительными нефтегазовыми доходами](/data-sources/federal/minfin-fedbud-oil)** и **[отчётностью по резервам](/reporting/fiscal-reserves-reporting)**; не заменяет **[кассовое исполнение](/glossary/treasury-execution)** по **[Казначейству](/data-sources/federal/roskazna-reports)** без проверки методики.

## Описание

Карточка описывает витрину **[Фонд национального благосостояния](https://minfin.gov.ru/ru/statistics/fonds/fnb/)** на сайте Минфина: публикуется документ **«Данные о движении средств и результатах управления средствами Фонда национального благосостояния»** с датами «Опубликовано» / «Изменено» и ссылкой **«Скачать»** на файл **XLSX**.

Сюда **не** входят полные **[открытые наборы Минфина](/data-sources/federal/minfin-opendata)** с паспортами `meta.csv` и машинным каталогом `list.csv` — этот раздел **«Статистика»** даёт **отдельную** ведомственную выгрузку по ФНБ, которая может **не дублироваться** тем же составом строк в реестре открытых данных.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/statistics/fonds/fnb/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Статистика» и вложений к документам.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Страница раздела (карточка документа, ссылка «Скачать») | https://minfin.gov.ru/ru/statistics/fonds/fnb/ | HTML; вложение **XLSX** |
| Родительский раздел «Статистика» | https://minfin.gov.ru/ru/statistics/ | HTML (навигация по смежным материалам) |

Прямой URL файла **XLSX** может меняться при обновлении вложения; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** на странице документа.

## Объекты данных и показатели

Состав листов, столбцов и строк задаётся **опубликованным XLSX**; по названию документа это:

- **Движение средств** ФНБ — притоки, оттоки и сальдо в разрезе, зафиксированном в таблице (сверяйте с **[289-ФЗ](/legal/nwf-289fz)** и **[законом о федеральном бюджете](/legal/budgetlaw)** на год, не подменяя юридическую трактовку одной выгрузкой).
- **Результаты управления** — показатели доходности и оценки активов **в логике раскрытия Минфина** на дату файла (не обязательно совпадают с рыночной отчётностью **[Банка России](/data-sources/federal/cbr-statistics)** или иными витринами без методики).

**Пробелы**: это **не** построчный учёт по **[КБК](/glossary/kbk)** и не замена **[форм исполнения Казначейства](/data-sources/federal/roskazna-reports)**; для связки с **источниками финансирования дефицита** используйте также **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)** и паспорты тематических наборов в **[открытых данных Минфина](/data-sources/federal/minfin-opendata)**.

## Периодичность обновления

Обновление привязано к **публикации новой редакции документа** и файла на странице; даты «Опубликовано» / «Изменено» указаны в карточке документа на сайте. Типового ежедневного API здесь нет — проверяйте страницу перед скачиванием.

См. также общие замечания по обновлению публичного сайта в разделе **«Ограничения и актуальность»** карточки **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)**.

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Некоторые автоматические запросы к сайту могут получать **страницу ограничения доступа** вместо контента — для парсинга учитывайте политику сайта и используйте **опубликованную пользователем ссылку «Скачать»** после проверки в браузере.
- Показатели в **XLSX** **не обязаны** совпадать построчно с **[агрегированными наборами открытых данных Минфина](/data-sources/federal/minfin-opendata)** — сверяйте методику, заголовок документа и дату файла.

## Примеры доступа

```bash
curl -sS "https://minfin.gov.ru/ru/statistics/fonds/fnb/" -o minfin-fnb-statistics.html
```

Дальше извлеките актуальный URL вложения **XLSX** из HTML (или скачайте вручную по кнопке «Скачать» на странице).

## Связанные страницы wiki

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — машиночитаемые наборы по ФНБ, долгу и смежным темам; состав полей и периодичность могут отличаться от страницы **statistics/fonds/fnb**.
- **[Минфин: дополнительные нефтегазовые доходы федерального бюджета (fedbud/oil)](/data-sources/federal/minfin-fedbud-oil)** — смежная витрина по **нефтегазовым доходам**, связанным с формированием ФНБ в бюджетной логике.
- **[Наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** — плановые и исполнительные срезы федерального бюджета для сопоставления с операциями и параметрами года по **[КБК](/glossary/kbk)** там, где методика соизмерима.
- **[Отчёты и материалы Федерального казначейства](/data-sources/federal/roskazna-reports)** — регламентированное **исполнение** и счета; интерпретация **остатков и движений ФНБ** требует методики **Минфина**.
- **[Отчётность по ФНБ, резервным фондам и управлению ликвидностью](/reporting/fiscal-reserves-reporting)** — как этот источник вкладывается в публичный контур отчётности по резервам.
- **[Фонд национального благосостояния (ФНБ)](/glossary/nwf)** — определение института и типичные ошибки смешения с **[ГВФ](/glossary/extrabudgetary-funds)**.
- **[Федеральный закон о Фонде национального благосостояния (289-ФЗ)](/legal/nwf-289fz)** — правовая рамка источников и использования средств ФНБ.

## Не путать с

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — **каталог наборов** с идентификаторами `7710168360-*` и `list.csv`, а не единичная XLSX-витрина под один заголовок документа на «Статистике».

## Частые вопросы

**В:** Достаточно ли этого файла для проверки всех операций ФНБ за год?

**О:** Файл даёт **официальную сводку** в формате Минфина на дату публикации; для **полноты** и **сверки с бюджетным исполнением** используйте **[открытые наборы](/data-sources/federal/minfin-opendata)**, **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)** и **[Казначейство](/data-sources/federal/roskazna-reports)** с учётом методики каждого канала.

**В:** Где искать те же темы в машиночитаемом каталоге Минфина?

**О:** Просматривайте **[сводный `list.csv`](https://minfin.gov.ru/opendata/list.csv)** и тематические наборы в **[открытых данных Минфина](/data-sources/federal/minfin-opendata)**; раздел **statistics/fonds/fnb** может **не дублироваться** тем же составом показателей и периодичностью.
