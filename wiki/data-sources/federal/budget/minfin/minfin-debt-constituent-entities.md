# Минфин: объём и структура госдолга субъектов РФ и долга муниципальных образований

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Субъекты РФ** (государственный долг) и **муниципальные образования** (муниципальный долг): агрегированные своды по объёму и структуре долга в публикациях Минфина |
| Тип доступа | **Файлы по ссылке**: HTML-страница раздела «Статистика → Государственный долг Российской Федерации» со списком документов и кнопкой «Скачать» для каждого **XLSX** |
| Основные форматы | **XLSX**; обёртка — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (скачивание с официальной страницы без регистрации) |

Коротко: **официальные сводные таблицы** по **государственному долгу субъектов** и **муниципальному долгу** в разрезе, заданном в файлах на дату публикации — удобная **макро**-точка входа для сравнения регионов и динамики структуры; для **регламентированной отчётности**, **долговых книг** и построчного учёта операций используйте **[отчётность по долгу](/reporting/public-debt-reporting)** и **[материалы Казначейства](/data-sources/federal/roskazna-reports)**.

## Описание

Карточка описывает витрину **[Государственный долг субъектов Российской Федерации](https://minfin.gov.ru/ru/statistics/gov_debt/debt_constituent_entities/)**: на странице перечислены редакции документа «Объём и структура государственного долга субъектов Российской Федерации и долга муниципальных образований» с датами публикации и вложениями **XLSX**. Обычно доступны **несколько файлов** с разным горизонтом лет (например отдельные выгрузки за скользящие периоды и архивные интервалы) — конкретный перечень и даты смотрите на странице.

Сюда **не** входят своды **исключительно по федеральному** государственному долгу РФ (другие подразделы сайта Минфина), а также **[каталог открытых наборов](/data-sources/federal/minfin-opendata)** с паспортами `meta.csv` — эта витрина оформлена как **статистический документ** со скачиваемыми таблицами, без обязательного включения в машинный реестр `list.csv`.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/statistics/gov_debt/debt_constituent_entities/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Статистика» и вложений к документам.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Страница раздела (список документов, «Скачать» для каждого файла) | https://minfin.gov.ru/ru/statistics/gov_debt/debt_constituent_entities/ | HTML; вложения **XLSX** |
| Родительский раздел «Государственный долг Российской Федерации» | https://minfin.gov.ru/ru/statistics/gov_debt/ | HTML (навигация по смежным материалам) |

Прямые URL файлов **XLSX** при обновлении вложений **меняются**; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** в карточке документа на странице раздела.

## Объекты данных и показатели

Состав строк и столбцов задаётся **опубликованными XLSX**; по смыслу заголовка документа это:

- **Агрегаты по субъектам РФ** — объём и структура **государственного долга** субъекта (конкретные разрезы и классификаторы территорий — в файле на дату скачивания).
- **Агрегаты по муниципальным образованиям** — объём и структура **муниципального долга** (в границах методики публикации).
- **Структура долга** — виды обязательств/инструментов в том виде, в каком они сведены в таблице (сопоставляйте с **[формами долга по БК РФ](/glossary/forms-of-state-debt)** и методическими пояснениями периода).

**Пробелы**: это **не** замена **[долговых книг](/data-sources/federal/roskazna-reports)** и регламентированных форм; **не** построчный реестр сделок; единицы измерения, валюта и точные определения показателей нужно брать из **шапки и листов** конкретного файла и сопутствующих методических материалов Минфина.

## Периодичность обновления

Публикации привязаны к **выходу новых редакций документов** на странице: у каждой карточки документа на сайте указаны даты «Опубликовано» и «Изменено». Отдельного API под выгрузку нет — перед массовой загрузкой проверяйте страницу.

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Автоматические запросы к сайту иногда отвечают **503** без типичного заголовка **User-Agent** клиента; для скриптов задайте, например, `User-Agent` браузера и соблюдайте разумную частоту обращений.
- Своды Минфина **не обязаны** построчно совпадать с **[открытыми наборами Минфина](/data-sources/federal/minfin-opendata)** или региональными порталами — сверяйте методику, заголовки и дату файла.

## Примеры доступа

Проверка доступности HTML-страницы раздела (без парсинга вложений):

```bash
curl -sS -A "Mozilla/5.0" -L -o /dev/null -w "%{http_code}\n" \
  "https://minfin.gov.ru/ru/statistics/gov_debt/debt_constituent_entities/"
```

Ссылки на **XLSX** забирайте из актуальной разметки страницы (кнопка «Скачать» у нужной редакции документа).

## Не путать с

- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — **каталог наборов** с `list.csv` и паспортами; тематическое пересечение по долгу не означает совпадение полей или периодичности с этой витриной.
- **Своды по государственному долгу Российской Федерации** на других страницах раздела «Государственный долг» на `minfin.gov.ru` — **федеральный** долг Правительства РФ, а не разрез «субъекты + МО» данной карточки.

## Частые вопросы

**В:** Достаточно ли этих XLSX для аудита долга конкретного муниципалитета?

**О:** Файлы дают **опубликованный свод** по методике Минфина на дату; для **первичного учёта**, договоров и полноты по МО опирайтесь на **региональные и муниципальные** источники и **[отчётность по долгу](/reporting/public-debt-reporting)** в связке с **[Казначейством](/data-sources/federal/roskazna-reports)**.

**В:** Почему скрипт получает 503?

**О:** Попробуйте запрос с **User-Agent** браузера и без агрессивной частоты; прямые ссылки на вложения лучше брать после ручной проверки страницы.
