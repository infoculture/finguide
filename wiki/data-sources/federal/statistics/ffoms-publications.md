# Публичные финансовые публикации ФФОМС

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный бюджет ОМС и документы фонда |
| Тип доступа | Разделы официального сайта и федеральные бюджетные производные |
| Основные форматы | HTML, PDF, отдельные таблицы |
| Оператор | Федеральный фонд ОМС |
| Режим доступа | Открытый просмотр |

Коротко: карточка ведёт к официальным документам ФФОМС и параллельным бюджетным публикациям. На дату проверки стабильная массовая построчная выгрузка всего финансового контура ФФОМС на сайте фонда не подтверждена.

## Описание

Для бюджета ФФОМС используйте закон о бюджете фонда, отчёт об исполнении и публикации Казначейства/Минфина. Реестры отрасли имеют иной объект наблюдения и не заменяют финансовую отчётность.

## Оператор

[ФФОМС](/organizations/ffoms).

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Официальный сайт фонда | https://ffoms.gov.ru/ | HTML, документы |
| Единый портал бюджетной системы | https://budget.gov.ru/ | HTML, JSON |

## Объекты данных и показатели

- закон о бюджете ФФОМС и изменения к нему;
- отчёт об исполнении бюджета фонда;
- агрегаты доходов, расходов и межбюджетных трансфертов;
- отраслевые реестры ОМС как отдельный нефинансовый контур.

## Периодичность обновления

По бюджетному и отчётному циклу; конкретная дата определяется документом.

## Ограничения и особенности

- отсутствие единого API требует инвентаризации документов по годам;
- федеральный фонд и ТФОМС анализируются раздельно;
- трансферты между бюджетами нельзя суммировать без консолидации.
