Коротко: официальные **годовые своды по регистрационной деятельности** (формы **1-ЮР**, **1-ИП**) и тематические файлы по **гендерному составу** новых ИП/ЮЛ — для анализа **динамики бизнеса**, сопоставления с **макро- и региональными рядами** и контекста **закупок и контрагентов**, без подмены **микроданных реестра** ЕГРЮЛ/ЕГРИП.

## Описание

Карточка описывает раздел **«Статистика по государственной регистрации»** на портале ФНС: перечень отчётов о работе регистрации **юрлиц** и **ИП/КФХ**, ссылки на **архив по годам** и блок **гендерной статистики**. В объём **не** входят: **налоговые статистические формы** (1-НМ, 1-НДС и др. — отдельный каталог), **поиск по реестру** и выписки **[ЕГРЮЛ/ЕГРИП](/information-systems/federal/egrul)**, **каталог открытых данных** с паспортами наборов по другому URL. По каждому набору открытых данных с идентификатором `7707329152-*` в wiki есть отдельная карточка (см. [оглавление федеральных источников](/data-sources/federal) и пример [7707329152-address](/data-sources/federal/7707329152-address)).

## Объекты данных и показатели

- **Строка набора** — как правило **агрегированные показатели регистрационной работы** (количественные и структурные срезы по методике отчёта) за **календарный год** или иной отчётный горизонт, заданный формой; не путать с **записью в реестре** по одному ОГРН.
- **Формы** — **1-ЮР** (сведения о работе по регистрации **юридических лиц**), **1-ИП** (сведения о работе по регистрации **ИП и КФХ**); состав граф и расшифровок — в текстах и файлах на страницах ФНС.
- **Гендерный состав при создании бизнеса** — отдельные файлы за отдельные годы (в таблице на витрине — прямые ссылки на документы на `data.nalog.ru`).
- **Связка с финансовой аналитикой** — через **динамику числа субъектов** и структуры регистрации; для склейки с контрактами и субсидиями по-прежнему нужны **ИНН/ОГРН** и реестровые или закупочные источники.
- **Пробелы:** нет единого API; URL вложенных страниц **год к году** могут отличаться; для сравнения с **официальной статистикой Росстата** требуется сверка методик.

## Периодичность обновления

Публикации привязаны к **годовому** циклу отчётности по регистрации и обновлению витрины: новые годы появляются в таблице на странице **[регистрационной статистики](https://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/regstats/)** по мере выхода материалов ФНС (дата обновления страницы указана в подвале сайта). Типичные ограничения портала см. в разделе **«Ограничения и актуальность»** в [карточке сайта ФНС](/information-systems/federal/fnssite).

## Ограничения и особенности

- **Условия использования:** ведомственные публикации; при переработке обычно требуется **указание источника** (формулировки — на страницах ФНС и в сопроводительных файлах).
- **Доступ автоматизацией:** как и для других разделов `nalog.gov.ru`, запросы без корректного **User-Agent** иногда получают **403**; используйте уважительный заголовок и умеренную частоту запросов.
- **Предмет учёта:** отчёты отражают **регистрационную деятельность и её итоги в агрегатах**, а не налоговую отчётность и не кассовое исполнение бюджета.

## Примеры доступа

Проверка доступности витрины (без парсинга HTML); при необходимости задайте заголовок `User-Agent`:

```python
import requests

url = "https://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/regstats/"
headers = {
    "User-Agent": "Mozilla/5.0 (compatible; research/1.0; +https://www.nalog.gov.ru/)",
}
response = requests.get(url, headers=headers, timeout=30)
response.raise_for_status()
print(response.status_code, response.headers.get("content-type", ""))
```

Файлы конкретных лет скачиваются по **ссылкам из таблицы на странице**; прямые URL на каждый год в карточке не фиксируются из‑за обновлений витрины.

- [ЕГРЮЛ и ЕГРИП](/information-systems/federal/egrul) — **помикровый** публичный доступ к реестру; регистрационная статистика даёт **своды**, а не выгрузку всех записей.
- [Официальная статистика Росстата (ЕМИСС, открытые данные)](/data-sources/federal/rosstat-opendata) — смежные макро- и официальные ряды; сопоставление только после проверки **определений показателей**.

## Не путать с

- **[Налоговая статистика ФНС](/data-sources/federal/nalog-statistics)** — налоги и взносы по формам НК РФ; другой раздел витрины (`…/statistics_and_analytics/forms/`).
- **[Публичный сервис ЕГРЮЛ/ЕГРИП](/information-systems/federal/egrul)** — выдача сведений по конкретному лицу; не заменяет **годовые агрегаты** регистрационной статистики.
