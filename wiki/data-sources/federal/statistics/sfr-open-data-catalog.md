# Каталог открытых данных Социального фонда России

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | СФР и архивы ПФР/ФСС в составе страницы |
| Тип доступа | Каталог паспортов, CSV и архивных версий |
| Основные форматы | CSV, HTML |
| Оператор | Социальный фонд России |
| Режим доступа | Открытый |

Коротко: официальный каталог содержит отдельные наборы СФР, включая исполнение бюджета по доходам и расходам. Периоды и даты обновления проверяются по паспорту каждого набора.

## Качество и верификация

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки | 2026-07-16 |
| Полнота | Частичная: перечень открытых наборов, не все данные фонда |
| Машиночитаемость | CSV с паспортами и структурами |
| Юридическая значимость | Официальный источник СФР |
| Задержка обновления | Различается по набору |
| Глубина архива | Есть версии СФР и архивные разделы ПФР/ФСС |
| Лицензия | На странице указаны типовые условия использования открытых данных |

## Описание

Каталог объединяет текущие наборы СФР и архивные публикации фондов-предшественников. Он дополняет статистическую витрину поступления и расходования средств: каталожные наборы имеют паспорта, а статистическая страница — собственную методику и Excel-файлы.

## Оператор

[Социальный фонд России](/organizations/sfr).

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Каталог | https://sfr.gov.ru/opendata/ | HTML, CSV |
| Набор расходов бюджета | https://sfr.gov.ru/opendata/pfr_opendata/data_set~7706016118-budget | HTML, CSV |

## Объекты данных и показатели

- паспорт набора, структура, дата актуальности и версии;
- агрегаты доходов или расходов бюджета по опубликованной структуре;
- тематические показатели выплат, программ и деятельности фонда.

## Периодичность обновления

Задаётся отдельно в паспорте набора; архивные файлы могут обновляться реже текущей витрины.

## Ограничения и особенности

- каталог включает наследуемые разделы ПФР и ФСС;
- идентификатор с прежним ИНН в URL не означает, что карточка описывает только старый фонд;
- суммы сверяются с бюджетной отчётностью по периоду и методике.

## Примеры доступа

Проверка каталога без догадок об URL конкретного CSV:

```bash
curl -L --fail --silent --show-error https://sfr.gov.ru/opendata/ -o sfr-opendata.html
```
