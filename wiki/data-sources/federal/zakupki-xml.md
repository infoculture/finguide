# XML-выгрузки ЕИС Закупки

## Описание

Полные ежедневные выгрузки в XML по 44-ФЗ и 223-ФЗ доступны через FTP. Архивы организованы по регионам и типам документов: контракты, извещения, протоколы, реестры.

## Оператор

Федеральное казначейство — ftp://ftp.zakupki.gov.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| FTP | ftp://ftp.zakupki.gov.ru/ | XML/ZIP |
| Описание схем | https://zakupki.gov.ru/epz/main/public/document/view.html | PDF |

## Состав данных

- Контракты по 44-ФЗ и 223-ФЗ
- Извещения и протоколы
- Планы-графики

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).

## Ограничения и особенности

- Лицензия: открытые данные (как правило, разрешена свободная переработка с указанием источника).
- Объёмы и форматы могут меняться; рекомендуется отслеживать страницу источника.
