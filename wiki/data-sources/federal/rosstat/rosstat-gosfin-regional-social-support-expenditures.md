Коротко: **одна книга на отчётный год** с **десятками листов** — на каждый **субъект** (и своды по стране/округам) таблица **расходов на меры социальной поддержки** в разрезе **категорий граждан** и **форм поддержки** (денежная выплата, оплата ЖКУ, льготный проезд, лекарственное обеспечение и т.д.). Удобно для **сравнения регионов** и **структуры мер**; не замена **формам казначейского исполнения** или **справочнику ОКТМО** без отдельной нормализации названий.

## Описание

Раздел **«Расходы консолидированных бюджетов субъектов Российской Федерации и муниципальных образований на реализацию мер социальной поддержки…»** на https://rosstat.gov.ru/statistics/finance содержит **архив годовых файлов** (`socpod_2015.xlsx` … `socpod_2021.xlsx` в проверенной витрине). Каждый файл — **самостоятельный годовой срез**; имя файла **включает год**.

**Структура книги (на примере `socpod_2021.xlsx`):**

- **`Содержание`** — список листов (= **перечень субъектов и сводов**).
- **Один лист = одна территория** для таблицы 1: «Российская Федерация», каждый **федеральный округ**, каждый **субъект РФ** (в т.ч. **города федерального значения**), при необходимости **варианты «область без АО»** и т.п. — точный перечень смотрите в **оглавлении конкретного года**.
- На листе — **одна широкая таблица**:
  - **Строки** — **категории населения** (инвалиды, ветераны, дети-инвалиды, лица, подвергшиеся радиации, и т.д.), с блоками **«всего»**, **«категории, отнесённые к компетенции РФ»** и подстроками «в том числе».
  - **Столбцы** — **суммарные расходы** и разложение по **видам мер** (колонки **2–13** в проверенной версии: денежная выплата, оплата ЖКУ, обеспечение жильём, топливо, телефон, проезд, ЛО и медуслуги, питание, санатории, прочие меры). Есть строка с **номерами колонок** (1…13) как **шаблон формы**.
  - **Единица:** в шапке указано **«тысяча рублей»** — при переводе в рубли умножайте на **1000**.
  - Заголовок периода в форме **«январь–декабрь `<год>`»** — то есть **полный год**.

**Натуральная форма** в названии таблицы на витрине означает, что в **других формах** той же тематики (не в этом файле) могут быть **дополнительные таблицы** по количественным показателям; внутри `socpod_*.xlsx` на листе **«Таблица 1»** в проверенной версии представлены **денежные суммы** по видам мер.

Сюда **не** входят: **федеральный** консолидированный бюджет целиком — см. **[gos_fin_byudget.xlsx](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf)**; **ЕМИСС 1.32.2** как отдельный веб-вход — см. ссылку на `fedstat.ru` рядом с файлами на витрине «Финансы».

**Росстат** — витрина: https://rosstat.gov.ru/statistics/finance; пример файла: https://rosstat.gov.ru/storage/mediabank/socpod_2021.xlsx

## Объекты данных и показатели

- **Строка** — **категория граждан** или **агрегат** («Всего», «Категории … компетенции РФ — всего»).
- **Столбец** — **вид меры социальной поддержки** (денежные и неденежные формы) либо служебные колонки с нумерацией.
- **Лист** — **территория**; для склейки с другими датасетами нужен **справочник субъектов** (например, **код субъекта**, **ОКТМО**) — **в файле обычно только текстовое название листа**, без кода.
- **Муниципальный уровень** в названии раздела означает, что **цифры по субъекту уже консолидированы** «субъект + все МО», а **не** построчно по каждому муниципалитету.

## Периодичность обновления

Публикация — **раз в год** на отчётный период (календарный год); новый год обычно добавляется **новым файлом**. Актуальный перечень файлов и даты размера/обновления смотрите на **HTML-витрине** (они могут **отставать** от календарного года на время подготовки формы).

## Ограничения и особенности

- **Широкая таблица с переносами в заголовках** — при импорте в SQL/Parquet удобнее **зафиксировать словарь** колонок по **номерам 1…13**, а человекочитаемые подписи хранить отдельно.
- **Пустые ячейки и пустые строки** вместо нуля — при суммировании используйте **`coalesce` в 0** с осторожностью к смыслу «нет данных» vs «ноль».
- **Сравнение лет** требует **отдельного скачивания каждого файла**; стабильного «одного URL на все года» нет.
- **Сопоставление с Казначейством** — только через **определения** статей соцподдержки в **отчётности** и **методику Росстата**; прямого поля КБК в этих книгах **нет**.

## Примеры доступа

Проверка доступности файла за конкретный год:

```bash
curl -sSk -A "Mozilla/5.0 (compatible; research/1.0)" -o /dev/null -w "%{http_code}\n" "https://rosstat.gov.ru/storage/mediabank/socpod_2021.xlsx"
```

Итерация по листам (имена листов = территории) на Python с `openpyxl`:

```python
from openpyxl import load_workbook

wb = load_workbook("socpod_2021.xlsx", read_only=True, data_only=True)
for name in wb.sheetnames:
    if name in ("Содержание",):
        continue
    ws = wb[name]
    print(name, ws.max_row, ws.max_column)
wb.close()
```

## Частые вопросы

**В:** Можно ли из этой книги получить расходы по каждому муниципалитету отдельно?

**О:** **Нет:** публикуется **консолидированный** уровень **субъекта РФ** (с включением МО в агрегат). Для муниципалитетов нужны **региональные открытые порталы** и/или **формы отчётности** с нужным уровнем детализации.

**В:** Почему на листе субъекта почти пустые строки по федеральным категориям?

**О:** Часть категорий **финансируется из федерального бюджета** и в региональной таблице может отражаться **ограниченно** или **нулём** в зависимости от **методики распределения расходов**; трактовку смотрите в **методических материалах** к форме и в **пояснениях Росстата** на витрине.
