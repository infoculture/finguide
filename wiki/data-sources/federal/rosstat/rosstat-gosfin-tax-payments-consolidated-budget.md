Коротко: **табличная сводка** по **крупным видам налоговых поступлений** в разрезе **консолидированного бюджета / федерального бюджета / консолидированных бюджетов субъектов РФ**, плюс **столбцы темпов роста** к предыдущему году. На **май 2026** в проверенной книге `gos_fin_nalog_2023.xls` **не** обнаружено отдельных листов с **задолженностью**, хотя **название кнопки** на витрине Росстата включает и задолженность — см. раздел «Ограничения».

## Описание

На странице https://rosstat.gov.ru/statistics/finance в блоке **«Государственные финансы»** публикуется кнопка с длинным заголовком про **поступление** и **совокупную задолженность** по налогам, сборам, страховым взносам, пеням, санкциям и процентам. Фактическая выгрузка по ссылке именуется **`gos_fin_nalog_<ГОД>.xls`**, где **год в имени файла** обычно совпадает с отчётным годом в заголовке таблицы.

Внутри книги (проверка содержимого на примере `gos_fin_nalog_2023.xls`):

- лист **`Содержание`** — перечень листов и реквизиты обновления;
- лист **`1`** — основная **матрица**: строки-виды налогов/групп, столбцы — **абсолютные значения** за отчётный год и **проценты к предыдущему году** для тех же разрезов;
- лист **`2`** — заготовка под **диаграмму динамики** (в проверенной версии — фактически только заголовок, без числовой таблицы);
- пустой служебный **`Лист1`** (может отсутствовать в других редакциях).

Страховые взносы на обязательное социальное страхование в основной таблице **не включены** (есть **справочная строка** внизу листа `1` с оговоркой по взносам).

Сюда **не** входят: **детальная налоговая статистика ФНС** по формам НК РФ — см. **[отдельный источник](/data-sources/federal/nalog-statistics)**; **страховые взносы** как полноценная матрица по фондам — см. **[исполнение бюджетов ГВФ](/data-sources/federal/rosstat-gosfin-extrabudgetary-funds-execution)** и **[консолидированный бюджет (свод Росстата)](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf)**.

**Росстат** — витрина: https://rosstat.gov.ru/statistics/finance; пример файла за 2023 год: https://rosstat.gov.ru/storage/mediabank/gos_fin_nalog_2023.xls

- Карточка ИС: **[ЕМИСС](/information-systems/federal/emiss)** — при необходимости **дополнительных рядов** (в т.ч. если появятся публикации по задолженности) ищите по каталогу показателей на `fedstat.ru`.

## Объекты данных и показатели

- **Строка** — **вид налога/группа** (например, налог на прибыль, НДФЛ, НДС, акцизы, налоги на имущество, НДПИ и др.) либо служебные подзаголовки («из них»).
- **Столбцы чисел (проверенная структура листа `1`)**:
  - абсолютные **млрд руб.** — **консолидированный бюджет**, **федеральный бюджет**, **консолидированные бюджеты субъектов РФ**;
  - **проценты к предыдущему году** — для тех же трёх разрезов.
- **Единицы измерения** указаны в шапке (**млрд рублей**); часть ячеек содержит **прочерк** вместо числа.
- **Связка с бюджетной отчётностью:** прямых кодов **КБК** нет; для стыковки с **[ГИИС](/data-sources/federal/budget-gov-ru-datasets)** нужны **промежуточные маппинги** по названиям статей и методическим пояснениям ФНС/Росстата.

## Периодичность обновления

Обновление привязано к **выходу очередной годовой таблицы**; дата публикации указана на **витрине** (метаданные файла рядом с кнопкой скачивания) и на листе **`Содержание`**. При смене отчётного года Росстат, как правило, выкладывает **новый файл** с новым годом в имени — старый URL с прежним годом может остаться **архивным**.

## Ограничения и особенности

- **Формат XLS:** для Python 3 часто нужен **`xlrd` 1.x** или конвертация в XLSX в LibreOffice; `pandas.read_excel` с `openpyxl` **напрямую** старый `.xls` не читает.
- **Расхождение заголовка кнопки и таблицы:** на **май 2026** в файле `gos_fin_nalog_2023.xls` таблично представлены **поступления по данным ФНС**; **блока задолженности** в листах не видно. Имеет смысл **перепроверять** каждую новую редакцию файла или искать **серии в ЕМИСС**, если нужны именно **задолженность/пени/санкции**.
- **Диаграмма:** лист `2` может быть **заглушкой** без данных — график иногда «живёт» только внутри Excel-клиента или в другой публикации.
- **Автозагрузка:** используйте **User-Agent браузера** и не полагайтесь на вечную стабильность **имени файла** без года — ориентируйтесь на **страницу витрины**.

## Примеры доступа

Проверка HTTP-кода для конкретной редакции файла:

```bash
curl -sSk -A "Mozilla/5.0 (compatible; research/1.0)" -o /dev/null -w "%{http_code}\n" "https://rosstat.gov.ru/storage/mediabank/gos_fin_nalog_2023.xls"
```

Чтение числового листа на Python (требуется `xlrd==1.2.0` для `.xls`):

```python
import xlrd

book = xlrd.open_workbook("gos_fin_nalog_2023.xls")
sh = book.sheet_by_name("1")
for row in range(6, min(17, sh.nrows)):
    print(sh.row_values(row))
```

## Частые вопросы

**В:** Где в этом файле задолженность по налогам, как в названии на сайте?

**О:** В проверенной версии **`gos_fin_nalog_2023.xls`** отдельной таблицы по задолженности **нет**; ориентируйтесь на **фактическое содержимое книги** после скачивания или на **серии ЕМИСС**. Заголовок кнопки на витрине может опережать состав файла.

**В:** Почему «акцизы» в таблице могут быть с отрицательным значением?

**О:** Это **особенность свода и методики отражения возмещений/зачётов** в администрируемых платежах; трактовку **нужно брать из методических комментариев Росстата/ФНС**, а не интерпретировать как «ошибку выгрузки» без проверки.
