Коротко: **мультилистовая книга** с **рядами поступлений и расходований** по каждому крупному **внебюджетному социальному фонду** и отдельным **блокам ОМС**, в разрезе **лет** (горизонтальная шапка-год) и **статей** (вертикаль: налоги и взносы, трансферты из федерального бюджета, выплаты населению, административные расходы и т.д.). Подходит для **долгой динамики** и **структуры фондов** в терминах **официальной статистики Росстата**, а не для **помесячной кассы**.

## Описание

Файл относится к блоку **«Государственные финансы»** на странице https://rosstat.gov.ru/statistics/finance. В отличие от **[сводного XLSX по консолидированному бюджету](/data-sources/federal/rosstat-gosfin-consolidated-budget-rf)** здесь — **отдельные таблицы по каждому фонду** (и отдельный лист для **СФР** после реорганизации), с **историей с 1995 г.** и **склейкой периодов** до/после реформ учёта.

Типичная структура (по содержанию листов в проверенной версии):

| Лист | Содержание (по оглавлению в книге) |
| --- | --- |
| `Содержание` | навигация, реквизиты, дата обновления |
| `1` | **ПФР** — поступление/расходование, в т.ч. **взносы**, **трансферты из федерального бюджета**, крупные статьи **выплат** (пенсии, ЕДВ, ОПС, маткапитал и др.) |
| `2` | **ФСС РФ** — аналогичная логика для периода до слияния в СФР |
| `3` | **СФР** — **свод за один год** (например, 2023) уже после объединения ПФР и ФСС |
| `4` | **ФФОМС** — динамика по годам |
| `5` | **ТФОМС** — динамика по годам |
| `6` | заготовка под **график** поступлений ПФР (как правило, без отдельной числовой матрицы на листе) |

**Единицы:** в шапках указано **«млн рублей»** с оговоркой, что **1995 год** может быть в **млрд рублей** — это критично для ETL, иначе порядок величин «ломается».

Сюда **не** входят: **региональные** расходы на соцподдержку из консолидированных бюджетов субъектов — см. **[отдельный XLSX](/data-sources/federal/rosstat-gosfin-regional-social-support-expenditures)**; **детальные формы казначейского исполнения** — **[Казначейство](/data-sources/federal/roskazna-reports)**.

## Объекты данных и показатели

- **Строка** — **статья движения средств** (поступление всего, налоги/взносы, трансферты из федерального бюджета, расходование, в т.ч. выплаты по крупным программам, административные расходы и т.п.).
- **Столбец** — **календарный год**; для **СФР** на листе `3` часто **один столбец** с отчётным годом.
- **Сноски** поясняют разрывы методики (например, **единый социальный налог**, **возвраты субсидий** в федеральный бюджет, включение/исключение отдельных трансфертов) — их нужно читать **до** сравнения с другими источниками.
- **Связка с организационным контуром:** после создания **[СФР](/organizations/sfr)** часть исторических рядов остаётся на **ПФР/ФСС** в этом файле как **статистическая преемственность** — см. **[глоссарий: внебюджетные фонды](/glossary/extrabudgetary-funds)** и **[взносы](/glossary/social-insurance-contributions)**.

## Периодичность обновления

Файл обновляется **нерегулярно**, когда Росстат пересобирает свод; дата на витрине и в **`Содержание`** (в проверенной копии встречалась отметка **июнь 2023** — она **не** означает, что все листы ограничены этим годом: часть листов уже содержит **2023** в шапке). После крупных институциональных реформ возможны **новые листы** или **перенумерация** — ориентируйтесь на **оглавление внутри книги**.

## Ограничения и особенности

- **Формат XLS:** для Python обычно нужен **`xlrd` 1.2.x**; следите за **типами** (год в ячейке может быть float `2023.0`).
- **Смешение единиц 1995 года** с «млн руб.» для остальных лет — обязательная **нормализация** при построении длинных рядов.
- **Текстовые числа с пробелами** (`'120 960'`) и пустые строки встречаются в старых фрагментах таблиц — потребуется **очистка**.
- **Не касса:** значения **агрегированы** по методике статистического наблюдения и отчётности фондов; для сверки с **бюджетом СФР в ГИИС** используйте **[наборы «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)** и **[казначейские формы](/data-sources/federal/roskazna-reports)**.

## Частые вопросы

**В:** Почему для 2023 года есть отдельный лист СФР, а ПФР/ФСС всё ещё ведутся длинными рядами?

**О:** Это отражает **переходный период** учёта и публикаций: **СФР** показан как **новая институциональная рамка**, а **исторические ряды** ПФР/ФСС сохраняют **сопоставимость** «до/после» для аналитики длинных горизонтов.

**В:** Можно ли из этой книги напрямую построить помесячный cash-flow фонда?

**О:** **Нет:** шаг агрегации — **год** (кроме отдельных справочных блоков). Для внутригодовой динамики ищите **отчётность фонда/Казначейства** или **специализированные наборы** в ГИИС.
