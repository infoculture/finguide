Коротко: **веб-приложение Show Data** на **`showdata.rosstat.gov.ru`** — поиск и подбор **официальных статистических показателей** в логике витрины Росстата. Для **госфинансов** это **ещё один вход** к макро- и региональным рядам (в т.ч. блокам про финансы и экономику), **не** замена **[казначейских](/data-sources/federal/roskazna-reports)** и **[бюджетных наборов ГИИС](/data-sources/federal/budget-gov-ru-datasets)** без сверки методик.

## Описание

Карточка описывает публичный URL **`/finder/`** сервиса **Show Data** Росстата: одностраничное приложение (**Backbone-стек**, загрузка через `/backbone/init-app`) для навигации по каталогу показателей и выгрузок. По смыслу это **смежный канал** с **[fedstat.ru (ЕМИСС)](/data-sources/federal/rosstat-opendata)** и **[открытыми данными на rosstat.gov.ru](/data-sources/federal/rosstat-opendata)**: данные относятся к **официальной статистике**, но точка входа, фильтры и форматы экспорта **свои**.

Сюда **не** входят: массовые стабильные URL файлов как в **`/storage/mediabank/`** без прохода через UI; **API** в смысле документированных endpoint’ов для машинного опроса (ориентир — **[ЕМИСС](/information-systems/federal/emiss)** и **[каталог ОД](https://rosstat.gov.ru/opendata/)**).

## Оператор

**Федеральная служба государственной статистики (Росстат)** — https://rosstat.gov.ru/

- Карточка ИС: **[ЕМИСС](/information-systems/federal/emiss)** — инфраструктура официальной статистики; Show Data — **публичная витрина-поиск** в том же контуре предметных данных, с отдельным интерфейсом.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Finder (поиск и подбор показателей) | https://showdata.rosstat.gov.ru/finder/ | HTML (SPA); экспорт из UI — по опциям выбранного продукта |
| Корень поддомена Show Data | https://showdata.rosstat.gov.ru/ | HTML; может перенаправлять или вести к обзорным разделам |

## Объекты данных и показатели

- **Единица выборки** — как правило **показатель официальной статистики** с разрезами по территории, периоду и классификаторам Росстата (конкретный набор полей задаётся в интерфейсе после подбора таблицы).
- **Связка с бюджетом** — через **код субъекта**, **ОКТМО** (где применимо) и **согласованные периоды**; прямого поля **КБК** в рядах Росстата обычно **нет**.
- **Пробелы для автоматизации:** зависимость от **сессии и cookie**, динамическая подгрузка контента; для регулярных пайплайнов надёжнее опираться на **паспорта наборов ОД** и прямые файловые URL из **[карточки ЕМИСС и ОД](/data-sources/federal/rosstat-opendata)**, если там дублируется тот же показатель.

## Периодичность обновления

Совпадает с **календарём публикаций** и **версиями таблиц** соответствующих продуктов официальной статистики; детали по конкретному показателю — в **метаданных витрины** после выбора набора или в **[ЕМИСС](/information-systems/federal/emiss)** (разделы про публикации и актуальность).

## Ограничения и особенности

- **Условия использования** — как у прочих публичных витрин Росстата; при выгрузке сверяйте сопроводительный текст и паспорт, если показатель дублируется в **[открытых данных](https://rosstat.gov.ru/opendata/)**.
- **Автоматизация:** сайт ориентирован на **человека в браузере**; массовый скрейп без официального API может упираться в сессии, изменения JS и структуры запросов.
- **Методика:** показатели Show Data **не тождественны** строкам **[ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** и **[налоговой статистики ФНС](/data-sources/federal/nalog-statistics)** без явной сверки определений.

## Примеры доступа

Проверка доступности точки входа (HTTP-код главной страницы finder):

```bash
curl -sSk -o /dev/null -w "%{http_code}\n" "https://showdata.rosstat.gov.ru/finder/"
```

## Не путать с

- **`fedstat.ru` как каталог показателей** — другой интерфейс и URL-пространство; наборы могут **пересекаться**, но стабильные **паспорта ОД** и типовые выгрузки чаще документируются в **[общей карточке Росстата](/data-sources/federal/rosstat-opendata)**.
