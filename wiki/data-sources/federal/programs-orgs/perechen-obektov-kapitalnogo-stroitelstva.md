# Перечень объектов капитального строительства (PDF, Правительство РФ)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный; **единичный PDF-файл** с перечнем объектов капитального строительства, опубликованный на **статическом хостинге** сайта Правительства РФ (`static.government.ru`) |
| Тип доступа | Файл по прямой ссылке (**без** каталога наборов и **без** API) |
| Основные форматы | PDF (таблично-текстовая вёрстка; для машинной обработки обычно нужен **разбор PDF** или ручной ввод) |
| Оператор | **[Правительство Российской Федерации](/organizations/pravgov)** — публикация в составе материалов на `government.ru` (реквизиты конкретной редакции перечня и сопроводительный контекст сверяйте по **титульному листу PDF** и разделу документов на дату анализа) |
| Режим доступа | Открытая загрузка по URL |

Коротко: **официальный перечень наименований и реквизитов объектов** (и связанных с ними сведений в зависимости от редакции документа) в форме **одного PDF** — удобен для **справочной привязки** к тематике **[национальных проектов](/glossary/national-projects)**, госпрограмм и обсуждению **капитальных вложений**; для **рядов план/факт и кассы по КБК** опирайтесь на **[наборы «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)** и **[открытые данные ГАСУ](/data-sources/federal/gasu-programs)**.

## Описание

Карточка фиксирует **прямую точку доступа** к PDF-версии перечня объектов капитального строительства на инфраструктуре публикации файлов Правительства РФ. Содержание и структура таблиц **задаются редакцией конкретного документа** (объекты, ответственные органы, этапы, объёмы финансирования и т. п. — если присутствуют в тексте).

Сюда **не** входят: машиночитаемые выгрузки с тем же составом полей, **реестры недвижимости**, **ГИС ЖКХ**, внутренние кабинеты ведомств и **юридически значимое опубликование НПА** (для актов — **[ОППИ](https://pravo.gov.ru/)** и официальные тексты постановлений).

## Оператор

Публикация файла — в контуре официального сайта **[Правительства Российской Федерации](/organizations/pravgov)**; см. таблицу URL ниже.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Прямая ссылка на PDF (как опубликовано на `static.government.ru`) | http://static.government.ru/media/files/cT5FNGvfdcak3rNqFz148Rfl3T4oLTWm.pdf | PDF |
| Тот же ресурс по HTTPS (если доступен в вашей среде) | https://static.government.ru/media/files/cT5FNGvfdcak3rNqFz148Rfl3T4oLTWm.pdf | PDF |

Имя файла в пути — **технический идентификатор** выгрузки; при обновлении перечня Правительством РФ **URL может смениться**. Для актуальной редакции имеет смысл искать материал в **[разделе документов](https://government.ru/docs/)** на `government.ru` по теме перечня объектов капитального строительства.

## Объекты данных и показатели

- **Строка перечня** — как правило объект капитального строительства с полями, заданными таблицей в PDF (наименование, место, ответственный орган, сроки, объёмы — **по факту колонок в файле**).
- **Связка с бюджетными и программными рядами** — через **наименование объекта**, **ведомство**, **нацпроект/госпрограмму** (если указаны в документе) и **[КБК](/glossary/kbk)** в других источниках; **сквозного стабильного ID** объекта в публичных бюджетных наборах **может не быть**.
- **Пробелы для автоматизации:** нет JSON/CSV, возможны **сканы**, сложная вёрстка таблиц, **разрывы строк** при копировании из PDF.

## Периодичность обновления

Обновление **не регламентировано** на уровне этого URL: выходит **новая редакция перечня** по решению Правительства РФ; после замены документа старый файл может остаться по старой ссылке или быть заменён — сверяйте **дату утверждения** в PDF.

## Ограничения и особенности

- **Один файл** — нет метаданных каталога открытых данных (автор, дата обновления на стороне сервера) в машиночитаемом виде рядом с файлом.
- **Сетевой доступ:** в отдельных средах возможны таймауты или политики TLS; при сбое HTTPS пробуйте HTTP-ссылку из таблицы выше.
- Содержание **не заменяет** **[исполнение бюджета](/data-sources/federal/budget-gov-ru-datasets)** и **[отчётность казначейства](/data-sources/federal/roskazna-reports)** — только справочный перечень объектов в редакции документа.

## Не путать с

- **[ЕИС в сфере закупок](/information-systems/federal/zakupki)** и **[XML-выгрузки закупок](/data-sources/federal/zakupki-xml)** — контракты и извещения по **закупкам** на стройку; это не автоматически тот же перечень, что в PDF Правительства РФ.
- **Реестры и учёт объектов капитального строительства** в профильных ГИС (например, разрешения, учёт в строительстве) — **иные** юридические контуры и идентификаторы.
