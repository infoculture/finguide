# programs.economy.gov.ru — веб-клиент АИС ГП (госпрограммы РФ)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный **рабочий** веб-интерфейс **аналитической информационной системы** обеспечения открытости деятельности федеральных органов исполнительной власти в рамках **государственных программ Российской Федерации** (в публичных материалах часто фигурирует как **АИС ГП**); оператор — **Минэкономразвития России** |
| Тип доступа | **Веб-приложение** (одностраничное; загрузка через `bundle-*.js`); машиночитаемые **файловые каталоги** для широкой публики ведутся на **[programs.gov.ru](/information-systems/federal/programs-gov)** |
| Основные форматы | **HTML** (интерфейс в браузере); отдельный хост **аутентификации**: `https://auth-programs.economy.gov.ru/` (см. `env-config.js` в корне сайта) |
| Оператор | Минэкономразвития России |
| Режим доступа | **Ограниченный** для полноценной работы: рабочие функции предполагают **учётную запись** участника процесса; корневая страница отдаётся без явной captcha, но это **не** замена публичного каталога открытых данных |

Коротко: **точка входа в служебный/деловой контур** той же предметной области, что и публичный **[programs.gov.ru](/information-systems/federal/programs-gov)**. Для **аналитики открытых данных** без доступа к кабинетам начинайте с **[открытых данных портала госпрограмм](https://programs.gov.ru/Portal/site/opendata)** и обзора **[госпрограммы и ФЦП](/data-sources/federal/fcp)**; этот URL полезен как **канонический адрес веб-клиента** АИС ГП и для понимания границы между публичной витриной и рабочим интерфейсом.

## Описание

Сайт `programs.economy.gov.ru` отдаёт оболочку **SPA** с заголовком страницы **«АИС ГП»** и подключает конфигурацию окружения (`/env-config.js`), где задаётся базовый URL сервиса аутентификации. По смыслу это **витрина рабочих функций** системы управления и аналитики по госпрограммам на стороне Минэкономразвития (согласование, отчётность, внутренние срезы — в редакции релиза на дату обращения), а не дублирующий каталог **CSV/XML** для анонимной массовой выгрузки.

Сюда **не** входят: **паспорта и открытые наборы для гражданина без регистрации** (это **[programs.gov.ru](/information-systems/federal/programs-gov)**), **мониторинг ГАСУ** (**[gasu.gov.ru/opendata](https://gasu.gov.ru/opendata)** — **[другой оператор и модель полей](/data-sources/federal/gasu-programs)**), **архив ФЦП** (**[fcp.economy.gov.ru](/information-systems/federal/sistema-fcp)**).

## Оператор

**Министерство экономического развития Российской Федерации** — https://programs.economy.gov.ru/

- Карточка ИС: **[programs.gov.ru — Государственные программы РФ](/information-systems/federal/programs-gov)** — публичный портал реестра и открытых данных в той же ведомственной линии; граница с этой карточкой: **публичные файлы и паспорта** — на `programs.gov.ru`, **рабочий веб-клиент** — на поддомене `programs.economy.gov.ru`.

## URL и точки доступа

| Тип доступа | URL | Формат |

| Сервис аутентификации (из конфигурации фронтенда) | https://auth-programs.economy.gov.ru/auth/ | перенаправления/HTML в контуре входа |

## Объекты данных и показатели

Для внешнего наблюдателя без учётной записи **состав объектов недоступен** так же, как на публичном API: ожидайте интерфейсные сущности уровня **программа**, **паспорт**, **показатели**, **планы-графики**, **отчётность** — в терминологии ведомства и в связке с **[ГРБС](/glossary/grbs)** и **[КБК](/glossary/kbk)** в смежных учётных контурах.

- **Публичная склейка** с бюджетом и закупками по-прежнему опирается на **[programs.gov.ru](/information-systems/federal/programs-gov)**, **[наборы «Электронного бюджета»](/data-sources/federal/budget-gov-ru-datasets)**, **[ЕИС](/data-sources/federal/procurement)** и **[ГАСУ](/data-sources/federal/gasu-programs)** — см. **[обзор по госпрограммам](/data-sources/federal/fcp)**.
- **Пробел для автоматизации:** не рассчитывайте на анонимные стабильные **REST/Open Data** endpoint’ы именно на `programs.economy.gov.ru`; проверяйте наличие выгрузок в **[каталоге programs.gov.ru](https://programs.gov.ru/Portal/site/opendata)**.

## Периодичность обновления

Задаётся **релизами веб-приложения** и регламентами ведомства; единого публичного регламента обновления «всех таблиц» на этом хосте для внешнего пользователя нет. Для периодичности **открытых файлов** реестра см. раздел **«Ограничения и актуальность»** в **[programs.gov.ru](/information-systems/federal/programs-gov)**.

## Ограничения и особенности

- **403/блокировки** при обращении некоторых автоматических клиентов (в т.ч. без типичного `User-Agent`) возможны; для проверки доступности используйте обычный браузер или HTTP-клиент с корректным заголовком.
- **CSP** сайта указывает разрешённые источники скриптов и **`auth-programs.economy.gov.ru`** — автоматизация должна учитывать политику безопасности браузера при попытках встраивания.
- Семантика полей в **внутренних** экранах **не обязана** совпадать с полями **[открытых наборов ГАСУ](/data-sources/federal/gasu-programs)** или **[казначейских форм](/data-sources/federal/roskazna-reports)**.

## Примеры доступа

Проверка, что корень отвечает (диагностика сети, не извлечение содержимого паспортов):

```bash
curl -sI -A "Mozilla/5.0" "https://programs.economy.gov.ru/"
```

Ожидаемый признак — код **200** и `Content-Type: text/html` (на дату проверки в репозитории).

## Частые вопросы

**В:** Где скачать CSV по госпрограммам без логина?

**О:** В **[каталоге открытых данных programs.gov.ru](https://programs.gov.ru/Portal/site/opendata)**; хост **programs.economy.gov.ru** рассчитан на **вход в веб-клиент** АИС ГП, а не на замену этого каталога.

**В:** Это «официальнее», чем programs.gov.ru?

**О:** Оба адреса относятся к ведомственной линии Минэкономразвития; для **цитируемых публичных выгрузок и паспортов** в открытых финансах обычно указывают **`programs.gov.ru`**, а **programs.economy.gov.ru** фиксируют, когда речь о **рабочем интерфейсе** или интеграции в контуре с аутентификацией.
