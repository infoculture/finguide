Коротко: публичный **мониторинг по организациям высшего образования**, где наряду с контингентом, направлениями подготовки и показателями деятельности раскрываются **финансовые сведения** (в том числе доходы/расходы и смежные показатели по методике мониторинга) — удобный **отраслевой** срез для сопоставления вузов и регионов, дополняющий **бухгалтерскую отчётность** на **[bus.gov.ru](/data-sources/federal/bus-opendata)**.

## Описание

Карточка описывает раздел **«Мониторинг (ВО)»** федеральной системы мониторинга на сайте **monitoring.miccedu.ru**: это витрина сводных показателей по **образовательным организациям высшего образования**, где пользователь выбирает территорию, организацию или иной разрез и получает табличные данные. Для анализа **бюджетных потоков и учётной отчётности** по формам казначейства мониторинг **не заменяет** **[открытые данные bus.gov.ru](/data-sources/federal/bus-opendata)** и **[своды Казначейства](/data-sources/federal/roskazna-federal-institutions-execution-summary)** — это разные методики и уровни агрегирования.

Сюда **не** входят: полный контур **[ЕИС Закупки](/information-systems/federal/zakupki)** по контрактам; **[ГИС «Наука»](/information-systems/federal/gisnauka)** по НИОКТР (научный контур **Минобрнауки**); внутренние отчёты учреждений, доступные только в закрытых кабинетах.

## Оператор

**Министерство просвещения Российской Федерации** — отраслевой заказчик мониторинга образования; канонический вход к описываемому разделу: https://monitoring.miccedu.ru/?m=vpo

## Объекты данных и показатели

Заполняйте выборку по актуальному интерфейсу — перечень вкладок и полей **меняется** вместе с регламентами мониторинга.

- **Образовательная организация высшего образования** — строка мониторинга: реквизиты и показатели по выбранному периоду/отчётной форме; для стыковки с другими источниками используйте **ИНН**, **ОГРН**, код региона.
- **Финансовые показатели** — отраслевые своды мониторинга (доходы, расходы, отдельные статьи или агрегаты по методике Минпросвещения); **не** путать с **[КОСГУ](/glossary/kosgu)** в отчётности учреждения и с **[казначейским исполнением](/glossary/treasury-execution)**.
- **Территориальные и отраслевые срезы** — агрегаты по субъекту РФ, типам организаций и направлениям; удобно для сравнения организаций и регионов «в одной таблице мониторинга».

Известные **пробелы**: методика мониторинга **не обязана** совпадать с формами **[плана ФХД](/reporting/pfhd)** и **[бухотчётности учреждений](/reporting/institution-financial-statements)**; при расхождении сумм сверяйте первичный источник по назначению (мониторинг против отчётности на bus).

## Периодичность обновления

Обновление показателей задаётся **регламентами отраслевого мониторинса** и календарём сбора данных по конкретным формам; на витрине ориентируйтесь на **дату среза** или год в интерфейсе выбора периода. Единого публичного «частота обновления одной строкой» в машиночитаемом паспорте набора здесь нет — уточняйте по экрану отчёта и методическим документам Минпросвещения на дату анализа.

## Ограничения и особенности

- Доступ **через браузер** и экспорт из UI; для регулярного ETL закладывайте хрупкость вёрстки и параметров запросов (`m=vpo` и др.).
- **Семантика показателя** берётся из справки мониторинга и расшифровок на странице, а не из предположений бухгалтерского учёта.
- Сопоставление с **федеральными бюджетными учреждениями** удобно дополнять **[сводом Казначейства по федеральным учреждениям](/data-sources/federal/roskazna-federal-institutions-execution-summary)** — иной уровень и методика, но тот же контур «казначейские учреждения».

## Примеры доступа

Проверка доступности страницы раздела ВО без авторизации:

```bash
curl -sS -o /dev/null -w "%{http_code}\n" "https://monitoring.miccedu.ru/?m=vpo"
```

Ожидаемый ответ при работе сервиса: код **200**.

## Не путать с

- **[Открытые данные bus.gov.ru](/data-sources/federal/bus-opendata)** — регламентированная отчётность учреждения и массовые выгрузки по паспортам наборов; **miccedu** даёт **мониторинговый** срез по правилам отраслевой статистики и показателей качества/ресурсов.
- **[Наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** — консолидированный бюджетный контур и реестры **бюджетного процесса**, а не показатели отдельного вуза в разрезе мониторинга образования.
