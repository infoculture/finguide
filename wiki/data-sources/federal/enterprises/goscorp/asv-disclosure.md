## Описание

Карточка — **навигатор по публичным витринам** ГК «Агентство по страхованию вкладов» (**АСВ**) для аналитики **банковской стабильности**: **реестр банков** — участников системы страхования вкладов (*asv.org.ru/banks*), **реестр НПФ** (*pension-funds*), **годовая отчётность** на **[e-disclosure](/data-sources/federal/e-disclosure-ru)** (id=38451), **закупки** в **[ЕИС](/data-sources/federal/procurement)**. АСВ проводит **санации** и **конкурсное** производство при отзыве лицензий у банков, выплачивает **страховое возмещение** вкладчикам.

Для **надзорной** статистики по банкам используйте **[Банк России](/organizations/cbr)**; АСВ — **оператор** механизма **страхования вкладов**, а не макропруденциальный регулятор.

Правовой статус: [АСВ](/organizations/asv).

## Об организации

| Признак | Значение |
| :--- | :--- |
| Полное наименование | Государственная корпорация «Агентство по страхованию вкладов» |
| Правовая форма | **Государственная корпорация** (с 2004 г., ФЗ «О страховании вкладов») |
| Отрасль | **Финансовая стабильность**: **страхование вкладов** физлиц и ИП, **санация** и **ликвидация** банков, управление **конкурсной** массой, **НПФ** в периметре гарантий (где применимо) |
| Госучастие | **Государственная** корпорация; финансирование через **взносы** банков — участников системы и **бюджетные** механизмы при крупных санациях |
| Функции | **Выплаты** вкладчикам при отзыве лицензии (в пределах **страховой суммы**), **временная администрация**, продажа активов, **аукционы** «покупателя» для обязательств |
| Публичные реестры | Список **банков**-участников и **НПФ** на asv.org.ru |
| ИНН | 7708514824 |
| Регуляторы (контекст) | **[Банк России](/organizations/cbr)** (лицензирование и отзыв лицензий), **[Минфин](/organizations/minfin)** (законодательная база) |

**АСВ** — важный **индикатор** стресса банковского сектора: рост **санаций** и **выплат** отражает волны **отзывов лицензий**. Для аналитики: **отчётность** АСВ на **e-disclosure**, **реестр** участников, **закупки** по ИНН **7708514824**. **Не** путать с **[Банком России](/organizations/cbr)** (надзор и **макростатистика** по кредитам) — у ЦБ иные витрины и периметр данных.

## Объекты данных и показатели

- **Реестр банков** — участники системы страхования вкладов; *asv.org.ru/banks*.
- **Реестр НПФ** — *asv.org.ru/pension-funds*.
- **Годовая отчётность** — баланс, расходы на санации, резервы — **e-disclosure** (id=38451).
- **Закупки** — **[ЕИС](/data-sources/federal/procurement)** по ИНН **7708514824** (агрегаторы Контур — вторично).
- **Нормативная база** — ФЗ «О страховании вкладов» (Consultant, base.garant.ru).
- **Связь с ЦБ** — отзывы лицензий и статистика на **[cbr.ru](https://www.cbr.ru/)** дополняют, но **не заменяют** отчётность АСВ.

## Периодичность обновления

**Событийная** и **календарная** (годовая отчётность, раскрытия, закупки); точные сроки — в разделах отчётности на сайте оператора и на [e-disclosure.ru](/data-sources/federal/e-disclosure-ru).

## Ограничения и особенности

- Официальный сайт asv.org.ru испытывает проблемы с доступом, что затруднило сбор информации. Использованы альтернативные надежные источники, где это было возможно. Разделы 'Корпоративное управление' и 'Устойчивое развитие' не имеют прямых официальных ссылок, но упоминаются в новостных статьях и на сторонних ресурсах.
- Строки бюджетной отчётности и **ПФХД** учреждений **не** продолжают корпоративный баланс этой организации.
- Часть URL в исходной таблице могла быть недоступна при автоматической проверке (5/12).
