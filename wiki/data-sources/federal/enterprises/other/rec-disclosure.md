## Описание

Карточка — **навигатор по раскрытию** **АО «Российский экспортный центр»** (**РЭЦ**, создан по **ПП РФ № 938** от 17.09.2015) для аналитики **государственной поддержки экспорта**: **страхование** и **финансирование** (в связке с **[Росэксимбанком](/data-sources/federal/roseximbank-disclosure)** и **[ЭКСАР](/data-sources/federal/exiar-disclosure)**), **продвижение** продукции, **обучение** экспортёров, **аналитика** рынков (*services/analitika-i-issledovaniya*).

На **exportcenter.ru** опубликованы **архивные** PDF бухгалтерской отчётности (**2015–2018**), **положения о закупках**, материалы **устойчивого развития** группы; актуальные годовые отчёты — проверять на **[e-disclosure](/data-sources/federal/e-disclosure-ru)** и в разделах **нормативных актов** (постановления Правительства в *upload/documents*).

Правовой статус: [РЭЦ](/organizations/rec).

## Об организации

| Признак | Значение |
| :--- | :--- |
| Полное наименование | Акционерное общество «Российский экспортный центр» (РЭЦ) |
| Правовая форма | **АО**; **институт развития** **экспорта** |
| Отрасль | **Несырьевой экспорт**: консультации, **маркетинг**, **страхование** (через партнёров), **образование**, **аналитика** |
| Госучастие | **Контроль** РФ; координация с **[Минэкономразвития](https://www.economy.gov.ru/)**, **[Минпромторгом](https://minpromtorg.gov.ru/)**, **[Минфином](/organizations/minfin)** |
| Экосистема | **[Росэксимбанк](/data-sources/federal/roseximbank-disclosure)** (кредиты и гарантии), **[ЭКСАР](/data-sources/federal/exiar-disclosure)** (страхование экспортных кредитов) |
| ИНН | 7703370230 |
| Регуляторы (контекст) | **[Минэкономразвития](https://www.economy.gov.ru/)** (экспортная политика), **[ЦБ РФ](/organizations/cbr)** (банковские партнёры) |

**РЭЦ** — **операционный хаб** «одного окна» для экспортёров; для аналитики: **объём услуг**, **закупки** (консалтинг, IT, мероприятия), **субсидии** из бюджета. **Не** путать с **[Росэксимбанком](/data-sources/federal/roseximbank-disclosure)** (баланс **банка**) и **[таможенной](/organizations/fts)** статистикой.

## Ограничения и особенности

- Информация собрана с официального сайта АО "Российский экспортный центр". Некоторые категории, такие как государственные реестры и профильные госорганы, были найдены через внутренний поиск по сайту, что может потребовать дополнительной проверки релевантности ссылок. Раздел для инвесторов не найден.
- Строки бюджетной отчётности и **ПФХД** учреждений **не** продолжают корпоративный баланс этой организации.
- Часть URL в исходной таблице могла быть недоступна при автоматической проверке (18/24).
