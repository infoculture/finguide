# ПАО «Россети»: корпоративное раскрытие

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | ПАО «Россети» и группа в зависимости от документа |
| Тип доступа | IR-сайт, годовые отчёты и документы эмитента |
| Основные форматы | HTML, PDF, XLSX |
| Оператор | ПАО «Россети» |
| Режим доступа | Открытый |

Коротко: официальный маршрут к структуре владения, РСБУ/МСФО, годовым отчётам и решениям акционеров. На дату проверки официальное годовое раскрытие подтверждает контроль Российской Федерации; долю перепроверяйте по последнему отчёту.

## Описание

Используйте годовой отчёт для структуры капитала и группы, МСФО — для консолидации, РСБУ — для головного общества, материалы собраний — для дивидендов. Закупки дочерних компаний ищутся отдельно.

## Объекты данных и показатели

- структура капитала и доля Российской Федерации;
- РСБУ/МСФО, выручка, прибыль, долг и инвестиции;
- дивиденды и решения собрания;
- закупки головного общества и компаний группы.

## Периодичность обновления

Годовая и промежуточная отчётность — по корпоративному календарю; закупки — по событиям.

## Ограничения и особенности

- периметр МСФО шире головного общества;
- ограничение раскрытия отдельных сведений не означает нулевое значение;
- закупки группы требуют списка юридических лиц и ИНН.
