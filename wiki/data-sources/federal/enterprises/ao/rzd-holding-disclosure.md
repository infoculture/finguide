# РЖД (корпоративный портал): раскрытие для инвесторов и отчётность

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральное **ОАО**; **монополист** инфраструктуры **железных дорог** общего пользования; **100%** в собственности РФ (типовая модель) |
| Тип доступа | Веб-портал с разделами для **инвесторов** и **акционеров**; файлы отчётности по ссылкам |
| Основные форматы | HTML, PDF, XLSX |
| Оператор | ОАО «Российские железные дороги» |
| Режим доступа | Открытый доступ к опубликованным материалам |

Коротко: **официальный корпоративный** слой **РЖД** для **отчётности** и **раскрытий** — стыкуется с **закупками** в **[ЕИС](/data-sources/federal/procurement)** и **структурой** группы в **[ЕГРЮЛ](/information-systems/federal/egrul)**; **не** источник **консолидированного бюджета** и **не** замена **[ГИИС](/information-systems/federal/giis-eb)**.

## Описание

Карточка задаёт **канонический вход** к **корпоративному раскрытию** ОАО «РЖД» на **company.rzd.ru**: годовой отчёт, разделы **«Инвесторам»** и **«Акционерам»**, **закупки**, **ESG**. Числовые **URL** внутри CMS **меняются** — ориентируйтесь на меню портала, а не на закладки. Для **бюджетных** трансфертов, **субсидий** на инфраструктуру и **тарифов** — **[субсидии](/data-sources/federal/subsidy)**, **[ФАС](/organizations/fas)** и **[ЕИС](/data-sources/federal/procurement)** по ИНН **головной** компании и **дочерних** обществ.

Правовой статус: [РЖД](/organizations/rzd).

## Об организации

| Признак | Значение |
| :--- | :--- |
| Полное наименование | Акционерное общество «Российские железные дороги» |
| Правовая форма | **ОАО** (не ПАО, но сопоставимое **корпоративное** раскрытие); **единственный акционер** — государство |
| Отрасль | **Железнодорожный транспорт**: **инфраструктура** (пути, станции, энергетика, связь), **грузовые** и **пассажирские** перевозки через **дочерние** компании, **логистика**, **инжиниринг** |
| Госучастие | **Стратегическое предприятие** с **полным госвладением**; инвестиционная программа согласуется с **транспортной политикой** и **бюджетными** мерами поддержки |
| Группа | **Холдинг РЖД**: головная **ОАО «РЖД»** + **ФПК** (пассажиры), **РЖД Логистика**, **РЖД-Технологии** и др.; консолидированная отчётность — **группа** |
| Тарифы и субсидии | **Регулируемые** тарифы (грузовая инфраструктура); **субсидии** и **госзадание** на отдельные **социально значимые** перевозки — сверка с **[субсидиями](/data-sources/federal/subsidy)** |
| ИНН (головная ОАО) | 7708503727 |
| Регуляторы (контекст) | **Росжелдор** (отраслевое управление), **[ФАС](/organizations/fas)** (тарифы), **[ЦБ](/organizations/cbr)** (облигации и раскрытие эмитента) |

**РЖД** — крупнейший **инфраструктурный** холдинг РФ. Для аналитики открытых данных: **корпоративные** PDF на **company.rzd.ru**, **закупки** — разделы **9352/9395** портала и **[ЕИС](/data-sources/federal/procurement)**; **не** путать с **pass.rzd.ru** / **cargo.rzd.ru** (операционные сервисы, не финансовая отчётность).

## Объекты данных и показатели

- **Годовой отчёт** — консолидированные показатели **группы** (выручка, EBITDA, инвестиционная программа); разделы **9388** / *Инвесторам*.
- **Грузопоток и инфраструктура** — операционные KPI в годовом отчёте и презентациях (тонно-км, погрузка).
- **Корпоративные события** — календарь раскрытий; **e-disclosure** (id=4543).
- **Закупки** — корпоративный портал (**9352**, **9395**) + **[ЕИС](/data-sources/federal/procurement)** по ИНН **7708503727** и дочерних.
- **ESG** — **9386**, отчёт **sr2024.rzd.ru**.
- **Связка с бюджетом** — **субсидии**, **госпрограммы**, **налоги** по **[ИНН](/glossary/inn)** дочерних обществ; **не** строки **федерального бюджета** как таковые.

## Периодичность обновления

**Календарная** (отчётность, собрания); при **смене** CMS портала проверяйте **новые** URL вручную.

## Ограничения и особенности

- **URL** внутренних страниц могут содержать **числовые идентификаторы** — хрупкие для долгосрочных скриптов без проверки.
- **Не** путать с **пассажирскими** и **грузовыми** операционными порталами (`pass.rzd.ru`, `cargo.rzd.ru`) — у них **другой** предмет данных.
