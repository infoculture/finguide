# ПАО «Газпром»: раскрытие информации для инвесторов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральное **ПАО**; **вертикально интегрированный газовый холдинг** с **контрольным пакетом РФ**; листинг на **MOEX** и иностранных площадках (исторически) |
| Тип доступа | Веб-разделы на **gazprom.ru**; **файлы** отчётности и презентаций по ссылкам с сайта |
| Основные форматы | HTML, PDF, XLSX |
| Оператор | ПАО «Газпром» |
| Режим доступа | Открытый доступ к опубликованным материалам; подписка на рассылки — по правилам сайта |

Коротко: **официальная витрина** «Газпрома» для **инвесторов** — годовая и промежуточная **отчётность**, презентации и календарь событий; полезно сопоставлять с **закупками** в **[ЕИС](/data-sources/federal/procurement)** и **агрегатами** бюджета, не подменяя **бюджетные** формы **[ГИИС](/information-systems/federal/giis-eb)**.

## Описание

Карточка фиксирует **канонический источник корпоративного раскрытия** ПАО «Газпром» — раздел **gazprom.ru/investors/** и связанные поддомены (**tenders.gazprom.ru**, **sustainability.gazpromreport.ru**). Для аналитики **открытых финансовых данных** сюда относят **отчётность эмитента**, презентации, **закупочный** след и **ESG**; **исполнение федерального бюджета** и **казначейские формы** — в **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** и **[отчётности Казначейства](/reporting/treasury-reports)**.

URL и имена файлов **меняются по годам** — при автоматизации сверяйте актуальную структуру сайта. Правовой статус: [Газпром](/organizations/gazprom).

## Об организации

| Признак | Значение |
| :--- | :--- |
| Полное наименование | Публичное акционерное общество «Газпром» |
| Правовая форма | **ПАО**; публичная компания с **доминирующим госучастием** |
| Отрасль | **Газовая отрасль**: геологоразведка и **добыча**, **транспорт** (магистральные газопроводы), **переработка**, **сбыт** в РФ и на экспорт, энергетические и сервисные активы группы |
| Госучастие | **Контрольный пакет акций** принадлежит Российской Федерации (раскрывается в годовом отчёте и на **e-disclosure**); ключевые решения согласуются с государственной политикой в газовой отрасли |
| Группа | Консолидированный периметр включает **дочерние и зависимые общества** (добыча, ГТС, региональные газовые компании, нефтегазовые и сервисные активы); в отчётности — **сегменты** по видам деятельности |
| Роль в публичных финансах | Крупнейший **налогоплательщик** и источник **дивидендов** в бюджет; **инвестиционная программа** и **тарифы** (в т.ч. регулируемые) влияют на макропоказатели, но **не** являются строками **федерального бюджета** |
| ИНН | 7736050003 — связка с **[ЕГРЮЛ](/information-systems/federal/egrul)** и **[ЕИС](/data-sources/federal/procurement)** |
| Регуляторы (контекст) | **[ЦБ РФ](/organizations/cbr)** (рынок бумаг), **[ФАС](/organizations/fas)** (тарифы транспорта газа), отраслевое регулирование в рамках **госполитики** |

**Газпром** — эталонный пример **ПАО с госучастием** для wiki: корпоративные показатели публикуются на **собственной IR-витрине** и дублируются на **[e-disclosure.ru](/data-sources/federal/e-disclosure-ru)**; бюджетный след ищите через **дивиденды**, **налоги**, **[субсидии](/data-sources/federal/subsidy)** и **контракты** в **[ЕИС](/data-sources/federal/procurement)** по ИНН группы.

## Объекты данных и показатели

- **Годовой / промежуточный отчёт (МСФО / РСБУ)** — выручка, EBITDA, капитальные затраты, **добыча и транспортировка газа**, долговой портфель; PDF/XLSX в *investors/disclosure/reports/*.
- **Операционные и отраслевые показатели** — объёмы добычи, прокачки, экспортные потоки (в презентациях и годовом отчёте).
- **Презентации для инвесторов** — не заменяют аудированную отчётность; не путать с **[отчётностью по госпрограммам](/reporting/state-program-reporting)**.
- **Закупки** — порталы **tenders.gazprom.ru**, **gazpromzakupki.ru** и зеркало в **[zakupki.gov.ru](/data-sources/federal/procurement)**.
- **ESG / устойчивое развитие** — **gazprom.ru/sustainability** и отдельный отчётный сайт **sustainability.gazpromreport.ru**.
- **Связка с реестрами** — **[ЕГРЮЛ](/information-systems/federal/egrul)** (структура группы), **[e-disclosure](/data-sources/federal/e-disclosure-ru)** (id=934).

## Периодичность обновления

**Событийная** и **календарная** (отчётность, собрания, раскрытия); точные даты — в разделе **календаря** на сайте эмитента.

## Ограничения и особенности

- **Не** источник **казначейского** исполнения бюджета и **не** замена **[отчётности учреждений](/reporting/institution-financial-statements)**.
- **Сверяйте** показатели с **официальным** текстом отчёта и методикой эмитента; wiki не дублирует **МСФО/РСБУ**-формулы.
