# Реестры ЕИС: контракты, извещения, РНП и независимые гарантии

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | ЕИС по правовым режимам и составу конкретного реестра |
| Тип доступа | Публичный поиск и машиночитаемые комплекты |
| Основные форматы | HTML, XML, ZIP |
| Оператор | Федеральное казначейство |
| Режим доступа | Открытый просмотр; массовый доступ по регламенту СОИ |

Коротко: четыре реестра отвечают на разные вопросы. Извещение описывает процедуру, контракт — обязательство сторон, РНП — решение о включении поставщика, независимая гарантия — способ обеспечения.

## Описание

| Маршрут | Что искать | Ключ связи |
| --- | --- | --- |
| Реестр контрактов | цена, стороны, исполнение, расторжение | реестровый номер контракта |
| Извещения и планы-графики | потребность, процедура, лоты и сроки | номер закупки |
| РНП | решение, заказчик, поставщик и срок включения | реестровый номер записи, ИНН |
| Независимые гарантии | гарант, принципал, сумма и срок | номер гарантии, номер закупки/контракта |

## Объекты данных и показатели

- закупка, лот, извещение, план-график;
- контракт, заказчик, поставщик, цена и исполнение;
- запись РНП и основание включения;
- независимая гарантия и обеспечиваемое обязательство.

## Периодичность обновления

Документы поступают по событиям участников; массовые комплекты следуют регламенту оператора.

## Ограничения и особенности

- связь между реестрами может быть один-ко-многим;
- отсутствие поставщика в РНП не подтверждает исполнение конкретного контракта;
- сумма гарантии не равна цене контракта;
- 44-ФЗ и 223-ФЗ обрабатываются раздельно.
