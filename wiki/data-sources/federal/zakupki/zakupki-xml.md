# XML-выгрузки ЕИС Закупки

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный; **полные машиночитаемые комплекты** документов **[ЕИС в сфере закупок](/information-systems/federal/zakupki)** по **44-ФЗ** и **223-ФЗ**, исторически разбитые по регионам и типам файлов |
| Тип доступа | **Смешанный**: актуальная выдача массовых комплектов через **сервисы отдачи информации (СОИ)** по регламенту оператора; **FTP** (`ftp://ftp.zakupki.gov.ru/`) с **01.07.2024** прекращён — только контекст для старых пайплайнов |
| Основные форматы | **XML** внутри **ZIP**-архивов; описание схем и методические материалы на портале — **PDF**/HTML |
| Оператор | Федеральное казначейство |
| Режим доступа | Публичная витрина и база документов на `zakupki.gov.ru` открыты; **условия доступа к СОИ** (учётная запись, лимиты, конечные URL) задаются **актуальной** инструкцией оператора |

Коротко: **сырой контрактный след** закупок — извещения, протоколы, контракты, планы-графики, реестровые сущности — в виде **XML**, чтобы построить собственный регистр контрактов и связать заказчиков и поставщиков по **ИНН/ОГРН** с **[ЕГРЮЛ](/information-systems/federal/egrul)** и бюджетными контурами.

## Описание

Карточка описывает **содержание и логику** массовых выгрузок ЕИС: те же правовые объекты, что на веб-портале, но в **нормализованных XML** по утверждённым схемам. Выгрузки рассчитаны на **регулярное обновление** (исторически — ежедневные архивы по FTP); после перехода на **СОИ** конкретные расписания, объёмы и способы аутентификации нужно брать из **текущей** документации на сайте ЕИС.

Сюда **не** входят интеграционный контур **загрузки** документов в ЕИС для ЭТП и заказчиков (**`/eis-integration/`**), а также **агрегированные** витрины исполнения бюджета — для лимитов и кассы см. **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** и **[витрины Казначейства](/data-sources/federal/roskazna-datamarts)**.

## Оператор

**Федеральное казначейство** — канонический узел документации и новостей оператора: https://zakupki.gov.ru/epz/main/public/document/view.html

- Карточка ИС: **[ЕИС в сфере закупок](/information-systems/federal/zakupki)** — там же таблица **точек доступа** (портал, СОИ, интеграция), раздел про **прекращение FTP** и ограничения выгрузок.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Документация, схемы, регламенты СОИ | https://zakupki.gov.ru/epz/main/public/document/view.html | PDF, HTML |
| Новость оператора о переходе с FTP на СОИ | https://zakupki.gov.ru/epz/main/public/news/news_preview.html?newsId=34208 | HTML |
| Публичная витрина (поиск без массовой выгрузки) | https://zakupki.gov.ru/epz/main/public/home.html | HTML |
| FTP (не использовать как рабочий канал с 01.07.2024) | ftp://ftp.zakupki.gov.ru/ | XML/ZIP (исторически) |

Актуальные **базовые URL сервисов СОИ**, правила регистрации и лимиты запросов в таблицу не выносятся: они меняются вместе с регламентом — ориентир на **[карточку ЕИС](/information-systems/federal/zakupki)** и базу знаний на портале.

## Объекты данных и показатели

- **Документы закупки** — извещения, заявки, протоколы (набор зависит от способа закупки и стадии).
- **Контракты** и связанные сведения по **44-ФЗ** и **223-ФЗ** — цена, предмет, сроки, стороны; для склейки с учётными системами полезны **ИНН**, **ОГРН**, идентификаторы заказчика и контракта в логике схемы.
- **Планы-графики закупок** — плановый контур; не путать с фактическим исполнением бюджета.
- **Реестровые и справочные сущности** — по актуальному перечню типов в выгрузке (в т.ч. сведения реестров в составе комплектов).

Типичные **ключи связи**: реестровый номер контракта/извещения, версии документов, коды **ОКПД2** в предмете, **КБК** там, где предусмотрено схемой. **44-ФЗ** и **223-ФЗ** в одном потоке — перед агрегацией нужен **отбор по признакам правового режима** (см. **[закупки и контракты](/data-sources/federal/procurement)** и термины **[44-ФЗ](/glossary/procurement-44fz)** / **[223-ФЗ](/glossary/procurement-223fz)**).

## Периодичность обновления

Исторически полные комплекты обновлялись **ежедневно** (FTP-архивы по регионам и типам документов). После перевода на **СОИ** периодичность и задержки относительно публикации на витрине задаются **регламентом отдачи** — см. разделы **«Точки доступа»** и **«Ограничения и актуальность»** в **[карточке ЕИС](/information-systems/federal/zakupki)**.

## Ограничения и особенности

- **FTP недоступен** как официальный канал с **01.07.2024**; инструкции и скрипты, заточенные только под `ftp.zakupki.gov.ru`, нужно переносить на **СОИ**.
- **Объёмы и схемы** меняются — перед продакшеном сверяйте **актуальные XSD/описания** на портале оператора.
- **Лицензирование**: открытые данные закупок обычно допускают переработку с указанием источника; уточняйте формулировки на витрине оператора для вашего сценария.
- **Смешение режимов**: без явного разделения **44-ФЗ** и **223-ФЗ** агрегаты по «всем XML ЕИС» легко трактуются неверно.
