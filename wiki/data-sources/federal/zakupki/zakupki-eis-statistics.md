# ЕИС Закупки: статистика на главной (госзакупки и контракты)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный; **агрегированные** показатели по закупкам и контрактам, размещаемым в **[ЕИС](/information-systems/federal/zakupki)**, с разделением **44-ФЗ** и **223-ФЗ** |
| Тип доступа | **Смешанный**: интерактивный блок на главной странице; **JSON** для графиков и карты (относительно `contextPath` портала); отдельный файл **методики** в **PDF** |
| Основные форматы | **HTML** (оболочка и вкладки), **JSON** (серии для слайдера и карты), **PDF** (методика расчёта) |
| Оператор | Федеральное казначейство |
| Режим доступа | Публичный просмотр; для **HTTP-запросов к JSON** на практике может потребоваться типовой **User-Agent** и заголовок **Referer** с главной страницы портала |

Коротко: **официальная сводная статистика ЕИС** на главной странице: экономия, планирование, извещения, контракты, способы закупки и оперативные показатели за текущий день — с **картой по регионам** и переключением **44-ФЗ** / **223-ФЗ**. Это **не** замена **[построчных данных и XML-выгрузок](/data-sources/federal/zakupki-xml)**, а слой **обзорной аналитики** и сверки с **методикой** оператора.

## Описание

Блок **«Статистика»** (`#statAnchor`) на **https://zakupki.gov.ru/epz/main/public/home.html** содержит:

- вкладку **«Общая»** — кнопки режима **44-ФЗ** / **223-ФЗ** и срезы **«Экономия»**, **«Планирование»**, **«Извещения»**, **«Контракты»**; отображение **«Карта»** или **«Цифры»** (соответствуют параметрам запросов к JSON);
- вкладку **«Способы закупки»** — распределение по способам (данные также подгружаются с сервера);
- вкладку **«Показатели за сегодня»** — оперативные счётчики по ЕИС;
- ссылку **«Методика расчета показателей»** на PDF с официальным описанием показателей.

Интерактив реализован скриптом портала (`statistics.js`): значения **не** зашиты в статический HTML, а приходят **после загрузки страницы**.

## Оператор

**Федеральное казначейство** — оператор **[ЕИС в сфере закупок](/information-systems/federal/zakupki)**; канонический вход на главную: https://zakupki.gov.ru/epz/main/public/home.html

- Карточка ИС: **[ЕИС в сфере закупок](/information-systems/federal/zakupki)** — границы системы, **СОИ** и машиночитаемые комплекты документов.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Главная страница, якорь раздела «Статистика» | https://zakupki.gov.ru/epz/main/public/home.html#statAnchor | HTML |
| Методика расчёта показателей (файл с портала) | https://zakupki.gov.ru/epz/main/public/documents/Calculation_methods.pdf | PDF |
| JSON для блока «Общая» (пример среза; `mode` и `tabType` совпадают с переключателями в UI) | `https://zakupki.gov.ru/epz/main/public/{mode}/{tabType}.json` с параметрами запроса (см. пример ниже) | JSON |

На странице задаётся `contextPath = "/epz/main/public"` — JSON-запросы строятся относительно этого префикса (как в клиентском коде портала).

## Объекты данных и показатели

- **Агрегаты по годам и регионам** (карта, легенда, слайдер «Цифры»): объёмы и доли в логике **[методики PDF](https://zakupki.gov.ru/epz/main/public/documents/Calculation_methods.pdf)**; идентификаторы срезов в JSON (например, ключи вида `44{год}`) нужно сопоставлять с подписями в ответе и в интерфейсе.
- **Способы закупки** — сводные счётчики/объёмы по способам для выбранного правового режима и параметров фильтра.
- **Показатели за сегодня** — краткие оперативные величины по состоянию на момент обновления сервера.

Типичные ключи для связи с микроданными: **код региона (КЛАДР/ФИАС)** в параметрах карты, разделение **FZ44** / **FZ223** (`placeOfSearch` в запросах), **год** отчёта. Прямой **реестровый номер контракта** в этих JSON обычно **не** является строкой набора — для контрактного следа используйте **[ЕИС и XML](/data-sources/federal/zakupki-xml)**.

## Периодичность обновления

**«Показатели за сегодня»** и часть общих рядов обновляются **в течение дня** (на клиенте может быть таймер обновления). Исторические годы в **«Общей»** статистике появляются по мере **закрытия отчётного периода** и пересчёта на стороне оператора — точные сроки смотрите в **методике** и по датам в JSON (`updateDateTime` в ответах вкладки «за сегодня»).

## Ограничения и особенности

- **Не документированный как публичный API** контракт JSON: пути, набор параметров (`placeOfSearch`, `year`, `kladr`, `contractStatus` и др.) и ответы могут меняться с обновлением портала; для воспроизводимых пайплайнов приоритет — **[XML СОИ](/data-sources/federal/zakupki-xml)**.
- Запросы к JSON без типичного **User-Agent** или без **Referer** с главной страницы иногда приводят к **ошибкам шлюза**; ориентируйтесь на поведение, близкое к браузеру.
- **44-ФЗ** и **223-ФЗ** в одном блоке — смешивать ряды без фильтра режима нельзя (см. **[44-ФЗ](/glossary/procurement-44fz)** и **[223-ФЗ](/glossary/procurement-223fz)**).
- Цифры блока — **витринная аналитика** ЕИС; сверка с **[агрегатами Казначейства](/data-sources/federal/roskazna-datamarts)** или **[отчётами Минфина](/data-sources/federal/minfin-purchases-monitoring)** требует учёта разных методик.

## Примеры доступа

Скачать **методику** показателей:

```bash
curl -sS -A "Mozilla/5.0" -L -o Calculation_methods.pdf \
  "https://zakupki.gov.ru/epz/main/public/documents/Calculation_methods.pdf"
```

Пример **JSON** для вкладки «Общая», режим «Цифры», срез «Экономия», с фильтром **44-ФЗ** (заголовок **Referer** снижает риск отказа шлюза):

```bash
curl -sS -A "Mozilla/5.0" \
  -e "https://zakupki.gov.ru/epz/main/public/home.html" \
  "https://zakupki.gov.ru/epz/main/public/all-stats/common-economy.json?placeOfSearch=FZ44"
```
