# Минфин: мониторинг закупок (отчёты)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный уровень: **аналитические отчёты** по результатам **мониторинга закупок** для **[государственных и муниципальных нужд](/glossary/procurement-44fz)** и закупок **[отдельных видов юридических лиц](/glossary/procurement-223fz)** (формулировка как на странице ведомства) |
| Тип доступа | **Файлы по ссылке**: HTML-страница раздела «Контрактная система» со списком документов и кнопкой «Скачать» для каждого **PDF** |
| Основные форматы | **PDF**; обёртка — HTML |
| Оператор | Минфин России |
| Режим доступа | Открытый (типовое скачивание без регистрации с официальной страницы) |

Коротко: **официальные сводные отчёты Минфина** по мониторингу сферы закупок (ежеквартально и **сводный годовой** аналитический отчёт) — источник **агрегированной методологии и выводов** ведомства; для **построчного контура закупок**, извещений и контрактов используйте **[ЕИС](/data-sources/federal/procurement)** и **[XML-выгрузки](/data-sources/federal/zakupki-xml)**.

## Описание

Карточка описывает витрину **[Мониторинг закупок](https://minfin.gov.ru/ru/perfomance/contracts/purchases/)**: на странице перечислены документы с датами публикации и ссылками на вложения. По состоянию на актуальные публикации на сайте доступны:

- **ежеквартальные отчёты** по результатам мониторинга закупок (I–IV квартал соответствующего года);
- **сводный аналитический отчёт** за календарный год (публикуется после окончания года, обычно весной следующего года).

Сюда **не** входят **[открытые наборы Минфина](/data-sources/federal/minfin-opendata)** из каталога с `list.csv` — раздел **perfomance/contracts/purchases** — отдельная **отчётная** подборка на сайте ведомства, без обязательного включения в машинный реестр открытых данных.

## Оператор

**Минфин России** — канонический URL раздела: https://minfin.gov.ru/ru/perfomance/contracts/purchases/

- Карточка ИС: **[Сайт Министерства финансов РФ](/information-systems/federal/minfinsite)** — публикация материалов раздела «Деятельность → Контрактная система» и вложений к отчётам.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Страница раздела (список отчётов, «Скачать» для каждого файла) | https://minfin.gov.ru/ru/perfomance/contracts/purchases/ | HTML; вложения **PDF** |
| Родительский раздел «Контрактная система» (навигация по смежным материалам) | https://minfin.gov.ru/ru/perfomance/contracts/ | HTML |

В адресе раздела используется сегмент **`perfomance`** (орфография сайта); при ручном наборе URL не заменяйте его на `performance`.

Прямые URL файлов **PDF** при обновлении вложений **меняются**; для автоматизации ориентируйтесь на **актуальную ссылку «Скачать»** в карточке документа на странице раздела.

## Объекты данных и показатели

Содержание задаётся **опубликованными PDF**; типично это:

- **обобщённые показатели и выводы** мониторинга (структура разделов и таблиц меняется от выпуска к выпуску);
- **правовая и процедурная** рамка **44-ФЗ** и **223-ФЗ** в изложении отчёта;
- ссылки на **исходные массивы** или методику — по тексту конкретного отчёта.

Типичные ключи для связки с другими источниками: **ИНН**, **ОГРН**, коды **ОКПД2**, территориальные коды — **если** они приведены в таблицах отчёта; это **не** замена реестру документов **[ЕИС](/data-sources/federal/procurement)**.

**Пробелы**: нет стабильного **CSV/XML/API**; для машинной обработки потребуется извлечение из **PDF** или опора на **исходные выгрузки ЕИС**.

## Периодичность обновления

Новые отчёты появляются **ежеквартально** (после окончания квартала, с задержкой на подготовку) и **ежегодно** (свод за год). Точные даты публикации смотрите в списке документов на **[странице раздела](https://minfin.gov.ru/ru/perfomance/contracts/purchases/)** — они указаны в карточках «Опубликовано / Изменено».

## Ограничения и особенности

- Материалы официального сайта Минфина распространяются на условиях **Creative Commons Attribution 4.0** (блок в подвале страниц `minfin.gov.ru`); при переработке указывайте источник.
- Некоторые HTTP-клиенты без типового **User-Agent** могут получать ответ **503**; при автоматической загрузке используйте корректный заголовок запроса (как в примере ниже).
- Цифры и интерпретации отчёта **не обязаны** построчно сходиться с **[агрегатами по закупкам](/data-sources/federal/roskazna-datamarts)** или с **[сырым ЕИС](/data-sources/federal/zakupki-xml)** — сверяйте методику в тексте PDF и дату выпуска.

## Примеры доступа

Сохранить HTML-обёртку страницы раздела (для парсинга списка документов и ссылок «Скачать»):

```bash
curl -sS -A "Mozilla/5.0" -o minfin-purchases-monitoring.html \
  "https://minfin.gov.ru/ru/perfomance/contracts/purchases/"
```

## Связанные страницы wiki

- **[Сведения о закупках и контрактах (ЕИС)](/data-sources/federal/procurement)** — основной **транзакционный** источник извещений, контрактов и смежных документов по **44-ФЗ** и **223-ФЗ**.
- **[ЕИС Закупки: статистика на главной](/data-sources/federal/zakupki-eis-statistics)** — **официальные своды** оператора ЕИС на главной портала (в т.ч. JSON для графиков); другая методика, чем у PDF Минфина.
- **[XML-выгрузки ЕИС Закупки](/data-sources/federal/zakupki-xml)** — машиночитаемые комплекты для воспроизводимого контура закупок без опоры только на PDF Минфина.
- **[ЕИС в сфере закупок](/information-systems/federal/zakupki)** — оператор, каналы доступа и ограничения официальной ИС.
- **[Отчётность по контрактам](/reporting/procurement-contract-reporting)** — регламентированные формы, смежные с реестрами контрактов и планами закупок.
- **[Открытые данные Минфина России](/data-sources/federal/minfin-opendata)** — каталог CSV/XML-наборов; отчёты мониторинга закупок в него **не обязаны** входить тем же составом.

## Не путать с

- **[Сведения о закупках и контрактах (ЕИС)](/data-sources/federal/procurement)** и **[XML-выгрузки](/data-sources/federal/zakupki-xml)** — **оперативные и массовые** данные закупок; страница Минфина даёт **ведомственный аналитический отчёт**, а не полный реестр процедур.
- **[Витрины данных Казначейства (datamarts)](/data-sources/federal/roskazna-datamarts)** — иные **агрегированные** срезы по закупкам от оператора **ЕИС**; пересечение по смыслу возможно, но форматы и методики различаются.

## Частые вопросы

**В:** Можно ли по этим отчётам восстановить все контракты заказчика?  
**О:** **Нет** — это **сводная отчётность** Минфина. Для полноты по контрактам и процедурам используйте **[ЕИС](/data-sources/federal/procurement)** и выгрузки по **[XML](/data-sources/federal/zakupki-xml)**.

**В:** Почему в URL «perfomance», а не «performance»?  
**О:** Так **задано на официальном сайте** Минфина; при копировании ссылок сохраняйте фактическое написание сегмента пути.
