# ПАО «Газпром»: раскрытие информации для инвесторов

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Публичное акционерное общество** с существенным участием государства; **корпоративное** раскрытие для инвесторов и широкой публики по правилам эмитента и регулятора рынка ценных бумаг |
| Тип доступа | Веб-разделы на **gazprom.ru**; **файлы** отчётности и презентаций по ссылкам с сайта |
| Основные форматы | HTML, PDF, XLSX |
| Оператор | ПАО «Газпром» |
| Режим доступа | Открытый доступ к опубликованным материалам; подписка на рассылки — по правилам сайта |

Коротко: **официальная витрина** «Газпрома» для **инвесторов** — годовая и промежуточная **отчётность**, презентации и календарь событий; полезно сопоставлять с **закупками** в **[ЕИС](/data-sources/federal/procurement)** и **агрегатами** бюджета, не подменяя **бюджетные** формы **[ГИИС](/information-systems/federal/giis-eb)**.

## Описание

Карточка фиксирует **источник корпоративного раскрытия** крупнейшей газовой компании с **госучастием**: где искать **официальные** PDF/XLSX и структуру разделов без копирования **ИС** или **НПА** в wiki. Для **машинного** скачивания URL и имена файлов меняются по годам и редакциям — автоматизируйте с **проверкой** актуальной структуры сайта.

## Оператор

**ПАО «Газпром»** — https://www.gazprom.ru/investors/

## Объекты данных и показатели

- **Годовой / промежуточный отчёт** — консолидированные показатели, пояснения к отчётности, сегменты.
- **Презентации и справочные материалы** — для сопоставления с **рыночными** ожиданиями; не путать с **[отчётностью по госпрограммам](/reporting/state-program-reporting)**.
- **Связка с открытыми госданными** — через **[ЕИС](/data-sources/federal/procurement)** (контрактный след), **[ЕГРЮЛ](/information-systems/federal/egrul)** (структура) и при необходимости **[e-disclosure](/data-sources/federal/e-disclosure-ru)**.

## Периодичность обновления

**Событийная** и **календарная** (отчётность, собрания, раскрытия); точные даты — в разделе **календаря** на сайте эмитента.

## Ограничения и особенности

- **Не** источник **казначейского** исполнения бюджета и **не** замена **[отчётности учреждений](/reporting/institution-financial-statements)**.
- **Сверяйте** показатели с **официальным** текстом отчёта и методикой эмитента; wiki не дублирует **МСФО/РСБУ**-формулы.
