# РЖД (корпоративный портал): раскрытие для инвесторов и отчётность

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | **Федеральный** оператор железнодорожной инфраструктуры; **корпоративное** раскрытие для инвесторов и широкой публики на **company.rzd.ru** и связанных поддоменах |
| Тип доступа | Веб-портал с разделами для **инвесторов** и **акционеров**; файлы отчётности по ссылкам |
| Основные форматы | HTML, PDF, XLSX |
| Оператор | ОАО «Российские железные дороги» |
| Режим доступа | Открытый доступ к опубликованным материалам |

Коротко: **официальный корпоративный** слой **РЖД** для **отчётности** и **раскрытий** — стыкуется с **закупками** в **[ЕИС](/data-sources/federal/procurement)** и **структурой** группы в **[ЕГРЮЛ](/information-systems/federal/egrul)**; **не** источник **консолидированного бюджета** и **не** замена **[ГИИС](/information-systems/federal/giis-eb)**.

## Описание

Карточка задаёт **канонический вход** к **раскрытию** холдинга на **company.rzd.ru** (структура меню и **числовые** пути страниц могут обновляться — сверяйте **актуальные** URL в интерфейсе «Инвесторам» / «Акционерам»). Для **бюджетных** трансфертов и **субсидий** используйте **[субсидии](/data-sources/federal/subsidy)** и **[госзадание](/data-sources/federal/govtasks)** в связке с **[ЕИС](/data-sources/federal/procurement)**.

## Оператор

**ОАО «Российские железные дороги»** — корпоративный портал https://company.rzd.ru/

## URL и точки доступа

| Корпоративный портал | https://company.rzd.ru/ | HTML |
| Пример страницы раскрытия (на момент оформления карточки) | https://company.rzd.ru/ru/9353/page/105104?id=1733 | HTML |

## Объекты данных и показатели

- **Годовой отчёт и финансовые вложения** — консолидированные показатели группы (см. актуальные файлы на портале).
- **Корпоративные события** — календарь раскрытий, корпоративное управление.
- **Связка с госфинансами** — через **закупки**, **субсидии** и **налоговую** статистику по **[ИНН](/glossary/inn)** дочерних обществ.

## Периодичность обновления

**Календарная** (отчётность, собрания); при **смене** CMS портала проверяйте **новые** URL вручную.

## Ограничения и особенности

- **URL** внутренних страниц могут содержать **числовые идентификаторы** — хрупкие для долгосрочных скриптов без проверки.
- **Не** путать с **пассажирскими** и **грузовыми** операционными порталами (`pass.rzd.ru`, `cargo.rzd.ru`) — у них **другой** предмет данных.

## Примеры доступа

```bash
curl -sS -o /dev/null -w "%{http_code}\n" -A "Mozilla/5.0" "https://company.rzd.ru/"
```
