# Статистика Банка России

## Описание

Открытые данные ЦБ РФ: денежно-кредитная политика, банковская статистика, платёжный баланс, финансовая отчётность кредитных организаций (XBRL).

## Оператор

Банк России — https://cbr.ru/statistics/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| База данных ЦБ | https://cbr.ru/hd_base/ | XLS/CSV/XML |
| API курсов | https://www.cbr-xml-daily.ru/ | JSON/XML |

## Состав данных

- Денежные агрегаты
- Платёжный баланс
- Банковская отчётность XBRL
- Курсы валют

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).

## Ограничения и особенности

- Лицензия: открытые данные (как правило, разрешена свободная переработка с указанием источника).
- Объёмы и форматы могут меняться; рекомендуется отслеживать страницу источника.
