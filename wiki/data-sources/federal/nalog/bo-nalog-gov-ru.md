# ГИР БО — публичная бухгалтерская (финансовая) отчётность организаций (bo.nalog.gov.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный; сведения по организациям, представившим отчётность в ФНС (состав публикуемых форм и исключения — по правилам раскрытия на стороне ФНС) |
| Тип доступа | Веб-портал с расширенным поиском и просмотром материалов отчётности |
| Основные форматы | HTML-интерфейс; типичные вложения отчётности — **PDF** и др. по факту публикации |
| Оператор | [ФНС России](/organizations/fns) |
| Режим доступа | Публичный просмотр опубликованных сведений; подача отчётности — отдельный контур (ЭДО, КЭП), не описывается в этой карточке |

Коротко: **bo.nalog.gov.ru** — государственный информационный ресурс **бухгалтерской отчётности** (ГИР БО): поиск организаций по реквизитам и отбор по отчётному периоду и **ОКВЭД**, затем просмотр размещённой отчётности. Для аналитики открытых финансовых данных это слой **микроданных по юрлицам** (не сводные таблицы налоговой статистики), пригодный для сопоставления с **ИНН**/**ОГРН** из [закупок](/data-sources/federal/procurement) и смежных реестров.

## Описание

На главной странице заявлено, что ресурс сформирован на основании информации, представленной составителями отчётности. Пользователь задаёт критерии (наименование, адрес, **ИНН**, **ОГРН**, отчётный год, тип периода — квартал / полугодие / девять месяцев, **ОКВЭД**) и получает выдачу со ссылками на опубликованные материалы.

В объём этой карточки **не** входят: личные кабинеты налогоплательщика, технические регламенты электронной сдачи отчётности и внутренние АИС ФНС.

## Оператор

[Федеральная налоговая служба](/organizations/fns) — публичная витрина: https://bo.nalog.gov.ru/

- Карточка ИС: [Сайт ФНС России — статистика и данные](/information-systems/federal/fnssite) — общий контур публикаций ФНС; ГИР БО — отдельный поддомен под бухгалтерскую отчётность организаций.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Публичный поиск и просмотр отчётности | https://bo.nalog.gov.ru/ | HTML; вложения отчётности чаще **PDF** |

Публичного **документированного API** витрины для массовой выгрузки «как каталог наборов» на момент описания карточки не заявлен; сценарий массового сбора потребует либо ручной работы через UI, либо оценки допустимости и устойчивости автоматизации с учётом правил сайта и нагрузочных ограничений.

## Объекты данных и показатели

- **Единица поиска** — организация (идентификация через **ИНН**, **ОГРН**, наименование, адрес) в связке с **отчётным периодом** и, при использовании фильтра, **ОКВЭД**.
- **Содержательный объект** — комплект форм **бухгалтерской (финансовой) отчётности** и сопутствующих документов, которые ФНС публикует в рамках ГИР БО (перечень форм и обязательность раскрытия зависят от категории организации и законодательства на дату отчётного периода).
- **Связка** с закупками и субсидиями — через **ИНН**/**ОГРН** контрагента; с котируемыми компаниями — дополнительно с витринами раскрытия эмитентов (см. [e-disclosure.ru](/data-sources/federal/e-disclosure-ru)).
- **Пробелы:** состав исключений из обязательной публикации и полнота архива по годам нужно сверять с актуальными разъяснениями ФНС; часть организаций может раскрываться иными каналами (например учреждения — через [bus.gov.ru](/data-sources/federal/bus-opendata)).

## Периодичность обновления

Данные появляются и обновляются **по мере представления и обработки** отчётности организациями и ФНС; частота не выравнивается по календарю как у типовой налоговой статистики. Общие ограничения публичных витрин ФНС см. в карточке [Сайт ФНС России](/information-systems/federal/fnssite) (разделы про стабильность URL и автоматизацию).

## Ограничения и особенности

- Назначение витрины — **оперативный поиск и просмотр**, а не единый «скачать все организации одним файлом» как у паспортов [открытых данных ФНС](https://www.nalog.gov.ru/rn77/opendata/).
- Для устойчивых пайплайнов ориентируйтесь на наборы с паспортами и прямыми файлами в каталоге opendata, если задача допускает такую подстановку по смыслу данных.
- Не смешивать показатели **бухгалтерской отчётности** с полями **агрегированной налоговой статистики** (формы 1-НМ и др.) — см. [Налоговая статистика ФНС](/data-sources/federal/nalog-statistics).

## Не путать с

- **[Открытые данные ФНС](https://www.nalog.gov.ru/rn77/opendata/)** и машиночитаемые наборы с идентификаторами `7707329152-*` (карточки в [федеральных источниках](/data-sources/federal/) — группа «Налоговая служба») — прежде всего **машиночитаемые выгрузки** по паспортам (реестры, справочники, тематические срезы), а не полный дубликат комплектов бухотчётности из ГИР БО.
- **[Налоговая статистика ФНС](/data-sources/federal/nalog-statistics)** — **агрегированные** формы налоговой отчётности и аналитика, другой предмет и типичная частота обновления.
- **[Открытые данные bus.gov.ru](/data-sources/federal/bus-opendata)** — плановые и отчётные срезы по **учреждениям** (ПФХД, госзадание, бухотчётность учреждений), не путать с коммерческими организациями в ГИР БО.
