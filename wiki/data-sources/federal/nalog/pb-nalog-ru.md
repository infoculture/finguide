# Сервис «Прозрачный бизнес» ФНС (pb.nalog.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный; сводные сведения по **юридическим лицам** и **индивидуальным предпринимателям** на основе данных, доступных ФНС для публикации |
| Тип доступа | Веб-портал с поиском, карточками, сравнением и встроенными сервисами (калькулятор, отраслевые ориентиры) |
| Основные форматы | HTML; часть сценариев может подгружать данные через внутренние запросы интерфейса без отдельного публичного каталога наборов |
| Оператор | [ФНС России](/organizations/fns) |
| Режим доступа | Публичный просмотр без регистрации для типовых сценариев поиска (условия и исключения — на стороне сервиса) |

Коротко: **pb.nalog.ru** («Прозрачный бизнес») — витрина ФНС для **агрегированной оценки налогоплательщика** и быстрой сверки реквизитов: поиск по **ИНН**/**ОГРН**/**ОГРНИП** и наименованию, карточка с показателями и справочной информацией, сопоставление с **отраслевыми нормативными показателями**, **калькулятор налоговой нагрузки**. Для аналитики открытых финансовых данных это слой **проверки контрагента** между реестрами, [закупками](/data-sources/federal/procurement) и [бухгалтерской отчётностью](/data-sources/federal/bo-nalog-gov-ru), а не замена машиночитаемых выгрузок [opendata ФНС](https://www.nalog.gov.ru/rn77/opendata/).

## Описание

Сервис объединяет в одном интерфейсе типовые задачи «посмотреть организацию целиком»: реквизиты и статусы из контуров, с которыми работает ФНС, ориентиры по отрасли, вспомогательные инструменты. На сайте также вынесены отдельные HTML-разделы (например [калькулятор налоговой нагрузки](https://pb.nalog.ru/calculator.html), [отраслевые нормативные показатели](https://pb.nalog.ru/indicator.html), [раздел про открытые данные](https://pb.nalog.ru/od.html), [сервисы оценки](https://pb.nalog.ru/grade.html)); часть пунктов меню ведёт на **внешние** адреса `nalog.gov.ru` и `service.nalog.ru` (отдельные реестры и сервисы).

В объём этой карточки **не** входят: личный кабинет налогоплательщика, АИС «Налог-3», а также сторонние коммерческие агрегаторы проверки контрагентов.

## Оператор

[Федеральная налоговая служба](/organizations/fns) — сервис: https://pb.nalog.ru/

- Карточка ИС: [Сайт ФНС России — статистика и данные](/information-systems/federal/fnssite) — общий контур публичных витрин ФНС; «Прозрачный бизнес» — отдельный поддомен под сводные карточки налогоплательщиков.

Публичного **документированного API** для массовой выгрузки карточек «как датасет» на момент описания не заявлено; для пайплайнов по срезам реестров и справочников используйте [открытые данные ФНС](https://www.nalog.gov.ru/rn77/opendata/) и карточки наборов с идентификаторами `7707329152-*` в [каталоге федеральных источников](/data-sources/federal/) (подкаталог `nalog/` в репозитории wiki).

## Объекты данных и показатели

- **Единица поиска** — организация или ИП (ключи связки: **ИНН**, **ОГРН**/**ОГРНИП**, наименование/ФИО в зависимости от типа субъекта).
- **Содержательный объект** — **сводная карточка** с показателями и блоками справочной информации, которые ФНС формирует для публичного сервиса (конкретный перечень полей и актуальность — по выдаче на сайте).
- **Связка** с закупками и субсидиями — через **ИНН**/**ОГРН** из контракта или реестра; для детализированной **бухгалтерской отчётности** по формам — переход к [ГИР БО](/data-sources/federal/bo-nalog-gov-ru) или к выпискам [ЕГРЮЛ/ЕГРИП](/information-systems/federal/egrul) по задаче.
- **Пробелы:** состав показателей и правила скрытия могут меняться; внутренние endpoint’ы страниц не следует считать контрактом для внешней интеграции.

## Периодичность обновления

Обновление карточек и показателей — **по мере поступления и обработки** сведений в учёте ФНС и смежных реестрах; единого календаря «как у годовой налоговой статистики» нет. Общие замечания по стабильности публичных витрин ФНС см. [Сайт ФНС России](/information-systems/federal/fnssite) (разделы про автоматизацию и актуальность URL).

## Ограничения и особенности

- Назначение сервиса — **оперативная проверка и ориентиры**, а не полная замена юридически значимой **выписки** из ЕГРЮЛ/ЕГРИП или комплекта форм бухгалтерской отчётности.
- Не подменять показатели карточки **агрегированными формами налоговой статистики** (1-НМ и др.) — см. [Налоговая статистика ФНС](/data-sources/federal/nalog-statistics): другой уровень агрегации и методика.
- Автоматизированный парсинг HTML возможен технически, но подвержен изменениям вёрстки, ограничениям по частоте запросов и политике сайта; для устойчивых рядов ориентируйтесь на [opendata](https://www.nalog.gov.ru/rn77/opendata/) и [XML ЕИС](/data-sources/federal/zakupki-xml) там, где задача это допускает.

## Не путать с

- **[Открытые данные ФНС](https://www.nalog.gov.ru/rn77/opendata/)** и карточки наборов `7707329152-*` — прежде всего **файлы и паспорта** для машинной обработки, а не интерактивная карточка «всё в одном окне».
- **[Налоговая статистика ФНС](/data-sources/federal/nalog-statistics)** — **региональные и отраслевые агрегаты** по формам отчётности; не дублирует построчную карточку организации в pb.nalog.ru.
