# Единый реестр субъектов МСП — публичный портал (rmsp.nalog.ru)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральный; сведения о **субъектах МСП** (юридические лица и индивидуальные предприниматели), включённых в единый реестр по правилам 209-ФЗ |
| Тип доступа | Веб-портал: поиск, статистика, экспорт выборки в **Excel**; отдельная страница **«Разработчикам»** с описанием интеграции |
| Основные форматы | HTML; табличная выгрузка поиска — **XLSX** (через кнопку экспорта в интерфейсе) |
| Оператор | [ФНС России](/organizations/fns) |
| Режим доступа | Публичный просмотр и выгрузки в объёме, разрешённом на портале |

Коротко: **rmsp.nalog.ru** — официальная **интерактивная витрина** единого реестра субъектов МСП: найти организацию или ИП по реквизитам и фильтрам, посмотреть **агрегированную статистику** по реестру (в т.ч. для нацпроекта по МСП) и выгрузить результаты поиска в Excel. Для **массовых машиночитаемых** срезов тот же реестр публикуется отдельным набором открытых данных [7707329152-rsmp](/data-sources/federal/7707329152-rsmp).

## Описание

Портал предназначен для **оперативного доступа** к сведениям реестра: простой и расширенный поиск, раздел статистики с показателями по категориям МСП, регионам, ОКВЭД и динамике, выбор **даты среза** реестра. На странице для разработчиков размещаются материалы по подключению (в т.ч. сценарии с обращением к серверным обработчикам портала вроде `search-proc.json` — параметры и ограничения нужно сверять с актуальной документацией на сайте).

В объём карточки **не** входят: учётные входы для внутренних ролей (если предусмотрены), администрирование реестра и **иной** реестр — получатели мер поддержки МСП на `rmsp-pp.nalog.ru` (см. [карточку ИС](/information-systems/federal/rmsp-pp-nalog) и набор [7707329152-rsmppp](/data-sources/federal/7707329152-rsmppp)).

## Объекты данных и показатели

- **Единица реестра** — субъект МСП (ЮЛ или ИП): наименование, категория (микро / малое / среднее), регион, даты включения и исключения из реестра, при поиске — иные поля в соответствии с формой выдачи.
- **Агрегаты портала** — количество субъектов и работников по выбранным фильтрам, разрезы по ОКВЭД и динамика по срезам дат (интерфейс «Статистика»).
- **Связка** с бюджетными и закупочными рядами — через **ИНН**, **ОГРН/ОГРНИП**, регион и **ОКВЭД**, если они доступны в выдаче; для полноты полей в машиночитаемом виде ориентируйтесь на выгрузку [7707329152-rsmp](/data-sources/federal/7707329152-rsmp).

## Периодичность обновления

Сведения в реестре и на портале обновляются **по правилам ведения единого реестра МСП** (формирование срезов, приём и обработка сведений налоговыми органами). Конкретные даты срезов выбираются в интерфейсе портала. Общие замечания по стабильности публичных витрин ФНС — в карточке [Сайт ФНС России](/information-systems/federal/fnssite).

## Ограничения и особенности

- Портал ориентирован на **поиск и отчётность в UI**; массовая автоматизация должна учитывать нагрузочные ограничения, сессии и правила, изложенные для разработчиков на https://rmsp.nalog.ru/razrabotchikam.html.
- Экспорт в Excel отражает **текущую выборку** поиска и может быть ограничен по числу строк (сообщения об ограничении — в интерфейсе портала).
- Не подменять **принадлежность к МСП** из этого реестра фактами **получения мер поддержки** — для последних используйте [rmsp-pp.nalog.ru](/information-systems/federal/rmsp-pp-nalog).
