# Маркетплейс федерального имущества

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Федеральное имущество и отдельные категории реализуемого имущества |
| Тип доступа | Публичная интерактивная витрина |
| Основные форматы | HTML/JavaScript |
| Оператор | Росимущество |
| Режим доступа | Открытый просмотр; отдельные заявления требуют входа |

Коротко: источник помогает найти объект федерального имущества по категории, региону и статусу вовлечения, увидеть карточку и перейти к торгам. Это не полная выгрузка реестра активов.

## Качество и верификация

| Измерение | Значение для этой карточки |
| --- | --- |
| Дата проверки | 2026-07-16 |
| Полнота | Частичная: только публично размещённые объекты |
| Машиночитаемость | Интерфейс без подтверждённого стабильного публичного API |
| Юридическая значимость | Официальная витрина Росимущества |
| Задержка обновления | Не документирована единым правилом |
| Глубина архива | Не подтверждена для массового получения |
| Лицензия | Условия повторного использования не указаны явно |

## Описание

Витрина охватывает земельные участки, здания, помещения, транспорт, акции, древесину и иное имущество. Карточка отражает доступный публичный статус объекта; документы процедуры и её итог проверяют по ссылке в ГИС «Торги».

## Оператор

[Росимущество](/organizations/rosim); карточка ИС — [ФГИАС ЕУСГИ / «Федеральное имущество онлайн»](/information-systems/federal/federal-property-online).

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Поиск объектов | https://fiol.rosim.gov.ru/mk/ | HTML/JavaScript |
| Описание сервиса | https://fiol.rosim.gov.ru/mk/info | HTML |

## Объекты данных и показатели

- объект имущества, категория, местоположение и кадастровый номер при наличии;
- статус вовлечения, начальная цена и сроки при наличии процедуры;
- ссылка на лот и контактные сведения продавца или арендодателя.

## Периодичность обновления

Карточки меняются по событиям управления и торгов; единый публичный регламент задержки для всей витрины не найден.

## Ограничения и особенности

- результат фильтра не доказывает полноту ведомственного реестра;
- URL состояния фильтра может быть нестабилен;
- финансовый результат сделки сверяется отдельно с ГИС «Торги» и бюджетным исполнением.
