# Витрины данных Казначейства (datamarts)

## Описание

Витрины аналитических данных Федерального казначейства. Содержат исторические данные по исполнению бюджета и закупкам в разрезе показателей.

## Оператор

Федеральное казначейство — https://datamarts.roskazna.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Веб-витрины | https://datamarts.roskazna.ru/ | HTML/XLSX |
| Открытые данные | https://datamarts.roskazna.ru/opendata/ | CSV |

## Состав данных

- Витрина «Доходы»
- Витрина «Расходы»
- Витрина «Закупки»
- Витрина «Госпрограммы»

- Лицензия: открытые данные (как правило, разрешена свободная переработка с указанием источника).
