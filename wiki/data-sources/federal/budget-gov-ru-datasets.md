# Наборы данных ГИИС «Электронный бюджет»

## Описание

Открытые наборы данных портала budget.gov.ru: исполнение федерального бюджета, бюджетная роспись, реестр участников бюджетного процесса, реестры субсидий и грантов.

## Оператор

Минфин России / Федеральное казначейство — https://budget.gov.ru/

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Открытые данные портала | https://budget.gov.ru/Развитие/Открытые-данные | CSV/JSON |
| API единого портала | https://budget.gov.ru/Развитие/API | JSON |

## Состав данных

- Бюджетная роспись
- Исполнение бюджета по доходам и расходам
- Реестр соглашений о субсидиях

## Периодичность обновления

См. соответствующий раздел в [информационной системе-операторе](/information-systems/).

## Ограничения и особенности

- Лицензия: открытые данные (как правило, разрешена свободная переработка с указанием источника).
- Объёмы и форматы могут меняться; рекомендуется отслеживать страницу источника.
