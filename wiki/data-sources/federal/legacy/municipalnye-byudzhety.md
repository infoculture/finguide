# Муниципальные бюджеты (сайты муниципальных образований)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Местный уровень: проекты и отчёты по бюджетам **муниципальных образований**, опубликованные **на официальных сайтах МО** и в смежных разделах (не единая федеральная выгрузка) |
| Тип доступа | Преимущественно веб-страницы и файлы по прямым ссылкам; стабильного единого API «по всем МО» нет |
| Основные форматы | HTML, PDF, DOC/DOCX, XLS/XLSX — как правило вложения к решениям и отчётам |
| Оператор | Органы местного самоуправления и финансовые органы МО (конкретный сайт — у каждого образования свой) |
| Режим доступа | Открытый просмотр и скачивание там, где документы выложены публично |

Коротко: ориентиры для **ручного поиска** полного пакета местных бюджетных материалов, когда их недостаточно или неудобно брать только с единого портала **«Электронного бюджета»**; полезно для сверки формулировок и PDF с **ОКТМО** и записями реестров.

## Описание

Муниципальный уровень часто даёт **наиболее детальные** таблицы и пояснения в составе проекта бюджета, отчёта об исполнении и решений представительного органа. При этом публикации **распределены** по сотням официальных сайтов, структура меню не унифицирована, а полнота и актуальность выкладки различаются.

С **1 января 2020 года** в рамках требований бюджетного законодательства к раскрытию бюджетной информации сведения о муниципальных бюджетах должны дублироваться на портале **ГИИС «Электронный бюджет»** (`budget.gov.ru`). На практике аналитику всё равно полезно проверять **первоисточник на сайте МО**: там могут быть дополнительные материалы (положения, приложения, протоколы слушаний), другой порядок разделов или более свежая редакция файла.

Эта карточка **не** описывает каталог открытых наборов портала — для машиночитаемых срезов и API см. **[наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets)** и **[муниципальные бюджеты (портал)](/data-sources/regional/municipal-budgets)**.

Типичные разделы официального сайта муниципалитета (городского округа, района, сельского поселения), где стоит искать бюджетные документы:

- Бюджет / Финансы / Финансовое управление (комитет, департамент).
- Документы; Нормотворческая деятельность; Муниципальные правовые акты.
- Решения муниципального совета / совета депутатов; повестка и приложения к заседаниям.

Имеет смысл просматривать **все** разделы, связанные с **документами** и деятельностью представительного органа, а не только блок «Бюджет».

Часто публикуют (в том числе в виде вложений):

- проект бюджета и пояснительная записка;
- бюджет, утверждённый решением представительного органа;
- корректировки (изменения) бюджета;
- отчёты об исполнении бюджета;
- муниципальные (целевые, адресные) программы;
- сведения о муниципальном заказе и финансовом контроле;
- материалы публичных слушаний по бюджету.

## Оператор

**Органы местного самоуправления** соответствующего муниципального образования и подведомственные им органы (в т.ч. финансовые) — публикации размещаются на **официальном сайте МО** (URL задаётся при создании/обновлении сайта в установленном порядке; единой федеральной ссылки на «все сайты» нет).

- Карточка ИС: **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** — федеральная витрина, куда по нормам раскрытия должны попадать сведения о бюджетах МО; для **стыковки кодов и загрузки** удобнее опираться на портал и связанные наборы данных, чем только на HTML сайта МО.

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Официальный сайт конкретного МО | Задаётся для каждого образования (например через региональный справочник или поиск) | HTML, вложения PDF/DOC/XLS |
| Единый портал бюджетной системы (сведения о бюджетах МО) | https://budget.gov.ru/ | HTML, CSV/JSON в открытых данных (см. отдельные карточки источников) |

Прямой «каталог всех муниципальных сайтов» в одном URL эта карточка не фиксирует: при отсутствии ссылки на сайт МО используйте **[реестр МО / ОКТМО](/information-systems/federal/munbase)** для идентификации образования и региональные справочники.

## Объекты данных и показатели

В типичной публикации на сайте МО строкой набора является **документ** (решение, отчёт, проект) или **таблица приложения** к нему:

- плановые и фактические показатели местного бюджета в разрезе **КБК**, администраторов доходов, распорядителей и получателей средств МО;
- показатели **муниципальных программ** и адресных программ;
- сведения об исполнении в формате отчётности местного уровня (конкретные формы — по действующим нормам на дату).

Ключи связки с другими источниками: **ОКТМО** территории МО, **ИНН** участников, коды **КБК/КОСГУ** в таблицах; наименование МО в PDF сверяйте со **[справочником территорий и реестром](/information-systems/federal/munbase)**, чтобы не смешивать упразднённые и действующие образования.

Пробелы: у части сайтов нет полнотекстового поиска, устаревшие разделы, нестабильные прямые ссылки на файлы; машиночитаемость ниже, чем у выгрузок с `budget.gov.ru`.

## Периодичность обновления

Зависит от **бюджетного цикла** конкретного МО и решений представительного органа (утверждение, корректировки, квартальные и годовые отчёты). На отдельных сайтах обновление запаздывает относительно загрузки в ГИИС.

Периодичность и полнота **федеральной** витрины по данным, переданным из МО, см. в разделе **«Ограничения и актуальность»** карточки **[ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb)** — без голой отсылки «см. ИС» без указания раздела.

## Ограничения и особенности

- Нет единого формата вёрстки и именования файлов; автоматизация сбора требует **правил по каждому сайту** или ручного отбора.
- Часть материалов только в сканах или в закрытых внутренних системах; для открытого анализа опирайтесь на подтверждённые публикации и выгрузки портала.
- Для сопоставления с отчётностью и консолидированными формами субъекта РФ учитывайте методику консолидации — см. **[отчётность об исполнении местных бюджетов](/reporting/municipal-budget-execution)**.
