# Всемирный банк — данные по России

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Международная витрина; **сопоставимые** временные ряды по Российской Федерации (код страны в API: `RUS`) на фоне других стран |
| Тип доступа | Смешанный: тематические страницы по стране, каталог индикаторов и **REST API** (`api.worldbank.org`) |
| Основные форматы | HTML на витрине; API — JSON и XML; выгрузки таблиц и датасетов — CSV и др. по выбранному каналу |
| Оператор | Группа Всемирного банка (World Bank) |
| Режим доступа | Открытый; массовые запросы к API разумно **дозировать** (см. документацию WB) |

Коротко: **макроэкономика, фискальные и социальные индикаторы** в едином формате для межстрановых сравнений и **длинных рядов**. Для учёта **исполнения бюджета РФ по КБК** это не замена [федеральным открытым контурам](/data-sources/federal/minfin-opendata), но удобный слой для **налоговой нагрузки, долга, ВВП, бедности** и связанных показателей в методике WB.

## Описание

Карточка описывает публичный контур **World Bank Open Data** для России: страницу страны, выборку индикаторов и программный доступ через **версию 2 API**. В объём **не** входят внутренние проектные кабинеты WB, условия займов и закрытые рабочие материалы; для **определений институтов группы WB** см. [глоссарий](/glossary/ibrd).

## Оператор

[Группа Всемирного банка](https://www.worldbank.org/) — витрина данных: https://data.worldbank.org/country/russian-federation

## URL и точки доступа

| Тип доступа | URL | Формат |
| --- | --- | --- |
| Страница страны (обзор индикаторов) | https://data.worldbank.org/country/russian-federation | HTML; ссылки на ряды и загрузки |
| Каталог показателей | https://data.worldbank.org/indicator | HTML |
| REST API (показатели по стране) | https://api.worldbank.org/v2/country/RUS/indicator/{INDICATOR_ID} | JSON, XML (`format`-параметр) |
| Справка по API | https://datahelpdesk.worldbank.org/knowledgebase/articles/898581-api-basic-call-structures | HTML |

В запросах API для России используйте код **`RUS`** (ISO 3166-1 alpha-3); список индикаторов и метаданные — через [документацию](https://datahelpdesk.worldbank.org/knowledgebase/articles/898581-api-basic-call-structures) и страницы показателей на витрине.

## Объекты данных и показатели

- **Строка набора** в API — как правило **наблюдение по году** (или доступному периоду): идентификатор показателя, страна, дата периода, значение; в JSON ответ обёрнут в массив с блоком пагинации и массивом наблюдений.
- **Типичные темы для госфинансов** — налоги и доходы бюджета в доле ВВП, государственный долг, расходы, социальные ряды (набор зависит от наличия данных у WB для РФ и от методики).
- **Связка с российскими реестрами** — не через ЕИНН/КБК: сопоставление на уровне **определений показателей, года и методики** с [официальной статистикой](/data-sources/federal/rosstat-opendata) и [витринами Минфина](/data-sources/federal/minfin-opendata).

**Пробелы:** значения могут быть **пустыми** для части лет; методики и пересчёты базы WB могут **отличаться** от отчётности Росстата и Минфина; перед выводами сверяйте страницу индикатора и первоисточник, указанный в метаданных WB.

## Периодичность обновления

Обновление **по показателям**: при публикации и пересмотре данных у первоисточников; в ответе API в блоке метаданных часто есть поле **`lastupdated`** (дата последнего обновления серии). Типично годовые ряды с **лагом** относительно оперативной отчётности страны; для срочных решений по бюджету опирайтесь на национальные операторы, а не только на WB.

## Ограничения и особенности

- **Условия использования:** открытые данные WB обычно распространяются на условиях, близких к **CC BY 4.0** с требованием атрибуции; точные формулировки — на [странице условий](https://www.worldbank.org/en/about/legal/terms-of-use-for-datasets) на дату использования.
- **Нагрузка на API:** следуйте [рекомендациям WB](https://datahelpdesk.worldbank.org/knowledgebase/topics/125589-developer-overview) по частоте запросов и кэшированию.
- **Стабильность:** идентификаторы индикаторов (`GC.TAX.TOTL.GD.ZS` и т.д.) держатся дольше, чем вспомогательные URL веб-интерфейса; для пайплайнов предпочтительно **API + список кодов** из документации.

## Примеры доступа

Налоговые доходы (% ВВП) по России за несколько лет через API v2:

```python
import requests

url = "https://api.worldbank.org/v2/country/RUS/indicator/GC.TAX.TOTL.GD.ZS"
params = {"format": "json", "date": "2020:2024", "per_page": 500}
r = requests.get(url, params=params, timeout=30)
r.raise_for_status()
wrapper = r.json()
meta, rows = wrapper[0], wrapper[1]
for row in rows:
    print(row["date"], row["value"])
```
