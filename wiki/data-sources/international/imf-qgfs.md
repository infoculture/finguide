# IMF Quarterly Government Finance Statistics (QGFS)

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Страны и несколько периметров государственного управления: бюджетное центральное, центральное и генеральное правительство |
| Тип доступа | IMF Data Portal, интерактивная выборка, выгрузка и SDMX API |
| Основные форматы | CSV/табличная выгрузка, JSON или SDMX через API, HTML |
| Оператор | [Международный валютный фонд](/organizations/imf) (IMF) |
| Режим доступа | Просмотр открыт; документация нового API и часть автоматизации могут требовать учётную запись IMF Data |

Коротко: QGFS нужен для квартального межстранового сравнения операций и финансовой позиции правительства. **Россия присутствует в контуре передачи квартальных GFS**, однако на дату проверки публичная карточка набора не показывает надёжную первую и последнюю российскую четверть: перед анализом период нужно подтвердить в выгрузке, а не выводить из общего календаря IMF.

## Качество и верификация

| Измерение | Значение для этой карточки |

| Машиночитаемость (`machine_readability`) | Выгрузка портала и SDMX 2.1/3.0 API; контракт запроса следует брать из актуального Swagger IMF |

## Описание

QGFS содержит доходы и расходы, чистое кредитование/заимствование, операции финансирования, активы и обязательства. IMF публикует показатели для разных уровней правительства; один и тот же показатель нельзя смешивать между периметрами `budgetary central government`, `central government` и `general government`.

Официальный материал IMF по российскому участию в Data Gaps Initiative сообщает, что Федеральное казначейство готовит квартальные GFS генерального правительства. Это подтверждает наличие процесса передачи, но не доказывает непрерывность конкретной серии на портале в 2026 году.

## Объекты данных и показатели

- Наблюдение задаётся страной, институциональным периметром, показателем GFS, единицей измерения и кварталом.
- Основные блоки: доходы, расходы, чистое кредитование/заимствование, финансовые операции, активы и обязательства.
- Российские КБК и формы Казначейства не являются ключами QGFS; мост возможен только после агрегирования и документированной перекодировки в GFSM.

## Периодичность обновления

Набор квартальный, но фактическая задержка и последняя четверть различаются по странам и сериям. Для России в рабочем журнале фиксируйте период, который реально вернула выгрузка на дату анализа.

## Ограничения и особенности

- До загрузки выберите уровень правительства и способ учёта; одинаковое название показателя при разных периметрах не означает одинаковую сумму.
- Текущая публичная карточка QGFS не даёт проверяемую без выборки границу российского ряда. Значение «не подтверждено» нельзя заменять предположением о текущем квартале.
- QGFS не заменяет российские казначейские формы для юридически значимой сверки исполнения бюджета.

## Не путать с

- **IMF Fiscal Monitor** — аналитическая база с годовыми оценками и прогнозами; QGFS — квартальная статистическая публикация по GFS.
- **Оперативным исполнением бюджета РФ** — строки КБК и формы Казначейства имеют другой периметр, классификацию и статус.
