# Статкомитет СНГ — хаб данных и метаданных

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Межгосударственный (СНГ); официальная статистика и метаданные по странам участницам, **включая Российскую Федерацию**, в единой витрине Статкомитета СНГ |
| Тип доступа | Смешанный: обзорная страница **хаба данных** и три связанных веб‑контура (ЕИАС, СУЗ‑СОД, база «Статистика СНГ») |
| Основные форматы | Преимущественно HTML‑интерфейсы и публикации на портале; машиночитаемые выгрузки зависят от конкретного раздела и набора |
| Оператор | Межгосударственный статистический комитет Содружества Независимых Государств (Статкомитет СНГ) |
| Режим доступа | Открытый для гостевых страниц портала; расширенные функции на сайте Статкомитета могут опираться на **личный кабинет** и иные регламенты |

Коротко: **сопоставимые межгосударственные ряды и метаданные** по темам официальной статистики СНГ — удобный слой для **региональных и макро‑сравнений** с РФ. Для **исполнения бюджета РФ по КБК, закупок и реестров** это не замена [федеральным контурам](/data-sources/federal/minfin-opendata), но полезный контекст рядом с [национальной статистикой Росстата](/data-sources/federal/rosstat-opendata).

## Описание

Карточка описывает публичный **хаб данных и метаданных** на портале Статкомитета СНГ: точку входа, с которой разводятся три системы (ЕИАС, СУЗ‑СОД, база показателей «Статистика СНГ»). В объём **не** входят внутренние регламенты межгосударственного обмена и закрытые кабинеты участников наблюдения; конкретные форматы выгрузок и API нужно смотреть **внутри выбранной системы** после перехода по ссылкам хаба.

## Оператор

**Межгосударственный статистический комитет СНГ** — каноническая страница хаба: https://new.cisstat.org/web/guest/datahub

## Объекты данных и показатели

- **Строка набора** на стороне хаба не фиксирована одним форматом: это **витрина‑навигатор**; фактическая единица данных задаётся выбранной системой (таблица показателя, публикация, набор открытых данных и т.п.).
- **Типичные темы** — макро‑ и отраслевые показатели по странам СНГ, демография, внешняя торговля, цены и др. в методике **межгосударственной** статистики; пересечение с российской отчётностью требует сверки **определений и баз**.
- **Связка с российскими наборами** — не через ЕИНН/КБК: сопоставление по **стране, показателю, периоду и методическим пояснениям** с [ЕМИСС и открытыми данными Росстата](/data-sources/federal/rosstat-opendata) и при необходимости с [агрегатами Всемирного банка](/data-sources/international/world-bank).

**Пробелы:** полнота рядов по годам и странам **различается**; стабильность прямых URL глубоких разделов нужно проверять при обновлении портала.

## Периодичность обновления

Задаётся **календарём публикаций** Статкомитета СНГ и национальными графиками предоставления данных; на уровне страницы хаба единого расписания выгрузок нет — смотрите актуальные даты внутри **ЕИАС**, **СУЗ‑СОД** и базы **«Статистика СНГ»**.

## Ограничения и особенности

- **Условия использования** — смотрите формулировки на портале Статкомитета СНГ и в сопроводительных материалах к конкретным продуктам; для коммерческого и массового переиспользования сверяйте текст на дату обращения.
- **Автоматизация:** навигация может опираться на сессии и структуру Liferay‑портала; для устойчивых пайплайнов фиксируйте **канонический вход** и проверяйте доступность подзаголовков после релизов сайта.
- **Не дублирует национальные регистры** — операционные бюджетные и закупочные данные РФ остаются во **[федеральном контуре каталога источников](/data-sources/)** (см. группу «Федеральные источники» в сайдбаре).

## Связанные страницы wiki

- [Официальная статистика Росстата: ЕМИСС и открытые данные](/data-sources/federal/rosstat-opendata) — национальные ряды РФ для сверки методик и значений с показателями СНГ.
- [Всемирный банк — данные по России](/data-sources/international/world-bank) — другой межстрановой контур сопоставимых макропоказателей; полезно держать рядом с хабом СНГ при международных сравнениях.
- [Открытые данные Минфина России](/data-sources/federal/minfin-opendata) — бюджет и долг РФ в национальном контуре; хаб СНГ не заменяет эти наборы при задачах по исполнению бюджета.
- [Росстат: статистика внешней торговли](/data-sources/federal/rosstat-external-trade) — внешняя торговля в классификации Росстата; пересекается по смыслу с межгосударственными публикациями СНГ, но иная методическая база.

## Не путать с

- **Федеральные порталы открытых данных РФ** ([data.gov.ru](/data-sources/federal/data-gov-ru), [Минфин](/data-sources/federal/minfin-opendata)) — национальный контур законодательства и учёта РФ, не портал Статкомитета СНГ.
- **ЕМИСС Росстата** — национальная инфраструктура официальной статистики РФ; название «ЕИАС» на сайте СНГ относится к **другой** системе Статкомитета СНГ.
