# UN Data — международная статистика ООН

| Признак | Значение |
| :--- | :--- |
| Уровень и охват | Международный агрегатор: показатели по странам и темам из публикаций агентств, фондов и программ системы ООН (включение РФ зависит от конкретного источника внутри ООН) |
| Тип доступа | Веб-портал поиска и переходы к страницам агентств; часть наборов с таблицами и выгрузками на стороне первоисточника |
| Основные форматы | HTML на портале; у отдельных наборов — CSV, XLS и др. по ссылке на агентство |
| Оператор | Департамент по экономическим и социальным вопросам ООН (UN DESA), витрина **UN Data** |
| Режим доступа | Открытый просмотр и поиск; условия переиспользования задаёт **первоисточник** набора, а не только заголовок на UN Data |

Коротко: **UN Data** — точка входа к множеству международных рядов; для российских бюджетных кодов, форм исполнения и административных регистров по-прежнему используйте [отчётность](/reporting/) и [федеральные источники](/data-sources/federal/).

## Описание

Карточка описывает публичный портал **https://data.un.org/** как каталог и поиск по темам (население, торговля, устойчивое развитие, социальные показатели и др.). UN Data **не** является единой машиночитаемой БД с общим API по всем рядам: фактическая структура набора, метаданные и лицензия обычно раскрываются на сайте **соответствующего** органа ООН после перехода по ссылке.

## Оператор

[Департамент ООН по экономическим и социальным вопросам](https://www.un.org/development/desa/) — витрина UN Data: https://data.un.org/

| --- | --- | --- |

| Каталог открытых данных ООН (перекрёстная витрина) | https://data.un.org/Explorer.aspx | HTML |

## Объекты данных и показатели

- **Единица публикации** на стороне UN Data — как правило **ссылка на тематическую таблицу или страницу агентства**; состав полей и периодичность задаются первоисточником.
- **Связка с российскими бюджетными данными** — не через КБК/КОСГУ: сопоставление на уровне **определения показателя, страны, года и методики** с [официальной статистикой Росстата](/data-sources/federal/rosstat-opendata) и [витринами Минфина](/data-sources/federal/minfin-opendata).
- **Типичные пробелы:** разные агентства используют разные классификации; часть рядов с **лагом** или с пересмотром задним числом; URL глубоких разделов на сайтах агентств могут меняться чаще, чем корень `data.un.org`.

## Периодичность обновления

Обновления **не единые по порталу**: расписание и дата последней ревизии указаны в карточке конкретного набора или на сайте агентства-оператора. Для оперативного мониторинга исполнения бюджета РФ ориентируйтесь на национальные источники и их регламенты публикации.

## Ограничения и особенности

- **Атрибуция и лицензия** — проверяйте условия на странице набора у агентства ООН; UN Data как навигатор не заменяет юридически значимую цитату первоисточника.
- **Автоматизация:** устойчивые пайплайны строят на **прямых URL агентств** и стабильных API там, где они есть; корневая страница UN Data удобна для ручного поиска и обзора тем.

## Примеры доступа

Проверка отклика главной страницы портала:

```bash
curl -sSk -o /dev/null -w "%{http_code}\n" "https://data.un.org/"
```

## Не путать с

- **Каталоги отдельных агентств ООН** (FAO, WHO, UNCTAD и др.) — UN Data помогает найти тему и перейти к первоисточнику, но не заменяет их собственные витрины, API и условия использования.
