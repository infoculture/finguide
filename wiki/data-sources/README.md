# Источники данных

Карточки источников строятся по одному шаблону: оператор, точки доступа, форматы, периодичность, ограничения и перекрёстные ссылки на ИС, отчётность и глоссарий.

## Для кого раздел

- Аналитики и журналисты, которым нужен **официальный** контур публикаций (план, исполнение, реестры).
- Разработчики выгрузок и исследователи, которым важны **форматы**, API и стабильность идентификаторов (КБК, ИНН, период).

## Подкатегории

- [Федеральные источники](./federal/README.md) — порталы и API федеральных органов власти.
- [Региональные источники](./regional/README.md) — бюджеты субъектов РФ и муниципальных образований.
- [Международные источники](./international/README.md) — официальные витрины и базы вне национального контура РФ с полезными рядами по России или сопоставлениями по странам.
- [Гражданские источники](./civil/README.md) — независимые проекты сообщества.

## Типовые вопросы

**Где машиночитаемые таблицы исполнения, а не только PDF?** Смотрите карточки с `data_source_kind: files` или `mixed` в федеральной группе и раздел [доступ к данным](/howto/access/).

**Как не смешать план и факт?** В каждой карточке зафиксированы границы описания; в наборе сверяйте поля периода, типа показателя и методичку на витрине оператора.

**Нужен ли международный ряд для сравнения с РФ?** [Международные источники](/data-sources/international/) — только как **внешний** слой; юридически значимые выводы по бюджету РФ опирайте на федеральные и региональные первоисточники.
