# Шаблон карточки отчётности

Служебная страница для авторов и агентов: при добавлении или правке карточки копируйте **блок ниже** (содержимое fenced-блока) в файл `wiki/reporting/<slug>.md`, замените плейсхолдеры и удалите неприменимые разделы целиком (без пустых заголовков).

Карточки лежат **в корне** `wiki/reporting/` (без вложенных подпапок). Публичный URL: `/reporting/<slug>`, где `<slug>` — имя файла без `.md`. Для новых карточек в frontmatter укажите `slug: /reporting/<slug>` тем же значением, чтобы ссылки с других страниц оставались стабильными.

### Поля frontmatter

| Поле | Обязательно | Пример / значения |
|------|-------------|-------------------|
| `title` | да | `"Форма 0503117 — Отчёт об исполнении федерального бюджета"` |
| `sidebar_label` | да | Короткая подпись для сайдбара (может совпадать с `title` или быть короче) |
| `tags` | да | Первая метка: `reporting`; далее контекст: `federal` \| `regional` \| `institution` \| `execution` \| `treasury` \| … |
| `last_updated` | да | `2026-05-10` |
| `slug` | да для новых карточек | `/reporting/<slug>` — тот же `<slug>`, что имя файла |
| `report_code` | по возможности | Код формы: `0503117`, `ПФХД`, текстовый идентификатор свода |
| `jurisdiction` | рекомендуется | `federal` \| `regional` \| `institution` \| `mixed` |
| `periodicity` | рекомендуется | `monthly` \| `quarterly` \| `annual` \| `on_demand` \| `mixed` |
| `execution_basis` | по возможности | `cash` \| `accrual` \| `mixed` \| `unspecified` — если применимо к форме |
| `primary_breakdown` | по возможности | Кратко: `КБК`, `КОСГУ`, `КБК+КОСГУ`, и т.п. |
| `pairs_with` | по возможности | Связанные коды форм или slug’и: `["0503387"]` или `["/reporting/0507011"]` |
| `operator_hint` | по возможности | Кто готовит/утверждает публичный контент (текстом; детали — в разделе про данные и ИС) |
