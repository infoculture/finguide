# Отчётность получателей бюджетных средств по субсидиям и грантам

| Признак | Значение |
| :--- | :--- |
| Код / вид отчётности | Отчёты об использовании субсидии (гранта), отчёты о достижении результата предоставления субсидии — по типовым условиям соглашений и НПА БК РФ |
| Уровень | Федеральный / региональный / муниципальный — в зависимости от источника субсидии |
| Периодичность | Задаётся соглашением (часто квартал и год; иные периоды для отдельных программ) |
| Основной разрез | Целевые показатели, фактические расходы по целям, соответствие КБК и условиям договора |
| Базис исполнения (если уместно) | Смешанный: учёт расходов по целям и подтверждение результатов |

Коротко: это отчётность **получателя** о том, как израсходованы средства и достигнуты условия предоставления; она связывает юридический контур соглашения с публичными реестрами и плановыми документами учреждения.

## Назначение и границы

Отчётность позволяет проверить целевое использование средств и выполнение условий субсидии. Она **не эквивалентна** кассовому исполнению бюджета по [форме 0503117](/reporting/0503117): казначейство фиксирует платежи, получатель — соответствие целям и показателям соглашения.

## Состав и разрезы

- фактические затраты по направлениям, предусмотренным соглашением;
- достижение значений результатов и показателей;
- остатки средств и возвраты — если предусмотрены условиями.

## Связанные формы и отчётность

- [План ФХД](/reporting/pfhd) — планируемые поступления и выплаты учреждения; сверяйте с отчётностью по субсидии по инструкции к соглашению.
- [Бухгалтерская отчётность учреждений](/reporting/institution-financial-statements) — финансовый результат и обязательства; не заменяет целевую отчётность по субсидии без методики соединения.

## Где встречается в данных

- [Соглашения (субсидии и субвенции)](/data-sources/federal/subsidy) — реестр и связанные сведения по договорам и суммам.
- [Наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets) — открытые реестры соглашений и смежные наборы (уточняйте актуальные идентификаторы наборов).
- [Открытые данные bus.gov.ru](/data-sources/federal/bus-opendata) — публикации учреждений-получателей, включая отчётность и связанные с госзаданием документы.

## Связанные информационные системы

- [ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb) — открытые данные и сервисы реестров соглашений на федеральном уровне.
- [bus.gov.ru](/information-systems/federal/bus-gov) — карточки учреждений и выгрузки отчётности.

## Связанные термины глоссария

- [Субсидии](/glossary/subsidies) — базовое определение и условия предоставления в модели БК РФ.
- [Гранты](/glossary/grants) — родственный институт; отличайте юридический режим при сводном анализе.
- [НПА о предоставлении субсидии](/glossary/npa-subsidy) — типовые требования к отчётности в нормативной модели.

## Особенности для аналитики

- Один получатель может иметь несколько соглашений с разной периодичностью отчётности; агрегируйте только после выверки по номеру и версии соглашения.
