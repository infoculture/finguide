# Сводная отчётность главных распорядителей средств федерального бюджета (ГРБС)

| Признак | Значение |
| :--- | :--- |
| Код / вид отчётности | Внутренняя и представительская отчётность ГРБС об исполнении доведённых лимитов и бюджетной росписи по ведомству; в открытых данных проявляется как ведомственные срезы плана и исполнения |
| Уровень | Федеральный (вертикаль распорядителя) |
| Периодичность | Операционная — месяц/квартал; годовая сводка по правилам отчётности |
| Основной разрез | ГРБС → распорядители → получатели; ЦСР и виды расходов по росписи |
| Базис исполнения (если уместно) | Смешанный: лимиты бюджетных обязательств и кассовое исполнение согласуют по разным регистрам |

Коротко: это **ведомственная проекция** федерального бюджета: как главный распорядитель исполнил доведённые суммы и программы; для гражданского аналитика доступ обычно через открытые наборы и витрины, а не через полный внутренний пакет отчётов ГРБС.

## Назначение и границы

Отчётность нужна для управления лимитами и контроля подведомственных учреждений и программ. Она **не тождественна** единой форме Казначейства [0507011](/reporting/0507011): казначейство даёт системную картину по КБК, ГРБС — управленческую по ведомству.

## Состав и разрезы

- исполнение по ведомственной структуре расходов;
- доведение лимитов БО до получателей;
- сверка с показателями госпрограмм и государственного задания там, где они заданы для ведомства.

## Связанные формы и отчётность

- [Отчётность по государственным программам](/reporting/state-program-reporting) — программные показатели ГРБС как ответственных исполнителей.
- [Форма 0507011](/reporting/0507011) — помесячное кассовое исполнение федерального бюджета для сверки системной картины с ведомственными сводами.

## Где встречается в данных

- [Наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets) — открытые таблицы исполнения и росписи с разрезом администраторов и участников бюджетного процесса.
- [Открытые данные Минфина России](/data-sources/federal/minfin-opendata) — бюджетная роспись и справочники, необходимые для сопоставления кодов с ведомственной структурой.
- [Бюджетная роспись](/data-sources/federal/budgetrosp) — плановый эталон по строкам росписи при наличии актуальных выгрузок (проверяйте состояние набора на дату анализа).

## Связанные информационные системы

- [ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb) — основной публичный контур плановых и исполнительных данных по федеральному бюджету.
- [АИС «Финансы»](/information-systems/federal/aisfinances) — закрытый ведомственный контур Минфина; публично доступны описания и регламенты, не полная операционная выгрузка.

## Связанные термины глоссария

- [ГРБС](/glossary/grbs) — роль главного распорядителя в бюджетном процессе.
- [Распорядитель бюджетных средств](/glossary/pbs) — звено между ГРБС и получателями при построении сводов.
- [КБК](/glossary/kbk) — классификационный каркас для сопоставления ведомственных таблиц с казначейской отчётностью.

## Особенности для аналитики

- Обозначение «ведомство» в открытых наборах может опираться на коды администраторов и реестр участников бюджетного процесса; проверяйте справочники на соответствующий год.
