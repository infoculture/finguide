# Официальная статистика сектора государственного управления и международная отчётность (СГФ)

| Признак | Значение |
| :--- | :--- |
| Код / вид отчётности | Показатели официальной статистики по финансовым потокам сектора госуправления и материалы международной отчётности по руководству МВФ (СГФ 2014 / GFSM 2014), включая таблицы соответствия отечественных кодов и международных категорий |
| Уровень | Федеральный и субнациональный — в публикациях Росстата; связка кодов — в методических материалах Минфина |
| Периодичность | Год и квартал — по календарю наблюдений; международные отчёты — по графику раскрытия |
| Основной разрез | Макроагрегаты и классификаторы; не операционные КБК отдельных платежей |
| Базис исполнения (если уместно) | Статистические определения расходов и доходов по методике наблюдения; отличаются от кассового исполнения бюджета |

Коротко: это **макро- и международный слой**, который используют для сравнения России с другими странами и для анализа структуры сектора; он **не подменяет** [формы Казначейства 0503…](/reporting/0503117) при построчном разборе бюджета.

## Назначение и границы

Материалы помогают понять масштаб и структуру операций сектора государственного управления в сопоставимых категориях. Для построения таблиц по контрагентам, платежам и лицевым счетам используйте бюджетные и казначейские источники, а не только статистические сводки.

## Состав и разрезы

- показатели финансов сектора госуправления в публикациях Росстата (конкретные формы и таблицы — по актуальному календарю);
- таблицы соответствия КБК категориям СГФ-2014 на стороне Минфина;
- справочные материалы по международным классификаторам расходов.

## Связанные формы и отчётность

- [Форма 0503117](/reporting/0503117) — операционное кассовое исполнение федерального бюджета; мост к СГФ строят через методики Минфина, а не прямым копированием строк.
- [Налоговая и таможенная статистика](/reporting/tax-and-customs-revenue-statistics) — смежный слой макродоходов при построении согласованных рядов.

## Где встречается в данных

- [Открытые данные Минфина России](/data-sources/federal/minfin-opendata) — наборы и справочники для сопоставления бюджетных показателей с международными категориями там, где они включены в реестр открытых данных.
- [Наборы данных ГИИС «Электронный бюджет»](/data-sources/federal/budget-gov-ru-datasets) — детальные ряды исполнения и планов для построения промежуточных агрегатов перед статистической группировкой.
- [Витрины данных Казначейства (datamarts)](/data-sources/federal/roskazna-datamarts) — аналитические срезы доходов и расходов как дополнение к официальной статистике при проверке масштабов.

## Связанные информационные системы

- [ЕМИСС](/information-systems/federal/emiss) — макроэкономические ряды Банка России; часто сопоставляют с бюджетными агрегатами в аналитике, не подменяя бюджетную отчётность.
- [ГИИС «Электронный бюджет»](/information-systems/federal/giis-eb) — операционный источник планов и исполнения для сверки со статистическими показателями по методике.

## Связанные термины глоссария

- [Бюджетная система](/glossary/budget-system) — контекст границ сектора государственного управления и бюджетов.
- [КБК](/glossary/kbk) — исходный классификационный язык отечественных бюджетных данных до трансляции в международные категории.
- [Казначейское исполнение](/glossary/treasury-execution) — операционный контур кассы, с которым статистические агрегаты сверяют по методике, а не построчно.

## Не путать с

- **Бюджетная отчётность Казначейства** — первичный операционный контур по кассе; статистика сектора строится по иным правилам наблюдения и может отличаться по суммам и периодам.

## Особенности для аналитики

- Сопоставляйте ряды только после явной проверки методики (касса vs начисления, консолидация уровней бюджетной системы).

См. также страницу [СГФ-2014 — соответствие классификаций МВФ](/budget-classification/sgf-2014) для связки КБК и международных категорий.
