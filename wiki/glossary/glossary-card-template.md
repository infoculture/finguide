# Шаблон карточки глоссария

Служебная страница для авторов и агентов: при добавлении или правке термина копируйте **блок ниже** (содержимое fenced-блока) в файл `wiki/glossary/<подпапка>/<slug>.md` (подпапка по теме — см. [оглавление глоссария](./README.md)), замените плейсхолдеры и удалите неприменимые разделы целиком (без пустых заголовков), в том числе **«Примеры»**, если иллюстрации не добавляют содержания к «Определение» и «Где встречается в учёте и данных». В frontmatter обязательно укажите `slug: /glossary/<slug>` (совпадает с именем файла без `.md`), чтобы URL не зависел от вложенности пути.

### Поля frontmatter

| Поле | Обязательно | Пример |
|------|-------------|--------|
| `title` | да | `"КБК — Код бюджетной классификации"` |
| `sidebar_label` | да | `"КБК"` |
| `tags` | да | `[glossary, classification]` — вторая метка: тема (`budget`, `classification`, `procurement`, `transfers`, …) |
| `last_updated` | да | `2026-05-10` |
| `slug` | да для новых карточек | `/glossary/<slug>` — тот же `<slug>`, что имя файла |
| `content_type` | да | `concept` (по умолчанию для терминов) |
| `entity_type` | да | `glossary` |
| `glossary_kind` | рекомендуется | `concept` \| `classification` \| `law` \| `process` \| `organization` |
| `source_url` | по возможности | прямая ссылка на НПА или страницу Минфина |
| `abbreviations` | при наличии | `["КБК"]` |

---

entity_type: glossary

# source_url: "https://..."
