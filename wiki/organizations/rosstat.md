# Федеральная служба государственной статистики

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Росстат |
| Официальный сайт | https://rosstat.gov.ru/ |
| Уровень | Федеральный |

## Описание

Росстат — федеральный орган исполнительной власти, осуществляющий формирование официальной статистической информации. Публикует данные через ЕМИСС (Единую межведомственную информационно-статистическую систему).

## Регулируемые / публикуемые данные

- Социально-экономическая статистика
- ЕМИСС
- Перепись и микроданные

## Точки доступа

- https://rosstat.gov.ru/opendata/
- https://fedstat.ru/
