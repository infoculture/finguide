# Федеральное казначейство

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Казначейство России |
| Официальный сайт | https://roskazna.gov.ru/ |
| Уровень | Федеральный |

## Описание

Федеральное казначейство — федеральный орган исполнительной власти, осуществляющий правоприменительные функции по обеспечению исполнения бюджетов всех уровней, кассовое обслуживание, ведение Электронного бюджета и Единой информационной системы в сфере закупок.

## Регулируемые / публикуемые данные

- Исполнение бюджетов
- Лицевые счета
- Реестры и витрины данных
- Контрактная система (оператор ЕИС)
