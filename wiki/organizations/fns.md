# Федеральная налоговая служба

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ФНС России |
| Официальный сайт | https://www.nalog.gov.ru/ |
| Уровень | Федеральный |

## Описание

ФНС России — федеральный орган исполнительной власти, осуществляющий функции по контролю за соблюдением законодательства о налогах и сборах, регистрацию юридических лиц и ИП. Ведёт ЕГРЮЛ/ЕГРИП, публикует обширную налоговую статистику и открытые данные.

## Регулируемые / публикуемые данные

- Налоговая статистика
- ЕГРЮЛ и ЕГРИП
- Реестр контрольно-кассовой техники
- Реестр МСП
