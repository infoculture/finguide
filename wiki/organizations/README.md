# Организации

Раздел описывает **ключевые органы и фонды**, которые задают или сопровождают публичные финансовые данные: кто публикует что, в какой ИС и как это стыкуется с [каталогом источников](/data-sources/).

## Для кого раздел

- Аналитикам, которым нужно понять **роль ведомства** (владелец витрины, оператор учёта, регулятор раскрытия).
- Авторам карточек: шаблон — `wiki/organizations/organizations-card-template.md` ([GitHub](https://github.com/infoculture/finguide/blob/master/wiki/organizations/organizations-card-template.md)); у шаблона `draft: true`, страница **не** в production-сборке. Правила — в `AGENTS.md`.

## Ключевые связки

| Вопрос | С чего начать |
| --- | --- |
| Бюджет, КБК, открытые наборы Минфина | [Министерство финансов РФ](./minfin.md) |
| Кассовое исполнение, лицевые счета, отчёты казначейства | [Федеральное казначейство](./federal-treasury.md) |
| Налоги и таможенные платежи в контуре доходов | [ФНС](./fns.md), [ФТС](./fts.md) |

| Сценарий без названия органа | [Карта данных](/intro/data-map) |

## Типовые вопросы

**Организация есть в реестре, но нет карточки в wiki?** См. [план наполнения](/intro/wiki-filling-plan); новые карточки — по шаблону в репозитории.

**Где юридическая основа полномочий?** Карточка организации ссылается на НПА; базовый каркас бюджетной системы — [Бюджетный кодекс](/legal/budget-code).
