# Министерство экономического развития РФ

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Минэкономразвития России |
| Официальный сайт | https://www.economy.gov.ru/ |
| Уровень | Федеральный |

## Описание

Минэкономразвития России отвечает за выработку государственной политики в сфере социально-экономического развития, поддержки малого и среднего предпринимательства, экспорта, госпрограмм развития. Курирует поддержку НКО и грантовые конкурсы.

## Регулируемые / публикуемые данные

- Госпрограммы развития
- Поддержка МСП
- Реестр СОНКО
- Прогноз социально-экономического развития
