# Федеральный фонд обязательного медицинского страхования (ФФОМС)

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ФФОМС |
| Официальный сайт | https://ffoms.gov.ru/ |
| Уровень | Федеральный |

## Описание

Федеральный фонд обязательного медицинского страхования финансирует систему ОМС. Публикует бюджет фонда, нормативы финансирования, реестры медицинских организаций — участников ОМС.

## Регулируемые / публикуемые данные

- Бюджет ФФОМС
- Нормативы финансового обеспечения ОМС
- Реестр медорганизаций ОМС
