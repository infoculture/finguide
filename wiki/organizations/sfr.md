# Социальный фонд России (СФР)

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | СФР |
| Официальный сайт | https://sfr.gov.ru/ |
| Уровень | Федеральный |

## Описание

Социальный фонд России образован в 2023 году путём объединения ПФР и ФСС. Администрирует пенсионное обеспечение, обязательное социальное страхование, выплаты по материнскому капиталу, пособия. Раскрывает бюджет фонда и отчёты об исполнении.

## Регулируемые / публикуемые данные

- Бюджет СФР
- Численность пенсионеров
- Размер выплат и пособий
- Реестр страхователей
