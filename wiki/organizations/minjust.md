# Министерство юстиции РФ

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Минюст России |
| Официальный сайт | https://minjust.gov.ru/ |
| Уровень | Федеральный |

## Описание

Министерство юстиции РФ ведёт реестры некоммерческих организаций, иностранных агентов, СМИ-иноагентов, нежелательных организаций. Публикует данные о НКО на edinstvennoe-okno.minjust.ru и предоставляет реестры через свой портал.

## Регулируемые / публикуемые данные

- Реестр зарегистрированных НКО
- Реестр НКО — иностранных агентов
- Реестр иностранных и международных НКО, нежелательных в РФ
- Реестр нотариусов
