# Счётная палата РФ

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | СП РФ |
| Официальный сайт | https://ach.gov.ru/ |
| Уровень | Федеральный |

## Описание

Счётная палата РФ — высший орган внешнего государственного аудита (контроля), подотчётный Федеральному Собранию. Проводит аудит исполнения федерального бюджета, отчёты публикует в открытом доступе.

## Регулируемые / публикуемые данные

- Аудит исполнения бюджета
- Бюллетени и отчёты
- Открытые данные бюджетной сферы

## Точки доступа

- https://ach.gov.ru/audit/
- https://ach.gov.ru/checks/
