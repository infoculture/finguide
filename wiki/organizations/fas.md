# Федеральная антимонопольная служба

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | ФАС России |
| Официальный сайт | https://fas.gov.ru/ |
| Уровень | Федеральный |

## Описание

ФАС России — федеральный орган исполнительной власти, осуществляющий контроль за соблюдением антимонопольного законодательства, законодательства о закупках. Ведёт реестр недобросовестных поставщиков (РНП), реестр естественных монополий.

## Регулируемые / публикуемые данные

- Реестр недобросовестных поставщиков
- Жалобы по 44-ФЗ и 223-ФЗ
- Реестр естественных монополий

## Точки доступа

- https://rnp.fas.gov.ru/
