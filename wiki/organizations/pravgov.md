# Правительство Российской Федерации

|  Признак | Значение |
| :--- | :--- |
| Краткое имя | Правительство РФ |
| Официальный сайт | http://government.ru/ |
| Уровень | Федеральный |

## Описание

Правительство РФ — высший исполнительный орган государственной власти. Готовит проект федерального бюджета, утверждает госпрограммы, координирует работу федеральных органов исполнительной власти.

## Регулируемые / публикуемые данные

- Проект федерального бюджета (вносит в Госдуму)
- Государственные программы
- Постановления и распоряжения
