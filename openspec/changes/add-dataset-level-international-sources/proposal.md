# Change: Добавить карточки конкретных международных наборов

## Why

Международный раздел сейчас в основном описывает порталы организаций верхнего уровня. Для воспроизводимого сравнения этого недостаточно: пользователь должен видеть конкретный набор, его охват России, временной диапазон, методологию и ограничения, а не только ссылку на главную страницу IMF, OECD или IBP.

## What Changes

- Добавить карточки IMF Quarterly Government Finance Statistics, IMF World Revenue Longitudinal Database и IMF Fiscal Monitor Database.
- Добавить карточку OECD Fiscal Decentralisation Database / релевантной витрины subnational finance.
- Добавить карточку Open Budget Survey для России 2023 как конкретного среза, а не универсального бюджетного ряда.
- Для каждого набора проверить фактический охват России, историческую глубину, частоту обновления, форматы доступа и ограничения сопоставимости.
- Связать общие карточки международных порталов с карточками конкретных наборов и обзором GFS/COFOG.

## Impact

- **Затронутые capabilities:** `international-fiscal-comparability`.
- **Затронутый контент:** `wiki/data-sources/international/`, `wiki/reference/international-gfs-cofog.md`, международный how-to и QA-корпус.
- **Не входит в change:** копирование международных данных в репозиторий или заявление о наличии российских наблюдений без проверки на дату публикации.
