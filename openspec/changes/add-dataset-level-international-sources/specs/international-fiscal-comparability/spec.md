## ADDED Requirements

### Requirement: Dataset-level international source cards

Проект SHALL содержать отдельные карточки источников для следующих наборов или их официальных актуальных преемников:

- IMF Quarterly Government Finance Statistics (QGFS);
- IMF World Revenue Longitudinal Database (WoRLD);
- IMF Fiscal Monitor Database;
- OECD Fiscal Decentralisation Database либо каноническая витрина OECD по subnational finance, если название/публикация изменены;
- International Budget Partnership Open Budget Survey: Russia 2023.

Изменение официального названия MUST быть объяснено в карточке; оно не является основанием молча подменить набор общим порталом организации.

#### Scenario: Reader lands on a concrete dataset

- **GIVEN** пользователь ищет квартальную статистику госфинансов или международный ряд доходов
- **WHEN** он открывает результат из международного каталога
- **THEN** страница описывает конкретный набор, а не только главную страницу организации

### Requirement: Russia coverage and time boundary

Каждая карточка международного набора MUST явно фиксировать по состоянию на `last_verified`:

- присутствует ли Россия в наборе;
- для каких лет/кварталов доступны наблюдения;
- является ли ряд текущим, историческим, приостановленным или отсутствующим;
- какие уровни государственного управления и единицы измерения используются.

Если российские данные отсутствуют, карточка MAY оставаться в каталоге для методологического сравнения, но MUST сообщать об отсутствии до раздела примеров доступа.

#### Scenario: Missing Russia series is visible early

- **GIVEN** набор полезен методологически, но не содержит актуальных наблюдений по России
- **WHEN** пользователь открывает карточку
- **THEN** ограничение видно до попытки загрузки и не маскируется общей формулировкой о покрытии стран

### Requirement: International access and comparability fields

Карточка каждого набора SHALL описывать официальный URL, формат/API, частоту обновления, глубину архива, лицензию/условия, методологический документ и ограничения прямого сопоставления с российскими КБК, формами и уровнями бюджетной системы.

Рабочий пример доступа MUST быть приведён для стабильной машиночитаемой точки входа; если такой точки нет, раздел примеров MUST быть удалён, а ограничение описано текстом.

#### Scenario: Example matches documented access

- **GIVEN** карточка содержит команду или код загрузки
- **WHEN** редактор проверяет пример на дату `last_verified`
- **THEN** пример использует официальный стабильный вход и возвращает данные либо документированную структуру ответа

### Requirement: Portal-to-dataset navigation

Общие карточки IMF, OECD и IBP SHALL ссылаться на соответствующие dataset-level карточки с пояснением выбора. Карточки наборов MUST ссылаться на обзор GFS/COFOG и релевантный how-to до аналитических примеров.

#### Scenario: Generic portal becomes a useful hub

- **GIVEN** пользователь начинает с общей карточки международной организации
- **WHEN** он выбирает тип задачи
- **THEN** он переходит к конкретному набору и видит правила сопоставимости до анализа
