# OpenSpec

Спецификации возможностей репозитория «Открытые государственные финансы» в формате, совместимом с [OpenSpec](https://openspec.dev/): нормативные требования (RFC 2119) и сценарии **GIVEN / WHEN / THEN**.

Источник целей: [_dev/final_report.md](../_dev/final_report.md).

- [`specs/`](specs/) — канонические `spec.md` по областям.
- [`changes/`](changes/) — дельты по фазам внедрения ([`phase-10-1`](changes/phase-10-1/proposal.md), [`phase-10-2`](changes/phase-10-2/proposal.md), [`phase-10-3`](changes/phase-10-3/proposal.md)).
- [`engineering-notes.md`](engineering-notes.md) — зафиксированные инженерные решения (стек экспорта, политика lychee).
