Статистика Банка России

ПризнакЗначение
Уровень и охватФедеральный; макроэкономическая и финансовая статистика регулятора, временные ряды по денежному обращению, банковскому сектору, внешнему сектору и рынкам
Тип доступаСмешанный: раздел Статистика, база данных показателей (hd_base), сервис получения данных (API временных рядов), файловые выгрузки и справочные сервисы (в т.ч. курсы валют)
Основные форматыCSV, XLS/XLSX, XML, JSON — в зависимости от серии и точки входа; страницы разделов — HTML со ссылками на файлы
ОператорБанк России
Режим доступаОткрытый для перечисленных публичных разделов; профильные отчётные кабинеты для кредитных организаций в карточку источника не входят

Коротко: официальные временные ряды и таблицы ЦБ РФ — денежные агрегаты, ключевая ставка, банковская и рыночная статистика, платёжный баланс, курсы валют, открытые наборы по финрынкам и надзору в объёме паспортов. Для анализа государственных финансов это прежде всего контекст (ставка, ликвидность, внешний сектор), а не исполнение бюджета по КБК.

Описание​

Карточка описывает публичный контур данных с сайта Банка России, через который берут статистику и машиночитаемые выгрузки: тематические блоки в разделе /statistics/, выборку показателей в /hd_base/, программный доступ через /statistics/data-service/, а также устойчивые веб-сервисы курсов (классический XML scripts/XML_daily.asp). В объём входят ссылки на методики и файлы по датам; внутренние кабинеты отчётности кредитных организаций и контуры надзора не являются предметом этой карточки.

Отдельный каталог https://cbr.ru/opendata/ на момент проверки не используется как стабильная витрина (прямой запрос отдаёт 404 — см. карточку сайта ЦБ); опирайтесь на разделы статистики, базу показателей и документированный API.

Оператор​

Банк России — каноническая витрина: https://cbr.ru/statistics/

  • Карточка ИС: Сайт Банка России — официальный портал-оператор тех же точек доступа; там же сведены ограничения по URL, API и сопоставление с бюджетными системами.

Объекты данных и показатели​

Заполняйте под задачу из метаданных серии на сайте ЦБ:

  • Строка набора — как правило наблюдение временного ряда: дата (или период), код/идентификатор показателя, значение, единица измерения и база по методике ЦБ.
  • Типичные блоки — денежные агрегаты; процентные ставки (включая ключевую); банковская статистика; платёжный баланс и внешний сектор; финансовые рынки; открытые наборы надзора; курсы валют.
  • Связка с бюджетными и налоговыми данными — не через единый реестр: сопоставление на уровне дат, методик и аналитической модели (КБК и казначейское исполнение ведут другие операторы).

Пробелы: единиц измерения и пересчёты заданы методиками ЦБ и могут отличаться от классификаций бюджета и Росстата; перед автоматизацией сверяйте страницу показателя и сопроводительные PDF.

Периодичность обновления​

  • Курсы официального списка — по торговым дням (сервис XML_daily.asp и аналогичные публикации на cbr.ru).
  • Статистические ряды — по календарю выхода отчётности ЦБ: часть показателей ежемесячно, часть ежеквартально; точная дата публикации приводится на странице серии или в релизе.
  • Детали состава разделов, идентификаторов серий в базе показателей и актуальности API см. разделы «Подсистемы и разделы», «Ограничения и актуальность» в карточке сайта Банка России.

Ограничения и особенности​

  • Условия использования: открытые данные регулятора; при переработке обычно требуется указание источника (формулировки — на страницах соответствующих наборов и в пользовательских документах ЦБ).
  • Стабильность: состав показателей, структура страниц и иногда URL меняются — для пайплайнов опирайтесь на идентификаторы серий в базе показателей и перечень API, а не только на закладки в браузере.
  • API: лимиты и форматы описаны в руководстве по API; внешние зеркала курсов не заменяют официальный домен для юридически значимой ссылки на курс.

Примеры доступа​

Официальный ежедневный файл курсов (XML) за указанную дату:

import requests
from datetime import date

date_req = date.today().strftime("%d/%m/%Y")
url = "https://www.cbr.ru/scripts/XML_daily.asp"
response = requests.get(url, params={"date_req": date_req}, timeout=30)
response.raise_for_status()
print(response.text[:500])

Для статистических рядов через API и SOAP используйте параметры из руководства по API Банка России и документацию на странице сервиса получения данных.

Связанные страницы wiki​

  • Сайт Банка России — полная карточка витрины, подсистем и ограничений доступа.
  • Статистический бюллетень Банка России (PDF) — PDF-выпуски со сводными таблицами; для автоматизации рядов предпочтительнее раздел ниже и база показателей.
  • Вестник Банка России — официальное издание ЦБ: нормативные и разъяснительные материалы по выпускам; не каталог временных рядов.
  • Годовой отчёт Банка России — PDF годовых отчётов и краткие сводки; нарратив и приложения к отчётному году, не каталог временных рядов.
  • Мониторинг отраслевых финансовых потоков — отдельная линейка аналитики ЦБ: PDF-выпуски и Excel stat_finflows.xlsx по платежам через ПС Банка России; не сводится к выборке показателей в hd_base.
  • Наборы данных портала «Электронный бюджет» — бюджетное планирование и исполнение; ряды ЦБ дополняют анализ макроокружением, но не содержат той же модели КБК.
  • Налоговая статистика ФНС (формы) — фискальные поступления; часто сопоставляют с денежными агрегатами и курсами ЦБ в обзорных моделях.
  • API Банка России — пошаговые примеры и подводные камни при работе с сервисами ЦБ.

Не путать с​

  • Неофициальные зеркала курсов (например https://www.cbr-xml-daily.ru/) — могут упрощать разработку, но не являются сайтом Банка России; для ссылки на официальный курс и методику публикации используйте cbr.ru и документацию регулятора.
