Налоговая статистика ФНС

ПризнакЗначение
Уровень и охватФедеральный; агрегированные показатели администрирования налогов и взносов, прежде всего в разрезе субъектов РФ и отчётных периодов по формам ФНС
Тип доступаФайловые публикации по ссылкам из раздела «Статистика и аналитика» на сайте ФНС; отдельного стабильного публичного API под сами формы нет
Основные форматыXLS/XLSX (типично для форм), страницы каталога форм — HTML со ссылками на файлы
ОператорФНС России
Режим доступаОткрытый просмотр и скачивание публичных форм; условия переработки — по пояснениям и паспортам на страницах источника

Коротко: готовые таблицы налоговой и смежной статистической отчётности (1-НМ, 1-НДС, 7-НП и др.) — основной публичный способ оценить собираемость, базы и задолженность по правилам ФНС. Для анализа кассового исполнения бюджета по КБК это вспомогательный слой: его сверяют с казначейскими и бюджетными контурами, не отождествляя строки форм с кассой.

Описание​

Карточка описывает витрину статистических форм на портале ФНС: перечень форм, методические пояснения и прямые ссылки на файлы (как правило XLS/XLSX) в разделе статистики. В объём не входят личные кабинеты налогоплательщика, каталог открытых данных по другому URL (открытые данные — отдельные наборы с паспортами), а также интерактивная аналитика и ZIP-выгрузки налоговых паспортов на аналитическом портале. По каждому машиночитаемому набору 7707329152-* в wiki заведена отдельная карточка в группе «Налоговая служба» (см. оглавление федеральных источников и пример 7707329152-address).

Оператор​

Федеральная налоговая служба — раздел статистики: https://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/

  • Карточка ИС: Сайт ФНС России — статистика и данные — каноническая витрина тех же разделов; там же перечислены открытые данные, аналитический портал и типичные ограничения по URL и автоматизации.

URL и точки доступа​

Тип доступаURLФормат
Раздел «Статистика и аналитика»https://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/HTML; навигация к формам и материалам
Каталог форм налоговой статистикиhttps://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/forms/HTML со ссылками на файлы; сами формы — XLS/XLSX и др. по позициям
Статистика госрегистрации ЮЛ и ИПhttps://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/regstats/HTML; годовые отчёты 1-ЮР, 1-ИП — см. отдельную карточку

Объекты данных и показатели​

  • Строка набора — как правило агрегат по субъекту РФ (и иногда по более детальным разрезам, заданным методикой конкретной формы) за отчётный период: суммы поступлений, начислений, задолженности, показатели по отдельным налогам — в зависимости от формы.
  • Типичные формы — 1-НМ (поступление налогов и взносов), 7-НП (задолженность), 1-НДС (НДС), 4-НМ, 5-ПМ и другие позиции каталога на дату скачивания; точный состав строк задаётся методическими указаниями к форме на сайте ФНС.
  • Связка с бюджетными данными — не через единый ключ с казначейскими файлами: сопоставление на уровне агрегатов, периода и экономической интерпретации (см. отчётность «налоги и таможня» в wiki).
  • Пробелы: нет единого машиночитаемого каталога с постоянными прямыми URL на каждый файл; структура страниц и набор форм меняются — перед пайплайном проверяйте актуальные ссылки на странице каталога.

Периодичность обновления​

Публикации выходят по календарю статистической отчётности ФНС: для разных форм — месяц, квартал, год; точные сроки и версии файлов указаны в метаданных и пояснениях на странице каждой формы. Общий контур публичной витрины и типичные ограничения по обновлению см. в разделах «Подсистемы и разделы» и «Ограничения и актуальность» в карточке сайта ФНС.

Ограничения и особенности​

  • Условия использования: открытые ведомственные публикации; при переработке обычно требуется указание источника (формулировки — на страницах форм и в сопроводительных документах ФНС).
  • Доступ автоматизацией: часть запросов без типичного User-Agent браузера может получать 403; для надёжной загрузки используйте уважительный заголовок и разумные паузы между запросами.
  • Содержание: формы отражают администрирование по НК РФ и методикам ФНС, а не кассу бюджета; при сверке с отчётами Казначейства учитывайте зачёты, распределение между уровнями бюджетной системы, возвраты и лаги публикаций.

Примеры доступа​

Проверка доступности страницы каталога форм (без парсинга HTML); при необходимости добавьте заголовок User-Agent, иначе сервер может ответить 403:

import requests

url = "https://www.nalog.gov.ru/rn77/related_activities/statistics_and_analytics/forms/"
headers = {
    "User-Agent": "Mozilla/5.0 (compatible; research/1.0; +https://www.nalog.gov.ru/)",
}
response = requests.get(url, headers=headers, timeout=30)
response.raise_for_status()
print(response.status_code, response.headers.get("content-type", ""))

Файлы конкретных форм скачиваются по ссылкам со страницы каталога; стабильные прямые URL на каждый XLS в карточке не фиксируются из‑за обновлений витрины.

Связанные страницы wiki​

  • Сайт ФНС России — статистика и данные — полная карточка портала: открытые данные, аналитический портал, ограничения.
  • Статистика налоговых и таможенных поступлений — как формы ФНС соотносятся с казначейским исполнением и чем отличается предмет учёта.
  • Отчёты Федерального казначейства — кассовое исполнение бюджетов для сопоставления с агрегатами налоговой статистики.
  • Hub of Data — агрегированный каталог, где индексируются наборы ФНС; удобен для поиска зеркал и API каталога, если нужна единая точка входа.
  • Статистика госрегистрации ЮЛ и ИП (ФНС) — годовые своды 1-ЮР, 1-ИП и тематические материалы по регистрации; иной предмет учёта, чем у налоговых форм.

Не путать с​

  • Статистика госрегистрации ЮЛ и ИП (ФНС) — отчёты о работе государственной регистрации (формы 1-ЮР, 1-ИП); не относится к собираемости налогов по формам 1-НМ, 7-НП и др.
  • Открытые данные на nalog.gov.ru — отдельный каталог наборов с паспортами (CSV/JSON и др. по набору); не смешивать с лентой статистических форм 1-НМ и аналогов в разделе форм.
  • Аналитический портал ФНС (источник данных) — ZIP-дампы налоговых паспортов субъектов по API и интерактивные графики на analytic.nalog.gov.ru; см. также карточку ИС. Не заменяет архив форм на основном сайте.
