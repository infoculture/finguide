Росстат: Российский статистический ежегодник

ПризнакЗначение
Уровень и охватФедеральный; основной официальный ежегодник официальной статистики по Российской Федерации — широкий круг разделов (население, экономика, социальная сфера, государство и финансы и др. в редакции Росстата)
Тип доступаВеб-каталог серии на rosstat.gov.ru (/folder/210/…): HTML-страницы выпусков, PDF полного текста или разделов, архивы (ZIP и др.), часто много мелких вложений (отдельные файлы по подразделам); программного JSON/API под всю серию нет
Основные форматыPDF, ZIP/RAR (состав задаётся на странице выпуска), при наличии XLS/XLSX, HTML
ОператорРосстат
Режим доступаОткрытый просмотр и скачивание публичных материалов; условия переработки — по сопроводительным файлам и общим правилам ведомства

Коротко: Российский статистический ежегодник — центральное печатное и файловое издание свода показателей по стране. Удобен для методических пояснений, оглавления по темам и архива прошлых лет; для пайплайнов почти всегда нужны скачивание страницы выпуска, обход списка вложений и нормализация после извлечения из PDF или Excel.

Описание​

Карточка описывает серию «Российский статистический ежегодник» и якорную страницу каталога Росстата: https://rosstat.gov.ru/folder/210/document/12994. Типичная публикация по году сочетает:

  • Карточку выпуска (HTML) со списком файлов: полный текст, приложения, иногда десятки отдельных небольших документов по разделам;
  • PDF (целиком или по частям — зависит от года);
  • Архивы с текстом, таблицами и приложениями в составе, заданном на странице конкретного выпуска.

Для задач открытых государственных финансов ежегодник даёт агрегированный контекст (бюджет, долг, налоги и смежные блоки в терминологии статистики) рядом с казначейскими отчётами и бюджетными наборами; прямой стыковки с КБК и регистрами исполнения нет без методической сверки определений.

Сюда не входят: «Россия в цифрах» (иной формат и охват краткого обзора), «Финансы России» (узкая финансовая серия), а также паспортированные наборы /opendata/ без привязки к выпуску ежегодника — см. открытые данные Росстата.

Оператор​

Федеральная служба государственной статистики (Росстат) — каталог серии «Российский статистический ежегодник»: https://rosstat.gov.ru/folder/210/document/12994

  • Карточка ИС: ЕМИСС — инфраструктура официальной статистики; многие ряды пересекаются с fedstat.ru, но страница выпуска ежегодника с PDF, архивами и вложениями не заменяется одной выгрузкой из ЕМИСС.

URL и точки доступа​

Тип доступаURLФормат
Каталог / якорная карточка серииhttps://rosstat.gov.ru/folder/210/document/12994HTML; переходы к выпускам по годам и спискам файлов
Родительская папка каталога (типично)https://rosstat.gov.ru/folder/210HTML; документы раздела

Прямые ссылки на отдельные PDF, ZIP и мелкие вложения при обновлениях каталога меняются; для скриптов парсьте актуальную страницу выпуска и забирайте href из списка материалов.

Объекты данных и показатели​

  • Единица публикации — выпуск за год (или редакция в заголовке): один или несколько крупных файлов плюс набор мелких вложений по структуре оглавления.
  • Строка аналитики после извлечения — как правило агрегат таблицы (показатель, период, территория, единица); единого машиночитаемого паспорта на всю серию нет.
  • Государство и финансы — сводные блоки официальной статистики; для сопоставления с ГИИС «Электронный бюджет» нужны явные определения и иногда сверка с ФНС по налоговым агрегатам.
  • Пробелы: нет единого API; фрагментированная выкладка (много файлов на одну страницу года); 403 при запросах без типичного User-Agent браузера.

Периодичность обновления​

Публикация привязана к готовности годового выпуска: обычно один основной выпуск в год и архив прошлых лет на странице серии. Даты появления вложений смотрите на карточке конкретного года; общий контур обновлений публичного сайта — в разделе «Ограничения и актуальность» карточки ЕМИСС.

Ограничения и особенности​

  • Масштаб вложений: на странице года часто много небольших файлов — закладывайте обход списка, очередь загрузок и идемпотентное хранение по URL/хешу.
  • Извлечение данных: PDF может быть версточным или сканом; Excel, если есть, ускоряет работу; для ZIP — учёт вложенных путей и кодировок имён файлов.
  • Содержание: ежегодник отражает официальную статистику Росстата; он не заменяет операционные регистры исполнения бюджета без отдельной методики.

Примеры доступа​

Проверка доступности якорной страницы каталога (HTTP-код; при 403 добавьте заголовок User-Agent):

curl -sSk -A "Mozilla/5.0 (compatible; research/1.0)" -o /dev/null -w "%{http_code}\n" "https://rosstat.gov.ru/folder/210/document/12994"

Связанные страницы wiki​

  • Официальная статистика Росстата: ЕМИСС и открытые данные — fedstat.ru, /opendata/; другой канал для готовых наборов и выгрузок по показателям.
  • Росстат: «Россия в цифрах» — краткий ежегодный обзор; пересечение тем с ежегодником есть, но объём, структура и набор файлов другие.
  • Росстат: сборник «Финансы России» — узкая финансовая серия; дополняет ежегодник по глубине финансового блока, но это отдельное издание.
  • Росстат: сборник «Национальные счета России» — специализированное издание по СНС; пересечение с ежегодником по теме макро есть, но комплект таблиц и методика рассчитаны на отдельную серию.
  • Росстат: Show Data — finder — поиск кодов показателей для сопоставления строк из ежегодника с рядами на fedstat.ru.
  • ЕМИСС — роль системы и ограничения автоматизации публичного доступа.

Не путать с​

  • Сборник «Россия в цифрах» — обзорный выпуск; не полный аналог полного ежегодника по охвату и комплекту материалов.
  • Сборник «Финансы России» — специализация на финансовом секторе и государственных финансах, не замена общего ежегодника.
  • Сборник «Национальные счета России» — отдельная серия по СНС; не путать с обзорными таблицами макро внутри ежегодника без полного комплекта счетов СНС из специализированного сборника.
  • Наборы открытых данных на rosstat.gov.ru — паспортированные наборы; ежегодник — прежде всего издание с PDF, архивами и многофайловой выкладкой по годам.

Частые вопросы​

В: Как автоматизировать скачивание всех файлов выпуска?

О: Парсите HTML страницы года, соберите все href на файлы (PDF, ZIP, XLSX и т.д.), скачивайте с User-Agent и паузами; версии фиксируйте по году выпуска и при возможности по хешу файла.

В: Нужен один показатель в динамике по годам — с ежегодника или с ЕМИСС?

О: Для ряда удобнее ЕМИСС и Show Data; ежегодник остаётся справочником с методикой и комментариями в редакции года.
