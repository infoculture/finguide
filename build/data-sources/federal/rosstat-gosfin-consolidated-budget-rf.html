Росстат (mediabank): консолидированный бюджет Российской Федерации (XLSX)

ПризнакЗначение
Уровень и охватФедеральный; консолидированный бюджет Российской Федерации и бюджеты государственных внебюджетных фондов за один отчётный год в статистической сводке (агрегаты по крупным статьям доходов)
Тип доступаПрямая ссылка на один файл в storage/mediabank/ с сайта раздела Финансы
Основные форматыXLSX
ОператорРосстат (в файле по доходам указаны первичные данные Федерального казначейства — см. сноску на листе 1)
Режим доступаОткрытое скачивание

Коротко: одна книга Excel с сводной таблицей доходов консолидированного бюджета РФ и ГВФ с разбивкой на федеральный бюджет и консолидированные бюджеты субъектов РФ, плюс лист с профицитом/дефицитом в процентах к ВВП за длинный горизонт лет. Удобно для быстрой макро-картинки; для расшифровки расходов по направлениям в самом файле стоит ограничение (см. ниже).

Описание​

Источник — не каталог открытых данных с meta.json, а презентационный файл, который Росстат вешает в блоке «Государственные финансы» на странице https://rosstat.gov.ru/statistics/finance. Имя файла на mediabank фиксировано (gos_fin_byudget.xlsx), а отчётный год и текст заголовков задаются внутри книги (при обновлении Росстат перезаписывает файл).

Сюда не входят: помесячное/поквартальное исполнение по КБК из ГИИС «Электронный бюджет»; операционные отчёты Казначейства; региональные детализации в разрезе мер социальной поддержки — для них см. отдельные файлы раздела.

Оператор​

Федеральная служба государственной статистики (Росстат) — витрина: https://rosstat.gov.ru/statistics/finance; файл: https://rosstat.gov.ru/storage/mediabank/gos_fin_byudget.xlsx

  • Карточка ИС: ЕМИСС — смежные показатели по государственным финансам ищите по каталогу на fedstat.ru; mediabank-файл дублирует часть нарратива в виде готовой таблицы.

URL и точки доступа​

Тип доступаURLФормат
Тематическая витрина «Финансы»https://rosstat.gov.ru/statistics/financeHTML; актуальная ссылка на файл и дата обновления
Файл «Консолидированный бюджет Российской Федерации»https://rosstat.gov.ru/storage/mediabank/gos_fin_byudget.xlsxXLSX

Объекты данных и показатели​

  • Единица наблюдения — строка сводной таблицы (статья дохода или итоговый блок) × три числовых столбца: по смыслу заголовков в книге это итого по консолидированному бюджету РФ и ГВФ, федеральный бюджет, консолидированные бюджеты субъектов РФ (точные подписи колонок берите из строки шапки листа 1 — при обновлениях верстка может слегка съезжать).
  • Лист Содержание — навигация по листам и служебные реквизиты (ответственный исполнитель, дата обновления).
  • Лист 1 — доходы по крупным налоговым и неналоговым группам, строка «Расходы — всего», «Дефицит(-)/Профицит»; внизу — сноски (в проверенной версии указано, что информация по доходам — по данным Федерального казначейства, а направления расходов не подлежат публикации в этой форме).
  • Лист 2 — ряд профицита/дефицита консолидированного и федерального бюджетов в процентах к ВВП за многолетний интервал (как правило — для графика).
  • Связка с другими источниками: для сопоставления с открытыми наборами Минфина/ГИИС нужна явная сверка определений «консолидированный бюджет» и состава фондов — см. глоссарий: консолидированный бюджет.

Периодичность обновления​

Файл обновляется эпизодически, когда Росстат выпускает очередную редакцию свода; точную дату смотрите на карточке файла на https://rosstat.gov.ru/statistics/finance (поле вида «обновлено …») и в листе Содержание. Отдельного регламентного API с версионированием нет: URL файла стабилен, содержимое — подменяемое.

Ограничения и особенности​

  • Не «сырые» бюджетные строки: нет КБК, нет помесячной кассы, нет муниципального среза в этом файле.
  • Расходы агрегированы: в проверенной версии книги присутствует только итог расходов и сальдо; детализация расходов в этой выгрузке намеренно не раскрывается (см. сноску на листе 1).
  • Верстка Excel: многослойные шапки, объединённые ячейки, нечисловые прочерки '-' — для автоматического импорта обычно нужен этап нормализации (поиск строки с подписью колонок, приведение типов).
  • HTTP-доступ: для скриптов используйте браузероподобный User-Agent и умеренные паузы; «голый» curl иногда получает 403 (см. общую карточку Росстата).

Примеры доступа​

Проверка ответа сервера для прямого скачивания (код HTTP):

curl -sSk -A "Mozilla/5.0 (compatible; research/1.0)" -o /dev/null -w "%{http_code}\n" "https://rosstat.gov.ru/storage/mediabank/gos_fin_byudget.xlsx"

  • Росстат: поступления налогов в консолидированный бюджет (mediabank) — соседний файл по налоговым поступлениям (ФНС) в том же блоке «Государственные финансы».
  • Росстат: исполнение бюджетов ГВФ (mediabank) — поступление и расходование средств ПФР/ФСС/СФР, ФФОМС и ТФОМС в другой книге gos_fin_fondy.xls.
  • Росстат: расходы на социальную поддержку по субъектам (mediabank) — региональный разрез расходов на меры соцподдержки, не федеральный консолидированный свод.
  • Официальная статистика Росстата: ЕМИСС и открытые данные — общий контур fedstat.ru, /statistics/finance, /opendata/.
  • Наборы ГИИС «Электронный бюджет» — машиночитаемые ряды исполнения и планов для сверки после согласования методик.
  • Отчёты и формы Федерального казначейства — кассовый контур для сопоставления с операционными данными.

Не путать с​

  • Росстат: «Финансы России» — годовой сборник (PDF, архивы, много вложений), другой носитель и комплект таблиц.
  • Статистика государственных финансов на портале Росказны — витрина казначейского оператора; иной учётный контур.

Частые вопросы​

В: Почему суммы доходов здесь и в открытых наборах budget.gov.ru не сходятся построчно?

О: Различаются определение консолидированного бюджета, момент учёта, классификация статей и охват фондов. Используйте этот файл как опорный статистический свод Росстата, а для юридически значимой отчётности — первоисточники ГИИС/Казначейства с явной методической сверкой.

В: Можно ли стабильно парсить лист 1 без правок кода при каждом обновлении?

О: Рискованно: меняются год в заголовке, состав строк и иногда раскладка шапки. Надёжнее искать якорные подписи («Доходы - всего», «Расходы - всего») и читать динамический диапазон колонок.
