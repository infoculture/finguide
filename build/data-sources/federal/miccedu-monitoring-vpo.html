Мониторинг в сфере образования: показатели организаций высшего образования (miccedu.ru)

ПризнакЗначение
Уровень и охватФедеральная витрина мониторинга сферы образования на домене miccedu.ru; раздел «Мониторинг (ВО)» ориентирован на организации высшего образования (вузы и иные организации ВО), при этом у корня портала могут быть смежные разделы мониторинга по другим ступеням и темам
Тип доступаСмешанный: интерактивный веб-интерфейс с фильтрами и таблицами; типичны выгрузки из интерфейса (например в табличные форматы — по возможностям текущей версии страницы), без обещания единого открытого API каталога наборов как у data.gov.ru
Основные форматыHTML (основной просмотр); при наличии в UI — XLSX/CSV через кнопки экспорта (проверяйте на актуальной версии раздела)
ОператорИнфраструктура мониторинга Министерства просвещения Российской Федерации и связанных подразделений отраслевого мониторинга (miccedu.ru)
Режим доступаОткрытый для опубликованных разделов без учётной записи (отдельные служебные кабинеты в общую карточку не входят)

Коротко: публичный мониторинг по организациям высшего образования, где наряду с контингентом, направлениями подготовки и показателями деятельности раскрываются финансовые сведения (в том числе доходы/расходы и смежные показатели по методике мониторинга) — удобный отраслевой срез для сопоставления вузов и регионов, дополняющий бухгалтерскую отчётность на bus.gov.ru.

Описание​

Карточка описывает раздел «Мониторинг (ВО)» федеральной системы мониторинга на сайте monitoring.miccedu.ru: это витрина сводных показателей по образовательным организациям высшего образования, где пользователь выбирает территорию, организацию или иной разрез и получает табличные данные. Для анализа бюджетных потоков и учётной отчётности по формам казначейства мониторинг не заменяет открытые данные bus.gov.ru и своды Казначейства — это разные методики и уровни агрегирования.

Сюда не входят: полный контур ЕИС Закупки по контрактам; ГИС «Наука» по НИОКТР (научный контур Минобрнауки); внутренние отчёты учреждений, доступные только в закрытых кабинетах.

Оператор​

Министерство просвещения Российской Федерации — отраслевой заказчик мониторинга образования; канонический вход к описываемому разделу: https://monitoring.miccedu.ru/?m=vpo

URL и точки доступа​

Тип доступаURLФормат
Мониторинг организаций высшего образования (основная карточка)https://monitoring.miccedu.ru/?m=vpoHTML (+ экспорт из UI при наличии)
Корень витрины мониторинга (прочие режимы и разделы — через меню сайта)https://monitoring.miccedu.ru/HTML

Объекты данных и показатели​

Заполняйте выборку по актуальному интерфейсу — перечень вкладок и полей меняется вместе с регламентами мониторинга.

  • Образовательная организация высшего образования — строка мониторинга: реквизиты и показатели по выбранному периоду/отчётной форме; для стыковки с другими источниками используйте ИНН, ОГРН, код региона.
  • Финансовые показатели — отраслевые своды мониторинга (доходы, расходы, отдельные статьи или агрегаты по методике Минпросвещения); не путать с КОСГУ в отчётности учреждения и с казначейским исполнением.
  • Территориальные и отраслевые срезы — агрегаты по субъекту РФ, типам организаций и направлениям; удобно для сравнения организаций и регионов «в одной таблице мониторинга».

Известные пробелы: методика мониторинга не обязана совпадать с формами плана ФХД и бухотчётности учреждений; при расхождении сумм сверяйте первичный источник по назначению (мониторинг против отчётности на bus).

Периодичность обновления​

Обновление показателей задаётся регламентами отраслевого мониторинса и календарём сбора данных по конкретным формам; на витрине ориентируйтесь на дату среза или год в интерфейсе выбора периода. Единого публичного «частота обновления одной строкой» в машиночитаемом паспорте набора здесь нет — уточняйте по экрану отчёта и методическим документам Минпросвещения на дату анализа.

Ограничения и особенности​

  • Доступ через браузер и экспорт из UI; для регулярного ETL закладывайте хрупкость вёрстки и параметров запросов (m=vpo и др.).
  • Семантика показателя берётся из справки мониторинга и расшифровок на странице, а не из предположений бухгалтерского учёта.
  • Сопоставление с федеральными бюджетными учреждениями удобно дополнять сводом Казначейства по федеральным учреждениям — иной уровень и методика, но тот же контур «казначейские учреждения».

Примеры доступа​

Проверка доступности страницы раздела ВО без авторизации:

curl -sS -o /dev/null -w "%{http_code}\n" "https://monitoring.miccedu.ru/?m=vpo"

Ожидаемый ответ при работе сервиса: код 200.

Связанные страницы wiki​

  • Открытые данные bus.gov.ru — планы ФХД, отчётность и бухформы конкретного учреждения; основной источник для сверки «как учёлось у организации» рядом с отраслевым мониторингом.
  • Казначейство: свод по федеральным бюджетным и автономным учреждениям — казначейское исполнение по учреждениям федерального уровня; дополняет мониторинг при анализе кассовых и бюджетных рядов.
  • bus.gov.ru — портал ГМУ — контекст портала учреждений и границы относительно открытых выгрузок.
  • ГИС «Наука» — научные работы и финансирование НИОКТР в вузах на стороне Минобрнауки; пересекается с частью вузов, но не подменяет мониторинг ВО по всем показателям.
  • Бухгалтерская (финансовая) отчётность учреждений — как соотносятся формы отчётности учреждения с отраслевыми сводами.

Не путать с​

  • Открытые данные bus.gov.ru — регламентированная отчётность учреждения и массовые выгрузки по паспортам наборов; miccedu даёт мониторинговый срез по правилам отраслевой статистики и показателей качества/ресурсов.
  • Наборы данных ГИИС «Электронный бюджет» — консолидированный бюджетный контур и реестры бюджетного процесса, а не показатели отдельного вуза в разрезе мониторинга образования.
