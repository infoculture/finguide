Минфин: краткая информация об исполнении консолидированных бюджетов субъектов РФ

ПризнакЗначение
Уровень и охватКонсолидированные бюджеты субъектов РФ: краткие своды по всей совокупности субъектов в трёх периодичностях публикации (месяц, квартал, год), в млрд руб.
Тип доступаФайлы по ссылке: HTML-страница раздела «Статистика → Бюджеты субъектов РФ» со списком документов и кнопкой «Скачать» для каждого XLSX
Основные форматыXLSX; обёртка — HTML
ОператорМинфин России
Режим доступаОткрытый (типовое скачивание без регистрации с официальной страницы)

Коротко: официальные агрегированные таблицы исполнения консолидированных бюджетов субъектов от Минфина — удобная макро-точка входа для сравнения регионов и динамики в одном файле; для регламентированной отчётности по формам, КБК и кассового контура по каждому субъекту используйте консолидированные бюджеты субъектов на сайте Казначейства и отчёты Казначейства.

Описание​

Карточка описывает витрину Краткая информация об исполнении бюджетов субъектов Российской Федерации: на странице перечислены документы с датами публикации и ссылками на выгрузки. По состоянию на актуальные публикации на сайте доступны три типа файлов:

  • ежемесячная краткая информация (показатели накопленно с начала года, млрд руб.);
  • ежеквартальная краткая информация (также накопленно с начала года, млрд руб.);
  • ежегодная краткая информация (итоги по финансовому году в млрд руб. — конкретный год см. в заголовке документа и в файле).

Сюда не входят открытые наборы Минфина из каталога с list.csv и паспортами meta.csv — раздел subbud/execute — отдельная статистическая подборка на сайте ведомства, без включения в машинный реестр открытых данных того же формата.

Оператор​

Минфин России — канонический URL раздела: https://minfin.gov.ru/ru/statistics/subbud/execute/

  • Карточка ИС: Сайт Министерства финансов РФ — публикация материалов раздела «Статистика» и вложений к документам.

URL и точки доступа​

Тип доступаURLФормат
Страница раздела (список документов, «Скачать» для каждого файла)https://minfin.gov.ru/ru/statistics/subbud/execute/HTML; вложения XLSX
Родительский раздел «Бюджеты субъектов Российской Федерации»https://minfin.gov.ru/ru/statistics/subbud/HTML (навигация по смежным материалам)

Прямые URL файлов XLSX при обновлении вложений меняются; для автоматизации ориентируйтесь на актуальную ссылку «Скачать» в карточке документа на странице раздела.

Объекты данных и показатели​

Состав строк и столбцов задаётся опубликованными XLSX; по смыслу заголовков страницы это:

  • Сводные показатели исполнения консолидированных бюджетов субъектов в млрд руб. — обычно по каждому субъекту РФ в одной таблице или наборе листов (конкретный перечень строк и статей — в файле на дату скачивания).
  • Накопленный с начала года итог — для ежемесячной и ежеквартальной серий (проверяйте подписи листов и шапку таблицы в файле).
  • Годовая серия — для сопоставления с формой 0503317 и иными официальными сводами за финансовый год (методика и детализация могут отличаться).

Пробелы: это не замена построчной отчётности Казначейства по 0503317 и не гарантированный разрез по каждому КБК; для сопоставления с кассовым фактом и формами используйте консолидированные бюджеты субъектов РФ.

Периодичность обновления​

Публикации привязаны к выходу новых редакций документов на странице: у каждой карточки документа указаны даты «Опубликовано» и «Изменено». Ежемесячный файл обновляется чаще, чем квартальный и годовой; стабильного API под выгрузку нет — перед массовой загрузкой проверяйте страницу.

Ограничения и особенности​

  • Материалы официального сайта Минфина распространяются на условиях Creative Commons Attribution 4.0 (блок в подвале страниц minfin.gov.ru); при переработке указывайте источник.
  • Автоматические запросы к сайту иногда упираются в ограничения доступа; для парсинга используйте опубликованные пользователем ссылки после проверки в браузере и учитывайте политику сайта.
  • Показатели Минфина в этих XLSX не обязаны построчно совпадать с открытыми наборами Минфина, срезами ГИИС и формами Казначейства — сверяйте методику, заголовки и дату файла.

Примеры доступа​

Сохранить HTML-страницу раздела для ручного или последующего разбора ссылок «Скачать»:

curl -sS "https://minfin.gov.ru/ru/statistics/subbud/execute/" -o minfin-subbud-execute.html

Связанные страницы wiki​

  • Минфин: данные по исполнению бюджетов субъектов РФ и местных бюджетов (PDF) — иной раздел сайта (perfomance/regions/operational): презентации и аналитика в PDF, не XLSX из subbud/execute.
  • Минфин: краткая информация об исполнении федерального бюджета — параллельная витрина Минфина для федерального бюджета (те же три периодичности, XLSX).
  • Минфин: краткая информация об исполнении бюджетов государственных внебюджетных фондов — параллельная витрина для бюджетов ГВФ (те же три периодичности, XLSX).
  • Минфин: краткая информация об исполнении консолидированного бюджета РФ и ГВФ (conbud/execute) — свод по Российской Федерации (консолидированный бюджет РФ и ГВФ), а не таблица по субъектам как в subbud/execute.
  • Консолидированные бюджеты субъектов РФ — регламентированные отчёты исполнения по субъектам (0503317 и сопутствующие материалы) на сайте Казначейства.
  • Отчёты Федерального казначейства — операционная карточка раздела «Исполнение бюджетов»: формы, форматы, ограничения автоматизации.
  • Региональные бюджеты — плановый контур и сайты субъектов; дополняет картину относительно факта в сводах Минфина и Казначейства.
  • Отчёты об исполнении бюджетов (обзор) — где в цепочке источников лежит факт исполнения относительно плана и казначейского исполнения.
  • Открытые данные Минфина России — каталог CSV/XML-наборов; раздел subbud/execute в него не обязан входить тем же составом полей.
  • Форма 0503317 и консолидированный бюджет — термины и назначение отчётности для интерпретации цифр.

Не путать с​

  • Открытые данные Минфина России — реестр наборов с list.csv и паспортами, а не список трёх XLSX на странице статистики subbud/execute.
  • Консолидированные бюджеты субъектов РФ — иной оператор (Казначейство), иная методика и обычно большая детализация по форме 0503317; свод Минфина удобен для обзора по регионам в одном файле, но не подменяет казначейскую отчётность.

Частые вопросы​

В: Можно ли по этим XLSX восстановить исполнение по каждому КБК по субъекту?

О: Обычно нет или не полностью: файлы краткой информации агрегируют показатели. Для разреза по КБК используйте формы исполнения Казначейства и материалы консолидированных бюджетов субъектов.

В: Почему цифра в годовом файле Минфина не совпадает с суммой по строкам 0503317?

О: Различаются определение показателя, момент среза, методика свёртки и источник агрегации (Минфин публикует краткую сводку, Казначейство — регламентированную форму). Сверяйте заголовки документов и пояснения к таблицам в обоих источниках.
