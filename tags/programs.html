# ГИС «Наука» (gisnauka.ru)

ГИС «Наука»: открытые данные НИОКТР, РИД и связь с бюджетным финансированием

источник для стыковки записей о научных работах и РИД с бюджетным финансированием, когда в паспорте набора, в выгрузках с портала или в ответах поискового API явно приведены ИНН, номера соглашений, коды программ или иные ключи, позволяющие соединить учёт науки с ГИИС «Электронный бюджет» и реестром соглашений.

Доступ к данным ГИС «Наука»

Как получить открытые сведения на gisnauka.ru: HTTP API выгрузок /api/egisu/open-data (year, month, card_type), глобальный поиск, «Сформировать выгрузку», внутренний /api/egisu/base/search из DevTools; завершённые периоды, User-Agent и стыковка с бюджетными наборами.
